## Desktop App Overview

Claude Code now has a **standalone desktop app**, providing full Claude Code capabilities without a terminal or IDE.

## Key Features

### Visual Diff Review

Review code changes visually. Compared to terminal-based text diffs, identifying changes is significantly easier.

### Multiple Simultaneous Sessions

Run multiple coding sessions side by side for different projects or tasks in parallel.

### Scheduled Tasks and Cloud Sessions

Schedule recurring tasks and launch cloud sessions for long-running processes.

## Supported Platforms

- macOS (Intel / Apple Silicon)
- Windows (x64 / ARM64)

## Installation

Download the installer from the official site, sign in, and start coding. A paid Claude subscription is required.

## Summary

The desktop app makes Claude Code more accessible to a wider audience, including those not comfortable with the terminal.
