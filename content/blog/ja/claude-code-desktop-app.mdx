## デスクトップアプリの概要

Claude Code に待望の **スタンドアロンデスクトップアプリ** が登場しました。ターミナルや IDE を使わずに Claude Code の全機能を利用できます。

## 主な機能

### ビジュアル差分レビュー

コードの変更をビジュアルに確認できます。ターミナルでのテキストベースの差分表示と比較して、変更箇所の把握が格段に容易になりました。

### 複数セッションの同時実行

サイドバイサイドで複数のコーディングセッションを同時に実行できます。異なるプロジェクトやタスクを並行して進めるのに便利です。

### スケジュールタスクとクラウドセッション

定期的に実行するタスクをスケジュール設定でき、クラウドセッションを起動して長時間の処理を実行することも可能です。

## 対応プラットフォーム

- macOS（Intel / Apple Silicon）
- Windows（x64 / ARM64）

## インストール方法

公式サイトからインストーラーをダウンロードし、サインインするだけで利用開始できます。Claude の有料サブスクリプションが必要です。

## まとめ

デスクトップアプリにより、Claude Code がより多くのユーザーにとってアクセスしやすくなりました。ターミナルに慣れていない方にもおすすめです。
