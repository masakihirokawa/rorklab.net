## Claude Opus 4.6 の登場

Anthropic は最新のフラッグシップモデル **Claude Opus 4.6** をリリースしました。エージェント構築とコーディングにおいて最も知的なモデルとして位置づけられています。

## 主な特徴

### 業界最高のコーディング性能

Claude Opus 4.6 はコーディングベンチマークにおいて業界最高水準のスコアを記録しています。複雑なマルチファイルの編集、アーキテクチャ設計、デバッグにおいて前バージョンから大幅に改善されました。

### 拡大された出力とコンテキスト

最大出力が **128K トークン** に拡大（前世代の Opus 4 は 32K）。さらにベータ版として **1M トークンのコンテキストウィンドウ** にも対応しています。

### Extended Thinking と Adaptive Thinking

従来の Extended Thinking に加え、新たに **Adaptive Thinking** モードが導入されました。`budget_tokens` を指定する必要がなく、問題の複雑さに応じて自動的に思考の深さを調整します。

## API での利用

API ID は `claude-opus-4-6` で、すべてのプラットフォーム（Claude API、AWS Bedrock、Google Vertex AI）で利用可能です。

## 料金

入力: $5 / 100万トークン、出力: $25 / 100万トークンです。Sonnet 4.6 と比較すると高価ですが、複雑なタスクではより少ないやり取りで解決できるため、結果的にコスト効率が良い場合もあります。

## まとめ

Claude Opus 4.6 は特にエージェント開発やコーディングの分野で大きな進歩を見せています。最高品質が求められるプロジェクトでは、ぜひ試してみてください。
