## #1: Why Environment Variables Are Undefined in Your Rork App — And How to Fix It

Getting `undefined` for environment variables at runtime is one of those problems that trips up almost every Rork developer at some point. This article covers the three most common causes: forgetting the `EXPO_PUBLIC_` prefix, EAS Secrets misconfiguration, and `.env` file placement errors — with step-by-step resolution for each.

```bash
# Correct setup: .env in the project root
EXPO_PUBLIC_API_URL=https://api.example.com
EXPO_PUBLIC_APP_NAME=MyApp

# Access in code
const apiUrl = process.env.EXPO_PUBLIC_API_URL;
```

[Why Environment Variables Are Undefined in Your Rork App — And How to Fix It](/articles/rork-dev/rork-env-variables-configuration-error-fix)
