## Rork — Building Mobile Apps with Prompts

Rork is an AI-powered platform for building mobile apps using just prompts. As of 2026, tens of thousands of developers use Rork to create thousands of apps published monthly to the App Store and Google Play.

**Key Differences from Traditional Development:**

- **No Coding Required**: Tell Rork "create an SNS-like app" and it generates code
- **Automatic Code Generation**: AI creates React Native or Swift (Rork Max) code
- **Live Editing**: Preview and refine changes in real-time
- **Simplified App Store Publishing**: Rork Max publishes with just a few clicks

This complete guide covers Rork from basics to advanced 2026 workflows.

## Building Your First App — From Prompt to Live

### Step 1: Sign Up

Visit Rork Lab (https://rorklab.net) and sign up with Google or GitHub.

### Step 2: Create a New Project

Click "New Project" and provide:

- **Project Name**: Example "MyWeatherApp"
- **Description**: Briefly describe the purpose
- **Platform**: React Native (Free/Pro) or Swift (Rork Max)
- **Template**: Start with "Blank" or "Hello World"

### Step 3: Write Your Prompt

Here's an example for a simple weather app:

```
Create:
- Search box at top for entering city name
- Current temperature, weather icon, feels-like temperature display
- 5-day weather forecast below (scrollable)
- Color scheme: blue and gray
- API: OpenWeatherMap
- Initial city: Tokyo
```

### Step 4: AI Generates Code

Rork AI analyzes your prompt and generates working code in 5-15 seconds. For React Native, it generates a structure like:

```javascript
import React, { useState, useEffect } from 'react';
import {
  View,
  Text,
  TextInput,
  ScrollView,
  ActivityIndicator,
  StyleSheet,
} from 'react-native';
import axios from 'axios';

export default function WeatherApp() {
  const [city, setCity] = useState('Tokyo');
  const [weather, setWeather] = useState(null);
  const [loading, setLoading] = useState(false);
  const [forecast, setForecast] = useState([]);

  useEffect(() => {
    fetchWeather(city);
  }, []);

  const fetchWeather = async (cityName) => {
    setLoading(true);
    try {
      const apiKey = 'YOUR_OPENWEATHER_API_KEY';
      const response = await axios.get(
        `https://api.openweathermap.org/data/2.5/weather?q=${cityName}&appid=${apiKey}&units=metric`
      );
      setWeather(response.data);

      const forecastRes = await axios.get(
        `https://api.openweathermap.org/data/2.5/forecast?q=${cityName}&appid=${apiKey}&units=metric`
      );
      setForecast(forecastRes.data.list.slice(0, 5));
    } catch (error) {
      console.error('Weather fetch error:', error);
    } finally {
      setLoading(false);
    }
  };

  const handleSearch = () => {
    fetchWeather(city);
  };

  if (loading) {
    return (
      <View style={styles.center}>
        <ActivityIndicator size="large" color="#0066cc" />
      </View>
    );
  }

  return (
    <ScrollView style={styles.container}>
      <View style={styles.searchBox}>
        <TextInput
          style={styles.input}
          placeholder="Enter city name..."
          value={city}
          onChangeText={setCity}
        />
      </View>

      {weather && (
        <View style={styles.weatherCard}>
          <Text style={styles.cityName}>{weather.name}</Text>
          <Text style={styles.temp}>{Math.round(weather.main.temp)}°C</Text>
          <Text style={styles.description}>{weather.weather[0].main}</Text>
          <Text style={styles.feelsLike}>
            Feels like: {Math.round(weather.main.feels_like)}°C
          </Text>
        </View>
      )}

      <View style={styles.forecastContainer}>
        <Text style={styles.forecastTitle}>5-Day Forecast</Text>
        {forecast.map((item, index) => (
          <View key={index} style={styles.forecastItem}>
            <Text style={styles.forecastDate}>
              {new Date(item.dt * 1000).toLocaleDateString('en-US')}
            </Text>
            <Text style={styles.forecastTemp}>
              {Math.round(item.main.temp)}°C
            </Text>
          </View>
        ))}
      </View>
    </ScrollView>
  );
}
```

## Rork Max — SwiftUI Native and 2-Click App Store Publishing

Rork Max, launched in 2026, is the premium plan for serious iOS development.

### SwiftUI Native Support

Rork Max generates **complete SwiftUI code** instead of React Native, delivering:

- **Performance**: 50-80% faster
- **App Store Approval**: Much higher pass rate
- **Apple Framework Integration**: Easy access to CoreData, HealthKit, etc.

```swift
import SwiftUI

struct ContentView: View {
  @State private var city = "Tokyo"
  @State private var weather: WeatherData?
  @State private var isLoading = false

  var body: some View {
    NavigationStack {
      VStack(spacing: 20) {
        HStack {
          TextField("Enter city name", text: $city)
            .textFieldStyle(.roundedBorder)
            .keyboardType(.default)

          Button(action: fetchWeather) {
            Image(systemName: "magnifyingglass")
              .foregroundColor(.white)
              .frame(width: 44, height: 44)
              .background(Color.blue)
              .cornerRadius(8)
          }
        }
        .padding()

        if isLoading {
          ProgressView()
        } else if let weather = weather {
          VStack(alignment: .center, spacing: 12) {
            Text(weather.city)
              .font(.title)
              .fontWeight(.semibold)

            Text("\(Int(weather.temp))°C")
              .font(.system(size: 56, weight: .bold))
              .foregroundColor(.blue)

            Text(weather.description)
              .font(.body)
              .foregroundColor(.gray)
          }
          .frame(maxWidth: .infinity)
          .padding(24)
          .background(Color(UIColor.systemGray6))
          .cornerRadius(12)
        }

        Spacer()
      }
      .navigationTitle("Weather App")
    }
    .task {
      await fetchWeather()
    }
  }

  func fetchWeather() async {
    isLoading = true
    // API call logic
    isLoading = false
  }
}

struct WeatherData {
  let city: String
  let temp: Double
  let description: String
}
```

## Backend Integration — Supabase, Firebase, Stripe

### Supabase Integration

Supabase is an open-source PostgreSQL backend. Direct integration with Rork:

```javascript
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = 'https://your-project.supabase.co';
const supabaseKey = 'your-anon-key';
const supabase = createClient(supabaseUrl, supabaseKey);

// User registration
const registerUser = async (email, password) => {
  const { user, error } = await supabase.auth.signUp({
    email,
    password,
  });
  return { user, error };
};

// Database insert
const addPost = async (userId, content) => {
  const { data, error } = await supabase
    .from('posts')
    .insert([{ user_id: userId, content, created_at: new Date() }]);
  return { data, error };
};

// Real-time listening
supabase
  .from('posts')
  .on('*', (payload) => {
    console.log('New post:', payload.new);
  })
  .subscribe();
```

### Firebase Integration

Google Firebase works great for notifications and real-time databases:

```javascript
import { initializeApp } from 'firebase/app';
import { getMessaging, getToken } from 'firebase/messaging';
import { getDatabase, ref, set, onValue } from 'firebase/database';

const firebaseConfig = {
  apiKey: 'YOUR_API_KEY',
  projectId: 'your-project-id',
};

const app = initializeApp(firebaseConfig);
const messaging = getMessaging(app);
const db = getDatabase(app);

// Get push notification token
const getNotificationToken = async () => {
  const token = await getToken(messaging);
  return token;
};

// Monitor real-time database
const listenToMessages = (chatRoomId) => {
  const messagesRef = ref(db, `chatRooms/${chatRoomId}/messages`);
  onValue(messagesRef, (snapshot) => {
    const data = snapshot.val();
    console.log('Messages:', data);
  });
};
```

### Stripe Integration

Payment and subscription implementation:

```javascript
import Stripe from 'stripe';

const stripe = new Stripe('sk_test_YOUR_SECRET_KEY');

// Create payment intent
const createPaymentIntent = async (amount, currency = 'usd') => {
  const paymentIntent = await stripe.paymentIntents.create({
    amount,
    currency,
    payment_method_types: ['card'],
  });
  return paymentIntent;
};

// Create subscription
const createSubscription = async (customerId, priceId) => {
  const subscription = await stripe.subscriptions.create({
    customer: customerId,
    items: [{ price: priceId }],
  });
  return subscription;
};
```

## Prompt Engineering Tips for Rork

Effective prompts are the key to success with Rork.

### 1. Enumerate Specific Features

**Good:**
```
Create a task management app with:
- Home screen: today's task list (with checkboxes)
- New task screen: title, description, due date inputs
- Calendar view: monthly and weekly filters
- Storage: SQLite local save, iCloud sync support
```

**Poor:**
```
Make a task management app
```

### 2. Specify Design Details

```
UI Color Scheme:
- Primary: #0066cc (blue)
- Accent: #ff6b35 (orange)
- Background: #f5f5f5 (light gray)

Font: System font (SF Pro / Roboto), headers bold (700)
```

### 3. Clarify API and Backend

```
Backend:
- Auth: Supabase Auth
- Database: Supabase PostgreSQL
- Real-time: Supabase Realtime

Image storage: Cloudinary or AWS S3
```

### 4. Iterate Gradually

First prompt generates basic features → refine with successive prompts:
1. "I want to preview" → basic features
2. "Make buttons bigger" → UI adjustments
3. "Add dark mode" → feature addition
4. "Support English and Japanese" → internationalization
