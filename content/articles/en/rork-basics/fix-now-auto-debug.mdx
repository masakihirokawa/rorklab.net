## What Is "Fix Now"?

When building apps in Rork, you'll occasionally run into build errors or runtime bugs. Fix Now is a feature that lets AI automatically fix these issues with a single click.

Even without coding experience or understanding of the error, you can resolve problems with just one button press — that's the core appeal of Fix Now.

## Types of Errors Fix Now Handles

### Build Errors

Automatically detects and fixes errors that occur during the build process:

- TypeScript / JavaScript syntax errors
- Incorrect import paths
- Type mismatches
- Dependency version conflicts

### Runtime Errors

Handles errors that occur while the app is running:

- Undefined / null reference errors
- API response format mismatches
- Navigation reference errors
- Infinite render loops

### Layout Issues

Fix Now can also address visual problems in some cases:

- Broken layouts
- Scroll issues
- Unresponsive to different screen sizes

## How to Use It

### Basic Flow

1. When an error occurs, a red banner appears at the top of the screen
2. Click the "Fix Now" button in the banner
3. AI analyzes the error and generates a fix
4. The fix is applied and the app automatically rebuilds

### Adding Context via Chat

When Fix Now alone doesn't solve the issue, providing more details through chat improves fix accuracy:

```
Fix Now didn't resolve the timer error.
The app crashes when the timer reaches 0.
I think the countdown might be missing
a stop condition when it hits zero.
```

### Running Fix Now Multiple Times

If one attempt doesn't work, you can run Fix Now again. The AI considers previous fix attempts and tries a different approach, often succeeding on the second or third try.

## How Fix Now Works

Fix Now operates through this process:

1. **Error log analysis**: Parses stack traces and error messages
2. **Code identification**: Locates the relevant files and code sections
3. **Fix generation**: AI generates the corrected code
4. **Auto-apply and test**: Applies the fix and rebuilds to verify

## Tips for Effective Use

### Fix Errors Immediately

Don't continue development with unresolved errors — problems compound quickly. Run Fix Now as soon as an error appears.

### Review What Changed

Fix Now is automatic, but make a habit of checking what was changed. Reviewing the diff in the code editor helps you prevent similar errors in the future.

### Combine with Chat

The most effective approach is using Fix Now and chat together. Use Fix Now for the initial fix, then fine-tune through chat for precise adjustments.

## When Fix Now Isn't Enough

Some situations are harder for Fix Now to handle on its own:

- Architecture-level issues requiring fundamental data flow redesign
- External API specification changes
- Platform-specific limitation issues

In these cases, describe the problem in detail via chat and ask the AI for a broader solution that may include architectural changes.

## Takeaway

Fix Now is a powerful ally for smooth app development in Rork. When you hit an error, don't panic — just press Fix Now first. More often than not, AI will resolve the issue in seconds.
