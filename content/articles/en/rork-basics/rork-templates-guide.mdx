## What Are Rork Templates?

Rork provides a library of **app templates** — pre-built starting points for common app types. Instead of writing prompts from scratch, choose a template and customize it. Prototypes come together in minutes.

## Available Templates

**Productivity & Business**:
- To-Do list app
- Expense tracker
- Time tracker
- Notes app

**Social & Community**:
- Chat app
- Social feed
- Event management

**Health & Fitness**:
- Workout tracker
- Meal logging
- Habit tracker

**E-commerce & Booking**:
- Online shop
- Booking system
- Restaurant menu

## How to Use Templates

### 1. Select a template

On the Rork project creation screen, choose "Start from template" and pick one that matches your use case.

### 2. Enter basic info

Provide your app name and a brief description. Rork generates code based on the selected template.

### 3. Customize with prompts

Once generated, customize via prompts:

```
"Change the color theme to blue"
"Add a category filter"
"Add dark mode support"
```

## Why Use Templates

**Save time**: Basic CRUD operations and navigation structure are already implemented.

**Best practices built in**: Templates follow Rork's recommended code structure, making them easy to extend.

**Pre-built features**: Many templates include authentication flows, data persistence, and common UI components.

## Templates vs. Starting from Scratch

| Situation | Recommendation |
|-----------|----------------|
| Common app type | Template |
| Unique UI/logic needed | From scratch |
| First app project | Template |
| Learning purposes | Either |

## Summary

Rork templates dramatically lower the barrier to starting app development. For first-time developers or rapid prototyping, they're the best starting point — build something working first, then add your unique touches.

For a deeper dive into this topic, we recommend [The Art of Prompt Engineering with ChatGPT](https://www.amazon.co.jp/dp/1394286791/?tag=pinocchio-22) by Nathan Hunter. It offers a comprehensive guide to prompt engineering techniques.
