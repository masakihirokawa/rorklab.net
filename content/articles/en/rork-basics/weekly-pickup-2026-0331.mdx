## 🥈 #2: Building a Real-Time Voice AI Assistant with Rork and the OpenAI Realtime API

**Category**: Rork AI ／ **Level**: Advanced

[Rork × OpenAI Realtime API：リアルタイム音声AIアシスタントアプリ完全実装ガイド](/articles/rork-ai/rork-openai-realtime-api-voice-assistant-app)

This premium guide is for developers who want to build a voice assistant that actually feels real — no noticeable latency, natural back-and-forth, and a polished experience. It covers the full pipeline: microphone input, speech recognition, AI response generation, and voice playback — all integrated into a Rork app.

```javascript
// Rork × OpenAI Realtime API — Basic structure
import { RealtimeClient } from '@openai/realtime-api-beta';

const client = new RealtimeClient({ apiKey: process.env.OPENAI_API_KEY });

// Start a session
await client.connect();
client.sendUserMessageContent([
  { type: 'input_audio', audio: base64AudioData }
]);

// Receive the AI's real-time voice response
client.on('response.audio.delta', ({ delta }) => {
  audioPlayer.play(delta);
});
```

If you've been wanting to add a voice interface to your app but weren't sure where to start, this is the guide.
