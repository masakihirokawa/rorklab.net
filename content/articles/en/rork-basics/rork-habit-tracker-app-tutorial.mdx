"Run every morning." "Read for ten minutes a day." "Drink eight glasses of water." These habits sound simple — but sticking with them is another story.

Having your own habit tracking app changes that. Today we'll build one with Rork in **30 minutes**, no programming experience needed.

## What You'll Build

The finished app will have:

- A personal habit list you can add to and manage
- Daily checkboxes to mark each habit complete
- A streak counter tracking your consecutive achievement days
- A clean, easy-to-use UI

## Step-by-Step Development in Rork

### Step 1: Create a New Project

Log in to Rork ([rork.app](https://rork.app)) and create a new project. Name it "HabitTracker" or whatever feels right, then hit Start.

### Step 2: Describe Your App to Rork

Paste the following prompt into Rork's chat input:

```
Build a habit tracker app.

Features needed:
1. Add habits
   - Enter a habit name via text input
   - Choose a category (Health, Learning, Work, Other)
   - Display an icon matching the category

2. Daily habit check-in
   - Show today's habit list on the home screen
   - Tap to toggle each habit on/off
   - Highlight completed habits in green

3. Streaks (consecutive days)
   - Show streak count for each habit
   - Display a celebration message at 7 and 30 consecutive days

4. Data persistence
   - Habits and check history survive app restarts
   - Automatically reset today's checks at midnight

Design:
- Clean, minimal UI
- White and green color scheme
- Clear visual distinction between completed and pending habits
```

**Main home screen:**

```typescript
// app/(tabs)/index.tsx
import React, { useState, useEffect } from 'react';
import {
  View, Text, FlatList, TouchableOpacity,
  StyleSheet, Alert
} from 'react-native';
import { useHabitStore } from '@/store/habitStore';
import { HabitCard } from '@/components/HabitCard';
import { getTodayString } from '@/utils/dateUtils';

export default function HomeScreen() {
  const { habits, completions, toggleCompletion } = useHabitStore();
  const today = getTodayString();

  const getCompletionStatus = (habitId: string) => {
    return completions.some(
      c => c.habitId === habitId && c.date === today && c.completed
    );
  };

  const completedCount = habits.filter(h => getCompletionStatus(h.id)).length;
  const completionRate = habits.length > 0
    ? Math.round((completedCount / habits.length) * 100)
    : 0;

  return (
    <View style={styles.container}>
      {/* Today's progress summary */}
      <View style={styles.summaryCard}>
        <Text style={styles.dateText}>
          {new Date().toLocaleDateString('en-US', {
            month: 'long', day: 'numeric', weekday: 'long'
          })}
        </Text>
        <Text style={styles.progressText}>
          {completedCount} / {habits.length} completed
        </Text>
        <View style={styles.progressBar}>
          <View
            style={[
              styles.progressFill,
              { width: `${completionRate}%` }
            ]}
          />
        </View>
        {completionRate === 100 && habits.length > 0 && (
          <Text style={styles.congratsText}>🎉 All habits completed today!</Text>
        )}
      </View>

      {/* Habit list */}
      <FlatList
        data={habits}
        keyExtractor={item => item.id}
        renderItem={({ item }) => (
          <HabitCard
            habit={item}
            isCompleted={getCompletionStatus(item.id)}
            onToggle={() => toggleCompletion(item.id, today)}
          />
        )}
        ListEmptyComponent={
          <View style={styles.emptyState}>
            <Text style={styles.emptyText}>No habits yet</Text>
            <Text style={styles.emptySubtext}>Tap "Add Habit" to get started</Text>
          </View>
        }
        contentContainerStyle={styles.listContent}
      />
    </View>
  );
}
```

**Habit card component:**

```typescript
// components/HabitCard.tsx
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';
import { Habit, CATEGORY_CONFIG } from '@/types/habit';

interface HabitCardProps {
  habit: Habit;
  isCompleted: boolean;
  onToggle: () => void;
}

export function HabitCard({ habit, isCompleted, onToggle }: HabitCardProps) {
  const config = CATEGORY_CONFIG[habit.category];

  return (
    <TouchableOpacity
      style={[styles.card, isCompleted && styles.cardCompleted]}
      onPress={onToggle}
      activeOpacity={0.7}
    >
      <View style={styles.leftSection}>
        <Text style={styles.icon}>{config.icon}</Text>
        <View style={styles.textSection}>
          <Text style={[styles.habitName, isCompleted && styles.habitNameCompleted]}>
            {habit.name}
          </Text>
          <Text style={styles.streakText}>
            🔥 {habit.streak}-day streak
          </Text>
        </View>
      </View>

      <View style={[styles.checkButton, isCompleted && styles.checkButtonCompleted]}>

}
```

## Frequently Asked Questions

**Q1. Can I build this on Rork's free plan?**

Yes — everything in this tutorial works on the free plan. Some native features like push notifications may require a paid plan or Rork Max. Start with the basics and upgrade when you need them.

**Q2. Can I share the finished app with friends and family?**

On the basic plan, you can share via QR code for installation on real devices. Publishing to the App Store or Google Play requires Rork Max ($200/month) for the two-click publish feature, or going through the standard developer account review process.

**Q3. How do I implement the midnight reset for daily check-ins?**

The simplest approach is comparing today's date with the last-checked date on each app launch. Just tell Rork: "When the app opens, check if the date has changed since the last check-in and reset today's completions if so." Rork will handle the implementation.
