## Why Animations Matter

Animations are what separate a polished app from a generic one. How a button responds to a tap, how smoothly a screen transitions, how the loading state feels — these details define perceived quality.

Rork Max is React Native-based, so `Animated` API and `react-native-reanimated` are available.

## Basic Fade Animation

Prompt Rork Max to add a fade-in:

```
"Add a fade-in animation to this component when it mounts.
Use the React Native Animated API."
```

Generated code example:

```javascript
import { useEffect, useRef } from 'react';
import { Animated, View } from 'react-native';

export function FadeInCard({ children }) {
  const opacity = useRef(new Animated.Value(0)).current;

  useEffect(() => {
    Animated.timing(opacity, {
      toValue: 1,
      duration: 400,
      useNativeDriver: true,
    }).start();
  }, []);

  return (
    <Animated.View style={{ opacity }}>
      {children}
    </Animated.View>
  );
}
```

## Spring Animation

For natural bouncy interactions (button press, card pop):

```javascript
const scale = useRef(new Animated.Value(1)).current;

const handlePress = () => {
  Animated.sequence([
    Animated.spring(scale, {
      toValue: 0.95,
      useNativeDriver: true,
      tension: 100,
    }),
    Animated.spring(scale, {
      toValue: 1,
      useNativeDriver: true,
      friction: 3,
    }),
  ]).start();
};
```

## Slide-In Panel

A bottom sheet that slides up:

```javascript
const translateY = useRef(new Animated.Value(300)).current;

const slideIn = () => {
  Animated.spring(translateY, {
    toValue: 0,
    useNativeDriver: true,
    tension: 65,
    friction: 8,
  }).start();
};
```

## Lottie Animations

For complex animations, use Lottie. Prompt Rork Max:

```
"Add a Lottie animation to the loading screen.
I'll provide a .json animation file from lottiefiles.com."
```

Place the animation JSON in `assets/animations/` and have Rork Max integrate it.

## Performance Tips

**Always use `useNativeDriver: true`**: Runs animations on the UI thread instead of the JS thread — dramatically smoother performance.

**Prevent unnecessary re-renders**: Use `React.memo` to avoid full component re-renders when animation values change.

## Summary

Animations in Rork Max:

- Fade, spring, slide — all handled with the `Animated` API
- Rork Max can generate these from natural language prompts
- `useNativeDriver: true` is essential for smooth performance
- Lottie for more complex animated sequences

Thoughtful animations make Rork Max apps feel significantly more polished.
