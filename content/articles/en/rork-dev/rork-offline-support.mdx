## Why Offline Support Matters

Modern mobile apps need to work without a network connection — on subways, planes, and in areas with poor signal. Offline capability dramatically improves user experience.

Rork Max apps are React Native-based, so native storage options are fully available.

## AsyncStorage — Lightweight Key-Value Storage

For settings, user profiles, and small data, `AsyncStorage` is the right tool.

```javascript
import AsyncStorage from '@react-native-async-storage/async-storage';

export async function saveSettings(settings) {
  await AsyncStorage.setItem('user_settings', JSON.stringify(settings));
}

export async function loadSettings() {
  const stored = await AsyncStorage.getItem('user_settings');
  return stored ? JSON.parse(stored) : null;
}
```

## expo-sqlite — Full Local Database

For tasks, notes, or offline-readable articles, use `expo-sqlite`:

```javascript
import * as SQLite from 'expo-sqlite';

const db = SQLite.openDatabase('myapp.db');

export function initDatabase() {
  db.transaction((tx) => {
    tx.executeSql(
      `CREATE TABLE IF NOT EXISTS tasks (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        title TEXT NOT NULL,
        completed INTEGER DEFAULT 0,
        created_at TEXT DEFAULT CURRENT_TIMESTAMP
      )`
    );
  });
}

export function addTask(title, callback) {
  db.transaction((tx) => {
    tx.executeSql(
      'INSERT INTO tasks (title) VALUES (?)',
      [title],
      (_, result) => callback(result.insertId)
    );
  });
}
```

## Network Status Monitoring

Show offline UI when there's no connection:

```javascript
import NetInfo from '@react-native-community/netinfo';
import { useState, useEffect } from 'react';

export function useNetworkStatus() {
  const [isOnline, setIsOnline] = useState(true);

  useEffect(() => {
    const unsubscribe = NetInfo.addEventListener((state) => {
      setIsOnline(state.isConnected && state.isInternetReachable);
    });
    return unsubscribe;
  }, []);

  return isOnline;
}
```

## Summary

For offline support in Rork Max:

- **AsyncStorage**: Settings and simple key-value data
- **expo-sqlite**: Structured data like tasks and notes
- **NetInfo**: Network status monitoring for offline UI feedback

All three can be added to your Rork Max app via natural language prompts.
