# Complete Guide to Apple Native APIs with Rork Max

Because Rork Max generates native Swift code, you get full access to Apple-exclusive APIs that were previously unavailable through the React Native-based original Rork. This guide covers the key native APIs you can leverage and how to implement them using natural language prompts.

## Why Native APIs Matter

The original Rork is built on React Native, which requires bridge layers to access Apple's native frameworks. Rork Max generates SwiftUI code directly, giving you seamless access to every API Apple provides.

This means advanced features that previously required native iOS development expertise can now be implemented with simple natural language instructions.

## ARKit — Building Augmented Reality Apps

ARKit is Apple's augmented reality framework that combines the camera and sensors to overlay digital content onto the real world.

### What You Can Build

- Furniture placement simulators that let users visualize items in their room
- AR navigation apps that display directions through the camera view
- 3D model viewers that showcase products or art in physical space
- AR measurement tools that calculate real-world dimensions

### Example Prompt

```
Build an app using ARKit that lets users place virtual furniture in their room
through the camera. Furniture should be rotatable and resizable, with the
ability to place multiple items simultaneously.
```

### LiDAR Scanner Support

iPhone Pro and iPad Pro models include a LiDAR scanner. Rork Max can leverage this for high-precision spatial recognition, including 3D room scanning, accurate distance measurement, and real-time mesh generation.

## HealthKit — Health & Fitness App Development

HealthKit is Apple's health data framework that provides secure read and write access to user health information.

### Available Data Types

- Steps, distance, calories burned
- Heart rate, blood pressure, blood oxygen
- Sleep analysis data
- Workout records
- Weight, BMI, body fat percentage

### Example Prompt

```
Create a fitness app that integrates with HealthKit to display daily steps
and calories burned in charts. Include weekly and monthly trend views,
plus a goal-setting feature.
```

### Privacy Considerations

HealthKit data requires explicit user permission. Code generated by Rork Max automatically includes proper permission request dialogs. For App Store review, you must clearly explain why your app needs access to health data.

## Dynamic Island — Immersive UI for iPhone 15 Pro and Later

Dynamic Island transforms the notch area at the top of the screen into an interactive UI element, perfect for displaying real-time information and elegant notifications.

### Use Cases

- Delivery tracking with real-time status updates
- Live sports scores during games
- Cooking or focus timers showing remaining time
- Music playback info and controls

### Example Prompt

```
Build a cooking timer app. When a timer is set, display the remaining time
in the Dynamic Island, with a notification when complete. Support
managing multiple timers simultaneously.
```

## Live Activities — Real-Time Lock Screen Updates

Live Activities work alongside Dynamic Island to display real-time information on the Lock Screen.

### Use Cases

- Food delivery order status tracking
- Sports match progress updates
- Ride-share vehicle arrival status
- Flight information with real-time updates

### Example Prompt

```
Build a food delivery order tracking app. Use Live Activities to display
order status (Preparing → On the Way → Arrived) on the Lock Screen,
with a map showing the driver's real-time location.
```

## Core ML — On-Device Machine Learning

Core ML is Apple's on-device machine learning framework that runs AI inference directly on the device without sending data to the cloud.

### What You Can Implement

- Image recognition and classification
- Natural language processing
- Speech recognition
- Object detection
- Style transfer

### Example Prompt

```
Create an app using Core ML that identifies plants from camera photos.
Display the plant name, characteristics, and care tips for each
identified species.
```

### Privacy Benefits

Core ML processes everything on-device, meaning user data never leaves the phone. This makes it ideal for privacy-focused applications.

## App Clips — Lightweight App Experiences Without Installation

App Clips are lightweight versions of your app that launch instantly from NFC tags, QR codes, or Safari links without requiring a full installation.

### Use Cases

- Restaurant menu ordering
- Parking payment
- Bike-sharing unlock
- Event ticket display

### Example Prompt

```
Build a cafe mobile ordering app with an App Clip. When users scan a
QR code, the ordering screen opens and they can pay with Apple Pay.
```

## HomeKit — Smart Home Integration

HomeKit lets you build apps that control Apple HomeKit-compatible smart home devices.

### Controllable Devices

- Smart lighting (brightness, color temperature)
- Thermostats (temperature settings)
- Security cameras (live feeds)
- Smart locks (lock/unlock)
- Smart plugs (on/off)

### Example Prompt

```
Build a smart home dashboard app with HomeKit integration. Organize
devices by room, with scene presets (Good Morning, Good Night) that
control multiple devices at once.
```

## Widgets — Home Screen Customization

Using SwiftUI-based WidgetKit, you can create widgets that display information on the Home Screen and StandBy mode.

### Widget Sizes

- Small (2x2): Compact information display
- Medium (4x2): Detailed information view
- Large (4x4): Rich content display
- Lock Screen: Lock Screen widgets
- StandBy: StandBy mode support

### Example Prompt

```
Add widgets to a weather app. Small widget shows current temperature
and weather icon, Medium widget shows 3-hour forecast, and Lock Screen
widget displays the temperature.
```

## Siri Intents — Voice Control Integration

SiriKit enables voice-operated app features through Siri.

### Supported Intent Types

- Sending messages
- Starting and ending workouts
- Payments and transfers
- List management
- Media playback

### Example Prompt

```
Add Siri Shortcuts to a task management app. "What are my tasks today"
reads the task list aloud, and "Add milk to my shopping list" adds
an item directly.
```

## Metal — High-Performance Graphics

Metal is Apple's low-level graphics API that provides direct GPU control for high-performance 3D graphics and computation.

### Use Cases

- 3D game development
- Real-time image and video effects
- Scientific visualization
- Custom rendering engines

### Example Prompt

```
Build a 3D solar system simulator using Metal. Each planet orbits
realistically, with pinch-to-zoom and swipe to change the viewing angle.
```

## Conclusion

The greatest advantage of Rork Max is its native Swift generation, which provides access to every Apple API. Advanced features like ARKit, HealthKit, Dynamic Island, and Core ML can be implemented simply by writing natural language prompts.

Features that were difficult or impossible with React Native-based apps can now be built in minutes with Rork Max. Start with the APIs that interest you most and deliver truly native experiences to your users.

By combining multiple native APIs, you can build richer, higher-performing apps. For example, combine ARKit with Core ML for AR object recognition, or HealthKit with Widgets for a health dashboard. The possibilities are endless.
