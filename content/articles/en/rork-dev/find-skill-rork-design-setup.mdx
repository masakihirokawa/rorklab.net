## Introduction

The biggest friction point in Rork development is **setup**. Installing the right design tools, testing frameworks, and deployment pipelines manually is overwhelming for beginners and time-consuming for experienced developers.

**Find Skill** solves this. It automatically detects what your project needs and installs matching skills—the specialized tools that extend Rork. In seconds, your environment is optimized.

This guide explains Find Skill from first principles, introduces the essential skills you'll use, and shows you complete workflows from design through deployment.

## What Is Find Skill?

### The Basic Idea

Find Skill performs three tasks:

1. **Analyzes your project** (project type, tech stack, target platforms)
2. **Recommends matching skills** ("These 5-8 tools are perfect for your project")
3. **Installs with one click** ("Install all recommended skills")

### Before vs. After Find Skill

**Without Find Skill:**
```
1. Decide what tools you need (30+ minutes thinking)
2. Search Rork Community for each tool individually
3. Install each skill manually
4. Configure dependencies
5. Start coding (already 1+ hour invested)
```

**With Find Skill:**
```
1. Click "Find Skill"
2. Review 5-8 recommended skills
3. Click "Install all"
4. Start coding (5 minutes total)
```

## How Does Find Skill Work?

### Intelligent Project Analysis

Find Skill examines:
- **Project type** (mobile app, web app, backend, etc.)
- **Tech stack** (React Native, Flutter, Swift, etc.)
- **Design tools** (Figma integration needed?)
- **Target platforms** (iOS, Android, Web, multiple)

A machine learning model analyzes these signals and suggests 5-10 optimal skills.

### Typical Recommendations for Mobile Apps

For a mobile app project, expect recommendations like:

## Essential Skills for Rork Development

### 1. UI Pro Max — Non-Negotiable Foundation

**What it does:** Converts Figma designs into app-ready code.

**Key Features:**
- Design system management
- Responsive layout automation
- Dark mode generation
- Animation code export

**When you use it:**
```
Rork generates UI → Figma refinement → UI Pro Max exports code → App integration
```

**Why it's critical:** Without UI Pro Max, you manually write all UI code from Figma designs—hours of work. UI Pro Max generates production-ready components instantly.

### 2. frontend-design — Component Optimization

**What it does:** Refines React/Vue component architecture.

**Key Features:**
- Automatic component splitting
- CSS-in-JS optimization
- Performance improvement (eliminate re-render waste)

**When you use it:**
After UI Pro Max outputs code, frontend-design makes it performant and maintainable.

### 3. sleek-design-mobile-apps — Platform Compliance

**What it does:** Ensures iOS HIG and Material Design compliance.

**Key Features:**
- iOS Human Interface Guidelines checker
- Material Design (Android) compliance
- Safe area auto-configuration
- Platform-specific conventions

**Why it matters:** Mobile apps have strict design standards. This skill auto-adapts your UI to each platform's expectations.

### 4. test-automation-mobile — Quality Assurance

**What it does:** Automated testing for UI and features.

**Key Features:**
- Screenshot-based visual regression testing
- Tap/swipe operation validation
- Multi-device simultaneous testing

**Speed:** Automated tests run 10x faster than manual testing.

### 5. app-deployment-swift / app-deployment-android — Publishing

**What it does:** Automates build, sign, and store upload.

**Key Features:**
- Build configuration automation
- TestFlight / Google Play Console auto-upload
- Version management automation

**When:** Final step before store launch.

## Using Find Skill: Step-by-Step

### Step 1: Launch Find Skill

From your Rork project dashboard, top-right menu → "Skills" → "Use Find Skill"

### Step 2: Input Project Information

Fill in (or let Rork auto-detect):
- **Project name** (auto-filled)
- **App type** (Mobile / Web / Backend)
- **Target platforms** (iOS / Android / Web / multiple)
- **Tech stack** (React Native / Flutter / Swift, etc.)

### Step 3: Review Recommendations

Find Skill presents 5-10 skills with descriptions:

```
Recommended Skills
════════════════════════════════════════
✅ UI Pro Max (Strongly Recommended)
   Convert Figma designs to production code

✅ frontend-design (Recommended)
   Optimize React components

✅ sleek-design-mobile-apps (Recommended)
   iOS/Android guideline compliance

⭕ test-automation-mobile (Optional)
   Automated UI testing

⭕ app-deployment-swift (Optional)
   iOS deployment automation
```

**Read descriptions and decide honestly:** Do you need each one?

### Step 4: One-Click Installation

Click "Install all recommended skills" → Skills activate automatically in seconds.

### Step 5: Explore and Learn

Visit "My Skills" page. Each skill has:
- Brief description
- Quick start guide (Japanese)
- Implementation examples

Read the quick start for each skill you'll use.

## Common Pitfalls and Solutions

### Pitfall 1: Installing Every Recommended Skill

Find Skill recommends what *might* help. **You choose what you actually need.**

**Solution:** Read each skill description. If irrelevant to your project, skip it.

### Pitfall 2: Never Revisiting Installed Skills

Skills sit unused, cluttering your environment.

**Solution:** Monthly cleanup. Visit "My Skills" and uninstall unused ones. Clean environment = better performance.

### Pitfall 3: Skipping Skill Documentation

You install a skill but don't know how to use it.

**Solution:** **Always** read the quick-start guide and examples for each new skill. Spend 10 minutes learning = hours saved later.

## Troubleshooting

### Q: Find Skill isn't recommending skills

**A:** Check:
- Internet connection to Rork
- Project settings accurate (tech stack, platforms)
- Rork app is up to date

### Q: Skill installed but not showing in Rork

**A:** Clear cache:
```
Rork menu → Settings → Clear cache → Restart
```

### Q: Can't find a specific skill

**A:** Find Skill shows only mainstream recommendations. Use Rork Community search to find niche skills.
