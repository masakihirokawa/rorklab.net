## Introduction

QR codes are essential in modern mobile apps — from payment systems and loyalty cards to inventory management and event ticketing. With Rork Max, you can implement a high-accuracy QR code scanner powered by iOS's native AVFoundation framework and a QR code generator using CoreImage, all through natural language prompts.

This guide covers:

- Building a QR code scanner with AVFoundation
- Generating QR codes with CoreImage
- Managing scan history with UserDefaults
- Real-world use cases (event check-in, WiFi sharing, and more)

## How QR Code Scanning Works on iOS

iOS QR code scanning uses the following pipeline:

1. **AVCaptureSession** initializes the camera input stream
2. **AVCaptureMetadataOutput** detects QR code patterns in the camera feed
3. The detected QR code data is extracted and processed
4. SwiftUI overlays provide visual scanning feedback to the user

Rork Max generates all of this native Swift code from a single prompt.

### Example of Generated Code

Here is the core structure Rork Max generates for the QR scanner:

```swift
import SwiftUI
import AVFoundation

// Main QR Scanner View
struct QRScannerView: View {
    @StateObject private var scanner = QRScannerViewModel()
    @State private var showResult = false
    @State private var scannedCode = ""

    var body: some View {
        ZStack {
            // Live camera preview
            CameraPreviewView(session: scanner.session)
                .ignoresSafeArea()

            // Scanning guide overlay
            ScannerOverlayView()

            // Control buttons
            VStack {
                Spacer()
                HStack {
                    // Flashlight toggle
                    Button(action: { scanner.toggleTorch() }) {
                        Image(systemName: scanner.isTorchOn
                            ? "bolt.fill" : "bolt.slash.fill")
                            .foregroundColor(.white)
                            .padding(12)
                            .background(.ultraThinMaterial)
                            .clipShape(Circle())
                    }
                    Spacer()
                }
                .padding()
            }
        }
        .onAppear { scanner.startScanning() }
        .onDisappear { scanner.stopScanning() }
        .onChange(of: scanner.scannedValue) { value in
            if let value = value {
                scannedCode = value
                showResult = true
            }
        }
        .sheet(isPresented: $showResult) {
            ScanResultView(code: scannedCode) {
                scanner.resetScan()
                showResult = false
            }
        }
    }
}

// QR Scanner ViewModel
class QRScannerViewModel: NSObject, ObservableObject,
    AVCaptureMetadataOutputObjectsDelegate {

    @Published var scannedValue: String?
    @Published var isTorchOn = false

    let session = AVCaptureSession()
    private let metadataOutput = AVCaptureMetadataOutput()

    func startScanning() {
        checkCameraPermission()
    }

    // Called when a QR code is detected
    func metadataOutput(_ output: AVCaptureMetadataOutput,
                        didOutput metadataObjects: [AVMetadataObject],
                        from connection: AVCaptureConnection) {
        guard let metadata = metadataObjects.first
                as? AVMetadataMachineReadableCodeObject,
              let value = metadata.stringValue else { return }

        // Trigger haptic feedback
        AudioServicesPlaySystemSound(SystemSoundID(kSystemSoundID_Vibrate))

        DispatchQueue.main.async {
            self.scannedValue = value
            self.session.stopRunning() // Stop to prevent duplicate reads
        }
    }
}
```

<Callout type="info">
You must add `NSCameraUsageDescription` to `Info.plist`. Rork Max automatically includes this key when generating camera-related code.
</Callout>

## Step 2: Implement QR Code Generation

Beyond scanning, many apps need to generate QR codes to display or share with other users.

### Prompt for Rork Max

```

- Share functionality using UIActivityViewController

```

### Generated QR Code Logic

```swift
import CoreImage
import UIKit

// QR Code generation utility
struct QRCodeGenerator {
    /// Generates a QR code image from a string
    /// - Parameters:
    ///   - string: The string to encode in the QR code
    ///   - size: Output image size (default: 256x256)
    /// - Returns: Generated QR code as UIImage, or nil on failure
    static func generate(from string: String,
                         size: CGSize = CGSize(width: 256, height: 256)) -> UIImage? {
        guard let data = string.data(using: .utf8) else { return nil }

        // Use CIQRCodeGenerator filter
        let filter = CIFilter(name: "CIQRCodeGenerator")
        filter?.setValue(data, forKey: "inputMessage")
        filter?.setValue("M", forKey: "inputCorrectionLevel") // Medium error correction

        guard let ciImage = filter?.outputImage else { return nil }

        // Scale up for high resolution output
        let scaleX = size.width / ciImage.extent.size.width
        let scaleY = size.height / ciImage.extent.size.height
        let scaledImage = ciImage.transformed(
            by: CGAffineTransform(scaleX: scaleX, y: scaleY))

        // Convert CIImage to UIImage
        let context = CIContext()
        guard let cgImage = context.createCGImage(
            scaledImage, from: scaledImage.extent) else { return nil }

        return UIImage(cgImage: cgImage)
    }
}

// Expected output: A 256x256 pixel black-and-white QR code image
// Example input: "https://rorklab.net" → QR code encoding the URL
```

### Displaying in SwiftUI

```swift
struct QRGeneratorView: View {
    @State private var inputText = ""
    @State private var qrImage: UIImage?

    var body: some View {
        VStack(spacing: 24) {
            TextField("Enter text or URL", text: $inputText)
                .textFieldStyle(.roundedBorder)
                .padding(.horizontal)
                .onChange(of: inputText) { text in
                    // Update QR code in real-time as user types
                    qrImage = QRCodeGenerator.generate(from: text)
                }

            if let qrImage = qrImage {
                Image(uiImage: qrImage)
                    .resizable()
                    .interpolation(.none) // No interpolation for sharp pixels
                    .frame(width: 256, height: 256)
                    .padding()
                    .background(Color.white)
                    .cornerRadius(12)
                    .shadow(radius: 8)

                // Share button using SwiftUI ShareLink
                ShareLink(
                    item: Image(uiImage: qrImage),
                    preview: SharePreview("QR Code",
                                         image: Image(uiImage: qrImage))
                ) {
                    Label("Share QR Code", systemImage: "square.and.arrow.up")
                        .frame(maxWidth: .infinity)
                        .padding()
                        .background(Color.blue)
                        .foregroundColor(.white)
                        .cornerRadius(10)
                        .padding(.horizontal)
                }
            }
        }
        .navigationTitle("Generate QR Code")
    }
}
```

<Callout type="tip">
Setting `.interpolation(.none)` is crucial for QR codes — it keeps pixel boundaries sharp and prevents the scanner from failing to read blurry codes.
</Callout>

## Step 3: Managing Scan History

Saving and displaying scan history improves usability for apps that scan codes frequently.

```swift
// Scan history manager using UserDefaults
class ScanHistoryManager: ObservableObject {
    private let maxHistoryCount = 20
    private let storageKey = "qr_scan_history"

    @Published var history: [ScanRecord] = []

    struct ScanRecord: Codable, Identifiable {
        let id: UUID
        let value: String       // The scanned string
        let scannedAt: Date     // Timestamp of scan
        var type: CodeType      // Detected type

        enum CodeType: String, Codable {
            case url, text, email, phone
        }
    }

    init() {
        loadHistory()
    }

    /// Add a new scan result to history
    func add(_ value: String) {
        let record = ScanRecord(
            id: UUID(),
            value: value,
            scannedAt: Date(),
            type: detectType(value)
        )

        history.insert(record, at: 0) // Newest item first

        // Trim to maximum history size
        if history.count > maxHistoryCount {
            history = Array(history.prefix(maxHistoryCount))
        }

        saveHistory()
    }

    private func detectType(_ value: String) -> ScanRecord.CodeType {
        if value.hasPrefix("http://") || value.hasPrefix("https://") {
            return .url
        } else if value.hasPrefix("mailto:") {
            return .email
        } else if value.hasPrefix("tel:") {
            return .phone
        }
        return .text
    }

    private func saveHistory() {
        if let encoded = try? JSONEncoder().encode(history) {
            UserDefaults.standard.set(encoded, forKey: storageKey)
        }
    }

    private func loadHistory() {
        guard let data = UserDefaults.standard.data(forKey: storageKey),
              let decoded = try? JSONDecoder().decode(
                  [ScanRecord].self, from: data) else { return }
        history = decoded
    }
}
```

```swift
// Generate a WiFi QR code compatible with both iOS and Android
extension QRCodeGenerator {
    /// Creates a WiFi configuration QR code using the standard WIFI: URI scheme
    static func generateWiFiQR(ssid: String,
                                password: String,
                                security: String = "WPA") -> UIImage? {
        // Standard WiFi QR format recognized by both iOS and Android
        let wifiString = "WIFI:T:\(security);S:\(ssid);P:\(password);;"
        return generate(from: wifiString)
    }
}
```

## Performance Best Practices

Keep these tips in mind when integrating QR code features:

**Battery efficiency:** Always call `session.stopRunning()` after a successful scan or when the view disappears. Rork Max automatically adds this to `onDisappear` lifecycle hooks.

**Duplicate scan prevention:** Add a brief cooldown (around 0.5 seconds) after a successful scan to avoid processing the same code multiple times.

**Thread safety:** AVFoundation metadata callbacks are delivered on a background thread. Always dispatch UI updates to the main thread using `DispatchQueue.main.async`.

**QR code rendering sharpness:** When displaying generated QR codes, always use `.interpolation(.none)` and white background — dark mode or gray backgrounds can cause scanners to fail to read the code.

## FAQ

**Q1. Can I implement QR code scanning with Rork (free tier)?**

Yes, but with a different tech stack. The free Rork tier uses Expo React Native, so it leverages `expo-barcode-scanner` for QR scanning. The AVFoundation-based native Swift implementation described in this article is specific to Rork Max (native Swift code generation).

**Q2. Can I scan barcodes other than QR codes (like EAN-13 or Code128)?**

Yes. In your Rork Max prompt, specify "multi-format scanner supporting EAN-13, QR Code, and Code128." Rork Max will add those formats to the `metadataObjectTypes` array in AVCaptureMetadataOutput, enabling recognition of those barcode types.

**Q3. Can I automatically open scanned URLs in Safari?**

Yes. You can ask Rork Max to add logic that calls `UIApplication.shared.open(url)` when the scanned value is a URL. However, as a best practice, it's better to show a confirmation dialog first — auto-launching external URLs without user confirmation creates a poor user experience and may cause App Store review issues.
