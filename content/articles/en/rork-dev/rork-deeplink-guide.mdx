## What Are Deep Links?

**Deep links** open a specific screen in your app directly from a URL. Tapping `myapp://product/123` launches your app and navigates to the product with ID 123 — without any manual navigation.

Deep links enable:
- Push notifications that navigate to specific screens
- Website-to-app handoff
- Cross-app integration
- Marketing link tracking

## URL Schemes vs. Universal Links

| Method | Format | Notes |
|--------|--------|-------|
| **URL Scheme** | `myapp://path` | Easy to set up. No fallback if app isn't installed. |
| **Universal Links** (iOS) | `https://myapp.com/path` | Seamless web/app handoff. Apple recommended. |
| **App Links** (Android) | `https://myapp.com/path` | Android equivalent of Universal Links. |

## Configuration in Rork Max

### app.json setup

```json
{
  "expo": {
    "scheme": "myapp",
    "ios": {
      "associatedDomains": ["applinks:myapp.com"]
    },
    "android": {
      "intentFilters": [
        {
          "action": "VIEW",
          "autoVerify": true,
          "data": [
            {
              "scheme": "https",
              "host": "myapp.com",
              "pathPrefix": "/app"
            }
          ],
          "category": ["BROWSABLE", "DEFAULT"]
        }
      ]
    }
  }
}
```

### React Navigation Integration

Prompt Rork Max:

```
"Add deep linking to the app with these routes:
- myapp://home → Home screen
- myapp://product/:id → Product Detail screen
- myapp://profile → Profile screen
Use React Navigation."
```

Generated linking config:

```javascript
const linking = {
  prefixes: ['myapp://', 'https://myapp.com'],
  config: {
    screens: {
      Home: 'home',
      ProductDetail: 'product/:id',
      Profile: 'profile',
    },
  },
};

<NavigationContainer linking={linking}>
  {/* ... */}
</NavigationContainer>
```

## Push Notification Integration

Include the deep link URL in notification data:

```javascript
// Server side
const notification = {
  title: "New message",
  body: "You have a message from Jane",
  data: { url: "myapp://chat/user/jane" }
};

// App side
Notifications.addNotificationResponseReceivedListener((response) => {
  const url = response.notification.request.content.data?.url;
  if (url) Linking.openURL(url);
});
```

## Summary

Deep links are a significant UX upgrade that Rork Max can implement from natural language prompts.

- Configure scheme and Associated Domains in `app.json`
- Define routes with React Navigation's `linking` config
- Combine with push notifications for direct screen navigation
