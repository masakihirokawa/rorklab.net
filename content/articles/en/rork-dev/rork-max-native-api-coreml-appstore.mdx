# Rork Max Advanced — Native APIs, Core ML & App Store Publishing

Building production apps with Rork Max goes far beyond basic UI. This guide covers advanced features—machine learning, augmented reality, hardware integration, and enterprise deployment—with practical code examples and strategies for successful App Store launches.

## Part 1: Core ML Machine Learning Integration

### Core ML Fundamentals

Core ML enables on-device machine learning inference. Models run locally, protecting user privacy while maintaining sub-100ms latency. Rork Max simplifies Core ML integration to a single prompt.

#### Model Preparation

```
Step 1: Create or obtain a .mlmodel file
  - Use Create ML for custom models
  - Or download pre-trained models from ML community
Step 2: Add model to Rork Max project assets
Step 3: Reference in your prompt
```

#### Example Prompt

```
"Build a plant disease detection app:
- Camera or photo library image selection
- Inference with PlantDisease.mlmodel
- Display diagnosis (disease name, confidence score)
- Show recommended treatments for the identified disease"
```

### Generated Code Example

```swift
import CoreML
import Vision

class PlantDiseaseClassifier {
    private let model: PlantDisease

    init() throws {
        self.model = try PlantDisease(configuration: MLModelConfiguration())
    }

    func classify(_ cgImage: CGImage) async throws -> (disease: String, confidence: Float) {
        let handler = VNImageRequestHandler(cgImage: cgImage)
        var result: (String, Float)?

        let request = VNCoreMLRequest(model: try model.model()) { request, error in
            guard let results = request.results as? [VNClassificationObservation],
                  let topResult = results.first else { return }
            result = (topResult.identifier, Float(topResult.confidence))
        }

        try handler.perform([request])
        return result ?? ("Unknown", 0)
    }
}

// Usage in SwiftUI
@main
struct PlantApp: App {
    @State private var selectedImage: UIImage?
    @State private var diagnosis: (String, Float)?
    @State private var isLoading = false

    var body: some Scene {
        WindowGroup {
            VStack(spacing: 20) {
                if let image = selectedImage {
                    Image(uiImage: image)
                        .resizable()
                        .scaledToFit()
                        .frame(maxHeight: 300)
                }

                Button(action: selectPhoto) {
                    Label("Select Photo", systemImage: "photo")
                }

                if isLoading {
                    ProgressView()
                } else if let (disease, confidence) = diagnosis {
                    VStack(alignment: .leading, spacing: 8) {
                        Text("Diagnosis: \(disease)")
                            .font(.headline)
                        Text("Confidence: \(String(format: "%.1f%%", confidence * 100))")
                            .font(.subheadline)
                    }
                    .padding()
                    .background(Color.gray.opacity(0.2))
                    .cornerRadius(8)
                }
            }
            .padding()
            .onChange(of: selectedImage) { newImage in
                Task {
                    isLoading = true
                    if let cgImage = newImage?.cgImage {
                        do {
                            let classifier = try PlantDiseaseClassifier()
                            diagnosis = try await classifier.classify(cgImage)
                        } catch {
                            print("Classification error: \(error)")
                        }
                    }
                    isLoading = false
                }
            }
        }
    }

    func selectPhoto() {
        // Photo picker logic
    }
}
```

### ARKit Code Structure

```swift
import ARKit
import RealityKit

struct FurnitureARView: UIViewControllerRepresentable {
    func makeUIViewController(context: Context) -> ARViewController {
        return ARViewController()
    }

    func updateUIViewController(_ controller: ARViewController, context: Context) {}
}

class ARViewController: UIViewController, ARViewDelegate {
    @IBOutlet weak var arView: ARView!

    override func viewDidLoad() {
        super.viewDidLoad()

        // Configure AR session with LiDAR support
        var configuration = ARWorldTrackingConfiguration()

        // Enable plane detection
        configuration.planeDetection = [.horizontal, .vertical]

        // Enable LiDAR depth for occlusion
        if ARWorldTrackingConfiguration.supportsFrameSemantics(.personSegmentationWithDepth) {
            configuration.frameSemantics.insert(.personSegmentationWithDepth)
        }

        arView.session.run(configuration)
    }

    func placeFurniture(modelName: String, at position: SIMD3<Float>) {
        let anchor = try? ModelEntity.loadModel(named: modelName)
        var transform = Transform(translation: position)
        anchor?.move(to: transform)

        let anchorEntity = AnchorEntity(world: transform)
        anchorEntity.addChild(anchor!)
        arView.scene.addAnchor(anchorEntity)
    }
}
```

## Part 3: Dynamic Island & Live Activities

Dynamic Island (iPhone 14 Pro+) displays real-time information in a pill-shaped area. Live Activities enable persistent notifications for ongoing events (sports scores, delivery tracking, workout progress).

### Prompt Example

```
"Build a sports scoreboard app:
- Display live match scores (Team A vs Team B)
- Show Dynamic Island widget with current score
- Live Activities for match time and score history
- Push notifications for goals and key moments
- Real-time updates via WebSocket"
```

### Implementation Code

```swift
import ActivityKit

// Define Live Activity attributes
struct SportsScoreActivityAttributes: ActivityAttributes {
    public struct ContentState: Codable, Hashable {
        var teamA: String
        var scoreA: Int
        var teamB: String
        var scoreB: Int
        var matchTime: String
        var period: String
    }

    var matchId: String
    var competition: String
}

// Request Live Activity
func startLiveActivity(matchId: String) {
    let attributes = SportsScoreActivityAttributes(
        matchId: matchId,
        competition: "Premier League"
    )

    let initialContent = ActivityContent(
        state: SportsScoreActivityAttributes.ContentState(
            teamA: "Arsenal",
            scoreA: 2,
            teamB: "Manchester City",
            scoreB: 1,
            matchTime: "45:30",
            period: "First Half"
        ),
        staleDate: Date(timeIntervalSinceNow: 300)
    )

    do {
        let activity = try Activity<SportsScoreActivityAttributes>.request(
            attributes: attributes,
            content: initialContent
        )
        print("Live Activity started: \(activity.id)")
    } catch {
        print("Failed to start Live Activity: \(error)")
    }
}

// Update Live Activity in real-time
func updateLiveActivity(with newScore: SportsScoreActivityAttributes.ContentState) async {
    let activities = Activity<SportsScoreActivityAttributes>.all

    for activity in activities {
        var updatedContent = activity.content
        updatedContent.state = newScore
        updatedContent.staleDate = Date(timeIntervalSinceNow: 300)

        await activity.update(updatedContent)
    }
}
```


### Code Implementation

```swift
import HealthKit

class HealthKitManager: ObservableObject {
    @Published var todaySteps: Int = 0
    @Published var heartRate: Int = 0
    @Published var sleepHours: Double = 0

    private let healthStore = HKHealthStore()

    func requestAuthorization() async throws {
        let readTypes: Set = [
            HKObjectType.quantityType(forIdentifier: .stepCount)!,
            HKObjectType.quantityType(forIdentifier: .heartRate)!,
            HKObjectType.categoryType(forIdentifier: .sleepAnalysis)!
        ]

        try await healthStore.requestAuthorization(
            toShare: [],
            read: readTypes
        )
    }

    func fetchTodaySteps() async {
        let stepsType = HKQuantityType.quantityType(forIdentifier: .stepCount)!
        let today = Calendar.current.startOfDay(for: Date())
        let predicate = HKQuery.predicateForSamples(withStart: today, end: Date())

        let query = HKStatisticsQuery(
            quantityType: stepsType,
            quantitySamplePredicate: predicate,
            options: .cumulativeSum
        ) { _, result, _ in
            guard let result = result,
                  let sum = result.sumQuantity() else { return }

            DispatchQueue.main.async {
                self.todaySteps = Int(sum.doubleValue(for: HKUnit.count()))
            }
        }

        healthStore.execute(query)
    }

    func fetchSleepData(for date: Date = Date()) async {
        let sleepType = HKObjectType.categoryType(forIdentifier: .sleepAnalysis)!
        let calendar = Calendar.current
        let startOfDay = calendar.startOfDay(for: date)
        let endOfDay = calendar.date(byAdding: .day, value: 1, to: startOfDay)!

        let predicate = HKQuery.predicateForSamples(
            withStart: startOfDay,
            end: endOfDay
        )

        let query = HKSampleQuery(
            sampleType: sleepType,
            predicate: predicate,
            limit: HKObjectQueryNoLimit,
            sortDescriptors: nil
        ) { _, results, _ in
            guard let sleepSamples = results as? [HKCategorySample] else { return }

            let totalSeconds = sleepSamples.reduce(0) { total, sample in
                total + sample.endDate.timeIntervalSince(sample.startDate)
            }

            DispatchQueue.main.async {
                self.sleepHours = totalSeconds / 3600
            }
        }

        healthStore.execute(query)
    }
}
```

## Part 5: HomeKit for Smart Home Control

HomeKit provides a unified protocol for controlling smart home devices—lights, thermostats, locks, cameras, and more. Rork Max handles HomeKit framework integration automatically.

### Prompt Example

```
"Create a smart home control app:
- List all HomeKit accessory devices
- Toggle lights and outlets on/off
- Adjust brightness and color temperature
- Control thermostat temperature
- Create and activate scenes (Good Morning, Goodnight)
- Siri integration for voice commands"
```

### HomeKit Implementation

```swift
import HomeKit

class HomeKitManager: NSObject, HMHomeManagerDelegate {
    @Published var homes: [HMHome] = []
    @Published var accessories: [HMAccessory] = []

    private let homeManager = HMHomeManager()

    override init() {
        super.init()
        homeManager.delegate = self
    }

    func homeManagerDidUpdateHomes(_ manager: HMHomeManager) {
        DispatchQueue.main.async {
            self.homes = manager.homes
            self.loadAccessories()
        }
    }

    func loadAccessories() {
        accessories = homes.flatMap { $0.accessories }
    }

    func toggleLight(_ accessory: HMAccessory) {
        guard let service = accessory.services.first(
            where: { $0.serviceType == HMServiceTypeLightbulb }
        ) else { return }

        guard let powerCharacteristic = service.characteristics.first(
            where: { $0.characteristicType == HMCharacteristicTypePowerState }
        ) else { return }

        let newValue = !(powerCharacteristic.value as? NSNumber)?.boolValue ?? false
        powerCharacteristic.writeValue(newValue) { error in
            if let error = error {
                print("Error toggling light: \(error.localizedDescription)")
            }
        }
    }

    func setBrightness(_ accessory: HMAccessory, to level: Int) {
        guard let service = accessory.services.first(
            where: { $0.serviceType == HMServiceTypeLightbulb }
        ) else { return }

        guard let brightCharacteristic = service.characteristics.first(
            where: { $0.characteristicType == HMCharacteristicTypeBrightness }
        ) else { return }

        brightCharacteristic.writeValue(level) { error in
            if let error = error {
                print("Error adjusting brightness: \(error.localizedDescription)")
            }
        }
    }
}
```

### Metal Setup Code

```swift
import MetalKit

class MetalViewController: UIViewController, MTKViewDelegate {
    var metalView: MTKView!
    var commandQueue: MTLCommandQueue?
    var renderPipelineState: MTLRenderPipelineState?

    override func viewDidLoad() {
        super.viewDidLoad()

        guard let device = MTLCreateSystemDefaultDevice() else {
            fatalError("Metal not supported on this device")
        }

        metalView = MTKView(frame: view.bounds, device: device)
        metalView.delegate = self
        view.addSubview(metalView)

        commandQueue = device.makeCommandQueue()
        setupRenderPipeline(device: device)
    }

    func setupRenderPipeline(device: MTLDevice) {
        guard let library = device.makeDefaultLibrary() else { return }

        let pipelineDescriptor = MTLRenderPipelineDescriptor()
        pipelineDescriptor.vertexFunction = library.makeFunction(name: "vertexShader")
        pipelineDescriptor.fragmentFunction = library.makeFunction(name: "fragmentShader")
        pipelineDescriptor.colorAttachments[0].pixelFormat = .bgra8Unorm

        renderPipelineState = try? device.makeRenderPipelineState(descriptor: pipelineDescriptor)
    }

    func mtkView(_ view: MTKView, drawableSizeWillChange size: CGSize) {}

    func draw(in view: MTKView) {
        guard let drawable = view.currentDrawable,
              let descriptor = view.currentRenderPassDescriptor,
              let commandBuffer = commandQueue?.makeCommandBuffer(),
              let renderEncoder = commandBuffer.makeRenderCommandEncoder(descriptor: descriptor),
              let pipelineState = renderPipelineState else { return }

        renderEncoder.setRenderPipelineState(pipelineState)
        // Render geometry here
        renderEncoder.endEncoding()

        commandBuffer.present(drawable)
        commandBuffer.commit()
    }
}
```

### Cloud Mac Fleet Deployment

```
Setup Steps:

2. Enable "Cloud Mac Fleet"
3. Select build configuration:
   - Development (for testing)
   - Release (for App Store)
4. Configure automatic builds on git push
5. Enable TestFlight auto-distribution
```

### GitHub Actions CI/CD Pipeline

```yaml
name: Rork Max CI/CD
on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: macos-latest
    steps:
      - uses: actions/checkout@v3
      - name: Build and Test
        run: |
          xcodebuild -scheme RorkApp \
          -configuration Debug \
          -destination 'platform=iOS Simulator,name=iPhone 15' \
          test

  build:
    needs: test
    if: github.ref == 'refs/heads/main'
    runs-on: macos-latest
    steps:
      - uses: actions/checkout@v3
      - name: Build Release
        run: |
          xcodebuild -scheme RorkApp \
          -configuration Release \
          -archivePath build/RorkApp.xcarchive \
          archive
      - name: Upload to TestFlight
        run: xcrun altool --upload-app -f build/RorkApp.ipa --apiKey ${{ secrets.APPSTORE_API_KEY }}
```
