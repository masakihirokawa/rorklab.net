```
Organized by:
```

```
Structure:

```


## Integrating UI Pro Max with Rork Max

### Step 1: Export Design Tokens

From UI Pro Max:
```
uipromax export \
  --format=rork \
  --include=colors,typography,components,spacing

Generated files:
✓ colors.json (97 palettes)
✓ typography.json (57 font pairs)
✓ components.json (50+ styles)
✓ spacing.json (scale definitions)
✓ shadows.json (elevation system)
✓ borders.json (radius system)
```

### Step 2: Import into Rork Max

```
rork integrate-design-system \
  --system=ui-pro-max \
  --import=./uipromax-export/

Automatic processing:
✓ Design token conversion
✓ React Native style generation
✓ TypeScript type definitions
✓ Component auto-generation
```

## Typography: Font Pairing Strategy

### The Three-Tier Typography System

**Tier 1: Heading Font** (Dominant visual element)
```
Typical choices:
- Inter (Modern, friendly)
- Poppins (Bold, trendy)
- SF Pro Display (Apple-native feel)
- Roboto (Google Material)

Weight: Bold (700-800) for emphasis
```

**Tier 2: Body Font** (Reading comfort)
```
Typical choices:
- Inter (same as headings for consistency)
- Open Sans (proven readability)
- SF Pro Text (Apple-native feel)
- Lato (Web-optimized)

Weight: Regular (400) for body text, Medium (500) for emphasis
```

**Tier 3: Monospace Font** (Code/numbers)
```
Typical choices:
- Fira Code (developer-friendly)
- Menlo (Apple default)
- Roboto Mono (Material default)
- JetBrains Mono (premium feel)

Usage: Prices, technical values, timestamps
```

## Spacing & Sizing System

UI Pro Max defines a consistent spacing scale:

```
Spacing scale (in pixels):
- xs: 4px      (extra small)
- sm: 8px      (small)
- md: 16px     (medium, default)
- lg: 24px     (large)
- xl: 32px     (extra large)
- 2xl: 48px    (2× large)
- 3xl: 64px    (3× large)

Usage guidelines:
- Between paragraphs: md (16px)
- Between sections: lg (24px) or xl (32px)
- Inside containers: sm (8px) to md (16px)
- Page margins: lg (24px) or xl (32px)

Implementation in Rork:
Define spacing constants and apply consistently
across all layouts.
```

```
Color Contrast Ratios:
- Text on background: 7:1 minimum
- UI components: 4.5:1 minimum
- Large text (18px+): 3:1 minimum

Font Readability:
- Dyslexia-friendly options available
- Monospace for technical content
- Adequate line height (1.5-1.75)

Spacing Accessibility:
- Touch targets minimum 44×44px
- Adequate spacing between interactive elements
- Clear visual hierarchy
```

## Performance Optimization

Implementing enterprise design systems effectively:

```
Best Practices:

   (don't create new arbitrary values)

   (from the 97 available)

4. Apply typography system strictly
   (don't mix font families excessively)
```
