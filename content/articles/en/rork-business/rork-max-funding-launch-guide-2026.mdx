## How to Join the Early Stage Program

### Step-by-Step Guide

#### Step 1: Create Rork Max Account

Visit [https://rork.com](https://rork.com) and sign up with your email.

```
1. Click "Sign Up"
2. Enter email address
3. Verify email link
4. Set password
```

#### Step 2: Access the Program Application

Open your Rork Max dashboard and navigate to **"Programs"** or **"Startup"** (left sidebar).

```
Dashboard > Programs > Early Stage
↓
Click "Apply Now"
```

#### Step 3: Complete Application Form

Required information:

- **App name** (English and Japanese if applicable)
- **App description** (200 characters)
- **Target user persona** (e.g., students, creators, professionals)
- **Current stage** (Ideation / In development / Beta / Ready to launch)
- **Problem statement** (Why build this?)
- **Competitive advantage** (Why Rork Max specifically?)

#### Step 4: Review Period

Expect response within 3-7 business days.

- **Approved**: Pro features unlock immediately
- **Not approved**: Feedback provided + reapplication option available

#### Step 5: Verify Pro Access

Check your dashboard to confirm Pro version is active:

```
Settings > Subscription > Current Plan: "Early Stage (Pro - Free)"
```

### Troubleshooting Common Issues

| Problem | Cause | Solution |
|---------|-------|----------|
| "You don't qualify" | Already purchased Pro | Contact support for refund |
| No confirmation email | Spam folder | Check Spam/Promotions |
| No response after 10 days | Application unclear | Email support@rork.com directly |

## Critical Launch Considerations

### Consideration 1: App Store Review Standards

Apple has tightened AI app reviews, especially for LLM integrations.

**What won't pass**:
- Fake "AI diagnosis" (medical regulation)
- Content generation from copyrighted material
- User data used for training without consent

**What does pass**:
- AI features with clear disclaimers
- Creative Commons-only content generation
- Explicit user consent for data usage

### Consideration 2: API Cost Management

ChatGPT API usage can become surprisingly expensive:

```
gpt-3.5-turbo: $0.001 per 1K tokens
gpt-4: $0.03 per 1K tokens

If each user uses 100 tokens/day:
Monthly cost per user: 30 × 100 × $0.001 = $3
100 users: $300/month
```

**Solutions**:
- [ ] Implement API caching
- [ ] Set usage limits per user
- [ ] Target 30%+ profit margin

### Consideration 3: Data Privacy (GDPR Compliance)

If you'll have EU users, compliance is required:

**Checklist**:
- [ ] Add GDPR clause to privacy policy
- [ ] Enable user data deletion rights
- [ ] Add cookie consent banner

Stripe handles most compliance automatically.

### Q1: I wasn't approved for Early Stage. Should I buy Pro?

**A**: Consider this priority:

1. **Is free sufficient?** Free tier allows 5 apps—often enough
2. **Testing phase?** 1-2 months testing? Free is fine
3. **Quick monetization needed?** Start free, upgrade after first revenue

**Cost optimization path**:
```
Months 1-2: Free tier testing
Month 3: Upgrade to Pro ($29/month)
Months 4-6: Build with Pro
Month 7+: Stay if revenue covers, drop if not
```

### Q2: Can Rork Max apps really ship on App Store?

**A**: Yes. With caveats:

- ✅ iOS: Full support
- ✅ Android: Full support
- ⚠️ Web/PWA: Possible but feature-limited

Apps built with Rork Max ship daily to App Store and Google Play. The platform quality level is "mid-market"—simple to moderate complexity apps work perfectly.

### Q3: How do I handle scaling and maintenance?

**A**: Plan for evolution:

```
Phase 1 (0-10k users): Rork Max is fine
Phase 2 (10k-100k): Rork Max + external APIs
Phase 3 (100k+): Consider custom development
```

Use Early Stage mentorship to guide these decisions.

### Q4: What about taxes when my app generates revenue?

**A**: You'll need to file as self-employed:

- **Threshold**: $100K annual revenue in most countries
- **Tax rate**: ~20% (varies by location)
- **Filing**: Use Stripe's revenue reports for accuracy

Consult a local accountant for your jurisdiction.
