## What is Claude Code?

Claude Code is a command-line tool from Anthropic that lets you delegate coding tasks to Claude directly from your terminal. It deeply integrates with your development workflow — reading and writing files, running Git operations, executing tests, and more.

<Callout type="info">
Claude Code requires a Max plan ($100/month) or an API key. It is not available on the Pro plan ($20/month).
</Callout>

## Installation

### Prerequisites

- Node.js 18 or later
- npm or yarn
- Anthropic account (Max plan or API key)

### Install Command

```bash
npm install -g @anthropic-ai/claude-code
```

Verify the installation:

```bash
claude --version
```

## Initial Setup

Running `claude` for the first time starts the authentication flow.

```bash
claude
```

A browser window will open asking you to log in with your Anthropic account. Once authenticated, Claude Code is ready to use in your terminal.

## Basic Usage

### Interactive Mode (Default)

```bash
claude
```

In interactive mode, you can have a back-and-forth conversation with Claude while working on your project.

### One-Shot Mode

```bash
claude -p "Translate the README.md in this directory to Japanese"
```

The `-p` flag executes a single task. This is useful for scripts and CI/CD pipelines.

### Common Commands

| Command | Description |
|---------|-------------|
| `claude` | Start interactive mode |
| `claude -p "..."` | Execute a one-shot task |
| `claude -r` | Resume the last session |
| `claude config` | View/change settings |
| `claude mcp` | Manage MCP servers |

## Slash Commands

Inside interactive mode, you can use slash commands:

```
/help        — Show help
/clear       — Clear conversation history
/compact     — Summarize and compress context
/cost        — Show token usage for current session
/doctor      — Diagnose configuration issues
/init        — Create a CLAUDE.md project config file
/review      — Start a code review
/terminal-setup — Configure terminal key bindings
```

<Callout type="tip">
The `CLAUDE.md` file created by `/init` is where you describe your project's coding conventions and architecture. Claude Code reads it automatically to generate context-aware code.
</Callout>

## Configure Your Project with CLAUDE.md

Create a `CLAUDE.md` file at your project root to give Claude Code context about your project.

```markdown
# Project Overview
Next.js 16 + TypeScript web application

# Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS v4
- Cloudflare Workers

# Coding Conventions
- Use function components
- TypeScript strict mode
- Tests with vitest
```

## MCP (Model Context Protocol)

Claude Code can connect to external services through MCP servers.

```bash
# Add an MCP server
claude mcp add slack -- npx -y @anthropic-ai/slack-mcp

# List registered servers
claude mcp list
```

<Callout type="warning">
MCP servers run in your local environment. Only install from trusted sources.
</Callout>

## Frequently Asked Questions

### Is Claude Code available on the Pro plan?

No, Claude Code requires a Max plan ($100/month or higher) or an API key. It's not available on Pro ($20/month). Individual developers can alternatively use the Claude API with pay-as-you-go pricing for Claude Code functionality.

### Is the CLAUDE.md file required?

No, it's optional but highly recommended. By documenting your project's structure, tech stack, and coding conventions in CLAUDE.md, Claude Code generates more accurate and consistent code. Use `/init` to create one when starting a new project.

### Can I use Claude Code offline?

No, Claude Code requires internet connectivity for API communication with Anthropic's servers. However, once code is generated and configuration files are created, you can edit and run them offline. The generated files remain on your local machine.

### Can I create custom MCP servers?

Yes, you can build your own MCP (Model Context Protocol) servers and integrate them with Claude Code. MCP follows an open standard, allowing custom implementations. Always verify the source and security of any MCP server before installation, especially third-party ones.
