## What Are Artifacts?

**Artifacts** is a feature available on Claude.ai that lets you generate content and preview it in real-time during your conversation. Code execution results, React components, SVG graphics, and Mermaid diagrams appear in a panel alongside the chat window.

## Supported Content Types

### Code Snippets

Write code in Python, JavaScript, TypeScript, and more, displayed with syntax highlighting and a convenient copy button.

### React Components

Render interactive UI components on the spot. Tailwind CSS utility classes are available, along with React Hooks like `useState` and `useEffect`.

```
"Create a counter app with dark mode toggle as an Artifact"
```

### SVG Graphics

Preview vector images directly. Great for logo design, icons, and simple illustrations.

### Mermaid Diagrams

Generate flowcharts, sequence diagrams, ER diagrams, and Gantt charts using Mermaid syntax, rendered visually.

```
"Create a user registration flowchart in Mermaid"
```

### HTML Pages

Preview complete HTML pages with CSS and JavaScript. Perfect for landing page prototypes and interactive demos.

### Markdown Documents

Display structured documents with formatting. Useful for drafting reports, guides, and documentation.

## Practical Use Cases

### Prototyping

Quickly visualize ideas by describing what you want. Tell Claude "what about a UI like this" and get a working React component.

### Data Visualization

Pass CSV data and ask for "a bar chart" to get an interactive Recharts visualization as an Artifact.

### Technical Documentation

Create architecture and sequence diagrams in Mermaid to embed in your technical docs.

### Learning and Education

Visualize algorithm behavior through animations or interactively explore mathematical concepts.

## Tips for Best Results

1. **Be specific**: "Create a dashboard with a line chart for sales data and a pie chart for goal achievement" works better than "make a dashboard"
2. **Iterate gradually**: Review the first version, then refine with instructions like "use more muted colors" or "add tooltips to the chart"
3. **Specify libraries**: Mention Recharts, D3, Three.js, or other libraries you want, and Claude will generate code using them

## Available Libraries

The following libraries can be imported in Artifacts:

- **React** — Components and Hooks
- **Recharts** — Charts and graphs
- **Lucide React** — Icon set
- **D3.js** — Data visualization
- **Three.js** — 3D graphics
- **Tailwind CSS** — Utility-first CSS
- **shadcn/ui** — UI components

## Summary

Artifacts transforms Claude from a text-based chat into an interactive creative tool. Try it for prototyping, data visualization, documentation, and more.
