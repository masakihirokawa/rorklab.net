# Building Your First AI Companion App with React Native

Have you ever wanted to create your own AI companion app? With Expo and React Native, you can build a chat interface that talks to an AI in just a few hours. In this guide, we'll create a simple but functional AI partner app from scratch.

---

## What We're Building

We'll create a basic mobile app where users can chat with an AI companion. The app will:
- Display chat messages in conversation bubbles
- Send text to Claude or OpenAI's API
- Persist conversations using AsyncStorage
- Show a simple persona for the AI character

Think of it like a beginner's version of Replika or Character.ai—nothing fancy, but genuinely functional.

---

## Project Setup

Start a new Expo project:

```bash
npx create-expo-app RorkLabCompanion
cd RorkLabCompanion
npx expo install
```

Install the dependencies we'll need:

```bash
npm install @react-native-async-storage/async-storage axios
npx expo install @react-native-async-storage/async-storage
```

## Basic Chat UI Component

First, let's create the core chat interface. This is a ScrollView with chat bubbles:

```typescript
// app/screens/ChatScreen.tsx
import React, { useState, useEffect } from 'react';
import {
  View,
  Text,
  ScrollView,
  TextInput,
  TouchableOpacity,
  StyleSheet,
  KeyboardAvoidingView,
  Platform,
} from 'react-native';
import AsyncStorage from '@react-native-async-storage/async-storage';

interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: number;
}

export default function ChatScreen() {
  const [messages, setMessages] = useState<Message[]>([]);
  const [input, setInput] = useState('');
  const [loading, setLoading] = useState(false);

  // Load saved messages on mount
  useEffect(() => {
    loadMessages();
  }, []);

  const loadMessages = async () => {
    try {
      const saved = await AsyncStorage.getItem('chatHistory');
      if (saved) {
        setMessages(JSON.parse(saved));
      }
    } catch (error) {
      console.error('Failed to load messages:', error);
    }
  };

  const saveMessages = async (msgs: Message[]) => {
    try {
      await AsyncStorage.setItem('chatHistory', JSON.stringify(msgs));
    } catch (error) {
      console.error('Failed to save messages:', error);
    }
  };

  const sendMessage = async () => {
    if (!input.trim()) return;

    const userMessage: Message = {
      id: Date.now().toString(),
      role: 'user',
      content: input,
      timestamp: Date.now(),
    };

    const updated = [...messages, userMessage];
    setMessages(updated);
    setInput('');
    setLoading(true);

    try {
      const response = await getAIResponse(updated);
      const assistantMessage: Message = {
        id: (Date.now() + 1).toString(),
        role: 'assistant',
        content: response,
        timestamp: Date.now(),
      };

      const finalMessages = [...updated, assistantMessage];
      setMessages(finalMessages);
      saveMessages(finalMessages);
    } catch (error) {
      console.error('API Error:', error);
      setMessages(updated);
    } finally {
      setLoading(false);
    }
  };

  return (
    <KeyboardAvoidingView
      style={styles.container}
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
    >
      <ScrollView style={styles.messageContainer}>
        {messages.length === 0 && (
          <View style={styles.welcomeContainer}>
            <Text style={styles.welcomeText}>Hey! I'm your AI companion.</Text>
            <Text style={styles.welcomeSubtext}>
              Say hello to start chatting!
            </Text>
          </View>
        )}
        {messages.map((msg) => (
          <View
            key={msg.id}
            style={[
              styles.messageBubble,
              msg.role === 'user' ? styles.userMessage : styles.aiMessage,
            ]}
          >
            <Text style={styles.messageText}>{msg.content}</Text>
          </View>
        ))}
        {loading && (
          <View style={styles.aiMessage}>
            <Text style={styles.messageText}>Thinking...</Text>
          </View>
        )}
      </ScrollView>

      <View style={styles.inputContainer}>
        <TextInput
          style={styles.input}
          placeholder="Type your message..."
          value={input}
          onChangeText={setInput}
          multiline
          editable={!loading}
        />
        <TouchableOpacity
          style={[styles.sendButton, loading && styles.sendButtonDisabled]}
          onPress={sendMessage}
          disabled={loading}
        >
          <Text style={styles.sendButtonText}>Send</Text>
        </TouchableOpacity>
      </View>
    </KeyboardAvoidingView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  messageContainer: {
    flex: 1,
    padding: 16,
  },
  welcomeContainer: {
    paddingVertical: 40,
    alignItems: 'center',
  },
  welcomeText: {
    fontSize: 18,
    fontWeight: '600',
    color: '#333',
    marginBottom: 8,
  },
  welcomeSubtext: {
    fontSize: 14,
    color: '#999',
  },
  messageBubble: {
    marginVertical: 8,
    maxWidth: '85%',
    paddingHorizontal: 12,
    paddingVertical: 10,
    borderRadius: 12,
  },
  userMessage: {
    alignSelf: 'flex-end',
    backgroundColor: '#007AFF',
  },
  aiMessage: {
    alignSelf: 'flex-start',
    backgroundColor: '#f0f0f0',
  },
  messageText: {
    fontSize: 16,
    color: '#000',
  },
  inputContainer: {
    flexDirection: 'row',
    padding: 12,
    borderTopWidth: 1,
    borderTopColor: '#eee',
    backgroundColor: '#fff',
  },
  input: {
    flex: 1,
    borderWidth: 1,
    borderColor: '#ddd',
    borderRadius: 20,
    paddingHorizontal: 16,
    paddingVertical: 10,
    marginRight: 8,
    maxHeight: 100,
  },
  sendButton: {
    width: 44,
    height: 44,
    borderRadius: 22,
    backgroundColor: '#007AFF',
    justifyContent: 'center',
    alignItems: 'center',
  },
  sendButtonDisabled: {
    opacity: 0.5,
  },
  sendButtonText: {
    color: '#fff',
    fontWeight: '600',
  },
});
```

## Connecting to Claude API

Now let's add the AI API integration. Create a `services/api.ts` file:

```typescript
// services/api.ts
import axios from 'axios';

const CLAUDE_API_KEY = 'your-api-key-here';
const SYSTEM_PROMPT = `You are a friendly AI companion. You're here to chat,
listen, and provide support. Keep responses concise and warm. Show genuine interest
in the user's life.`;

interface Message {
  role: 'user' | 'assistant';
  content: string;
}

export async function getAIResponse(messages: Message[]): Promise<string> {
  try {
    const response = await axios.post(
      'https://api.anthropic.com/v1/messages',
      {
        model: 'claude-3-5-sonnet-20241022',
        max_tokens: 1024,
        system: SYSTEM_PROMPT,
        messages: messages.map((m) => ({
          role: m.role,
          content: m.content,
        })),
      },
      {
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': CLAUDE_API_KEY,
          'anthropic-version': '2023-06-01',
        },
      }
    );

    return response.data.content[0].text;
  } catch (error) {
    console.error('Error calling Claude API:', error);
    throw error;
  }
}
```

## Important: Handling API Keys Safely

**Never hardcode API keys in your app!** For production, you need a backend service. For now, this is just for learning. In the beginner phase, consider:

- Creating a simple Node.js backend that proxies requests
- Using environment variables with eas build
- Never shipping API keys in your app binary

## Persistence with AsyncStorage

We've already added AsyncStorage to save conversations. Users can close the app and their chat history will be there when they return:

```typescript
const saveMessages = async (msgs: Message[]) => {
  try {
    await AsyncStorage.setItem('chatHistory', JSON.stringify(msgs));

  }
};
```

## Important: AI Limitations & Honesty

Remember: your AI companion is just a language model. It doesn't truly understand emotions, have memory across sessions (unless you implement it), or feel affection. Be honest with your users about these limitations. This builds trust and sets healthy expectations.
