## What Is Aqua Voice?

Aqua Voice is an AI-powered voice dictation tool for Mac and Windows, developed by a Y Combinator (W24) startup. It features a proprietary speech recognition model called "Avalon" that sets it apart from generic dictation tools.

The key differentiator is its ability to **read screen context to improve recognition accuracy**. For programming terms and technical jargon, Aqua Voice achieves approximately 17x better accuracy than Siri or Google Voice Typing, making it especially popular among developers and technical writers.

## Why It Works Great with Claude

Whether you're using Cowork or Claude Code, interacting with Claude is fundamentally text-based. The more complex your instructions, the more typing you need to do.

Aqua Voice inserts text directly into whatever text field has focus — Cowork's chat input, Claude Code's terminal prompt, or any browser input field. It works anywhere your cursor is.

Since speaking averages around 150 words per minute (roughly 3-4x faster than typing), the time savings become significant when crafting detailed prompts.

## Setup

### 1. Installation

Download and install from the official website (aquavoice.com). It supports Mac (Apple Silicon and Intel) and Windows 10/11.

### 2. Basic Usage

After installation, Aqua Voice runs in the menu bar (Mac) or system tray (Windows). Press your configured shortcut key to start dictation.

The shortcut is fully customizable. Startup time is under 50ms, and text insertion takes about 1 second (as fast as 450ms).

### 3. Custom Dictionary

Register technical terms and proper nouns in the custom dictionary for better recognition. For example:

- `Claude Code` — AI coding tool name
- `Cowork` — Claude desktop feature
- `MCP` — Model Context Protocol abbreviation
- `SKILL.md` — Skill definition file name

## Practical Usage

### With Cowork

Place your cursor in Cowork's chat input, activate Aqua Voice, and simply speak your instructions.

For example, you can say something like "Read all the PDFs in this folder and create an Excel file summarizing their contents" — and it types it out for you. Aqua Voice automatically adds punctuation and formatting, so the output reads naturally.

### With Claude Code

Claude Code runs in the terminal, and Aqua Voice works directly with terminal input fields.

Technical terms, commands, and file paths are recognized with high accuracy thanks to screen context reading. Programming terms like `useEffect`, `kubectl`, and `PyTorch` achieve 97% recognition accuracy.

### Paste Shortcut

A dedicated shortcut lets you re-paste the last transcribed text:

- **Mac**: `Cmd + Ctrl + V`
- **Windows**: `Ctrl + Shift + →`

This is useful when you want to send the same instruction to a different window.

## Comparison with Claude's Built-in Voice Modes

Claude already offers several voice input options.

### Claude.ai Voice Mode

Claude.ai (web and mobile) includes a built-in conversational voice mode for real-time voice conversations, but it's limited to the web interface.

### Claude Code /voice Command

Claude Code has a built-in `/voice` command — hold Space, speak, and release to transcribe. It's convenient but relies on generic speech recognition.

### Where Aqua Voice Excels

Aqua Voice differentiates itself in several ways:

1. **Works everywhere**: Cowork, Claude Code, browsers, email, Slack — anywhere there's a text cursor
2. **Technical term recognition**: Reads screen context to accurately transcribe programming terminology
3. **49 languages**: Natural switching between languages including Japanese and English
4. **Low latency**: Under 50ms startup, approximately 1 second text insertion

That said, Claude Code's `/voice` requires no installation and works out of the box, making it perfectly adequate for quick, simple prompts. Aqua Voice shines when you need to dictate longer prompts or require precise technical terminology.
