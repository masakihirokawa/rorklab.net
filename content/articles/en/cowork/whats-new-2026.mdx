## Scheduled Tasks — Automation at Scale

One of the most powerful additions to Cowork in 2026 is **scheduled tasks**. What once required manual execution can now run automatically on a regular schedule.

### Flexible Scheduling with Cron Expressions

Scheduled tasks support standard cron syntax for precise control over when tasks run:

```
0 9 * * *        # Every day at 9:00 AM
0 9 * * 1-5      # Weekdays (Mon-Fri) at 9:00 AM
30 8 * * 1       # Every Monday at 8:30 AM
0 0 1 * *        # First day of each month at midnight
```

### On-Demand Execution

Beyond scheduled runs, you can **manually trigger any task** at any time. Need to run something right now? Execute it immediately without waiting for the next scheduled window.

### Stored as SKILL.md Files

All tasks are saved as `SKILL.md` Markdown files, making task definitions text-based and easily manageable. This approach enables:

- Version control integration
- Simple sharing with team members
- Easy editing in any text editor
- Clear documentation of task logic

```
# Daily Sales Report

Schedule: 0 9 * * *
Timezone: America/New_York

This task generates yesterday's sales report
and emails it to the leadership team with
key metrics and insights.
```

<Callout type="info">
Because SKILL.md files are plain text, you can manage complex task definitions using your favorite editor, and track changes using Git or other version control systems.
</Callout>

## Global Folder Instructions — Customize Your Workspace

New in 2026: **Global Folder Instructions** let you define how AI should behave in specific directories.

### Folder-Specific Guidelines

Create a `.claude-instructions` file in any folder to provide context-specific guidance:

```
# .claude-instructions

This folder contains client presentations.

When working with files here:
- Follow corporate brand guidelines
- Use only Arial or Helvetica fonts
- Stick to brand colors (RGB: 0, 102, 204)
- Include company logo on every slide
- Add legal disclaimer footer
- Proofread for accuracy before finalizing
```

### Benefits

- **Consistency** — Automatic adherence to style guides and standards
- **Efficiency** — No need to repeat instructions for every task
- **Team Alignment** — Ensures uniform quality and branding across projects
- **Knowledge Preservation** — Guidelines are documented and shared
