## What Is Pollo AI?

Pollo AI is an all-in-one platform for generating high-quality AI videos from text and images. It integrates multiple industry-leading video generation models — including Kling AI, Hailuo AI, Runway, PixVerse, Vidu AI, and Luma AI — into a single interface, letting you choose the best model for your purpose and style.

The platform also features Pollo AI's proprietary model, Pollo 2.5, known for its high prompt fidelity and cost-effectiveness. As of 2026, over 40 AI video effects and templates are available, making it easy for beginners to create professional-quality videos.

The main generation modes include:

- **Text to Video**: Generate videos from text prompts
- **Image to Video**: Add motion to still images to create videos
- **Video to Video**: Apply style transformations to existing videos
- **AI Animation**: Automatically generate character animations
- **Consistent Character**: Generate continuous scenes maintaining the same character

Pollo AI also supports Japanese prompts, making it accessible to non-English speakers.

## Automating with Claude Code × Pollo AI API

### Getting Your API Key

Create an account on the Pollo AI API platform (pollo.ai/api-platform) and obtain your API key. The API provides programmatic access to the same features available in the web UI, including Text to Video and Image to Video generation.

### Creating a Generation Script with Claude Code

Ask Claude Code to build an API integration script:

```
Write a Python script that generates videos from text prompts
using the Pollo AI API. Include:
- API key loaded from environment variables
- Prompt and model name as arguments
- Generation status polling
- Download of completed video
```

Here's the basic script structure Claude Code might generate:

```python
import os
import time
import requests

API_KEY = os.environ.get("POLLO_API_KEY")
BASE_URL = "https://api.pollo.ai/v1"

def generate_video(prompt: str, model: str = "pollo-2.5",
                   aspect_ratio: str = "16:9",
                   duration: int = 5) -> str:
    """Generate a video from text using the Pollo AI API."""
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    }

    # Submit generation request
    response = requests.post(
        f"{BASE_URL}/video/generate",
        headers=headers,
        json={
            "prompt": prompt,
            "model": model,
            "aspect_ratio": aspect_ratio,
            "duration": duration,
        }
    )
    response.raise_for_status()
    task_id = response.json()["task_id"]

    # Poll for status
    while True:
        status = requests.get(
            f"{BASE_URL}/video/status/{task_id}",
            headers=headers
        ).json()

        if status["state"] == "completed":
            return status["video_url"]
        elif status["state"] == "failed":
            raise RuntimeError(f"Generation failed: {status.get('error')}")

        print(f"Generating... ({status['state']})")
        time.sleep(10)


def download_video(url: str, output_path: str) -> None:
    """Download the generated video."""
    response = requests.get(url, stream=True)
    response.raise_for_status()
    with open(output_path, "wb") as f:
        for chunk in response.iter_content(chunk_size=8192):
            f.write(chunk)
    print(f"Saved: {output_path}")


if __name__ == "__main__":
    prompt = (
        "A serene Japanese garden with cherry blossoms "
        "gently falling, koi pond reflecting sunlight, "
        "slow camera pan, cinematic 4K quality"
    )

    print("Generating video...")
    video_url = generate_video(prompt)
    download_video(video_url, "output.mp4")
```

### Building a Batch Generation Pipeline

You can also build a pipeline with Claude Code to process multiple prompts at once.

```python
import json

def batch_generate(scene_plan_path: str,
                   output_dir: str = "./output") -> list:
    """Batch-generate videos from a scene plan JSON file."""
    os.makedirs(output_dir, exist_ok=True)

    with open(scene_plan_path) as f:
        scenes = json.load(f)

    results = []
    for i, scene in enumerate(scenes):
        print(f"\nScene {i + 1}/{len(scenes)}: {scene['description']}")

        video_url = generate_video(
            prompt=scene["prompt"],
            model=scene.get("model", "pollo-2.5"),
            aspect_ratio=scene.get("aspect_ratio", "16:9"),
            duration=scene.get("duration", 5),
        )

        output_path = os.path.join(
            output_dir, f"scene_{i + 1:02d}.mp4"
        )
        download_video(video_url, output_path)
        results.append(output_path)

    print(f"\nAll {len(results)} scenes generated successfully")
    return results
```

Prepare your scene plan as a JSON file:

```json
[
  {

    "model": "kling-ai",
    "aspect_ratio": "16:9",

  },
  {
    "description": "Interior",
    "prompt": "Interior of a minimalist cafe with natural lighting...",
    "model": "kling-ai",
    "aspect_ratio": "16:9",
    "duration": 5
  }
]
```

You can also ask Claude to create this JSON file for you, so all you need to do is describe the theme and the entire generation pipeline comes together.

## Prompt Engineering Tips

To generate high-quality videos with Pollo AI, prompt design is crucial. When asking Claude to craft prompts, keeping these points in mind will significantly improve results.

### Be Specific About Composition

Clearly describe what is where and how. Avoid vague expressions and use words that indicate distance from the camera and subject placement.

Good: `A single red rose in a glass vase, centered in frame, placed on a white marble table`

Bad: `A flower on a table`

### Specify Camera Movement

Explicitly stating whether you want a static or dynamic shot stabilizes the generated output.

Camera movement terms you can use: `slow dolly-in`, `smooth pan right`, `gentle zoom out`, `360 orbit`, `static shot`, `tracking shot`.

### Define Lighting and Color Tone

Lighting and color have the biggest impact on the mood of your footage.

Lighting types: `golden hour sunlight`, `soft studio lighting`, `dramatic side lighting`, `neon glow`.

Color: `warm color grading`, `cool blue tones`, `desaturated cinematic`.

### Quality and Style Keywords

Add quality and style keywords at the end of your prompt: `cinematic 4K`, `photorealistic`, `anime style`, `oil painting style`, `slow motion`, etc.

## Pricing and Per-Model Costs

Pollo AI uses a credit-based system. The free plan provides daily credits, while paid plans offer additional credit purchases.

Different models consume different amounts of credits, so choosing the right model for each use case matters. A cost-effective strategy is to use the affordable Pollo 2.5 during prototyping and testing, then switch to Kling AI or Runway for final production videos.

You can ask Claude to "suggest cost-effective model and parameter combinations" to maximize results within your budget.

## FAQ

### Is Pollo AI free to use?

Yes, a free plan is available. You can generate videos within the daily credit allowance. However, high-quality models and longer videos require more credits, so a paid plan is recommended for serious production work.

### Can I use prompts in languages other than English?

Pollo AI supports prompts in multiple languages, including Japanese. However, most video generation AIs perform best with English prompts. By asking Claude to generate English prompts from your native-language descriptions, you get the best of both worlds.

### Who owns the copyright of generated videos?

Usage rights for videos generated with Pollo AI vary by plan. Paid plans generally allow commercial use. Check Pollo AI's terms of service for details.

### How do I integrate the API with Claude Code?

Register on the Pollo AI API platform (pollo.ai/api-platform) to get your API key, then set it as the `POLLO_API_KEY` environment variable. Refer to the "Automating with Claude Code × Pollo AI API" section in this article for sample code.
