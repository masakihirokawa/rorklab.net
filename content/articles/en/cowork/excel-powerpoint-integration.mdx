# Cowork × Excel / PowerPoint Integration

Cowork mode allows Claude to directly manipulate files in a local environment, making it possible to create and analyze Excel spreadsheets and auto-generate PowerPoint presentations. This guide covers practical use cases and techniques.

## Cowork's Office File Support

Cowork runs on a lightweight Linux VM and leverages Python libraries to directly manipulate Office files:

- **openpyxl**: Reading and writing Excel (.xlsx) files
- **python-pptx**: Creating and editing PowerPoint (.pptx) files
- **pandas**: Data analysis with Excel integration

## Excel Integration in Practice

### Creating Spreadsheets

Simply ask Cowork to "create a sales report Excel" and it generates a formatted spreadsheet.

Example request:

```
Create an Excel report from the monthly sales data.
Apply colored formatting to headers, add a totals row,
and include a month-over-month comparison column.
```

Cowork automatically handles:

- Structuring data and placing it in cells
- Header formatting (fonts, background colors, borders)
- Inserting formulas (SUM, percentage changes, etc.)
- Auto-adjusting column widths
- Applying conditional formatting

### Data Analysis and Visualization

Pass an existing Excel file to Cowork for analysis:

```
Analyze the sales data in this Excel file and add
a monthly trend chart and a top 10 products ranking table.
```

What Cowork can analyze and visualize:

- Creating pivot tables
- Inserting line charts, bar charts, and pie charts
- Calculating statistics (mean, median, standard deviation)
- Sorting and filtering data
- Highlighting with conditional formatting

### CSV Conversion

Converting CSV files to formatted Excel is straightforward:

```
Convert this CSV file to Excel.
Format it as a table and apply comma-separated
number formatting to numeric columns.
```

## PowerPoint Integration in Practice

### Automated Presentation Generation

Cowork can create presentations from scratch based on a topic:

```
Create a PowerPoint for "Q1 2026 Business Report."
Structure it in 5 sections: title slide, table of contents,
performance summary, department results, and future plans.
```

Generated slides include:

- Title slides and section dividers
- Auto-arranged text content
- Embedded tables and charts
- Consistent design themes
- Speaker notes

### Editing Existing Slides

You can also edit existing PowerPoint files:

```
Update the chart data on page 3 of this PowerPoint.
Also, add a summary slide at the end.
```

Supported edit operations:

- Modifying and adding text
- Adding, removing, and reordering slides
- Inserting and replacing images
- Updating tables
- Adjusting design elements

### Generating Slides Directly from Data

Create chart-filled slides directly from Excel data or CSVs:

```
Create a management meeting presentation from this sales Excel.
Include a KPI dashboard slide and detailed department slides.
```

- In-cell data analysis and summarization

Claude in Excel uses the Opus 4.6 model and supports native Excel operations.

## Tips and Best Practices

### Effective Request Tips

- **Be specific**: Instead of "make it nice," specify "blue header with white text, right-align numbers"
- **Describe the end goal**: Explain who will see the file and what it should communicate
- **Build incrementally**: For complex files, make incremental requests rather than one large ask
- **Select your folder**: Pre-select your working folder for smooth file reading and writing

### Considerations

- Large files (tens of thousands of rows) may take longer to process
- Macro-enabled .xlsm files have limited macro editing support
- PowerPoint animation settings have limited support
- Fonts depend on the environment; use caution with specialty fonts
