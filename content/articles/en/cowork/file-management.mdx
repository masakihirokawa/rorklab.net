## What is Cowork?

Cowork is Anthropic's desktop application that brings Claude's capabilities directly to your file system and computer. It allows you to manage files, perform batch operations, convert formats, and automate repetitive desktop tasks without leaving your workspace.

<Callout type="info">
Cowork runs locally on your computer and has access to your file system. You maintain full control over which directories Claude can access and what operations it can perform.
</Callout>

## Installing Cowork

### System Requirements

- macOS 11+, Windows 10+, or Linux (Ubuntu 20.04+)
- 4GB RAM minimum
- 500MB free disk space
- Anthropic account with API key

### Installation Steps

1. **Download** the Cowork application from [anthropic.com](https://anthropic.com)
2. **Install** using the platform-specific installer
3. **Launch** Cowork from your applications folder
4. **Authenticate** with your Anthropic account
5. **Grant folder access** to the directories you want to work with

## Understanding Folder Access

One of Cowork's key security features is granular folder access control. You explicitly choose which directories Claude can access.

### Requesting Folder Access

When you ask Claude to work with files:

1. Claude requests permission to access a specific folder
2. A dialog appears asking you to confirm
3. You can browse and select the folder
4. Claude gains access only to that folder and its contents
5. You can revoke access at any time in settings

### Best Practices for Folder Access

- **Grant minimal access**: Only allow access to folders you need Claude to work with
- **Organize your files**: Keep related files in the same folder
- **Avoid sensitive folders**: Don't grant access to credential or configuration directories
- **Review permissions regularly**: Check which folders Claude can access periodically

## Common File Operations

### Reading Files

Ask Claude to read and analyze your files:

```
Read the CSV file "sales_data.csv" and summarize the key metrics.
```

Claude can read:
- Text files (.txt, .md)
- Code files (.py, .js, .java, etc.)
- Structured data (.csv, .json)
- Documents (.pdf, .docx)

### Creating and Writing Files

Claude can create new files in authorized folders:

```
Create a Python script called "analysis.py" that processes the data.
```

Claude will:
- Create the file with specified content
- Use proper formatting and syntax
- Place the file in the correct location
- Show you the content before saving

### Copying and Moving Files

Organize your files by asking Claude to reorganize them:

```
Copy all PNG files from the Downloads folder to a new "Images" subfolder.
```

### Renaming Files in Bulk

Claude can rename multiple files at once:

```
Rename all PDF files in this folder to include today's date in the filename.
Change "document.pdf" to "document_2026-03-09.pdf"
```

## File Format Conversion

One of Cowork's most powerful features is batch file format conversion. Claude can convert between many formats automatically.

### Supported Conversions

| From | To | Common Use Cases |
|------|----|----|
| CSV | JSON, Excel, PDF | Data migration, API integration |
| Markdown | HTML, PDF, DOCX | Publishing, documentation |
| Images | PNG, JPG, WEBP | Compression, format standardization |
| JSON | CSV, Excel | Data analysis, spreadsheet import |
| Audio | MP3, WAV, OGG | Compression, compatibility |

### Example: CSV to JSON

```
Convert "sales_data.csv" to JSON format.
Keep the structure flat and use descriptive field names.
Save as "sales_data.json".
```

Claude will handle the conversion automatically while preserving data integrity.

### Example: Markdown to PDF

```
Convert all .md files in this folder to PDF.
Apply professional formatting with a standard font.
Save PDFs with the same filenames.
```

### Batch Format Conversion

Convert multiple files at once:

```
I have 20 TIFF images that need to be converted to JPG format.
Compress them to 80% quality and keep them in the same folder.
Add "_compressed" to each filename.
```

## Security Considerations

### Protecting Sensitive Files

- **Never grant access** to folders containing passwords or credentials
- **Use encryption** for sensitive documents before processing
- **Review files** that Claude will process before sharing access
- **Limit scope**: Give access only to necessary folders

### Best Practices

1. **Keep credentials separate**: Store API keys and passwords outside Cowork's access
2. **Audit operations**: Review significant file operations before they're executed
3. **Backup important files**: Keep backups before bulk operations
4. **Revoke access**: Remove folder permissions when no longer needed
5. **Monitor activity**: Check the operation log periodically

<Callout type="warning">
Cowork performs real file operations. Always review what Claude will do before confirming batch operations that involve deletion or large-scale changes.
</Callout>

## Troubleshooting Common Issues

### Permission Denied Errors

**Problem**: "Permission denied" when accessing a folder

**Solution**:
- Grant folder access in Cowork settings
- Check file/folder permissions on your system
- Ensure you have read/write permissions

### File Encoding Issues

**Problem**: Text files display incorrectly

**Solution**:
- Ensure files are UTF-8 encoded
- Try converting to UTF-8 before processing
- Check file extensions match actual format

### Conversion Failures

**Problem**: File conversion doesn't complete

**Solution**:
- Check file integrity (file isn't corrupted)
- Ensure sufficient disk space
- Try smaller batches if processing many files
- Check file format compatibility

## Performance Tips

1. **Work with reasonable batch sizes**: Process 50-100 files at a time, not thousands
2. **Close unneeded applications**: Free up system resources for conversions
3. **Use SSD when possible**: Faster storage improves performance
4. **Monitor system resources**: Watch CPU and disk usage during large operations
5. **Split large projects**: Break into smaller, manageable tasks
