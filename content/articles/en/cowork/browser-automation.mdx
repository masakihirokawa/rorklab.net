## What Is Cowork Browser Automation?

Cowork's **Claude in Chrome** feature lets you delegate browser operations to Claude. You can automate web page browsing, form filling, information gathering, screenshot capture, and more using natural language instructions.

## Basic Usage

### Page Browsing

Have Claude open websites and extract content for you.

```
"Open the Anthropic website and tell me the titles of the latest blog posts"
```

Claude operates browser tabs to access pages, analyzes content, and returns the information you need.

### Form Filling

Automate web form input. Note that sensitive data like passwords and banking information is restricted for security.

```
"Fill in my name and address on this form"
```

### Screenshots

Capture the current state of a page for visual verification.

```
"Take a screenshot of this page"
```

## Use Cases

### Information Gathering

Ideal for collecting and comparing information across multiple websites.

```
"Check the prices for Tokyo to Osaka flights on three travel sites and compare them"
```

### Website Testing

Have Claude verify your website's display and functionality.

```
"Open my site example.com and check if the mobile layout looks correct"
```

### Data Entry Automation

Automate repetitive data entry tasks.

```
"Use the data from this spreadsheet to fill in the web form entries one by one"
```

## Security and Limitations

Browser automation comes with several important restrictions.

### Protected Operations

- **Password entry**: Login credentials must be entered manually
- **Financial data**: Credit card numbers and bank details are never handled by Claude
- **Account creation**: New accounts must be created manually
- **Permission changes**: Document sharing and access control changes require manual action

### Operations Requiring Confirmation

Claude will ask for confirmation before performing these actions:

- File downloads
- Form submissions (clicking "Submit" or "Post" buttons)
- Sending emails or messages
- Accepting terms and conditions
- Completing purchases or payments

## Tips for Effective Use

1. **Be specific**: Clearly describe the target buttons, links, or elements
2. **Work in steps**: Break complex operations into stages and verify at each step
3. **Verify results**: Use screenshots to confirm each step's outcome

## Summary

Cowork's browser automation is a powerful tool for streamlining everyday web tasks. Understanding its security limitations will help you leverage it effectively for information gathering and data entry.
