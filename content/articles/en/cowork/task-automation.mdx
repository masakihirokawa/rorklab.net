## What is Cowork?

Cowork is a desktop feature of Claude designed for file operations and task automation. You can execute complex file processing and data transformations simply by giving natural language instructions — no coding required.

<Callout type="info">
Cowork is available as part of the Claude Desktop app. It's currently in research preview.
</Callout>

## Getting Started

In Cowork, you select a local folder and grant Claude access. Claude can then read and write files within that folder, handling a variety of tasks automatically.

### What You Can Do

- **File organization** — Sort files by type into folders
- **Data conversion** — Transform CSV to Excel, Markdown to Word, and more
- **Report generation** — Analyze data and auto-generate documents or presentations
- **Code execution** — Create and run Python scripts
- **Image processing** — Resize, convert formats, extract metadata

## Scheduled Tasks

Cowork lets you schedule tasks to run at regular intervals using cron expressions.

### Schedule Examples

| Purpose | Cron Expression | Description |
|---------|-----------------|-------------|
| Morning briefing | `0 9 * * 1-5` | Weekdays at 9:00 AM |
| Weekly report | `0 17 * * 5` | Every Friday at 5:00 PM |
| Daily backup | `0 0 * * *` | Every day at midnight |
| Monthly summary | `0 10 1 * *` | First of each month at 10:00 AM |

<Callout type="tip">
Cron expressions are evaluated in your local machine's timezone. No UTC conversion needed.
</Callout>

## The Skill System

Skills extend Cowork's capabilities. Each skill bundles optimized prompts and processing steps for specific task types.

### Built-in Skills

- **docx** — Create and edit Word documents
- **xlsx** — Work with Excel spreadsheets
- **pptx** — Build PowerPoint presentations
- **pdf** — Generate, merge, and split PDFs
- **frontend-design** — Build frontend UI components

### How Skills Work

Skills activate automatically. Say "summarize this in Excel" and the xlsx skill loads. Ask for "a presentation about Q3 results" and the pptx skill kicks in.

## Practical Examples

### 1. Organize Your Downloads Folder

```
Organize files in the Downloads folder by these rules:
- PDFs → Documents/PDF/
- Images (jpg, png, gif) → Pictures/
- Excel/CSV → Documents/Data/
- Everything else → Documents/Other/
```

### 2. Analyze CSV Data into a Report

```
Analyze sales_data.csv and create an Excel report including:
- Monthly sales trend chart
- Top 10 products by revenue
- Year-over-year comparison table
```

### 3. Auto-Generate Weekly Email Drafts

```
Every Friday at 4 PM, summarize this week's work log (work_log.md)
and generate a weekly report email draft in Markdown.
```

## Connector Integrations

Cowork connects to external services through MCP connectors:

- **Slack** — Search and summarize messages
- **Asana / Jira** — Task management integration
- **Google Drive** — Read and write files
- **GitHub** — Repository operations

<Callout type="warning">
Connectors require authentication with each service. Connect them through Cowork's settings panel.
</Callout>
