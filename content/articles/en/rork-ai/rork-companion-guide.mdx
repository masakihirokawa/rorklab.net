## What Is Rork Companion?

Rork Companion is your **AI pair programming partner** built into the Rork platform. It provides real-time code review, optimization suggestions, and intelligent debugging while you develop, accelerating your app creation process.

Unlike traditional code generators that excel at creating from scratch, Rork Companion specializes in **improving, optimizing, and debugging existing code**. Think of it as a senior developer sitting beside you, always ready to offer insights.

### Key Features of Rork Companion

- **Real-time Code Analysis**: Instant feedback as you write code
- **Context-Aware Suggestions**: Understands your project structure and architecture
- **UI Component Optimization**: Seamless Figma design synchronization
- **Performance Diagnostics**: Automatically detects hidden bottlenecks

## Real-World Companion Use Cases

### Case 1: Modernizing Legacy Code

Show Companion old async code, and it suggests:

```javascript
// Before: Promise-based approach
function fetchUserData(id) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      const user = { id: id, name: "User" };
      resolve(user);
    }, 1000);
  });
}
```

Companion recommends:

```typescript
// After: Modern async/await with types
async function fetchUserData(id: string): Promise<User> {
  await delay(1000);
  return { id, name: "User" };
}
```

## Best Practices for Companion

### 1. Be Specific and Clear

**Poor**: "Make this better"
**Good**: "This list rendering is slow. Suggest performance optimizations for large datasets"

### 2. Provide Context

- Specify your tech stack (React, SwiftUI, etc.)
- State performance goals (load in under 2 seconds, etc.)
- Mention target devices (iPhone SE or newer, Android 8+, etc.)

### 3. Approach Improvements Gradually

Instead of overwhelming Companion, improve in stages:

1. Structural improvements
2. Performance optimization
3. Design quality enhancement

This incremental approach yields higher-quality AI suggestions.

## Important Considerations

### AI Isn't Perfect

Companion achieves 99% accuracy, but complex business logic always deserves human review. Use Companion as guidance, not gospel.

### Don't Share Sensitive Data

Never show API keys, customer data, or confidential code. Use sanitized data or local processing for testing.

### Understand Before Implementing

Ask "Why?" when Companion suggests changes. Understanding the reasoning helps you learn and make better decisions.

## Frequently Asked Questions

### Q: Does Companion work offline?

A: No. It's cloud-based and requires an internet connection.

### Q: Can I use Companion on multiple projects simultaneously?

A: Yes. Companion works across all projects in your account.

### Q: Do I need a paid Rork plan for Companion?

A: Yes. Companion requires Rork Pro or higher. It's not available on the free tier.

### Q: Does Companion auto-apply changes to my code?

A: No. All suggestions require your approval. Nothing changes without your explicit confirmation.
