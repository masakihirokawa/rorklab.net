The difference between a chatbot that users love and one they abandon often comes down to a single UX decision: **does the response appear instantly, or does the user wait staring at a spinner?** Streaming—where tokens appear word-by-word as they're generated—is what separates a polished AI product from a frustrating one.

This guide shows you how to build a production-quality AI chatbot with **Rork Max** and **Gemini API streaming**. By the end, you'll have a fully functional app with real-time text rendering, conversation memory, markdown support, and solid error handling.

## Prerequisites

- A Rork Max account (powered by Claude Opus 4.6)
- A Gemini API key from Google AI Studio (free: https://aistudio.google.com)
- Basic familiarity with Expo SDK 53+ and React Native

## Gemini API Streaming Integration

### Building the API Client

```typescript
// services/gemini.ts
import { GoogleGenerativeAI, HarmCategory, HarmBlockThreshold } from "@google/generative-ai";

const API_KEY = process.env.EXPO_PUBLIC_GEMINI_API_KEY!;

// Safety settings — tune these based on your app's audience
const SAFETY_SETTINGS = [
  { category: HarmCategory.HARM_CATEGORY_HARASSMENT, threshold: HarmBlockThreshold.BLOCK_MEDIUM_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_HATE_SPEECH, threshold: HarmBlockThreshold.BLOCK_MEDIUM_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_SEXUALLY_EXPLICIT, threshold: HarmBlockThreshold.BLOCK_MEDIUM_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_DANGEROUS_CONTENT, threshold: HarmBlockThreshold.BLOCK_MEDIUM_AND_ABOVE },
];

export interface ChatMessage {
  id: string;
  role: "user" | "model";
  content: string;
  timestamp: Date;
  isStreaming?: boolean;
}

export class GeminiChatClient {
  private genAI: GoogleGenerativeAI;
  private model: ReturnType<GoogleGenerativeAI["getGenerativeModel"]>;

  constructor(systemInstruction?: string) {
    this.genAI = new GoogleGenerativeAI(API_KEY);
    this.model = this.genAI.getGenerativeModel({
      model: "gemini-2.5-pro",
      systemInstruction: systemInstruction ?? "You are a helpful and knowledgeable AI assistant.",
      safetySettings: SAFETY_SETTINGS,
      generationConfig: {
        temperature: 0.7,
        topP: 0.95,
        maxOutputTokens: 2048,
      },
    });
  }

  private toGeminiHistory(messages: ChatMessage[]) {
    return messages
      .filter((m) => !m.isStreaming)   // Exclude incomplete streaming messages
      .map((m) => ({
        role: m.role,
        parts: [{ text: m.content }],
      }));
  }

  /**
   * Send a message with streaming, invoking callbacks for each chunk.
   */
  async sendMessageStreaming(
    userMessage: string,
    history: ChatMessage[],
    onChunk: (text: string) => void,
    onComplete: (fullText: string) => void,
    onError: (error: Error) => void,
  ): Promise<void> {
    try {
      const chat = this.model.startChat({
        history: this.toGeminiHistory(history),
      });

      const result = await chat.sendMessageStream(userMessage);
      let fullText = "";

      for await (const chunk of result.stream) {
        const chunkText = chunk.text();
        fullText += chunkText;
        onChunk(chunkText);   // Update UI immediately
      }

      onComplete(fullText);
    } catch (error) {
      onError(error instanceof Error ? error : new Error(String(error)));
    }
  }
}
```

### Chat State Management Hook

```typescript
// hooks/useChat.ts
import { useState, useCallback, useRef } from "react";
import { GeminiChatClient, ChatMessage } from "@/services/gemini";
import { nanoid } from "nanoid";

const MAX_HISTORY = 20;

export function useChat(systemInstruction?: string) {
  const [messages, setMessages] = useState<ChatMessage[]>([]);
  const [isLoading, setIsLoading] = useState(false);
  const clientRef = useRef(new GeminiChatClient(systemInstruction));

  const sendMessage = useCallback(async (text: string) => {
    if (!text.trim() || isLoading) return;

    // Show user message immediately
    const userMessage: ChatMessage = {
      id: nanoid(),
      role: "user",
      content: text.trim(),
      timestamp: new Date(),
    };

    // Placeholder for the streaming AI response
    const aiMessageId = nanoid();
    const aiPlaceholder: ChatMessage = {
      id: aiMessageId,
      role: "model",
      content: "",
      timestamp: new Date(),
      isStreaming: true,
    };

    setMessages((prev) => {
      const updated = [...prev, userMessage, aiPlaceholder];
      // Trim to max history (remove oldest pair first)
      return updated.length > MAX_HISTORY * 2
        ? updated.slice(updated.length - MAX_HISTORY * 2)
        : updated;
    });

    setIsLoading(true);
    const historyForApi = messages.slice(-MAX_HISTORY * 2);

    await clientRef.current.sendMessageStreaming(
      text.trim(),
      historyForApi,
      // Append each incoming chunk to the placeholder
      (chunk) => {
        setMessages((prev) =>
          prev.map((m) =>
            m.id === aiMessageId ? { ...m, content: m.content + chunk } : m
          )
        );
      },
      // Mark streaming complete
      (_fullText) => {
        setMessages((prev) =>
          prev.map((m) =>
            m.id === aiMessageId ? { ...m, isStreaming: false } : m
          )
        );
        setIsLoading(false);
      },
      // Handle errors gracefully
      (error) => {
        setMessages((prev) =>
          prev.map((m) =>
            m.id === aiMessageId
              ? { ...m, content: "⚠️ Something went wrong. Please try again.", isStreaming: false }
              : m
          )
        );
        setIsLoading(false);
        console.error("Gemini API error:", error);
      }
    );
  }, [messages, isLoading]);

  const clearHistory = useCallback(() => {
    setMessages([]);
    clientRef.current = new GeminiChatClient(systemInstruction);
  }, [systemInstruction]);

  return { messages, isLoading, sendMessage, clearHistory };
}
```

## Building the Chat UI

```typescript
// screens/ChatScreen.tsx
import React, { useRef, useCallback, useState } from "react";
import {
  View, FlatList, TextInput, TouchableOpacity,
  KeyboardAvoidingView, Platform, StyleSheet, ActivityIndicator, Text,
} from "react-native";
import { ChatBubble } from "@/components/ChatBubble";
import { TypingIndicator } from "@/components/TypingIndicator";
import { useChat } from "@/hooks/useChat";

export default function ChatScreen() {
  const { messages, isLoading, sendMessage } = useChat(
    "You are a helpful Rork Max expert. Answer mobile app development questions clearly and concisely."
  );
  const [inputText, setInputText] = useState("");
  const flatListRef = useRef<FlatList>(null);

  const handleSend = useCallback(() => {
    if (!inputText.trim()) return;
    sendMessage(inputText);
    setInputText("");
    setTimeout(() => flatListRef.current?.scrollToEnd({ animated: true }), 100);
  }, [inputText, sendMessage]);

  return (
    <KeyboardAvoidingView
      style={styles.container}
      behavior={Platform.OS === "ios" ? "padding" : "height"}
      keyboardVerticalOffset={90}
    >
      <FlatList
        ref={flatListRef}
        data={messages}
        keyExtractor={(item) => item.id}
        renderItem={({ item }) => <ChatBubble message={item} />}
        ListFooterComponent={
          isLoading && !messages[messages.length - 1]?.isStreaming
            ? <TypingIndicator />
            : null
        }
        onContentSizeChange={() => flatListRef.current?.scrollToEnd({ animated: true })}
        contentContainerStyle={styles.messageList}
      />

      <View style={styles.inputContainer}>
        <TextInput
          style={styles.input}
          value={inputText}
          onChangeText={setInputText}
          placeholder="Type a message..."
          multiline
          maxLength={2000}
        />
        <TouchableOpacity
          style={[styles.sendButton, (!inputText.trim() || isLoading) && styles.disabled]}
          onPress={handleSend}
          disabled={!inputText.trim() || isLoading}
        >
          {isLoading
            ? <ActivityIndicator color="#fff" size="small" />
            : <Text style={styles.sendIcon}>↑</Text>}
        </TouchableOpacity>
      </View>
    </KeyboardAvoidingView>
  );
}
```

## Performance Optimization

### Preventing Unnecessary Re-renders During Streaming

Because `setMessages` is called for every token during streaming, preventing cascade re-renders is critical for smooth performance on lower-end Android devices.

```typescript
// components/ChatBubble.tsx
import React, { memo } from "react";
import { View, StyleSheet } from "react-native";
import Markdown from "react-native-markdown-display";

interface Props {
  message: ChatMessage;
}

export const ChatBubble = memo(({ message }: Props) => {
  const isUser = message.role === "user";

  return (
    <View style={[styles.bubble, isUser ? styles.userBubble : styles.aiBubble]}>
      <Markdown style={isUser ? userMarkdownStyle : aiMarkdownStyle}>
        {message.content || (message.isStreaming ? "▊" : "")}
      </Markdown>
    </View>
  );
}, (prev, next) => {
  // Only re-render when content or streaming state changes
  return (
    prev.message.content === next.message.content &&
    prev.message.isStreaming === next.message.isStreaming
  );
});
```

## Frequently Asked Questions

**Q1. How do I handle Gemini API rate limits in a real app?**

The free tier allows 15 requests per minute for Gemini 2.5 Pro. For production, combine three approaches: (1) debounce the send button with a 2-second minimum interval between requests, (2) upgrade to a paid API tier for higher quotas, and (3) detect HTTP 429 errors and show a user-friendly "Please wait a moment and try again" toast. Never silently swallow rate limit errors—users get confused when nothing happens after tapping Send.

**Q2. How does conversation history affect API costs?**

Every request sends the full conversation history, so longer chats cost more tokens. The `MAX_HISTORY = 20` setting in this guide keeps a practical balance between context retention and cost. For cost-sensitive apps, drop this to 10–15 messages. Switching to `gemini-2.5-flash` reduces per-token pricing by roughly 80% compared to Pro, at the expense of some reasoning quality.

**Q3. Can users cancel a streaming response mid-generation?**

Yes. Pass an `AbortController` signal into the streaming request. In your `useChat` hook, create `abortRef = useRef<AbortController | null>(null)`, initialize it on each `sendMessage` call, and expose a `cancelStream` function that calls `abortRef.current.abort()`. In the UI, show a stop button whenever `isLoading` is true. This requires Gemini SDK 0.5+ which supports the standard Fetch `AbortSignal`.

## Wrapping Up

Building a great AI chatbot with Rork Max and Gemini API comes down to a few key principles:

1. **Stream from the first token** — never make users wait for the full response.
2. **Manage conversation history deliberately** — cap it to control costs without sacrificing coherence.
3. **Optimize rendering during streaming** — `React.memo` with custom comparison functions keeps the UI smooth even on budget Android devices.
4. **Handle failures visibly** — replace placeholder messages with clear error text so users know what happened and what to do next.
