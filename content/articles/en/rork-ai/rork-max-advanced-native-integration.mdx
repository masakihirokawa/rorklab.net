## Vision API: Image Recognition

Vision provides on-device machine learning for image analysis—face detection, text recognition, barcode scanning, object classification.

**Use case: Business card scanner**

```swift
import Vision

func performOCR(image: UIImage) {
    guard let cgImage = image.cgImage else { return }

    let request = VNRecognizeTextRequest { request, _ in
        let observations = request.results as? [VNRecognizedTextObservation] ?? []
        let text = observations
            .compactMap { $0.topCandidates(1).first?.string }
            .joined(separator: "\n")

        DispatchQueue.main.async {
            self.recognizedText = text
        }
    }

    let handler = VNImageRequestHandler(cgImage: cgImage)
    try? handler.perform([request])
}
```

## ARKit: Augmented Reality

AR applications are practical and deployable. Examples: furniture preview, sports stats display, construction visualization.

**Pattern:**

```swift
import ARKit
import RealityKit

class ARViewController: UIViewController, ARViewDelegate {
    var arView: ARView!

    override func viewDidLoad() {
        super.viewDidLoad()

        arView = ARView(frame: view.bounds)
        view.addSubview(arView)

        loadAndPlaceModel()
    }

    func loadAndPlaceModel() {
        guard let modelAnchor = try? ModelEntity
            .loadModel(named: "furniture_model") else { return }

        var transform = modelAnchor.transform
        transform.translation.z = -0.5
        modelAnchor.move(to: transform, relativeTo: arView.scene)
    }
}
```
