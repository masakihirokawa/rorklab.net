## What Is the OpenAI Whisper API?

**Whisper** is OpenAI's speech recognition model, renowned for its accuracy across more than 99 languages. The `whisper-1` model is available via API and delivers near-human-level transcription quality.

| Feature | Details |
|---------|---------|
| Languages | 99+ (Japanese, English, Chinese, and more) |
| Accuracy | Near-human level |
| Pricing | $0.006 / minute (as of March 2026) |
| Max file size | 25MB |
| Supported formats | mp3, mp4, mpeg, mpga, m4a, wav, webm |

**Why Whisper over native APIs?** Apple's Speech Framework is iOS-only. Whisper runs in the cloud and works identically on iOS and Android — a perfect fit for Rork's cross-platform apps.

## How It Works: The Architecture

Here's the data flow for a Rork-built voice recognition app:

```
[User] → speaks into microphone
   ↓
[Expo Audio] → records and saves as .m4a file
   ↓
[FormData] → attaches audio file with model settings
   ↓
[OpenAI Whisper API] → transcribes speech to text
   ↓
[App UI] → displays transcription result
```

The key is getting Rork to generate the **bridge code** between Expo Audio (React Native's audio library) and the Whisper API endpoint.

### Step 2: The Core Recording and Transcription Code

Rork generates code similar to this for the core functionality:

```typescript
// hooks/useVoiceRecorder.ts
import { useState, useRef } from 'react';
import { Audio } from 'expo-av';

const WHISPER_API_URL = 'https://api.openai.com/v1/audio/transcriptions';
const OPENAI_API_KEY = process.env.EXPO_PUBLIC_OPENAI_API_KEY ?? '';

export function useVoiceRecorder() {
  const [isRecording, setIsRecording] = useState(false);
  const [isTranscribing, setIsTranscribing] = useState(false);
  const [transcription, setTranscription] = useState('');
  const [error, setError] = useState<string | null>(null);
  const recordingRef = useRef<Audio.Recording | null>(null);

  const startRecording = async () => {
    try {
      const { granted } = await Audio.requestPermissionsAsync();
      if (!granted) throw new Error('Microphone permission denied');

      await Audio.setAudioModeAsync({
        allowsRecordingIOS: true,
        playsInSilentModeIOS: true,
      });

      const { recording } = await Audio.Recording.createAsync(
        Audio.RecordingOptionsPresets.HIGH_QUALITY
      );
      recordingRef.current = recording;
      setIsRecording(true);
      setError(null);
    } catch (err) {
      setError(`Recording error: ${err}`);
    }
  };

  const stopAndTranscribe = async () => {
    if (!recordingRef.current) return;
    setIsRecording(false);
    setIsTranscribing(true);

    try {
      await recordingRef.current.stopAndUnloadAsync();
      const uri = recordingRef.current.getURI();
      if (!uri) throw new Error('Recording file not found');

      const result = await transcribeAudio(uri);
      setTranscription(result);
    } catch (err) {
      setError(`Transcription error: ${err}`);
    } finally {
      setIsTranscribing(false);
      recordingRef.current = null;
    }
  };

  return { isRecording, isTranscribing, transcription, error,
           startRecording, stopAndTranscribe };
}

async function transcribeAudio(audioUri: string): Promise<string> {
  const formData = new FormData();

  // React Native FormData accepts URI-based file attachments
  formData.append('file', {
    uri: audioUri,
    type: 'audio/m4a',
    name: 'recording.m4a',
  } as any);
  formData.append('model', 'whisper-1');
  formData.append('language', 'en'); // Set to 'ja' for Japanese, '' for auto-detect

  const response = await fetch(WHISPER_API_URL, {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${OPENAI_API_KEY}`,
      // Do NOT manually set Content-Type — let FormData handle it
    },
    body: formData,
  });

  if (!response.ok) {
    const errorData = await response.json();
    throw new Error(errorData.error?.message ?? 'API error');
  }

  const data = await response.json();
  return data.text;
  // Expected output: "The meeting starts at 3pm. Please review the agenda beforehand."
}
```

### Step 3: Configure Environment Variables

Add your API key configuration to `app.config.js`:

```javascript
// app.config.js
export default {
  expo: {
    name: 'VoiceNote',
    slug: 'voice-note',
    plugins: [
      [
        'expo-av',
        {
          microphonePermission:
            'Allow VoiceNote to access your microphone for voice recording.',
        },
      ],
    ],
  },
};
```

Create a `.env` file in your project root (**never commit this to Git**):

```bash
EXPO_PUBLIC_OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxx
```

> ⚠️ **Security note**: The `EXPO_PUBLIC_` prefix exposes the value client-side. For production apps, route all API calls through a backend (Cloudflare Workers or Supabase Edge Functions) to keep your API key private. See [Rork + Cloudflare AI Backend Guide](/articles/rork-dev/rork-cloudflare-ai-backend).

## Common Errors and Fixes

### Error: `Audio.Recording.createAsync is not a function`

**Cause:** `expo-av` is not installed correctly.
**Fix:** Tell Rork: *"Add the expo-av package to this project"*

### Error: `Invalid file format` (from Whisper API)

**Cause:** Unsupported audio format sent to the API.
**Fix:** Always use `Audio.RecordingOptionsPresets.HIGH_QUALITY` — it guarantees the `.m4a` format Whisper accepts.

```typescript
// ✅ Correct
const { recording } = await Audio.Recording.createAsync(
  Audio.RecordingOptionsPresets.HIGH_QUALITY
);

// ❌ Custom options may produce unsupported formats
```

### Error: `Request too large` (file > 25MB)

**Cause:** Recording is longer than approximately 4 hours.
**Fix:** Add a maximum recording duration:

```typescript
const MAX_RECORDING_MS = 3 * 60 * 1000; // 3 minutes

const timer = setTimeout(() => {
  if (isRecording) stopAndTranscribe();
}, MAX_RECORDING_MS);
```

### Error: Microphone permission denied on Android

**Cause:** `RECORD_AUDIO` permission not declared in `AndroidManifest.xml`.
**Fix:** The `expo-av` plugin in `app.config.js` handles this automatically. If you built manually, add the permission entry through Rork.

## FAQ

**Q1. How much does the Whisper API cost?**

The price is $0.006 per minute of audio (as of March 2026). Transcribing 100 minutes per day costs about $0.60/day or $18/month. Always check [OpenAI's pricing page](https://openai.com/pricing) for the latest rates.

**Q2. Does it work offline?**

No — Whisper API requires an internet connection. For offline support, consider Apple's Speech Framework (iOS only) via Rork Max's native Swift capabilities, or explore on-device Core ML models via the [Rork Max Core ML Guide](/articles/rork-dev/rork-max-core-ml-guide).

**Q3. Is recorded audio stored anywhere?**

The `expo-av` library saves recordings to the device's temporary storage. Recordings are deleted after the API call. Note that OpenAI may retain submitted audio for up to 30 days per their privacy policy — disclose this in your app's privacy policy.

**Q4. Can I use this with the free Rork plan?**

You can start on Rork's free plan, but complex API integration code generation is more reliable on Pro. If you hit limits, upgrade or break your prompt into smaller steps.

**Q5. Does it work on Android too?**

Yes. `expo-av` is fully cross-platform. The `app.config.js` plugin setup automatically handles the `RECORD_AUDIO` permission for Android builds.
