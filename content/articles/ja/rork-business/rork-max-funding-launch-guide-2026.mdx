## Rork Max とは何か？

### 簡潔な定義

Rork Max は、**ノーコード/ローコードを活用したAIアプリビルダー** です。Figmaでデザイン、Rork MaxでAIロジックを追加、数時間でアプリ完成という、従来の開発フローを革新する技術として注目されています。

### なぜ注目されているのか？

1. **開発時間の短縮**: 1ヶ月かかるアプリが数日で完成
2. **コスト削減**: エンジニアを雇わずに起業可能
3. **AI統合**: ChatGPT、Gemini、Claude などの LLM を ノーコードで統合
4. **AppStore対応**: iOS/Android に直接デプロイ可能

これらの特徴から、**個人アプリ起業家** や **スモールチーム** にとって革命的なツールとして認識されています。

## Early Stage プログラムとは何か？

### 定義と目的

**Early Stage プログラム** は、Rork Max が個人開発者・スタートアップ向けに提供する **完全無料の支援制度** です。

| 項目 | 内容 |
|------|------|
| **対象** | 初期段階（<$1M ARR）のスタートアップ、個人開発者 |
| **提供内容** | Rork Max Pro版の無料アクセス（期間限定） |
| **支援期間** | 6ヶ月〜1年（スタートアップの進捗により延長可能） |
| **その他特典** | メンターシップ、コミュニティアクセス、課金収益の一部還元 |

### 参加資格

以下の条件のいずれかを満たす必要があります（2026年時点）:

- [ ] アプリがまだ App Store に公開されていない
- [ ] 月間アクティブユーザー（MAU）が 1,000 未満
- [ ] 月額経常収益（MRR）が $100 未満
- [ ] 初期シード投資を受けたばかり（受けていない）

### 参加のメリット

**金銭的メリット**:
- Rork Max Pro（月額 $29）を無料で 6ヶ月使用 = $174 の節約
- AI コンポーネント無制限使用（通常版は制限あり）
- 優先サポート

**ビジネスメリット**:
- Rork Max コミュニティからの投資家紹介
- 起業家メンターへのアクセス
- イベント・ピッチコンテストへの優遇参加枠

## Early Stage プログラムへの参加方法

### ステップバイステップガイド

#### Step 1: Rork Max アカウントの作成

公式サイト [https://rork.com](https://rork.com) にアクセスし、メールアドレスで登録します。

```
1. Sign Up をクリック
2. メールアドレス入力
3. メール認証リンクをクリック
4. パスワード設定
```

#### Step 2: Early Stage プログラムへの申込み

Rork Max ダッシュボード内の **「Programs」** または **「Startup」** メニュー（左サイドバー）を開きます。

```
Dashboard > Programs > Early Stage
↓
「Apply Now」をクリック
```

#### Step 3: 申込みフォームの記入

以下の情報が必須です：

- **アプリ名**（英語と日本語）
- **アプリの説明**（200文字程度）
- **想定ユーザー層**（例: 学生、クリエイター、ビジネスマン等）
- **現在の進捗度**（アイデア段階 / 開発中 / ベータ版 / 公開準備中）
- **解決する問題**（Why: なぜこのアプリを作るのか？）
- **競合との差別化点**（Why Rork Max？）

#### Step 4: 審査期間

通常 3〜7 日以内に Rork Max チームから メール返信があります。

- **承認時**: Early Stage ステータスが自動的にアカウントに付与
- **不承認時**: フィードバック付きで連絡（再申込み可）

#### Step 5: Pro 版へのアップグレード確認

承認後、ダッシュボード内で Pro 版機能がすべてアンロックされていることを確認します。

```
Settings > Subscription > Current Plan: "Early Stage (Pro - Free)"
```

### よくあるエラーと対処法

| エラー | 原因 | 対処法 |
|------|------|--------|
| 「You don't qualify for this program」 | 既にPro有料版を購入済み | サポートに連絡（Pro費用返金可能） |
| 申込み完了メールが来ない | スパムメールボックスに入っている | Spam フォルダを確認 |
| 承認待ちが1週間以上 | 申込み情報が曖昧 | support@rork.com に直接問い合わせ |

- [ ] Rork Max Pro でアプリロジックを実装

### Phase 2: ローンチ前期間（2〜4週間）

- [ ] Apple の審査期間: 通常 1〜3日（時期により 1週間）

### Phase 3: ローンチ（Week 13）

**ローンチ翌日〜2週間**:

### Phase 4: マネタイズ（1ヶ月以降）

**Week 5〜8（ローンチ後）**:
- [ ] 有料プランの販売開始

## マネタイズ戦略 — Rork Max アプリで月 $1,000+ を稼ぐ方法

### 戦略 1: Freemium モデル（推奨）

**構成**:
- 基本機能は完全無料（ユーザー獲得重視）
- プレミアム機能は有料（広告除去、高度なAI機能、クラウド保存等）
- 価格: $2.99/月 または $19.99/年

**典型的な LTV（ライフタイムバリュー）**:
```
月間ユーザー: 10,000 人
有料率: 3%（300人）
平均購入額: $50/年
月間売上: 300 × $50 / 12 = $1,250
```

### 戦略 2: 広告 + 有料プランの組み合わせ

**収入源の多元化**:
- AdMob（Googleバナー広告）: $0.5〜2/1000 impression
- Admob リワード広告: $1〜3 / view
- 有料プラン: $9.99/月

**アプリで月 10,000 session の場合**:
```
バナー広告: 50,000 impression × $1 / 1000 = $50
リワード広告: 500 view × $2 = $1,000
有料プラン: 300人 × $10/月 = $3,000
月間売上: $4,050
```

### 戦略 3: B2B SaaS（エンタープライズ向け）

高度な AI 機能を持つアプリの場合、企業向けプランを追加：

- **Standard**: $9.99/月（個人向け）
- **Business**: $49.99/月（小規模企業向け）
- **Enterprise**: $399/月（大企業向け・カスタマイズ対応）

## Rork Max ローンチの注意点と落とし穴

### 注意点 1: App Store レビュー基準の厳格化

Apple は AI アプリ、特に LLM 連携アプリに対する審査を厳しくしています。

**NG例**:
- 詐欺的な「AI 診断」（医療行為とみなされる可能性）
- 著作権侵害疑いのあるコンテンツ生成機能
- ユーザーの個人情報を無断で学習に使用

**OK例**:
- 医療免責事項を明記した「AI 健康ナビ」
- Creative Commons コンテンツのみで学習した「AI 創作支援」
- ユーザーが明示的にオプトインした学習機能

### 注意点 2: AI API コストの上昇

Rork Max で ChatGPT API を使用する場合、API 呼び出しコストが想定外に膨らむ可能性があります。

**典型的なコスト**:
```
gpt-3.5-turbo: $0.001 / 1K tokens
gpt-4: $0.03 / 1K tokens

ユーザー当たり 1日100 token 使用の場合:
月間 30日 × 100 × $0.001 = $3/ユーザー/月

100ユーザーで月間 $300 のコスト
```

**対策**:
- [ ] API キャッシング機能を活用
- [ ] レート制限（1ユーザー 1日100回まで等）を実装
- [ ] 有料プランの収益で API コストをカバー（利益率 > 30%を目指す）

### 注意点 3: ユーザーデータ保護（GDPR 対応）

EU ユーザーからのアクセスがある場合、GDPR コンプライアンスが必須です。

**対応チェックリスト**:
- [ ] プライバシーポリシーに GDPR 条項を記載
- [ ] ユーザーの削除・ダウンロード権を実装
- [ ] Cookie 同意バナーを実装

Rork Max は Stripe 統合を推奨しており、Stripe は GDPR 対応を自動で処理します。追加実装はほぼ不要です。

## FAQ — 資金調達とローンチについてのよくある質問

### Q1: Early Stage プログラムに参加できませんでした。有料版を購入すべき？

**A**: 以下の優先順位で検討してください：

1. **無料版で十分か？** → 無料版で機能 (5App まで) があれば十分。Pro へのアップグレード不要。
2. **Rork Max でのテスト期間か？** → 1〜2ヶ月のテストなら無料版でも OK。本格開発なら Pro を検討。
3. **すぐにマネタイズを狙うか？** → App Store 登録直後であれば、無料版から始めて、売上が出てから Pro に有料化するのが合理的。

**コスト最適化戦略**:
```
Month 1-2: 無料版で開発テスト
Month 3: Pro 版へのアップグレード（月 $29）
Month 4-6: Pro で本格開発
Month 7+: 売上がカバーできれば継続、難しければ無料版に戻す
```

### Q2: Rork Max で作ったアプリは本当に App Store で公開できる？

**A**: はい、公開できます。ただし以下の条件があります：

- ✅ iOS（iPhone / iPad）: 完全サポート
- ✅ Android（Google Play）: 完全サポート
- ⚠️ PWA（Web アプリ）: 可能だが、ネイティブアプリ比で機能制限あり

Rork Max は React Native + Expo をベースにしており、公式に App Store / Google Play へのビルド出力機能を持っています。ただし、Rork Max の品質レベルは「中級向け」です。本格的な機能（オフライン対応、複雑なアニメーション等）が必要な場合は、エンジニアに引き継ぐことも視野に。

### Q3: Rork Max で作ったアプリのメンテナンス・スケーリングはどうする？

**A**: Rork Max はノーコード特性上、スケーリングに制限があります。

**想定シナリオ**:
```
Phase 1（0-1万ユーザー）: Rork Max で十分
Phase 2（1-10万ユーザー）: Rork Max + 外部API（AWS Lambda 等）で拡張
Phase 3（10万+ユーザー）: カスタム開発への移行を推奨
```

Early Stage メンターシップを活用して、適切なタイミングでエンジニアリング決定を相談するのがベストです。

### Q4: アプリの売上が出た場合、税務申告は？

**A**: 個人事業主登録 + 売上申告が必須です。日本の場合：

- **対象**: 年間売上 > ¥100,000
- **申告期限**: 翌年 2月15日 - 3月15日
- **税率**: 売上の約 20%（所得税 + 住民税）

実装の詳細は、個人向け税務相談サイト（国税庁 Web サイト）または税理士に相談してください。Stripe の「業績レポート」機能で売上を集計できます。
