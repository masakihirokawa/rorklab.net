## 第1章: Rork × Figmaで「デザイン→実装」を高速化

### Figmaでの設計フェーズ

**ステップ1: コンポーネント設計（1～2時間）**

Figmaでアプリの全画面をコンポーネント化します。ボタン、フォーム、カード、モーダルなど、再利用可能な単位に分割。

**例：習慣トラッキングアプリ**
- ホーム画面: 日付、本日のタスク表示、プログレスバー
- 詳細画面: タスク入力、履歴表示
- 設定画面: 通知設定、ユーザー情報

各画面を5～10個のコンポーネントに分割し、色・フォント・サイズを統一。

**ステップ2: インタラクティブプロトタイプ（1～2時間）**

Figmaのプロトタイプ機能で、画面遷移とアニメーションをシミュレート。ボタンをクリック→詳細画面へ遷移、など。

これにより、Rorkへの指示が明確になり、生成されるアプリの完成度が大幅に向上します。

### Rorkでの実装フェーズ

**プロンプト例:**

```
以下のFigmaデザイン（URL: https://figma.com/...）を参考に、
React NativeのアプリとしてRorkで実装してください。

要件:
- ホーム画面: 本日の習慣リスト表示、チェック機能
- 詳細画面: 習慣の詳細情報と編集
- 設定: 通知時間設定
- カラー: 主色 #6366F1、サブ色 #EC4899
- フォント: Inter（見出し）、Roboto（本文）
- アニメーション: 画面遷移はフェードイン

データベース: Firebase Firestore
認証: Firebase Auth
```

RorkはこのプロンプトとFigmaデザインから、完全に機能するアプリを3～6時間で生成します。

### 品質チェックと調整

生成されたアプリをRork Maxの「Fix Now」で自動テスト。エラーが自動修正されます。

微調整（色の調整、スペーシング、アニメーション速度など）が必要な場合は、再度プロンプトで指示。

**平均完成時間: 8～12時間**（従来の設計→実装なら4～8週間）

## 第2章: Rork × Canvaでマーケティング素材を量産

### App Storeスクリーンショット＆プロモ画像

**制作例（1アプリあたり30分）:**

```
1. Canvaで「App Store Screenshot」テンプレートを開く
2. アプリの最重要機能を3～5つの画面で説明
3. テキスト: 「毎日の習慣を視覚化」「通知で習慣を継続」など
4. デザイン: アプリのUI色とフォントに統一
5. スクリーンショット5枚を生成
6. App Storeの「スクリーンショット」セクションにアップロード
```

**効果:**
- 高品質なスクリーンショットでASO（App Store最適化）向上
- ダウンロード数が通常の1.5～2倍増加

### YouTube サムネイル＆チャンネルアート

Canvaのyoutubeテンプレートを使用：

```
1. 新規動画ごとに「YouTubeサムネイル」テンプレートでカスタマイズ
2. 動画のテーマとアプリ機能を視覚的に表現
3. テキストは太字・大きめで、動画内のキーフレームを静止画化
4. チャンネルアート（バナー）は1回制作で使い回し
```

### Kindle 表紙＆セールスページ

```
1. 「Kindleカバーデザイン」テンプレートで表紙を制作
2. タイトル: 「Rork Max × Swift でアプリを作る」
3. 背景：テック系グラデーション、アプリスクリーンショットのコラージュ
4. Gumroadセールスページもテンプレでカスタマイズ
```

### 効率化のコツ

**テンプレートブランド化:**
```
- フォント: 全素材でInter or Roboto統一
- 色: プライマリ#6366F1、セカンダリ#EC4899で統一
- ロゴ: 各素材の隅に配置
→ ブランド認知度向上
```

**複数アプリの同時対応:**
```
Canvaの「フォルダ」機能で、アプリごとにテンプレートを保存。
新規アプリのリリース時は、前のテンプレートをコピー＆色変更。
20分でアプリ全体のマーケティング素材が完成。
```

## 第3章: Stripe連携でサブスクリプション課金を実装

### Rork Max での StoreKit 2 実装

Rork Max は、Swift ネイティブの StoreKit 2 を自動統合可能。以下のプロンプトで実装：

```
このアプリにサブスクリプション課金を追加してください：

プラン1: 「月額プラン」 ¥980/月
- 全機能アンロック
- 優先サポート

プラン2: 「年額プラン」 ¥9,800/年（お得）
- 全機能アンロック
- 優先サポート
- 7日間の無料トライアル

支払いはApple In-App Purchaseで処理
```

Rork Max がこれを自動実装。曲線実装時間は2～3時間。

### ウェブ版での Stripe Checkout 実装

ウェブアプリ版の場合、Stripe Checkout を使用：

```typescript
// pages/api/create-checkout-session.ts
import Stripe from 'stripe';
import { NextApiRequest, NextApiResponse } from 'next';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!);

export default async function handler(
  req: NextApiRequest,
  res: NextApiResponse
) {
  if (req.method === 'POST') {
    try {
      const session = await stripe.checkout.sessions.create({
        line_items: [
          {
            price: 'price_xxxxx', // サブスク月額プラン
            quantity: 1,
          },
        ],
        mode: 'subscription',
        success_url: 'https://example.com/success?session_id={CHECKOUT_SESSION_ID}',
        cancel_url: 'https://example.com/cancel',
      });
      res.json({ id: session.id });
    } catch (err) {
      res.status(500).json({ error: (err as Error).message });
    }
  }
}
```

### 課金フロー

```
1. アプリ内で「プレミアムプランへアップグレード」ボタンをタップ
2. Stripe Checkoutが開く
3. ユーザーがクレジットカード情報を入力
4. サブスク開始→毎月自動課金
5. Webhook で サーバーに購入情報を通知
6. ユーザーがプレミアム機能を解放
```

## 第5章: YouTube マネタイズ戦略

### チャンネル立ち上げと初期コンテンツ

**第1～4週:**
```
1. チャンネルアート＆アイコンを Canva で制作
2. 初回動画（5～10分）: 「このアプリを1時間で作りました」
3. 次の動画: アプリの使い方チュートリアル
4. 定期投稿: 週1～2本のペース
```

### YouTube Partner Program 到達ロードマップ

```
【第1～3ヶ月】
- 毎週2本の動画投稿
- チャンネル登録者 100～200
- 総再生時間 500～1000時間
- →まだ収益化不可

【第4～6ヶ月】
- 毎週2本継続投稿
- バイラル動画が1～2本出現
- チャンネル登録者 500～800
- 総再生時間 2000～4000時間
- →まだ収益化不可

【第7～12ヶ月】
- チャンネル登録者 1000人 達成 ✓
- 総再生時間 4000時間 達成 ✓
- →YouTube Partner Program 認可
- 広告収益開始
```

### 収益額シミュレーション

```
チャンネル登録者 2000人、総再生時間 8000時間

アルゴリズムが安定化した場合：
- 月間平均再生数: 50,000～100,000
```

## 第8章: よくある質問（FAQ）

### Q1: 全チャネル対応は本当に可能ですか？

**A:** 段階的であれば可能です。

**フェーズ1（0～2ヶ月）:**
- App Store/Google Play に1～2アプリ公開
- 月5～15万円の広告収入

**フェーズ2（2～4ヶ月）:**
- ウェブ版を追加（Stripe連携）
- テンプレート販売を開始
- 月20～50万円へ

**フェーズ3（4～6ヶ月）:**
- YouTube チャンネル開始
- Kindle 出版
- 月50～100万円へ

焦らず段階的に進めることが成功のコツです。

### Q2: Figma＋Canvaの学習に時間がかかりませんか？

**A:** 初期学習は1～2週間で十分です。

**基本操作:**
- Figmaの画面設計: YouTube チュートリアル3本（3時間）
- Canva テンプレート編集: 実戦2～3回で習得（2時間）

その後は「テンプレートをコピーして色変更」だけなので、実務時間は30分/アプリ。

### Q3: Stripe の手数料は高くないですか？

**A:** App Store 比較で大幅に安い。

```
App Store での月額課金:
- アプリ内課金: 30% 手数料
- ¥980/月のプラン × 100ユーザー = ¥98,000
- 手数料: ¥29,400
- 手取り: ¥68,600

Stripe での月額課金:
- Stripe 手数料: 3.6% + 固定料
- ¥980/月 × 100ユーザー = ¥98,000
- 手数料: 約¥3,528
- 手取り: ¥94,472

→ 手数料だけで月2.6万円の差
```
