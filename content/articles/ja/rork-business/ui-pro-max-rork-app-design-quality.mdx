## UI Pro Max × Rork — エンタープライズグレードのデザイン品質

デザインの一貫性と高い品質は、ユーザー満足度と企業のブランドイメージに直結します。UI Pro Max は 97 種類の配色パレット、57 フォントペア、50 種類以上の UI スタイルを提供する、デザインシステムの決定版です。2026年、Rork との統合により、これらのプロ品質なデザイン資産を、アプリ開発プロセスに直接組み込むことが可能になりました。本記事では、UI Pro Max のデザイン標準を Rork アプリに完全に適用する方法を、実践的かつ詳細に解説します。

## Rork への UI Pro Max の統合

### Step 1: UI Pro Max デザイシステムのエクスポート

```bash
# UI Pro Max から Rork 互換形式でエクスポート
uipromax export \
  --format=rork \
  --include=colors,typography,components,spacing

# エクスポート内容:
# ✓ colors.json       - 97 配色パレット
# ✓ typography.json   - 57 フォントペア

# ✓ spacing.json      - スペーシング系
```

```bash
# Rork プロジェクトで UI Pro Max を統合
```

### フォントの実装例

```typescript
// src/components/Typography.tsx
// UI Pro Max のタイポグラフィ標準を使用

import React from 'react';
import { Text, StyleSheet } from 'react-native';
import { typographyPairs, typographyScale, lineHeights } from '@/design-tokens/typography';

interface TypographyProps {
  variant: 'h1' | 'h2' | 'h3' | 'body' | 'small' | 'caption';
  weight?: 'light' | 'normal' | 'medium' | 'bold';
  color?: string;
  children: React.ReactNode;
}

// UI Pro Max の "Apple Design" フォントペアを選択
const selectedPair = typographyPairs.appleDesign;

const styles = StyleSheet.create({
  h1: {
    fontFamily: selectedPair.heading.family,
    fontSize: typographyScale['4xl'],      // 32px
    fontWeight: '700',
    lineHeight: lineHeights.tight,         // 1.2
    letterSpacing: letterSpacings.tight,   // -0.02
  },
  h2: {
    fontFamily: selectedPair.heading.family,
    fontSize: typographyScale['3xl'],      // 28px
    fontWeight: '600',
    lineHeight: lineHeights.tight,
  },
  h3: {
    fontFamily: selectedPair.heading.family,
    fontSize: typographyScale['2xl'],      // 24px
    fontWeight: '600',
    lineHeight: lineHeights.normal,        // 1.5
  },
  body: {
    fontFamily: selectedPair.body.family,
    fontSize: typographyScale.base,        // 16px
    fontWeight: '400',
    lineHeight: lineHeights.relaxed,       // 1.75
    letterSpacing: letterSpacings.normal,
  },
  small: {
    fontFamily: selectedPair.body.family,
    fontSize: typographyScale.sm,          // 14px
    fontWeight: '400',
    lineHeight: lineHeights.normal,
  },
  caption: {
    fontFamily: selectedPair.body.family,
    fontSize: typographyScale.xs,          // 12px
    fontWeight: '500',
    lineHeight: lineHeights.tight,
  },
});

export const Typography: React.FC<TypographyProps> = ({
  variant,
  weight = 'normal',
  color = '#000000',
  children
}) => {
  return (
    <Text
      style={[
        styles[variant],
        { color },
        weight && { fontWeight: weight }
      ]}
    >
      {children}
    </Text>
  );
};
```

## UI スタイル・コンポーネントの適用

### UI Pro Max の 50+ UIスタイル

```typescript
// src/components/Button.tsx
// UI Pro Max の Button スタイル（自動生成）

import React from 'react';
import { Pressable, Text, StyleSheet } from 'react-native';
import { colors, spacing } from '@/design-tokens';

type ButtonVariant = 'primary' | 'secondary' | 'tertiary' | 'danger' | 'success';
type ButtonSize = 'sm' | 'md' | 'lg';

interface ButtonProps {
  variant?: ButtonVariant;
  size?: ButtonSize;
  disabled?: boolean;
  onPress: () => void;
  children: string;
}

// UI Pro Max の Button スタイル定義
const buttonStyles = {
  primary: {
    background: colors.primary,
    text: colors.white,
    border: 'none',
    shadow: '0 4px 6px rgba(0,0,0,0.1)',
  },
  secondary: {
    background: colors.secondary,
    text: colors.text,
    border: `1px solid ${colors.border}`,
    shadow: 'none',
  },
  tertiary: {
    background: 'transparent',
    text: colors.primary,
    border: 'none',
    shadow: 'none',
  },
  danger: {
    background: colors.error,
    text: colors.white,
    border: 'none',
    shadow: '0 4px 6px rgba(255,59,48,0.2)',
  },
  success: {
    background: colors.success,
    text: colors.white,
    border: 'none',
    shadow: '0 4px 6px rgba(52,199,89,0.2)',
  },
};

const sizeStyles = {
  sm: {
    padding: `${spacing.xs} ${spacing.sm}`,
    fontSize: 12,
    borderRadius: 6,
  },
  md: {
    padding: `${spacing.sm} ${spacing.md}`,
    fontSize: 16,
    borderRadius: 8,
  },
  lg: {
    padding: `${spacing.md} ${spacing.lg}`,
    fontSize: 18,
    borderRadius: 12,
  },
};

export const Button: React.FC<ButtonProps> = ({
  variant = 'primary',
  size = 'md',
  disabled = false,
  onPress,
  children
}) => {
  const buttonStyle = buttonStyles[variant];
  const sizeStyle = sizeStyles[size];

  return (
    <Pressable
      style={[
        {
          backgroundColor: disabled ? colors.disabled : buttonStyle.background,
          paddingVertical: parseInt(sizeStyle.padding.split(' ')[0]),
          paddingHorizontal: parseInt(sizeStyle.padding.split(' ')[1]),
          borderRadius: sizeStyle.borderRadius,
          opacity: disabled ? 0.5 : 1,
        }
      ]}
      onPress={onPress}
      disabled={disabled}
    >
      <Text
        style={{
          color: buttonStyle.text,
          fontSize: sizeStyle.fontSize,
          fontWeight: '600',
          textAlign: 'center',
        }}
      >
        {children}
      </Text>
    </Pressable>
  );
};
```
