# Rork Max × AI機能統合ガイド

AI機能を組み込むことは、Rork Max アプリの競争力を大きく高めます。このガイドでは、オンデバイス機械学習（TensorFlow Lite / Core ML）とクラウドベースのAI API（OpenAI / Gemini）の二つのアプローチを詳しく解説します。

## オンデバイス ML vs クラウド AI の選択基準

| 要素 | オンデバイス ML | クラウド AI API |
|------|-----------------|-----------------|
| レイテンシ | 低（<100ms） | 中程度（300ms～） |
| プライバシー | ✓ ローカルのみ | ⚠️ サーバー送信 |
| バンドルサイズ | 大（10～100MB） | 小 |
| 精度 | 限定的 | 高い |
| 更新 | 手動ダウンロード | 自動（サーバー側） |
| 初期投資 | モデル開発費 | API利用料 |
| インターネット | 不要 | 必須 |

<Callout type="info">
**推奨戦略**: オンデバイス ML でプライバシー重視の基本機能を、クラウド AI で高精度が必要な機能を提供します。例えば、顔認識はデバイス内、自然言語理解は API。
</Callout>

### TensorFlow Lite モデルのロード

**src/utils/tflite.ts**
```typescript
import * as tf from '@tensorflow/tfjs';
import '@tensorflow/tfjs-react-native';

export async function loadModel(modelUrl: string) {
  try {
    // TFLITE_WASM_BACKEND を使う（推奨）
    await tf.setBackend('wasm');
    const model = await tf.loadLayersModel(modelUrl);
    console.log('モデルロード成功:', model.summary());
    return model;
  } catch (error) {
    console.error('モデルロード失敗:', error);
    throw error;
  }
}

export async function predictImage(
  model: tf.LayersModel,
  imageData: number[][]
) {
  try {
    const input = tf.tensor(imageData, [1, 224, 224, 3]);
    const prediction = model.predict(input) as tf.Tensor;
    const result = await prediction.data();
    return Array.from(result);
  } catch (error) {
    console.error('推論失敗:', error);
    throw error;
  }
}
```

```typescript
  'Cat',
  'Bird',
  'Fish',

  'Hamster',
  'Parrot',
  'Turtle',

export default function ImageClassification() {
  const [model, setModel] = useState<tf.LayersModel | null>(null);
  const [prediction, setPrediction] = useState<string | null>(null);
  const [confidence, setConfidence] = useState<number | null>(null);

      setModel(loadedModel);
      setIsLoading(false);

      console.error('モデル初期化失敗:', error);
      setIsLoading(false);
    }
  };

  const handleCapture = async () => {
    if (!cameraRef || !model) return;

    setIsLoading(true);
    try {
      // カメラ画像をキャプチャ

        base64: true,
        quality: 0.8,
      });

      // Base64 を数値配列に変換
      const imageData = decodeBase64Image(photo.base64 || '');

      // 推論実行
      const predictions = await predictImage(model, imageData);

      // 最高スコアの結果を取得
      const topIndex = predictions.indexOf(Math.max(...predictions));
      const topScore = Math.max(...predictions);

      setPrediction(CLASS_LABELS[topIndex] || '不明');
      setConfidence((topScore * 100).toFixed(2));
    } catch (error) {
      console.error('キャプチャ失敗:', error);
    } finally {
      setIsLoading(false);
    }
  };

  const decodeBase64Image = (base64String: string): number[][] => {
    // 実装: Base64 をデコードして数値行列に変換
    // 簡略版
    return Array(224)
      .fill(null)
      .map(() => Array(224).fill(0));
  };

  if (isLoading && !model) {
    return (
      <View style={{ ...styles.container, justifyContent: 'center' }}>
        <ActivityIndicator size="large" />
        <Text style={{ marginTop: 12, textAlign: 'center' }}>
          モデルを読み込み中...
        </Text>
      </View>
    );
  }

  return (
    <View style={styles.container}>
      <CameraView
        ref={setCameraRef}
        style={styles.camera}
      />

      <TouchableOpacity
        style={styles.button}
        onPress={handleCapture}
        disabled={isLoading}
      >
        {isLoading ? (
          <ActivityIndicator color="#fff" />
        ) : (
          <Text style={styles.buttonText}>撮影して認識</Text>
        )}
      </TouchableOpacity>

      {prediction && (
        <View style={styles.resultContainer}>
          <Text style={styles.resultText}>認識結果: {prediction}</Text>
          <Text style={styles.confidenceText}>
            信頼度: {confidence}%
          </Text>
        </View>
      )}
    </View>
  );
}
```

## パート2: テキスト埋め込み & セマンティック検索

オンデバイスでテキスト埋め込みを生成し、セマンティック検索を実装します。

**src/utils/textEmbedding.ts**
```typescript
import * as tf from '@tensorflow/tfjs';

interface EmbeddingModel {
  model: tf.LayersModel;
  tokenizer: Map<string, number>;
}

export async function loadEmbeddingModel(): Promise<EmbeddingModel> {
  const model = await tf.loadLayersModel(
    'https://example.com/models/universal_sentence_encoder.json'
  );

  // トークナイザーをロード（簡略版）
  const tokenizer = new Map<string, number>();
  const words = [
    'hello',
    'world',
    'machine',
    'learning',
    'ai',
  ];
  words.forEach((word, idx) => tokenizer.set(word, idx + 1));

  return { model, tokenizer };
}

export function cosineSimilarity(a: number[], b: number[]): number {
  const dotProduct = a.reduce((sum, val, i) => sum + val * b[i], 0);
  const normA = Math.sqrt(a.reduce((sum, val) => sum + val * val, 0));
  const normB = Math.sqrt(b.reduce((sum, val) => sum + val * val, 0));

  return dotProduct / (normA * normB);
}

export async function getTextEmbedding(
  text: string,
  embeddingModel: EmbeddingModel
): Promise<number[]> {
  const tokens = text.toLowerCase().split(' ').map((word) => {
    return embeddingModel.tokenizer.get(word) || 0;
  });

  const input = tf.tensor2d([tokens], [1, tokens.length]);
  const output = embeddingModel.model.predict(input) as tf.Tensor;
  const embedding = await output.data();

  tf.dispose([input, output]);
  return Array.from(embedding);
}
```

## パート3: クラウド AI API との統合

### OpenAI API（GPT-4o）統合

**src/services/openaiService.ts**
```typescript
interface ChatMessage {
  role: 'user' | 'assistant' | 'system';
  content: string;
}

export class OpenAIService {
  private apiKey: string;
  private baseURL = 'https://api.openai.com/v1';

  constructor(apiKey: string) {
    this.apiKey = apiKey;
  }

  async chat(messages: ChatMessage[], model = 'gpt-4o'): Promise<string> {
    try {
      const response = await fetch(`${this.baseURL}/chat/completions`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          Authorization: `Bearer ${this.apiKey}`,
        },
        body: JSON.stringify({
          model,
          messages,
          max_tokens: 500,
          temperature: 0.7,
        }),
      });

      if (!response.ok) {
        throw new Error(`API Error: ${response.status}`);
      }

      const data = await response.json();
      return data.choices[0].message.content;
    } catch (error) {
      console.error('OpenAI API error:', error);
      throw error;
    }
  }

  async transcribeAudio(audioPath: string): Promise<string> {
    const formData = new FormData();
    formData.append('file', {
      uri: audioPath,
      type: 'audio/wav',
      name: 'audio.wav',
    } as any);
    formData.append('model', 'whisper-1');

    const response = await fetch(`${this.baseURL}/audio/transcriptions`, {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${this.apiKey}`,
      },
      body: formData,
    });

    const data = await response.json();
    return data.text;
  }

  async generateImage(prompt: string): Promise<string> {
    const response = await fetch(`${this.baseURL}/images/generations`, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        Authorization: `Bearer ${this.apiKey}`,
      },
      body: JSON.stringify({
        prompt,
        model: 'dall-e-3',
        n: 1,
        size: '1024x1024',
      }),
    });

    const data = await response.json();
    return data.data[0].url;
  }
}
```

### チャット UI の実装

**src/screens/ChatGPT.tsx**
```typescript
import React, { useState, useRef, useEffect } from 'react';
import {
  View,
  FlatList,
  TextInput,
  TouchableOpacity,

  ActivityIndicator,
  StyleSheet,
} from 'react-native';
import { OpenAIService } from '../services/openaiService';

const styles = StyleSheet.create({
  container: { flex: 1, backgroundColor: '#fff' },
  messageList: { flex: 1, padding: 16 },
  messageRow: { marginBottom: 12 },
  userMessage: {
    alignSelf: 'flex-end',
    backgroundColor: '#007AFF',
    padding: 12,
    borderRadius: 8,
    maxWidth: '80%',
  },
  assistantMessage: {
    alignSelf: 'flex-start',
    backgroundColor: '#E5E5EA',
    padding: 12,
    borderRadius: 8,
    maxWidth: '80%',
  },
  messageText: { color: '#000', fontSize: 14 },
  userText: { color: '#fff' },
  inputContainer: {
    flexDirection: 'row',
    padding: 16,
    borderTopWidth: 1,
    borderTopColor: '#E5E5EA',
  },
  input: {
    flex: 1,
    borderWidth: 1,
    borderColor: '#E5E5EA',
    borderRadius: 20,
    paddingHorizontal: 16,
    paddingVertical: 8,
    marginRight: 8,
  },
  sendButton: {
    width: 40,
    height: 40,
    borderRadius: 20,
    backgroundColor: '#007AFF',
    justifyContent: 'center',
    alignItems: 'center',
  },
});

export default function ChatGPT() {
  const [messages, setMessages] = useState<
    Array<{ role: string; content: string }>
  >([]);
  const [input, setInput] = useState('');
  const [isLoading, setIsLoading] = useState(false);
  const listRef = useRef(null);
  const openai = new OpenAIService(process.env.OPENAI_API_KEY || '');

  const handleSend = async () => {
    if (!input.trim()) return;

    const userMessage = { role: 'user', content: input };
    setMessages((prev) => [...prev, userMessage]);
    setInput('');
    setIsLoading(true);

    try {
      const response = await openai.chat([
        ...messages,
        userMessage,
      ]);

      setMessages((prev) => [
        ...prev,
        { role: 'assistant', content: response },
      ]);
    } catch (error) {
      console.error('チャットエラー:', error);
    } finally {
      setIsLoading(false);
    }
  };

  useEffect(() => {
    if (listRef.current) {
      listRef.current.scrollToEnd({ animated: true });
    }
  }, [messages]);

  return (
    <View style={styles.container}>
      <FlatList
        ref={listRef}
        data={messages}
        keyExtractor={(_, idx) => idx.toString()}
        renderItem={({ item }) => (
          <View style={styles.messageRow}>
            <View
              style={
                item.role === 'user'
                  ? styles.userMessage
                  : styles.assistantMessage
              }
            >
              <Text
                style={[
                  styles.messageText,
                  item.role === 'user' && styles.userText,
                ]}
              >
                {item.content}
              </Text>
            </View>
          </View>
        )}
        style={styles.messageList}
        contentContainerStyle={{ justifyContent: 'flex-end' }}
      />

      {isLoading && (
        <View style={{ padding: 16, alignItems: 'center' }}>
          <ActivityIndicator />
        </View>
      )}

      <View style={styles.inputContainer}>
        <TextInput
          style={styles.input}
          placeholder="メッセージを入力..."
          value={input}
          onChangeText={setInput}
          editable={!isLoading}
        />
        <TouchableOpacity
          style={styles.sendButton}
          onPress={handleSend}
          disabled={isLoading}
        >
          <Text style={{ color: '#fff', fontSize: 18 }}>
            {isLoading ? '' : '→'}
          </Text>
        </TouchableOpacity>
      </View>
    </View>
  );
}
```

## API キー管理のベストプラクティス

**src/services/apiClient.ts**
```typescript
export class APIClient {
  private baseURL = 'https://your-backend.com/api';

  async callAI(
    endpoint: string,
    payload: any
  ): Promise<any> {
    // バックエンドを経由して API を呼び出す
    const response = await fetch(
      `${this.baseURL}/${endpoint}`,
      {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          Authorization: `Bearer ${await this.getAuthToken()}`,
        },
        body: JSON.stringify(payload),
      }
    );

    if (!response.ok) {
      throw new Error('API request failed');
    }

    return response.json();
  }

  private async getAuthToken(): Promise<string> {
    // auth token をセキュアストレージから取得
    // (expo-secure-store を使用)
    return 'secure_token_here';
  }
}
```

## Gemini API 統合

Google の Gemini API でも同様に統合できます。

**src/services/geminiService.ts**
```typescript
export class GeminiService {
  private apiKey: string;
  private baseURL = 'https://generativelanguage.googleapis.com/v1beta/models';

  constructor(apiKey: string) {
    this.apiKey = apiKey;
  }

  async generateContent(prompt: string): Promise<string> {
    const response = await fetch(
      `${this.baseURL}/gemini-1.5-pro:generateContent?key=${this.apiKey}`,
      {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          contents: [
            {
              parts: [{ text: prompt }],
            },
          ],
        }),
      }
    );

    const data = await response.json();
    return data.candidates[0].content.parts[0].text;
  }
}
```

## まとめ

### オンデバイス ML の利点
- ✓ プライバシー第一
- ✓ インターネット不要
- ✓ 低レイテンシ
- ✗ モデルサイズ大
- ✗ 精度限定的

### クラウド AI の利点
- ✓ 高精度
- ✓ 継続的改善
- ✓ バンドルサイズ小
- ✗ インターネット必須
- ✗ API コスト発生

**推奨戦略**: 両者を組み合わせる。基本的な機能はオンデバイス、高度な処理はクラウド API。
