## Photoshop依存の現実 — 個人開発者の課題

Photoshop（Creative Cloud）の月額費用は約3,000円。年間36,000円です。加えて：

- **学習曲線が急**：UIデザインに特化した個人開発者には、Photoshopの高度な機能は不要
- **ワークフロー煩雑**：毎回手作業でレイアウト、テキスト配置、サイズ調整が必要
- **アップデート負担**：新しいOSに対応するたびに互換性問題が発生

こうした課題を解決するのが、**生成AI + 専門ツールの組み合わせ**です。本ロードマップでは、以下のツールで完全移行します：

| 用途 | ツール | 特徴 |
|------|--------|------|
| テキスト＆レイアウト案 | Claude | プロンプトだけでUIレイアウト設計 |

## ワークフロー比較表 — 従来型 vs AI型

### 従来型ワークフロー（Photoshop中心）

```
1. Photoshop開く → テンプレート読み込み
2. テキスト入力 → フォント選定 → 色調整
3. 背景画像をダウンロード → 配置 → トリミング
4. 各サイズ（1242×2208, 1080×1920等）に手動変換
5. PNG/JPEGエクスポート
6. 品質チェック → 修正が発生したら1に戻る
→ 1枚のスクリーンショット制作に 30～60分
```

### AI統合型ワークフロー

```
1. Claudeにプロンプト送信：「iOS App Store向けスクリーンショット。機能説明はXXX。背景はYYY」
2. Claude → HTMLレイアウト案 + 指示内容を返す
3. Canva / Figma で即座に実装
4. Midjourney で背景画像を並列生成
5. 統合 → 複数サイズ自動変換 → ダウンロード
→ 1枚のスクリーンショット制作に 5～10分（6枚セットなら30分以内）
```

**コスト比較**（年間、個人開発者）：
- Photoshop：36,000円 + 時間コスト
- AI統合型：Canva Pro（1,680円）+ Claude（月額20ドル） ≒ 6,000円 + 時間削減

## スクリーンショット制作のAI移行手順

App Store・Google Playでのアセット審査は、スクリーンショットの品質で大きく左右されます。以下は、Rorkアプリ実装例です。

### ステップ1：Claudeでレイアウト設計

Claudeに以下のプロンプトを送信：

```
あなたはApp Store UXデザイナーです。
以下の条件でiOS用スクリーンショット（1242×2208px）の構成案を、HTMLコンポーネント形式で提案してください：

【機能説明】
- アプリ名：Rork
- ターゲット：iOS個人開発者
- スクリーンショット1：「コード自動生成」機能
  - ヘッダーテキスト：「コード生成を自動化」
  - サブテキスト：「プロンプトを入力するだけで、iOSアプリのコードが5秒で完成」
  - 背景色：グラデーション（深青→紫）
  - CTA：「今すぐ試す」

【出力形式】
```html
<div class="screenshot-1">
  <!-- 背景画像 -->
  <!-- テキストレイアウト -->
  <!-- 色彩指定 -->
</div>
```

【追加指示】
- 白地にテキストではなく、背景に視覚的ハイライトを含める
- モバイル画面内に余白を適切に設定（上下マージン60px）
- フォント：San Francisco（iOS標準）, Poppins（見出し用）
```

Claude の返答例：

```html

### ステップ2：CanvaでUI実装（5分）

1. [Canva Pro](https://www.canva.com) を開く
2. 「カスタムサイズ」→ **1242 × 2208px** を指定
3. Claudeが提案したレイアウトを参考に、Canvaテンプレートでドラッグ＆ドロップで実装
4. 背景にMidjourney画像を挿入（下記参照）
5. 複数言語版は「テキストレイヤー置換」機能で一括変更

### ステップ3：Midjourneyで背景画像生成（3分並列）

```
/imagine prompt:
Abstract gradient background, deep blue to purple, tech aesthetic,
smooth gradients, minimalist, App Store screenshot background,
high quality, 1242x2208px ratio, modern app design, soft lighting
```

生成された画像をCanvaに挿入し、透明度・トリミングで調整。

### ステップ4：複数サイズ自動変換

Canvaの「サイズ変更」機能で以下を一度に出力：
- iPhone 6.5インチ（1242×2688px）
- iPhone 5.5インチ（1242×2208px）
- iPad（1536×2048px）

**結果：1枚のスクリーンショット完成までに約8分**（6枚セットなら40分以内）

## アイコン・バナーのAI移行手順

App StoreアイコンやGoogle Play バナーは、静止画よりもシンプルで、AI生成がより効果的です。

### ステップ1：Midjourneyでアイコンラフを生成

```
/imagine prompt:
App icon for code generation tool named Rork,
minimalist design, gradient background blue to purple,
coding symbol, pixel perfect, 1024x1024px,
modern flat design, professional, App Store icon
```

複数の変種を生成し、ダウンロードして比較。

### ステップ2：Figmaで微調整・エクスポート

生成されたアイコンをFigmaにインポート：
1. 背景を修正
2. 透明度・シャドウを追加
3. 複数サイズ（1024×1024 / 512×512 / 180×180 など）で自動エクスポート

**iOS要件**：角丸なし、背景透明PNG（背景設定はOS側で自動適用）

### ステップ3：Canvaでバナー・プロモ画像制作

Google Playの「バナー」は1200×628px。テンプレートから5分で制作可能。

```
Canvaテンプレート：「Google Play バナー」
→ アイコン＋テキストを配置
→ PNGエクスポート
```

**成果：アイコン＋バナー＋複数サイズで、従来1時間 → AI利用で15分以内**

CapCutで字幕テンプレートを複製、言語を変更。複数言語版を一括エクスポート。

## FAQ

### Q1. AI生成画像はApp Storeの審査で承認されますか？

**A.** はい、問題ありません。Apple・Googleの審査ガイドラインには「AI生成画像の禁止」という項目はなく、むしろ多くのアプリがMidjourneyやDALL-E 3を利用したアセットで承認されています。ただし、**著作権侵害・成人向けコンテンツは禁止**なので、プロンプトが具体的すぎないか確認してください。

### Q2. Photoshopの代替ツールは、細かい調整に対応していますか？

**A.** 対応しています。Figma・Canvaは基本的な調整（色、透明度、配置、フォント）に優れています。**ただし、複雑なマスク処理や色抽出などは、Figma + Pluginで実装するか、Adobe Expressを検討してください**。Rork開発では、Figmaで99%のニーズをカバーしています。

### Q3. 複数言語版（日本語・英語・中国語）を効率的に制作するには？

**A.** Canvaの「テンプレート複製」機能を使います。日本語版スクリーンショットを作成後、「プロジェクト複製」→ 各言語版で「テキスト置換」するだけ。背景・レイアウトは統一されるため、1セット作成後3言語版なら追加20分以内で完成します。
