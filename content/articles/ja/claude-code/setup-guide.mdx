## Claude Code とは

Claude Code は Anthropic が提供するコマンドラインツールで、ターミナルから直接 Claude にコーディングタスクを委任できます。ファイルの読み書き、Git操作、テスト実行など、開発ワークフローに深く統合されます。

<Callout type="info">
Claude Code は Max プラン（$100/月）または API キーで利用できます。Pro プラン（$20/月）では利用できません。
</Callout>

## インストール

### 前提条件

- Node.js 18 以上
- npm または yarn
- Anthropic アカウント（Max プランまたは API キー）

### インストールコマンド

```bash
npm install -g @anthropic-ai/claude-code
```

インストール後、以下のコマンドで動作確認：

```bash
claude --version
```

## 初回セットアップ

初めて `claude` コマンドを実行すると、認証フローが始まります。

```bash
claude
```

ブラウザが開き、Anthropic アカウントでのログインを求められます。認証が完了すると、ターミナルでClaude Codeが使えるようになります。

## 基本的な使い方

### 対話モード（デフォルト）

```bash
claude
```

対話モードでは、チャットのように Claude と会話しながら作業を進められます。

### ワンショットモード

```bash
claude -p "このディレクトリのREADME.mdを日本語に翻訳して"
```

`-p` フラグで単発のタスクを実行できます。スクリプトやCI/CDパイプラインとの連携に便利です。

### よく使うコマンド

| コマンド | 説明 |
|---------|------|
| `claude` | 対話モードを開始 |
| `claude -p "..."` | ワンショットでタスク実行 |
| `claude -r` | 直前のセッションを再開 |
| `claude config` | 設定の確認・変更 |
| `claude mcp` | MCP サーバーの管理 |

## スラッシュコマンド

対話モード内では、スラッシュコマンドが使えます：

```
/help        — ヘルプを表示
/clear       — 会話履歴をクリア
/compact     — 会話を要約してコンテキストを圧縮
/cost        — 現在のセッションのトークン使用量を表示
/doctor      — 設定の問題を診断
/init        — CLAUDE.md プロジェクト設定ファイルを作成
/review      — コードレビューを開始
/terminal-setup — ターミナルのキーバインドを設定
```

<Callout type="tip">
`/init` で作成される `CLAUDE.md` ファイルに、プロジェクトのコーディング規約やアーキテクチャの説明を書いておくと、Claude Code がプロジェクトの文脈を理解した上でコードを生成してくれます。
</Callout>

## CLAUDE.md でプロジェクトを設定する

プロジェクトルートに `CLAUDE.md` を作成すると、Claude Code がプロジェクトの文脈を自動的に読み取ります。

```markdown
# プロジェクト概要
Next.js 16 + TypeScript のウェブアプリケーション

# 技術スタック
- Next.js 16 (App Router)
- Tailwind CSS v4
- Cloudflare Workers

# コーディング規約
- 関数コンポーネントを使用
- TypeScript strict モード
- テストは vitest を使用
```

## MCP（Model Context Protocol）

Claude Code は MCP サーバーを通じて外部サービスと連携できます。

```bash
# MCP サーバーの追加
claude mcp add slack -- npx -y @anthropic-ai/slack-mcp

# 登録済みサーバーの一覧
claude mcp list
```

<Callout type="warning">
MCP サーバーはローカル環境で実行されます。信頼できるソースからのみインストールしてください。
</Callout>

## 次のステップ

- **バックグラウンドタスク** — GitHub Actions との連携で CI/CD に Claude を組み込む
- **カスタムスラッシュコマンド** — プロジェクト固有のワークフローを自動化
- **プラグイン** — MCP サーバーやスキルを組み合わせた高度な自動化

## よくある質問

### Claude Code は Pro プランで使えますか？

いいえ、Claude Code は Max プラン（$100/月以上）または API キーが必須です。Pro プラン（$20/月）では利用できません。個人開発者で限定的に使う場合、API キーを発行して従量課金で利用する方法もあります。

### CLAUDE.md ファイルは必須ですか？

いいえ、必須ではありませんが、推奨されます。CLAUDE.md にプロジェクトの説明、技術スタック、コーディング規約を記述すると、Claude Code がより正確で一貫性のあるコードを生成できます。プロジェクト開始時に `/init` で作成するのが便利です。

### オフラインで Claude Code は使えますか？

いいえ、Claude Code はクラウドベースのサービスなため、API 通信が必須です。インターネット接続が必要です。ただし、生成されたコードと設定ファイルはローカルに保存され、オフラインでも編集・実行できます。

### MCP サーバーを自作できますか？

はい、可能です。MCP は標準プロトコルであり、独自の MCP サーバーを実装して Claude Code に接続できます。詳細は MCP サーバー構築ガイドを参照してください。信頼できるソースのサーバーのみインストールしてください。
