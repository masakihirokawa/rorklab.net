## MCP とは

**Model Context Protocol（MCP）** は、AI モデルが外部ツールやデータソースと連携するための標準プロトコルです。Claude Code は MCP サーバーを通じて、データベースへのクエリ、API の呼び出し、ファイルシステムの操作など、あらゆる機能を追加できます。

## MCP の仕組み

MCP はクライアント（Claude Code）とサーバー（ツール提供側）の間の通信プロトコルです。

1. **サーバー**がツール（関数）の一覧と説明を公開
2. **Claude Code** がその一覧を認識し、必要に応じてツールを呼び出し
3. **サーバー**がツールを実行し、結果を返送
4. **Claude Code** が結果を理解して次のアクションを決定

## MCP サーバーの設定

Claude Code の設定ファイルに MCP サーバーを追加します。

```json
{
  "mcpServers": {
    "my-database": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres", "postgresql://localhost/mydb"]
    }
  }
}
```

設定ファイルの場所は以下の通りです。

- プロジェクト単位: `.claude/settings.json`
- ユーザー単位: `~/.claude/settings.json`

## 主要な公式 MCP サーバー

Anthropic とコミュニティが提供する公式サーバーの例です。

### ファイルシステム

ファイルの読み書き、ディレクトリ操作を安全に行います。

### PostgreSQL / SQLite

データベースへの接続、クエリ実行、スキーマ情報の取得ができます。

### GitHub

イシューの管理、PR の操作、リポジトリ情報の取得が可能です。

### Slack

メッセージの送受信、チャンネル管理が行えます。

### Web ブラウザ

ウェブページの取得とスクレイピングが可能です。

## カスタム MCP サーバーの構築

TypeScript で独自の MCP サーバーを構築できます。

```typescript
import { Server } from "@modelcontextprotocol/sdk/server";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio";

const server = new Server({
  name: "my-custom-server",
  version: "1.0.0",
});

// ツールの定義
server.setRequestHandler("tools/list", async () => ({
  tools: [{
    name: "get_weather",
    description: "指定した都市の天気を取得",
    inputSchema: {
      type: "object",
      properties: {
        city: { type: "string", description: "都市名" }
      },
      required: ["city"]
    }
  }]
}));

// ツールの実行
server.setRequestHandler("tools/call", async (request) => {
  if (request.params.name === "get_weather") {
    const city = request.params.arguments.city;
    // 天気 API を呼び出す処理
    return { content: [{ type: "text", text: `${city}の天気: 晴れ 22°C` }] };
  }
});

const transport = new StdioServerTransport();
await server.connect(transport);
```

## セキュリティに関する注意

MCP サーバーは強力ですが、セキュリティに注意が必要です。

- **信頼できるサーバーのみ使用する**: 第三者の MCP サーバーはコードを確認してから導入
- **最小権限の原則**: データベース接続は読み取り専用アカウントを使用
- **機密情報の管理**: API キーは環境変数で管理し、設定ファイルにハードコードしない

## まとめ

MCP は Claude Code を単なるコーディング支援ツールから、あらゆるシステムと連携できる万能エージェントへと進化させます。公式サーバーから始めて、必要に応じてカスタムサーバーを構築していきましょう。
