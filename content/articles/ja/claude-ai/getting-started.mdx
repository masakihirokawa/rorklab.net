## Claude とは

Claude は Anthropic が開発した AI アシスタントです。文章の作成、コードの生成、翻訳、要約、分析など、幅広いタスクをこなすことができます。

<Callout type="info">
Claude は claude.ai のウェブアプリ、モバイルアプリ、API、Claude Code（CLI）、Cowork（デスクトップ）など、複数の方法で利用できます。
</Callout>

## アカウントを作成する

1. [claude.ai](https://claude.ai) にアクセス
2. 「サインアップ」をクリック
3. メールアドレスまたは Google アカウントで登録
4. メール認証を完了

無料プランでも Claude Sonnet を使った会話が可能です。より高性能な Claude Opus を使いたい場合は Pro プラン（月額 $20）へのアップグレードを検討してください。

## 最初の会話を始める

ログイン後、チャット画面が表示されます。テキストボックスにメッセージを入力するだけで会話が始まります。

### 良いプロンプトの例

```
以下のPythonコードのバグを見つけて修正してください。
修正箇所には「# 修正」とコメントを付けてください。

def calculate_average(numbers):
    total = 0
    for n in numbers:
        total += n
    return total / len(numbers)  # 空リストの場合にZeroDivisionError
```

### 効果的なプロンプトのコツ

- **具体的に書く**: 「良い文章を書いて」ではなく「300文字以内のビジネスメールを書いて」
- **コンテキストを与える**: 背景情報、対象読者、目的を伝える
- **出力形式を指定する**: 「箇条書きで」「表形式で」「JSON で」など
- **ステップバイステップを促す**: 「段階的に考えてください」と伝える

<Callout type="tip">
Claude は長い文脈を理解できます。関連するドキュメントや参考資料をそのまま貼り付けて、それについて質問することも効果的です。
</Callout>

## プランの比較

| プラン | モデル | 価格 | 特徴 |
|--------|--------|------|------|
| Free | Sonnet | 無料 | 基本的な会話、回数制限あり |
| Pro | Opus / Sonnet | $20/月 | 高性能モデル、優先アクセス |
| Team | Opus / Sonnet | $25/月/人 | チーム管理、共有ワークスペース |
| Enterprise | 全モデル | 要相談 | SSO、監査ログ、SLA |
