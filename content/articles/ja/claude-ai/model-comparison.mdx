## Claude のモデルラインナップ

Anthropic は用途や予算に応じて選べる3つのモデルファミリーを提供しています。それぞれに特徴があり、タスクの複雑さ、速度要件、コストのバランスで最適な選択が変わります。

## Claude Opus 4 — 最高性能モデル

**Opus** は Claude ファミリーの最上位モデルです。

### 得意な用途

- 複雑な推論やマルチステップの分析
- 高品質なコード生成・レビュー
- 学術論文レベルの長文執筆
- ニュアンスの繊細な翻訳

### 特徴

Opus は他のモデルと比較して応答時間は長めですが、特に難しい問題での正確性と深い洞察が際立ちます。コーディングベンチマークでは業界最高水準のスコアを記録しています。

## Claude Sonnet 4 — バランス型モデル

**Sonnet** は性能とコストの最適なバランスを実現するモデルです。

### 得意な用途

- 日常的なチャットとQ&A
- ビジネス文書の作成・編集
- コード生成と簡単なデバッグ
- データ分析とレポート作成

### 特徴

Opus に迫る性能を持ちながら、応答速度は大幅に高速です。多くのユースケースで最もコストパフォーマンスが高い選択肢となります。Claude.ai のデフォルトモデルとして採用されています。

## Claude Haiku 3.5 — 高速・低コストモデル

**Haiku** は速度とコスト効率に最適化されたモデルです。

### 得意な用途

- リアルタイムチャットボット
- 大量のテキスト分類・ラベリング
- 簡単な要約やフォーマット変換
- API を使った大規模バッチ処理

### 特徴

応答速度が極めて高速で、API 利用時のコストも最も低く抑えられます。シンプルなタスクであれば Sonnet と遜色ない品質を発揮します。

## モデル選択フローチャート

用途に応じた選び方の目安です。

1. **最高品質が必要**（論文、重要なコードレビュー）→ **Opus**
2. **品質と速度のバランスが重要**（日常業務、一般的なコーディング）→ **Sonnet**
3. **速度・コストが最優先**（チャットボット、大量処理）→ **Haiku**

## API での料金体系

API 利用時の料金はモデルによって大きく異なります。入力トークンと出力トークンで別々に課金されます。

- **Opus**: 最も高額だが、1回のやり取りで解決できるため結果的にコスト効率が良い場合も
- **Sonnet**: 中程度の料金設定。多くのプロジェクトで推奨
- **Haiku**: 最も安価。大量処理やプロトタイピングに最適

最新の料金情報は [Anthropic の公式サイト](https://www.anthropic.com/pricing) で確認してください。

## まとめ

迷ったらまず **Sonnet** から始めるのがおすすめです。品質が足りなければ Opus に、コストを下げたければ Haiku に切り替えるという段階的なアプローチが効率的です。
