## Artifacts とは

**Artifacts** は Claude.ai で利用できる、対話の中でコンテンツを生成しリアルタイムにプレビューできる機能です。コードの実行結果、React コンポーネント、SVG 図表、Mermaid ダイアグラムなどを会話ウィンドウの右側に表示できます。

## 対応コンテンツタイプ

### コードスニペット

Python、JavaScript、TypeScript などのコードを記述し、シンタックスハイライト付きで表示します。コピーボタンですぐに利用できます。

### React コンポーネント

インタラクティブな UI コンポーネントをその場でレンダリングします。Tailwind CSS のユーティリティクラスが使用可能で、`useState` や `useEffect` などの React Hooks にも対応しています。

```
「ダークモード切り替え付きのカウンターアプリを Artifact で作って」
```

### SVG グラフィック

ベクター画像を直接プレビューできます。ロゴデザイン、アイコン、簡単なイラストレーションに便利です。

### Mermaid ダイアグラム

フローチャート、シーケンス図、ER 図、ガントチャートなどを Mermaid 記法で生成し、ビジュアルにレンダリングします。

```
「ユーザー登録のフローチャートを Mermaid で作って」
```

### HTML ページ

CSS と JavaScript を含む完全な HTML ページをプレビューできます。ランディングページのプロトタイプやインタラクティブなデモに最適です。

### マークダウン文書

構造化された文書をフォーマット付きで表示します。レポート、ガイド、ドキュメントの下書きに使えます。

## Artifacts の活用例

### プロトタイピング

アイデアを素早くビジュアル化できます。「こんな UI はどうか」と Claude に伝えるだけで、動作する React コンポーネントが生成されます。

### データの可視化

CSV データを渡して「棒グラフにして」と指示すれば、Recharts を使ったインタラクティブなチャートが Artifact として表示されます。

### 技術文書の作成

アーキテクチャ図やシーケンス図を Mermaid で作成し、技術文書に組み込めます。

### 学習・教育

アルゴリズムの動作をビジュアルアニメーションで確認したり、数学の概念をインタラクティブに探索できます。

## 効果的な使い方のコツ

1. **具体的に指示する**: 「ダッシュボードを作って」より「売上データの折れ線グラフと目標達成率の円グラフを並べたダッシュボードを作って」の方が良い結果になります
2. **段階的に改善する**: 最初のバージョンを見てから「色をもっと落ち着いたトーンに」「グラフにツールチップを追加して」と調整していきましょう
3. **ライブラリを指定する**: Recharts、D3、Three.js など使いたいライブラリを明示すると、それに合わせたコードが生成されます

## 利用可能なライブラリ

Artifacts では以下のライブラリがインポート可能です。

- **React** — コンポーネントとフック
- **Recharts** — グラフ・チャート
- **Lucide React** — アイコンセット
- **D3.js** — データビジュアライゼーション
- **Three.js** — 3D グラフィック
- **Tailwind CSS** — ユーティリティファーストの CSS
- **shadcn/ui** — UI コンポーネント

## まとめ

Artifacts は Claude を単なるテキストチャットから、インタラクティブなクリエイティブツールへと進化させる機能です。プロトタイピング、データ可視化、ドキュメント作成など、さまざまな場面で活用してみてください。
