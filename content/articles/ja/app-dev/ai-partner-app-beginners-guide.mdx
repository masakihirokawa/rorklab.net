# 初級ガイド：React NativeでAIコンパニオンアプリを作ろう

自分だけのAIコンパニオンアプリを作ってみたいと思いませんか？ExpoとReact Nativeなら、数時間でチャットインターフェースを作ることができます。このガイドでは、基本的なAIパートナーアプリをゼロから構築していきます。

---

## 何を作るのか

このアプリは以下の機能を持ちます：
- チャット形式でAIと会話
- ClaudeやOpenAIのAPIにリクエスト送信
- AsyncStorageで会話履歴を保存
- シンプルなAIキャラクターのペルソナ設定

ReplicaやCharacter.aiの初級版といった感じです。派手ではありませんが、実際に動作します。

---

## プロジェクトセットアップ

まずは新しいExoプロジェクトを作ります：

```bash
npx create-expo-app RorkLabCompanion
cd RorkLabCompanion
npx expo install
```

必要な依存関係をインストール：

```bash
npm install @react-native-async-storage/async-storage axios
npx expo install @react-native-async-storage/async-storage
```

## 基本的なチャットUI

まずはメインのチャットインターフェースを作成します。ScrollViewでチャット泡を表示：

```typescript
// app/screens/ChatScreen.tsx
import React, { useState, useEffect } from 'react';
import {
  View,
  Text,
  ScrollView,
  TextInput,
  TouchableOpacity,
  StyleSheet,
  KeyboardAvoidingView,
  Platform,
} from 'react-native';
import AsyncStorage from '@react-native-async-storage/async-storage';

interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: number;
}

export default function ChatScreen() {
  const [messages, setMessages] = useState<Message[]>([]);
  const [input, setInput] = useState('');
  const [loading, setLoading] = useState(false);

  // マウント時に保存された会話を読込
  useEffect(() => {
    loadMessages();
  }, []);

  const loadMessages = async () => {
    try {
      const saved = await AsyncStorage.getItem('chatHistory');
      if (saved) {
        setMessages(JSON.parse(saved));
      }
    } catch (error) {
      console.error('メッセージ読込失敗:', error);
    }
  };

  const saveMessages = async (msgs: Message[]) => {
    try {
      await AsyncStorage.setItem('chatHistory', JSON.stringify(msgs));
    } catch (error) {
      console.error('メッセージ保存失敗:', error);
    }
  };

  const sendMessage = async () => {
    if (!input.trim()) return;

    const userMessage: Message = {
      id: Date.now().toString(),
      role: 'user',
      content: input,
      timestamp: Date.now(),
    };

    const updated = [...messages, userMessage];
    setMessages(updated);
    setInput('');
    setLoading(true);

    try {
      const response = await getAIResponse(updated);
      const assistantMessage: Message = {
        id: (Date.now() + 1).toString(),
        role: 'assistant',
        content: response,
        timestamp: Date.now(),
      };

      const finalMessages = [...updated, assistantMessage];
      setMessages(finalMessages);
      saveMessages(finalMessages);
    } catch (error) {
      console.error('APIエラー:', error);
      setMessages(updated);
    } finally {
      setLoading(false);
    }
  };

  return (
    <KeyboardAvoidingView
      style={styles.container}
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
    >
      <ScrollView style={styles.messageContainer}>
        {messages.length === 0 && (
          <View style={styles.welcomeContainer}>
            <Text style={styles.welcomeText}>こんにちは！僕はあなたのAIパートナーです。</Text>
            <Text style={styles.welcomeSubtext}>
              話しかけてチャットを始めましょう！
            </Text>
          </View>
        )}
        {messages.map((msg) => (
          <View
            key={msg.id}
            style={[
              styles.messageBubble,
              msg.role === 'user' ? styles.userMessage : styles.aiMessage,
            ]}
          >
            <Text style={styles.messageText}>{msg.content}</Text>
          </View>
        ))}
        {loading && (
          <View style={styles.aiMessage}>
            <Text style={styles.messageText}>考え中...</Text>
          </View>
        )}
      </ScrollView>

      <View style={styles.inputContainer}>
        <TextInput
          style={styles.input}
          placeholder="メッセージを入力..."
          value={input}
          onChangeText={setInput}
          multiline
          editable={!loading}
          placeholderTextColor="#999"
        />
        <TouchableOpacity
          style={[styles.sendButton, loading && styles.sendButtonDisabled]}
          onPress={sendMessage}
          disabled={loading}
        >
          <Text style={styles.sendButtonText}>送信</Text>
        </TouchableOpacity>
      </View>
    </KeyboardAvoidingView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  messageContainer: {
    flex: 1,
    padding: 16,
  },
  welcomeContainer: {
    paddingVertical: 40,
    alignItems: 'center',
  },
  welcomeText: {
    fontSize: 18,
    fontWeight: '600',
    color: '#333',
    marginBottom: 8,
  },
  welcomeSubtext: {
    fontSize: 14,
    color: '#999',
  },
  messageBubble: {
    marginVertical: 8,
    maxWidth: '85%',
    paddingHorizontal: 12,
    paddingVertical: 10,
    borderRadius: 12,
  },
  userMessage: {
    alignSelf: 'flex-end',
    backgroundColor: '#007AFF',
  },
  aiMessage: {
    alignSelf: 'flex-start',
    backgroundColor: '#f0f0f0',
  },
  messageText: {
    fontSize: 16,
    color: '#000',
  },
  inputContainer: {
    flexDirection: 'row',
    padding: 12,
    borderTopWidth: 1,
    borderTopColor: '#eee',
    backgroundColor: '#fff',
  },
  input: {
    flex: 1,
    borderWidth: 1,
    borderColor: '#ddd',
    borderRadius: 20,
    paddingHorizontal: 16,
    paddingVertical: 10,
    marginRight: 8,
    maxHeight: 100,
    color: '#000',
  },
  sendButton: {
    width: 44,
    height: 44,
    borderRadius: 22,
    backgroundColor: '#007AFF',
    justifyContent: 'center',
    alignItems: 'center',
  },
  sendButtonDisabled: {
    opacity: 0.5,
  },
  sendButtonText: {
    color: '#fff',
    fontWeight: '600',
  },
});
```

## Claude APIに接続

次はAI APIの統合です。`services/api.ts`を作成：

```typescript
// services/api.ts
import axios from 'axios';

const CLAUDE_API_KEY = 'your-api-key-here';
const SYSTEM_PROMPT = `あなたは親友のようなAIコンパニオンです。
ユーザーの話に耳を傾け、励まし、サポートします。
回答は簡潔で温かみのあるものにしてください。`;

interface Message {
  role: 'user' | 'assistant';
  content: string;
}

export async function getAIResponse(messages: Message[]): Promise<string> {
  try {
    const response = await axios.post(
      'https://api.anthropic.com/v1/messages',
      {
        model: 'claude-3-5-sonnet-20241022',
        max_tokens: 1024,
        system: SYSTEM_PROMPT,
        messages: messages.map((m) => ({
          role: m.role,
          content: m.content,
        })),
      },
      {
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': CLAUDE_API_KEY,
          'anthropic-version': '2023-06-01',
        },
      }
    );

    return response.data.content[0].text;
  } catch (error) {
    console.error('Claudeエラー:', error);
    throw error;
  }
}
```

## 重要：APIキーの安全な管理

**APIキーを直接コードに書き込まないでください！** 本番環境では、バックエンドサービスが必要です。初心者段階では：

- シンプルなNode.jsバックエンドをプロキシとして使用
- EAS buildで環境変数を使用
- APIキーをアプリバイナリに含めないこと

## AsyncStorageで会話を保存

既にAsyncStorageで会話を保存する機能を組込みました。ユーザーがアプリを閉じても、チャット履歴は戻ってくるときに残ります：

```typescript
const saveMessages = async (msgs: Message[]) => {
  try {
    await AsyncStorage.setItem('chatHistory', JSON.stringify(msgs));
  } catch (error) {
    console.error('保存失敗:', error);
  }
};
```

## 重要：AIの限界と正直さ

AIコンパニオンは単なる言語モデルです。感情を本当に理解しておらず、セッション間でメモリを持たず、本当の愛情を感じることもできません。ユーザーに正直に伝えることが大切です。信頼を作り、健全な期待値を設定できます。
