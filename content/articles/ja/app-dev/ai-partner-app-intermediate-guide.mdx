# 中級ガイド：洗練されたAIコンパニオン - UI、メモリ、ペルソナ

初めてのチャットアプリを作りましたね。次は本番レベルにしましょう。複数のAIペルソナ、永続的なメモリ、スムーズなアニメーション、プッシュ通知を追加します。ReplicaやCharacter.aiのような雰囲気になります。

## アーキテクチャ概要

中級のコンパニオンアプリには複数の層があります：

1. **UIレイヤー** — 洗練されたチャット泡、タイピングインジケータ、アバター
2. **ストレージレイヤー** — 会話履歴用SQLite
3. **メモリシステム** — プロンプトに注入される会話要約
4. **ペルソナシステム** — AIキャラクター間の切り替え
5. **通知システム** — チャット催促のリマインダー

各層を構築していきましょう。

## SQLiteのセットアップ

まずSQLiteをインストール：

```bash
npx expo install expo-sqlite
```

データベースサービスを作成：

```typescript
// services/database.ts
import * as SQLite from 'expo-sqlite';

const db = SQLite.openDatabase('companion.db');

export interface StoredMessage {
  id: string;
  conversationId: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: number;
}

export interface Conversation {
  id: string;
  personaId: string;
  title: string;
  createdAt: number;
  lastMessageAt: number;
}

export async function initializeDatabase() {
  return new Promise<void>((resolve, reject) => {
    db.transaction((tx) => {
      tx.executeSql(
        `CREATE TABLE IF NOT EXISTS conversations (
          id TEXT PRIMARY KEY,
          personaId TEXT NOT NULL,
          title TEXT NOT NULL,
          createdAt NUMBER NOT NULL,
          lastMessageAt NUMBER NOT NULL
        );`,
        [],
        () => console.log('会話テーブル作成完了'),
        (_, error) => reject(error)
      );

      tx.executeSql(
        `CREATE TABLE IF NOT EXISTS messages (
          id TEXT PRIMARY KEY,
          conversationId TEXT NOT NULL,
          role TEXT NOT NULL,
          content TEXT NOT NULL,
          timestamp NUMBER NOT NULL,
          FOREIGN KEY (conversationId) REFERENCES conversations(id)
        );`,
        [],
        () => console.log('メッセージテーブル作成完了'),
        (_, error) => reject(error)
      );

      tx.executeSql(
        `CREATE TABLE IF NOT EXISTS personas (
          id TEXT PRIMARY KEY,
          name TEXT NOT NULL,
          systemPrompt TEXT NOT NULL,
          avatar TEXT,
          color TEXT
        );`,
        [],
        () => {
          console.log('ペルソナテーブル作成完了');
          resolve();
        },
        (_, error) => reject(error)
      );
    });
  });
}

export async function saveMessage(message: StoredMessage): Promise<void> {
  return new Promise((resolve, reject) => {
    db.transaction((tx) => {
      tx.executeSql(
        `INSERT INTO messages (id, conversationId, role, content, timestamp)
         VALUES (?, ?, ?, ?, ?);`,
        [
          message.id,
          message.conversationId,
          message.role,
          message.content,
          message.timestamp,
        ],
        () => resolve(),
        (_, error) => reject(error)
      );
    });
  });
}

export async function getConversationMessages(
  conversationId: string
): Promise<StoredMessage[]> {
  return new Promise((resolve, reject) => {
    db.transaction((tx) => {
      tx.executeSql(
        `SELECT * FROM messages WHERE conversationId = ? ORDER BY timestamp ASC;`,
        [conversationId],
        (_, result) => {
          resolve(result.rows._array as StoredMessage[]);
        },
        (_, error) => reject(error)
      );
    });
  });
}
```

## 要約によるメモリシステム

古いメッセージを要約して会話メモリを実装：

```typescript
// services/memory.ts
import axios from 'axios';

const CLAUDE_API_KEY = process.env.CLAUDE_API_KEY;

export async function summarizeConversation(
  messages: Array<{ role: string; content: string }>
): Promise<string> {
  const messagesToSummarize = messages.slice(0, -10);

  if (messagesToSummarize.length < 5) {
    return '';
  }

  try {
    const response = await axios.post(
      'https://api.anthropic.com/v1/messages',
      {
        model: 'claude-3-5-sonnet-20241022',
        max_tokens: 500,
        messages: [
          {
            role: 'user',
            content: `この会話を2～3文で要約してください。
ユーザーについて学んだことに焦点を当ててください：\n\n${messagesToSummarize
              .map((m) => `${m.role}: ${m.content}`)
              .join('\n')}`,
          },
        ],
      },
      {
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': CLAUDE_API_KEY,
          'anthropic-version': '2023-06-01',
        },
      }
    );

    return response.data.content[0].text;
  } catch (error) {
    console.error('要約失敗:', error);
    return '';
  }
}

export async function getAIResponseWithMemory(
  messages: Array<{ role: string; content: string }>,
  persona: any,
  memorySummary: string
): Promise<string> {
  const memoryContext = memorySummary
    ? `\n\n以前の会話要約：${memorySummary}`
    : '';

  const systemPrompt = persona.systemPrompt + memoryContext;

  try {
    const response = await axios.post(
      'https://api.anthropic.com/v1/messages',
      {
        model: 'claude-3-5-sonnet-20241022',
        max_tokens: 1024,
        system: systemPrompt,
        messages: messages,
      },
      {
        headers: {
          'Content-Type': 'application/json',
          'x-api-key': CLAUDE_API_KEY,
          'anthropic-version': '2023-06-01',
        },
      }
    );

    return response.data.content[0].text;
  } catch (error) {
    console.error('APIエラー:', error);
    throw error;
  }
}
```

```typescript
// screens/PersonaScreen.tsx
import React from 'react';
import { View, TouchableOpacity, Text, StyleSheet, ScrollView } from 'react-native';
import { PERSONAS } from '../services/personas';

interface Props {
  onSelectPersona: (personaId: string) => void;
  currentPersonaId: string;
}

export default function PersonaScreen({
  onSelectPersona,
  currentPersonaId,
}: Props) {
  return (
    <ScrollView style={styles.container}>
      <Text style={styles.title}>コンパニオンを選ぶ</Text>

      {PERSONAS.map((persona) => (
        <TouchableOpacity
          key={persona.id}
          style={[
            styles.personaCard,
            currentPersonaId === persona.id && styles.selectedCard,
          ]}
          onPress={() => onSelectPersona(persona.id)}
        >
          <Text style={styles.avatarEmoji}>{persona.avatarEmoji}</Text>
          <View style={styles.personaInfo}>
            <Text style={styles.personaName}>{persona.name}</Text>
```

```typescript
// services/notifications.ts
import * as Notifications from 'expo-notifications';

export async function requestNotificationPermission() {
  const { status } = await Notifications.requestPermissionsAsync();
  return status === 'granted';
}

export async function scheduleDailyNotification() {
  await Notifications.scheduleNotificationAsync({
    content: {
      title: 'あなたのコンパニオンがあなたのことを考えています！',
      body: 'チャットして最新情報を共有しましょう',
      sound: 'default',

      minute: 0,
      repeats: true,
```

## ダークモード対応

Contextを使ったテーマサポート：

```typescript
// context/ThemeContext.tsx
import React, { createContext, useState } from 'react';

export interface Theme {
  dark: boolean;
  colors: {
    background: string;
    text: string;
    border: string;
    userBubble: string;
  };
}

const LIGHT_THEME: Theme = {
  dark: false,
  colors: {
    background: '#fff',
    text: '#000',
    border: '#eee',
    userBubble: '#007AFF',
  },
};

const DARK_THEME: Theme = {
  dark: true,
  colors: {
    background: '#1a1a1a',
    text: '#fff',
    border: '#333',
    userBubble: '#0084FF',
  },
};

export const ThemeContext = createContext(LIGHT_THEME);

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  const [theme, setTheme] = useState<Theme>(LIGHT_THEME);

  const toggleTheme = () => {
    setTheme(theme.dark ? LIGHT_THEME : DARK_THEME);
  };

  return (
    <ThemeContext.Provider value={theme}>
      {children}
    </ThemeContext.Provider>
  );
}
```
