## あなたの「あったらいいな」が、作れる

前回の記事「[30分でアプリ完成！](/articles/rork-basics/rork-first-app-30min)」で最初のアプリを作った方は、もう一つの大事なことに気づいたはずです。「思ったより簡単だった！」ということに。

今回は、日常生活で「こんなアプリがあったら便利なのに」と感じる場面ごとに、Rorkでどう実現するかをご紹介します。

## 健康管理に役立つアプリ

### 血圧記録アプリ

```
あなた：「毎日の血圧（上・下）と脈拍を記録できる
         アプリを作ってください。
         ・日付と時間は自動入力
         ・過去の記録を一覧で見られる
         ・グラフで推移がわかるようにして
         ・文字は大きめにして見やすく」
```

病院に行くときに記録を見せれば、お医者さんとの会話もスムーズになります。

### 服薬リマインダー

```
あなた：「薬の飲み忘れを防ぐアプリを作って。
         ・朝・昼・夕・就寝前の4回の時間帯を設定
         ・飲んだらチェックできる
         ・飲み忘れがあったら表示してくれる
         ・今週の服薬状況が一覧できる」
```

### 歩数・運動記録

```
あなた：「毎日の歩数と散歩時間を記録するアプリ。
         ・手動で数値を入力する方式
         ・週間と月間のまとめが見られる
         ・目標歩数を設定してクリアしたら祝福メッセージ」
```

## AIへの指示のコツ

### コツ1：使う場面を伝える

```
△「リストアプリ作って」
◎「スーパーで買い物するとき使う
   リストアプリを作ってください」
```

### コツ2：使う人の特徴を伝える

```
「70代のシニア向けなので、
 文字は大きめ、ボタンも大きく、
 操作はシンプルにしてください」
```

### コツ3：少しずつ機能を追加する

最初はシンプルに作って、使いながら「こんな機能も欲しいな」と思ったら追加していく。この進め方が一番うまくいきます。

## よくある質問（FAQ）

### Q: 作ったアプリのデータはどこに保存されますか？

アプリの種類によりますが、スマートフォン上に保存されるものもあれば、クラウドに保存されるものもあります。AIに「データをクラウドに保存して」とお願いすれば対応してくれます。

### Q: 家族にアプリの使い方を教えるのが大変そうです

シンプルに作ることが大切です。「ボタンは3つ以内」「画面は1〜2画面」を心がけると、説明なしでも使えるアプリになります。

### Q: アプリの見た目を変えたいのですが

「色をピンクに変えて」「背景に桜の模様を入れて」「フォントをもっと丸い感じにして」——こんなふうにAIに伝えれば、デザインを変更してくれます。

## まとめ — あなたのアイデアが、誰かの「便利」になる

「こんなアプリがあったらいいのに」——その気持ちが、Rorkではそのまま形になります。

健康管理、家族のつながり、趣味の充実。あなたが作ったアプリは、あなた自身の暮らしを楽にするだけでなく、家族や地域の方にも喜ばれるかもしれません。
