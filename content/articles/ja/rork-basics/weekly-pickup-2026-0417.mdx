## 1位：Rorkアプリの環境変数が読み込めない原因と解決法

Rork で開発したアプリを実機テストしようとしたら `undefined` が返ってくる——そんな状況に直面した方は少なくないと思います。この記事では `EXPO_PUBLIC_` プレフィックスの付け忘れ、EAS Secrets の設定ミス、`.env` ファイルの置き場所の誤りという典型的な3パターンをStep by Stepで解説しています。

環境変数まわりのトラブルは情報が散らばっていて調べにくいので、この記事にまとめることにしました。迷ったときのリファレンスとして手元に置いておいていただけると嬉しいです。

```bash
# Expo で環境変数を正しく設定する例
# .env ファイル（プロジェクトルート直下）
EXPO_PUBLIC_API_URL=https://api.example.com
EXPO_PUBLIC_APP_NAME=MyApp

# コード内では以下でアクセス
const apiUrl = process.env.EXPO_PUBLIC_API_URL;
```
