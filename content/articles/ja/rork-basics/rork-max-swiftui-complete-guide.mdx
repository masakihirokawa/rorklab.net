## はじめに

Rork Maxが登場した時、多くの開発者は驚きました。なぜなら「プロンプトを書くだけでSwiftUIのネイティブiOSアプリが生成される」という、それまで不可能に近かった夢が現実になったからです。

実際、Rork Maxは**わずか3日で$1.5M ARR（年間経常収益）を達成**しました。この数字が示すのは、開発者やビジネスマンが「できればSwiftUIでネイティブアプリを作りたい」という強い需要を持っていたということ。

しかし「SwiftUIが生成される」というだけではなく、**Dynamic Island対応、Live Activities、HealthKit、ARKit、Metal 3D**など、Appleが提供する最新のネイティブAPIまで、ほぼ全て対応しているという点が本当にすごい。

このガイドでは、Rork MaxのSwiftUI生成機能を隅々まで解説し、あなたがプロフェッショナルなiOSアプリを短時間で完成させられるようにサポートします。

## Rork MaxのSwiftUI対応ネイティブAPI一覧

Rork Maxが対応しているネイティブiOS APIの全体像を、まずリスト形式で整理しましょう。これらは全て、自動でSwiftUIコードとして生成されます。

### UIと画面表示
- **Dynamic Island** — iPhoneの動的アイランド対応、リアルタイム情報表示
- **Live Activities** — ロック画面・動的アイランドでのアクティビティ表示
- **Widget** — ホーム画面・ロック画面ウィジェット（最大3サイズ対応）
- **App Clips** — 軽量アプリスライス機能

### ヘルス＆フィットネス
- **HealthKit** — 心拍数、歩数、運動量、睡眠データの読み書き
- **WorkoutKit** — ワークアウト認識・トラッキング
- **Vision Framework** — 顔認識、テキスト認識、オブジェクト検出

### スマートホーム＆IoT
- **HomeKit** — スマートホームデバイスの制御
- **NFC** — NFC読み取り・ペアリング

### AR・3D・ゲーム
- **ARKit** — 拡張現実アプリケーション
- **RealityKit** — 3Dレンダリング、AR シーン構築
- **Metal** — 高速GPU処理、3Dゲーム開発
- **LiDAR** — 深度計測、3Dスキャン

### 音声・AI
- **Speech Recognition** — 音声入力・文字変換
- **Siri Intents** — Siriショートカット連携

### シェアリング・通信
- **SharePlay** — 複数ユーザー間のリアルタイム共有
- **Messages** — iMessage App拡張
- **CallKit** — VoIP通話アプリ

### デバイスコントロール
- **Vision Pro** — Apple Vision Pro対応アプリ
- **Apple Watch** — Watch OSアプリ
- **Apple TV** — tvOSアプリ
- **Mac Catalyst** — Mac用ポート

### セキュリティ・決済
- **LocalAuthentication** — Face ID・Touch ID認証
- **Security Framework** — キーチェーン、暗号化
- **StoreKit 2** — アプリ内課金、サブスクリプション

### その他
- **PushKit** — プッシュ通知（VoIP、ファイル転送対応）
- **CloudKit** — iCloud同期
- **CoreData** — ローカルデータベース

このリスト全てが、Rork Maxのプロンプトで自動生成対応しています。

## 基本的な使い方：最初のSwiftUIアプリ作成

では実際に、Rork Maxでアプリを作ってみましょう。基本的なステップは非常にシンプルです。

### Step 1: Rork MaxのWebインターフェースを開く

Rork Maxの公式サイトにアクセスして、ログインします。無料トライアルまたはサブスクリプション（$200/月）でアクセス可能です。

### Step 2: 新規プロジェクトを作成

「Create New Project」をクリックして、以下の情報を入力します：

- **プロジェクト名**: 例「Weather App」
- **説明**: 「Show current weather, hourly forecast, 7-day forecast with SwiftUI」
- **プラットフォーム**: iOS / watchOS / tvOS を選択

### Step 3: AI プロンプトを入力

プロンプトの例：
```
Create a weather app using SwiftUI with:
- Current weather display using Location and Weather frameworks
- Hourly forecast with horizontal scroll
- 7-day forecast with custom charts
- Dark mode support
- Dynamic Island live activity for current temp
- Add HealthKit integration to show activity rings
```

### Step 4: SwiftUIコード生成開始

「Generate Code」をクリックすると、Rork Maxのクラウドコンパイラが数秒～数十秒でSwiftUIコードを生成します。

### Step 5: ブラウザシミュレータで確認

生成されたコードは、ブラウザ内のiPhoneシミュレータで即座に確認できます。レイアウト、アニメーション、ネイティブAPI呼び出しもシミュレータ内で動作します。

## クラウドMacでXcodeなしビルド：2クリック公開の仕組み

従来の開発では、Xcode（約12GB）をインストールして、ローカルマシンをコンパイルマシンとして使う必要がありました。しかしRork Maxは異なります。

### クラウドコンパイルの流れ

1. **プロンプト入力** → SwiftUIコード生成（ブラウザ内のAI）
2. **クラウドMac予約** → Rork Maxのバックエンド上の仮想Macを自動確保
3. **リモートビルド** → クラウドMac上でXcodeが自動実行、IPA生成
4. **QRコードダウンロード** → テスト用アプリをQRスキャンでインストール

### クラウドMacの具体的な流れ

Rork Maxダッシュボード上の「Build」ボタンをクリックすると：

1. Rork MaxのバックエンドがmacOS仮想マシンを起動（初回のみ数秒待ち）
2. 最新版Xcodeが自動インストール（初回のみ数分）
3. あなたのSwiftUIプロジェクトをその仮想Mac上に展開
4. `xcodebuild` コマンドで自動ビルド開始
5. IPA（iOS App Package）ファイルが生成される
6. IPAのダウンロードリンク＆QRコードがダッシュボードに表示

生成されたQRコードを、**開発者登録済みのiPhone で読み取る**と、ビルドしたばかりのアプリが数秒でインストールされます。

### TestFlightへの自動配布

さらに進むと、以下も自動化できます：

- **App Certificate & Provisioning Profile** の自動生成
- **TestFlight へのアップロード**（内部テスター・外部テスター対応）
- **App Store Connect との連携**

つまり、Rork Max上で「公開」ボタンを押すだけで、App StoreのTestFlightやApp Store本審査への自動提出まで実現できる、ということです。

## Rork Companion アプリ：リアル実機でのデバッグ

クラウドシミュレータだけでなく、**実際のiPhone・iPad上でのテスト**も必須です。そこで活躍するのが「Rork Companion」です。

### Rork Companionとは

Rork Companionは、App Storeで無料配布されているアプリ。あなたが作ったRork Max のアプリを、実機上で直接テストするために使用します。

### 使い方

1. iPhone/iPad上で「Rork Companion」アプリを起動
2. Rork MaxダッシュボードのプロジェクトページにQRコード生成ボタンがあるので、それをタップ
3. iPhone CompanionアプリでそのQRコードをスキャン
4. **数秒後、あなたが作ったアプリが実機上で起動**

リアル実機で以下が確認できます：

- **Dynamic Island アニメーション** — 実際の動き
- **HealthKit/HomeKit 連携** — デバイスとの通信
- **パフォーマンス** — 実際の処理速度
- **バッテリー消費** — 最適化の必要性判定

### TestFlightとの違い

| テスト方法 | メリット | デメリット |
|---|---|---|
| Rork Companion | 数秒で最新版を試せる、デバッグ情報を取得可 | 内部テストのみ |
| TestFlight | App Store審査に最も近い環境 | 毎回ビルド＆アップロード（数分） |
| App Store本番 | 完全に本番環境 | アップデート不可（却下されたら終わり） |

実際の開発フローは「Rork CompanionでQA → TestFlightで最終確認 → App Store提出」という流れになります。

## よくあるエラーと対処法

Rork Maxで開発していると、時々ビルドエラーが出ます。ここでは主なエラーと解決策を紹介します。

### エラー①：「Provisioning Profile not found」

**原因**: Apple Developer登録が不完全、または証明書が期限切れ

**解決法**:
1. Apple Developer アカウントにログイン
2. Certificates, Identifiers & Profiles で「iOS Development」証明書を確認
3. 有効期限が1年以内なら OK。期限切れなら新規作成
4. Rork Max設定で「Re-sync Apple credentials」を実行

### エラー②：「SwiftUI compilation failed」

**原因**: 古いSwiftUI構文を使用している、またはSDKバージョン不整合

**解決法**:
1. Rork Maxダッシュボード → Settings → iOS SDK Version で iOS 17.0 以上を選択
2. プロンプトで「Use latest SwiftUI 6 syntax」と明示
3. 生成されたコードをAI修正（「Fix SwiftUI compiler errors」ボタン）

### エラー③：「Build timeout」

**原因**: クラウドMacの一時的な混雑、またはコンパイルが長すぎる

**解決法**:
1. 数分待ってから「Retry Build」
2. 不要な機能を削除して再度生成
3. 複数のWidgetを同時生成していないか確認（Metal + Widget + ARKit は同時生成不可）

### エラー④：「App Store rejection: Missing privacy manifest」

**原因**: iOS 17.4以降の要件で、プライバシー宣言（NSPrivacyTracking等）がないと却下

**解決法**:
1. Rork Maxの「Privacy Configuration」パネルで必要なAPIをチェック
2. HealthKit使用 → NSHealthShareUsageDescription を設定
3. 位置情報使用 → NSLocationWhenInUseUsageDescription を設定
4. 自動で Privacy Manifest（PrivacyInfo.xcprivacy）を生成し、プロジェクトに追加

## FAQ

### Q: Rork Max の料金はいくらですか？

A: 月額 **$200/月**（年払い割引あり）で、無制限のプロジェクト作成・クラウドコンパイルが利用できます。追加のTestFlightビルドは追加費用なし。

### Q: SwiftUI以外のFramework（UIKit等）は対応していますか？

A: 現在のRork Maxは **SwiftUI専用** です。UIKitでの生成は現在未対応ですが、将来のロードマップに含まれているとのこと。

### Q: 生成されたコードは自分で編集できますか？

A: はい。ブラウザのコードエディタ（Monaco Editor搭載）で、生成後のSwiftUIコードを手動で編集できます。編集後「Re-compile」で再ビルド可能。

### Q: 複数プロジェクトの同時開発はできますか？

A: できます。Rork Maxダッシュボード上で複数プロジェクトを管理し、それぞれ独立してビルド・公開可能です。

### Q: Vision Pro用アプリの開発はどのくらい簡単ですか？

A: Rork Maxで「Generate Vision Pro app」を選択すると、SwiftUIのVision Pro API（RealityKit、Hand Tracking、Eye Tracking）に対応した3Dアプリが自動生成されます。VisionOS開発の複雑さが大幅に軽減されます。
