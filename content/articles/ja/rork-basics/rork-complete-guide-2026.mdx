## Rork とは — プロンプトでアプリを作る時代

Rork は AI の力を活用して、プロンプトだけでモバイルアプリを開発できるプラットフォームです。2026年現在、数万人のデベロッパーが Rork を使い、App Store や Google Play で公開されるアプリを毎月数千個作っています。

**従来の開発との違い：**

- **コード記述が不要**：Rork のチャットに「SNS のようなアプリを作って」と指示するだけ
- **自動コード生成**：AI が React Native または Swift（Rork Max）でコード生成
- **リアルタイム編集**：プレビューを見ながら修正・改善が可能
- **App Store 公開の簡略化**：Rork Max なら数クリックで公開

この完全ガイドでは、Rork の基本から応用まで、2026年の最新ワークフローを網羅します。

## 最初のアプリを作る — プロンプトからアプリ生成まで

### ステップ 1：Rork にサインアップ

Rork Lab の公式サイト（https://rorklab.net）から Google または GitHub アカウントでサインアップします。

### ステップ 2：新規プロジェクトを作成

ダッシュボードで「New Project」をクリックし、以下の情報を入力：

- **プロジェクト名**：例「MyWeatherApp」
- **プロジェクト説明**：簡潔に目的を説明
- **プラットフォーム**：React Native（Free/Pro）または Swift（Rork Max）を選択
- **テンプレート**：「Blank」または「Hello World」から開始

### ステップ 3：プロンプトを書く

以下はシンプルな天気アプリのプロンプト例：

```
作成してください：
- 画面上部に都市名を入力できる検索ボックス
- 現在の気温、天気アイコン、体感温度を表示
- 下部に 5 日間の天気予報（スクロール可能）
- カラースキーム：青とグレー
- API：OpenWeatherMap を使用
- 初期値：東京の天気
```

### ステップ 4：AI がコードを生成

Rork AI はプロンプトを解析し、約 5～15 秒で完全に動作するアプリのコードを生成します。React Native の場合、以下のような構造が自動生成されます：

```javascript
import React, { useState, useEffect } from 'react';
import {
  View,
  Text,
  TextInput,
  ScrollView,
  ActivityIndicator,
  StyleSheet,
} from 'react-native';
import axios from 'axios';

export default function WeatherApp() {
  const [city, setCity] = useState('Tokyo');
  const [weather, setWeather] = useState(null);
  const [loading, setLoading] = useState(false);
  const [forecast, setForecast] = useState([]);

  useEffect(() => {
    fetchWeather(city);
  }, []);

  const fetchWeather = async (cityName) => {
    setLoading(true);
    try {
      const apiKey = 'YOUR_OPENWEATHER_API_KEY';
      const response = await axios.get(
        `https://api.openweathermap.org/data/2.5/weather?q=${cityName}&appid=${apiKey}&units=metric`
      );
      setWeather(response.data);

      const forecastRes = await axios.get(
        `https://api.openweathermap.org/data/2.5/forecast?q=${cityName}&appid=${apiKey}&units=metric`
      );
      setForecast(forecastRes.data.list.slice(0, 5));
    } catch (error) {
      console.error('Weather fetch error:', error);
    } finally {
      setLoading(false);
    }
  };

  const handleSearch = () => {
    fetchWeather(city);
  };

  if (loading) {
    return (
      <View style={styles.center}>
        <ActivityIndicator size="large" color="#0066cc" />
      </View>
    );
  }

  return (
    <ScrollView style={styles.container}>
      <View style={styles.searchBox}>
        <TextInput
          style={styles.input}
          placeholder="都市名を入力..."
          value={city}
          onChangeText={setCity}
        />
      </View>

      {weather && (
        <View style={styles.weatherCard}>
          <Text style={styles.cityName}>{weather.name}</Text>
          <Text style={styles.temp}>{Math.round(weather.main.temp)}°C</Text>
          <Text style={styles.description}>{weather.weather[0].main}</Text>
          <Text style={styles.feelsLike}>
            体感温度: {Math.round(weather.main.feels_like)}°C
          </Text>
        </View>
      )}

      <View style={styles.forecastContainer}>
        <Text style={styles.forecastTitle}>5日間予報</Text>
        {forecast.map((item, index) => (
          <View key={index} style={styles.forecastItem}>
            <Text style={styles.forecastDate}>
              {new Date(item.dt * 1000).toLocaleDateString('ja-JP')}
            </Text>
            <Text style={styles.forecastTemp}>
              {Math.round(item.main.temp)}°C
            </Text>
          </View>
        ))}
      </View>
    </ScrollView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#f0f4f8',
    paddingHorizontal: 16,
    paddingVertical: 20,
  },
  center: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  searchBox: {
    marginBottom: 20,
  },
  input: {
    borderWidth: 1,

    borderRadius: 8,
    padding: 12,
    fontSize: 16,
    backgroundColor: '#fff',
  },
  weatherCard: {
    backgroundColor: '#0066cc',
    borderRadius: 12,
    padding: 24,
    marginBottom: 20,
    alignItems: 'center',
  },
  cityName: {
    fontSize: 28,

    color: '#fff',
    marginBottom: 8,
  },
  temp: {
    fontSize: 48,
    fontWeight: 'bold',
    color: '#fff',
  },
  description: {
    fontSize: 18,
    color: '#e0e0e0',
    marginTop: 8,
  },
  feelsLike: {
    fontSize: 14,
    color: '#b3d9ff',
    marginTop: 12,
  },
  forecastContainer: {
    marginTop: 16,
  },
  forecastTitle: {
    fontSize: 18,
    fontWeight: '600',
    marginBottom: 12,
    color: '#333',
  },
  forecastItem: {
    flexDirection: 'row',
```

## Rork Max — SwiftUI ネイティブとApp Store 2クリック公開

Rork Max は 2026年に導入された最高段階のプランで、iOS 開発の本格化を目指すデベロッパー向けです。

### SwiftUI ネイティブ対応

Rork Max では、React Native ではなく **SwiftUI で完全ネイティブなコードを生成** できます。これにより：

- パフォーマンス：50～80% の高速化
- App Store での審査通過率：大幅に向上
- Apple 標準フレームワーク（CoreData、HealthKit など）の活用が容易

```swift
import SwiftUI

struct ContentView: View {
  @State private var city = "Tokyo"
  @State private var weather: WeatherData?
  @State private var isLoading = false

  var body: some View {
    NavigationStack {
      VStack(spacing: 20) {
        HStack {
          TextField("都市名を入力", text: $city)
            .textFieldStyle(.roundedBorder)
            .keyboardType(.default)

          Button(action: fetchWeather) {
            Image(systemName: "magnifyingglass")
              .foregroundColor(.white)
              .frame(width: 44, height: 44)
              .background(Color.blue)
              .cornerRadius(8)
          }
        }
        .padding()

        if isLoading {
          ProgressView()
        } else if let weather = weather {
          VStack(alignment: .center, spacing: 12) {
            Text(weather.city)
              .font(.title)
              .fontWeight(.semibold)

            Text("\(Int(weather.temp))°C")
              .font(.system(size: 56, weight: .bold))
              .foregroundColor(.blue)

            Text(weather.description)
              .font(.body)
              .foregroundColor(.gray)
          }
          .frame(maxWidth: .infinity)
          .padding(24)
          .background(Color(UIColor.systemGray6))
          .cornerRadius(12)
        }

        Spacer()
      }
      .navigationTitle("天気アプリ")
    }
    .task {
      await fetchWeather()
    }
  }

  func fetchWeather() async {
    isLoading = true
    // API 呼び出しロジック
    isLoading = false
  }
}

struct WeatherData {
  let city: String
  let temp: Double
  let description: String
}
```

### App Store 2クリック公開

Rork Max では、以下のワークフローで数分で App Store に公開できます：

## バックエンド連携 — Supabase・Firebase・Stripe

### Supabase 連携

Supabase は PostgreSQL ベースのオープンソースバックエンドサービスです。Rork から直接統合できます：

```javascript
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = 'https://your-project.supabase.co';
const supabaseKey = 'your-anon-key';
const supabase = createClient(supabaseUrl, supabaseKey);

// ユーザー登録
const registerUser = async (email, password) => {
  const { user, error } = await supabase.auth.signUp({
    email,
    password,
  });
  return { user, error };
};

// データベース挿入
const addPost = async (userId, content) => {
  const { data, error } = await supabase
    .from('posts')
    .insert([{ user_id: userId, content, created_at: new Date() }]);
  return { data, error };
};

// リアルタイムリスニング
supabase
  .from('posts')
  .on('*', (payload) => {
    console.log('New post:', payload.new);
  })
  .subscribe();
```

### Firebase 連携

Google Firebase はプッシュ通知や実時間データベースに最適：

```javascript
import { initializeApp } from 'firebase/app';
import { getMessaging, getToken } from 'firebase/messaging';
import { getDatabase, ref, set, onValue } from 'firebase/database';

const firebaseConfig = {
  apiKey: 'YOUR_API_KEY',
  projectId: 'your-project-id',
};

const app = initializeApp(firebaseConfig);
const messaging = getMessaging(app);
const db = getDatabase(app);

// プッシュ通知トークン取得
const getNotificationToken = async () => {
  const token = await getToken(messaging);
  return token;
};

// リアルタイムデータベース監視
const listenToMessages = (chatRoomId) => {
  const messagesRef = ref(db, `chatRooms/${chatRoomId}/messages`);
  onValue(messagesRef, (snapshot) => {
    const data = snapshot.val();
    console.log('Messages:', data);
  });
};
```

### Stripe 統合

課金・決済機能の実装：

```javascript
import Stripe from 'stripe';

const stripe = new Stripe('sk_test_YOUR_SECRET_KEY');

// 決済インテント作成
const createPaymentIntent = async (amount, currency = 'jpy') => {
  const paymentIntent = await stripe.paymentIntents.create({
    amount,
    currency,
    payment_method_types: ['card'],
  });
  return paymentIntent;
};

// サブスクリプション作成
const createSubscription = async (customerId, priceId) => {
  const subscription = await stripe.subscriptions.create({
    customer: customerId,
    items: [{ price: priceId }],
  });
  return subscription;
};
```

## Rork vs Flutter vs React Native — 2026年の比較

| 項目 | Rork | Flutter | React Native |
|------|------|---------|--------------|
| **開発時間** | 最速（数日） | 中程度（数週間） | 中程度（数週間） |
| **学習曲線** | 最小（プロンプトのみ） | 中程度（Dart必須） | 中程度（JavaScript必須） |
| **iOS ネイティブ対応** | Rork Max で対応 | 部分的 | 制限あり |
| **パフォーマンス** | 良好（React Native/Swift） | 優秀 | 良好 |
| **カスタマイズ性** | 中程度 | 高い | 高い |
| **App Store 公開** | 2クリック（Max） | 複雑 | 複雑 |
| **コミュニティ** | 急速成長 | 大規模 | 非常に大規模 |
| **料金** | 月 $15～$49 | 無料 | 無料 |
| **用途** | MVP・スタートアップ | 本格アプリ | 本格アプリ |

**結論**：迅速な開発と App Store 公開を目指すなら **Rork**、高度なカスタマイズが必要なら **Flutter** または **React Native** が最適です。

## アプリ開発のプロンプトのコツ

効果的なプロンプトを書くことが、Rork で成功する鍵です。

### 1. 具体的な機能を列挙する

**良い例：**
```
以下の機能を含むタスク管理アプリを作成してください：
- ホーム画面：本日のタスク一覧（チェックボックス付き）
- 新規タスク追加画面：タイトル、説明、期限を入力可能
- カレンダー表示：月別・週別フィルタリング
- ストレージ：SQLite でローカル保存、iCloud 同期対応
```

**悪い例：**
```
タスク管理アプリを作って
```

### 2. デザインを詳細に指定する

```
UI カラースキーム：
- プライマリ：#0066cc（青）
- アクセント：#ff6b35（オレンジ）
- 背景：#f5f5f5（ライトグレー）

フォント：システムフォント (SF Pro / Roboto)、見出しは太字（700）
```

### 3. API・バックエンドを明確にする

```
バックエンド：
- 認証：Supabase Auth
- データベース：Supabase PostgreSQL
- リアルタイム更新：Supabase Realtime

画像ストレージ：Cloudinary または AWS S3
```

### 4. 段階的に改善する

初期プロンプトで基本機能を実装した後、細かい改善を重ねます：

1. 「プレビューを見たい」→ 基本機能生成
2. 「ボタンを大きくして」→ UI 調整
3. 「ダークモード対応」→ 機能追加
4. 「多言語対応（英語・日本語）」→ 国際化
