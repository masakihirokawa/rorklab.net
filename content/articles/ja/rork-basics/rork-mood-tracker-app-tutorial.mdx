「最近なんとなく調子が悪い」「先月は気持ちが安定していた気がする」——でも、何がその差を生み出しているのか、なかなか把握できないものです。

自分の感情を記録することで、睡眠・運動・天気・人間関係など、気分に影響する要因が見えてきます。今回はRorkを使って、**ムードトラッカーアプリ**を作るチュートリアルをお届けします。

## 今回作るアプリの概要

完成するアプリの機能：

- 5段階の気分スコアを毎日記録（最高〜最悪）
- 気分に合った絵文字アイコンで選びやすいUI
- 今日の気分に対してひとこと日記メモを追加
- 過去の記録をカレンダー形式で振り返り
- 週間・月間の気分グラフで傾向を把握

## Rorkでの開発手順

### ステップ1：プロジェクト作成

Rork（[rork.app](https://rork.app)）にログインして「新しいプロジェクト」を作成します。プロジェクト名は「MoodTracker」や「気分記録アプリ」としましょう。

### ステップ2：アプリの仕様をRorkに伝える

Rorkのチャット入力欄に以下のプロンプトを入力してください：

```
気分・感情を記録するムードトラッカーアプリを作ってください。

【必要な機能】
1. 今日の気分記録
   - 5段階の気分スコアを選択（1=最悪、2=悪い、3=普通、4=良い、5=最高）
   - 各スコアに対応する絵文字アイコン（😢😟😐🙂😊）を表示
   - ひとこと日記メモを自由入力できる（任意）
   - カテゴリタグを複数選択できる（仕事・家族・健康・趣味・その他）

2. 記録の履歴表示
   - カレンダービューで日ごとの気分を色分け表示
   - リストビューで最近30件の記録を表示
   - 各記録タップで詳細（スコア・メモ・タグ）を確認

3. 傾向グラフ
   - 直近7日間の気分スコアを折れ線グラフで表示
   - 月別の平均スコアを棒グラフで表示

4. データ保存
   - ローカルに全データを保存
   - アプリを閉じても記録が消えない

【デザイン】
- 落ち着いたパステルカラー（気持ちが和らぐ配色）
- 絵文字アイコンを大きく表示して選びやすいUI
- 1日1回の入力をストレスなく行えるシンプルさ
```

**今日の気分入力画面:**

```typescript
// app/(tabs)/index.tsx
import React, { useState } from 'react';
import {
  View, Text, TouchableOpacity, TextInput,
  StyleSheet, ScrollView, Alert
} from 'react-native';
import { useMoodStore } from '@/store/moodStore';
import { MOOD_CONFIG, CATEGORY_CONFIG } from '@/types/mood';
import type { MoodScore, MoodCategory } from '@/types/mood';
import { getTodayString } from '@/utils/dateUtils';

export default function TodayScreen() {
  const { entries, addEntry } = useMoodStore();
  const today = getTodayString();
  const todayEntry = entries.find(e => e.date === today);

  const [selectedScore, setSelectedScore] = useState<MoodScore | null>(null);
  const [note, setNote] = useState('');
  const [selectedCategories, setSelectedCategories] = useState<MoodCategory[]>([]);

  const toggleCategory = (cat: MoodCategory) => {
    setSelectedCategories(prev =>
      prev.includes(cat) ? prev.filter(c => c !== cat) : [...prev, cat]
    );
  };

  const handleSave = () => {
    if (!selectedScore) {
      Alert.alert('気分を選んでください');
      return;
    }
    addEntry({
      id: Date.now().toString(),
      date: today,
      score: selectedScore,
      note,
      categories: selectedCategories,
      createdAt: new Date().toISOString(),
    });
    Alert.alert('✅ 記録しました！');
  };

  if (todayEntry) {
    const config = MOOD_CONFIG[todayEntry.score];
    return (
      <View style={[styles.container, { backgroundColor: config.bgColor }]}>
        <Text style={styles.savedTitle}>今日の気分を記録済み</Text>
        <Text style={styles.savedEmoji}>{config.emoji}</Text>
        <Text style={[styles.savedLabel, { color: config.color }]}>
          {config.label}
        </Text>
        {todayEntry.note ? (
          <Text style={styles.savedNote}>「{todayEntry.note}」</Text>
        ) : null}
      </View>
    );
  }

  return (
    <ScrollView style={styles.container}>
      <Text style={styles.dateText}>{today}</Text>
      <Text style={styles.questionText}>今日の気分はどうですか？</Text>

      {/* 気分スコア選択 */}
      <View style={styles.moodRow}>
        {([1, 2, 3, 4, 5] as MoodScore[]).map(score => {
          const config = MOOD_CONFIG[score];
          const isSelected = selectedScore === score;
          return (
            <TouchableOpacity
              key={score}
              style={[
                styles.moodButton,
                { backgroundColor: isSelected ? config.color : '#F5F5F5' }
              ]}
              onPress={() => setSelectedScore(score)}
            >
              <Text style={styles.moodEmoji}>{config.emoji}</Text>
              <Text style={[
                styles.moodLabel,
                { color: isSelected ? '#fff' : '#666' }
              ]}>
                {config.label}
              </Text>
            </TouchableOpacity>
          );
        })}
      </View>

      {/* カテゴリ選択 */}
      <Text style={styles.sectionTitle}>今日の気分に関係すること（任意）</Text>
      <View style={styles.categoryRow}>
        {(Object.entries(CATEGORY_CONFIG) as [MoodCategory, { label: string; icon: string }][]).map(
          ([cat, { label, icon }]) => {
            const isSelected = selectedCategories.includes(cat);
            return (
              <TouchableOpacity
                key={cat}
                style={[styles.categoryChip, isSelected && styles.categoryChipSelected]}
                onPress={() => toggleCategory(cat)}
              >
                <Text>{icon} {label}</Text>
              </TouchableOpacity>
            );
          }
        )}
      </View>

      {/* メモ入力 */}
      <Text style={styles.sectionTitle}>ひとこと日記（任意）</Text>
      <TextInput
        style={styles.noteInput}
        placeholder="今日あったことや感じたことを..."
        multiline
        value={note}
        onChangeText={setNote}
        maxLength={200}
      />

      <TouchableOpacity
        style={[styles.saveButton, !selectedScore && styles.saveButtonDisabled]}
        onPress={handleSave}
        disabled={!selectedScore}
      >
        <Text style={styles.saveButtonText}>記録する</Text>
      </TouchableOpacity>
    </ScrollView>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1, backgroundColor: '#FAFAFA', padding: 20 },
  dateText: { fontSize: 14, color: '#888', marginBottom: 4 },
  questionText: { fontSize: 22, fontWeight: '700', color: '#1a1a1a', marginBottom: 20 },
  moodRow: { flexDirection: 'row', justifyContent: 'space-between', marginBottom: 24 },
  moodButton: {
    flex: 1, marginHorizontal: 4, borderRadius: 12, padding: 10,
    alignItems: 'center',
  },
  moodEmoji: { fontSize: 28, marginBottom: 4 },
  moodLabel: { fontSize: 11, fontWeight: '600' },
  sectionTitle: { fontSize: 15, fontWeight: '600', color: '#333', marginBottom: 10 },
  categoryRow: { flexDirection: 'row', flexWrap: 'wrap', gap: 8, marginBottom: 24 },
  categoryChip: {
    paddingHorizontal: 14, paddingVertical: 8, borderRadius: 20,
    backgroundColor: '#F0F0F0',
  },
  categoryChipSelected: { backgroundColor: '#C8E6C9' },
  noteInput: {
    backgroundColor: '#fff', borderRadius: 12, padding: 14,
    minHeight: 80, fontSize: 15, textAlignVertical: 'top',
    borderWidth: 1, borderColor: '#E0E0E0', marginBottom: 24,
  },
  saveButton: {
    backgroundColor: '#42A5F5', borderRadius: 14, padding: 16, alignItems: 'center',
  },
  saveButtonDisabled: { backgroundColor: '#BDBDBD' },
  saveButtonText: { color: '#fff', fontSize: 17, fontWeight: '700' },
  savedTitle: { fontSize: 18, color: '#555', textAlign: 'center', marginBottom: 16 },
  savedEmoji: { fontSize: 72, textAlign: 'center', marginBottom: 12 },
  savedLabel: { fontSize: 24, fontWeight: '700', textAlign: 'center', marginBottom: 8 },
  savedNote: { fontSize: 16, color: '#666', textAlign: 'center', fontStyle: 'italic' },
});
```

## よくある質問（FAQ）

**Q1. 毎日の入力が二重に記録されるのを防ぐにはどうすればいいですか？**

本チュートリアルのコード例では、既に当日の記録が存在する場合は「記録済み」画面を表示する設計にしています。ストアの `addEntry` 関数内で「同じ日付の記録がある場合は上書きする」という処理を追加することもできます。Rorkに「同じ日に記録した場合は上書きする仕様にしてください」と伝えれば実装してくれます。

**Q2. 気分スコアの粒度を5段階から10段階に変えたいです。**

`MoodScore` 型と `MOOD_CONFIG` の定義を変更することで対応できます。Rorkのチャットで「気分スコアを10段階に変更し、UIもそれに合わせて更新してください」と伝えると、型・設定・UI全体をまとめて修正してくれます。

**Q3. 複数人で使えるよう、ユーザー認証を追加できますか？**

Rorkは Supabase や Firebase との連携をサポートしています。「Supabase Auth でメールアドレス認証を追加し、気分データをユーザーごとにクラウド保存してください」と伝えると、認証フローとデータベース設計をまとめて実装してくれます。
