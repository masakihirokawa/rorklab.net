## AI アプリ開発ツール

### Rork / Rork Max — AI モバイルアプリ開発

**概要**

Rork は AI による自動モバイルアプリ開発プラットフォームです。自然言語での説明から、iOS・Android アプリを自動生成できます。Rork Max はより高度な機能を提供し、複雑なアプリ構造の実装に対応しています。

**用途**

- iOS・Android アプリの自動生成
- プロトタイピングの高速化
- MVP（最小成功版）開発の加速
- アプリ UI・UX の自動デザイン
- レガシーアプリのモダナイゼーション

**料金**

- Rork Free：無料（基本機能、月額限定）
- Rork Pro：月額$29 ～
- Rork Max：月額$99 ～

**Rork との連携ポイント**

Rork が基盤となるため、本記事で紹介する他の AI ツール・API は Rork で生成されたアプリに組み込まれます。Rork が生成したコードに対して、Claude Code や Cursor で追加実装・カスタマイズを加えることが一般的です。

### Antigravity — AI コーディングエディタ

**概要**

Antigravity は Google が開発した AI コーディングエディタで、Python、JavaScript、Kotlin などの言語に対応しています。モバイル開発では、Kotlin（Android）や Swift（iOS）の補完にも対応しており、Rork 生成コードの改善に活用できます。

**用途**

- Kotlin / Swift コード生成・補完
- API 統合コードの自動生成
- バグ修正・リファクタリング
- テストコード自動生成
- ネイティブモバイル開発の加速

**料金**

- Antigravity：一部無料機能
- Premium プラン：詳細は Google 公式サイト参照

**Rork との連携ポイント**

Rork で生成された Swift / Kotlin コードを Antigravity で最適化・拡張することで、より高度なネイティブ機能の組み込みが容易になります。ネイティブ API の統合、パフォーマンス最適化などに活用できます。

**関連記事**

- [ネイティブ API 統合実装](/)

### Claude Code — ターミナル AI コーディング

**概要**

Claude Code は Anthropic の AI コーディングアシスタントで、ターミナル・CLI での実装に特に強みがあります。複雑なバックエンド実装、API サーバー構築、スクリプト生成などに最適です。

**用途**

- バックエンド API サーバー実装
- Node.js / Python スクリプト
- データベーススキーマ設計
- 複雑なビジネスロジック実装
- DevOps スクリプト

**料金**

- Claude Pro：月額20ドル
- Claude Code：Pro 加入時に利用可能
- Claude API 経由：従量課金

**Rork との連携ポイント**

Rork でモバイルアプリのフロントエンドを開発し、Claude Code でバックエンド API サーバーを実装することで、完全なモバイルアプリケーション開発が実現できます。

**関連記事**

- [Claude Code でバックエンド API 実装](/)
- [Rork ×バックエンド統合開発](/)

### Cursor — AI コード補完エディタ

**概要**

Cursor は VS Code ベースの AI 統合エディタで、複数の AI モデル対応、リアルタイム補完、マルチファイル編集などが特徴です。モバイルアプリ開発の付帯実装（サーバー、スクリプト）に適しています。

**用途**

- マルチファイル編集・リファクタリング
- バックエンド実装
- 自動テストコード生成
- CI/CD スクリプト作成
- デザイン資産処理スクリプト

**料金**

- Cursor Free：無料（月間制限あり）
- Cursor Pro：月額20ドル

**Rork との連携ポイント**

Rork で生成されたアプリに対して、Cursor で付帯的なバックエンド処理、ユーティリティスクリプトなどを追加実装できます。Rork と Cursor の組み合わせにより、フルスタック開発が効率化されます。

## AI アシスタント

### Claude AI

**概要**

Claude は Anthropic の高性能 AI チャットボットで、安全性と信頼性を重視した設計です。複雑な推論、長文分析、創作に定評があり、モバイルアプリ開発のコンサルティングに最適です。

**用途**

- アプリ企画・仕様検討
- API 仕様設計相談
- ユーザーフロー設計
- バグ原因分析
- 技術課題の相談

**料金**

- Claude Free：無料
- Claude Pro：月額20ドル
- Claude API：従量課金

**Rork との連携ポイント**

Rork でアプリ実装前に Claude と企画を詰めることで、より精度の高い実装指示を Rork に与えられます。アプリ完成後のトラブルシューティングも Claude が得意です。

**関連記事**

- [Claude でアプリ企画を詰める](/)

### Gemini

**概要**

Gemini は Google の高性能対話型 AI で、高速処理、マルチモーダル対応が特徴です。画像解析、テキスト処理、コード生成に幅広く対応します。

**用途**

- アプリのクイック実装提案
- UI・UX アイデア出し
- 市場リサーチ・競合分析
- マーケティング施策検討
- テストシナリオ生成

**料金**

- Gemini Free：無料
- Gemini Advanced：月額2,000円
- Gemini API：従量課金

**Rork との連携ポイント**

Gemini で高速にアイデア出し・提案をしながら、詳細な実装は Rork で進めるアプローチが効果的です。マルチターン対話により、反復的にアプリ仕様を洗練できます。

**関連記事**

- [Gemini でアプリプロトタイピング](/)

### ChatGPT

**概要**

ChatGPT は OpenAI の最も利用者が多い AI アシスタントで、自然な会話、初心者向けの説明が特徴です。幅広いユースケースに対応し、学習リソースも豊富です。

**用途**

- 初心者向けの技術学習
- アプリ開発の一般的な相談
- 文章・ドキュメント作成
- デバッグのアイデア出し
- 価格・ビジネス相談

**料金**

- ChatGPT Free：無料（制限あり）
- ChatGPT Plus：月額20ドル
- ChatGPT API：従量課金

**Rork との連携ポイント**

初心者向けの相談、学習リソース検索などは ChatGPT が最適です。Rork 導入初期段階での基礎知識習得に有効です。

## デザイン・UI ツール

### Figma / Figma Make

**概要**

Figma はクラウドベースのデザインツールで、業界標準です。Figma Make は新しい機能で、AI による自動デザイン生成、デザイン改善提案などが可能です。モバイルアプリ UI デザインに最適です。

**用途**

- モバイルアプリ UI デザイン
- プロトタイピング
- デザインシステム管理
- チーム協働設計
- AI による自動デザイン生成

**料金**

- Figma Free：無料（基本機能）
- Figma Professional：月額12ドル
- Figma Make：Premium 機能

**Rork との連携ポイント**

Figma で詳細な UI デザインを先に作成し、その仕様から Rork でアプリ実装することで、より正確な UI 実装が実現できます。デザイン仕様の共有が容易で、チーム開発に適しています。

**関連記事**

- [Figma でモバイルアプリ UI デザイン](/)
- [Figma デザインから Rork での実装](/)

### Google Stitch

**概要**

Google Stitch は Google の AI デザインツールで、テキスト説明から UI デザインを自動生成します。モバイルアプリ開発では、プロトタイピング段階での UI 生成に活用できます。

**用途**

- テキストから UI デザイン自動生成
- デザイン案のラピッドプロトタイピング
- アクセシビリティを考慮した UI 生成
- モバイル・ウェブ対応デザイン
- デザインバリエーション生成

**料金**

- Google Stitch：詳細は Google 公式発表待ち
- 無料版・有料版両立予定

**Rork との連携ポイント**

Google Stitch で生成したデザインを Figma にエクスポートし、Rork に連携することで、テキスト → AI UI → アプリ実装の完全自動化パイプラインが実現できます。

**関連記事**

UI Pro Max は AI 駆動のデザイン支援ツールで、デザインシステム管理、自動レイアウト最適化、アクセシビリティチェックなどを提供します。モバイルアプリの高度な UI 管理に活用できます。

**用途**

- デザインシステム管理
- 自動レイアウト調整
- アクセシビリティ対応チェック

**Rork との連携ポイント**

## AI コンテンツ制作

### Pollo AI — アプリ紹介動画の自動生成

**概要**

Pollo AI は AI 動画生成ツールで、テキスト説明から短編動画を自動生成します。モバイルアプリの紹介動画、マーケティング動画の制作に最適です。

**用途**

- アプリ紹介動画の自動生成
- App Store / Google Play 用プレビュー動画
- SNS マーケティング動画
- ユーザーチュートリアル動画
- アプリ更新情報の動画化

**料金**

- Pollo AI Free：無料（制限あり）
- Pollo AI Pro：月額25ドル

**Rork との連携ポイント**

Rork で完成したアプリの機能を説明してから、Pollo AI でアプリ紹介動画を生成することで、マーケティング資料の作成が効率化されます。App Store・Google Play での視認性向上に直結します。

**関連記事**

- [Pollo AI でアプリ紹介動画作成](/)
- [App Store 最適化とアプリ紹介動画](/)

### Suno AI — BGM 生成

**概要**

Suno AI はテキストプロンプトから音楽を自動生成するツールです。アプリ内 BGM、紹介動画の背景音楽、ポッドキャスト用 BGM などに活用できます。著作権もユーザーに帰属するため、ビジネス利用も安心です。

**用途**

- アプリ内背景音楽（BGM）
- 紹介動画の背景音楽
- ゲーム・教育アプリの効果音
- ポッドキャスト BGM
- アプリプロモーション動画音楽

**料金**

- Suno AI Free：無料（毎月 50 クレジット）
- Suno AI Pro：月額12ドル（毎月 500 クレジット）
- Suno AI Premium：月額24ドル（毎月 1000 クレジット）

**Rork との連携ポイント**

Suno AI で生成した BGM を Rork アプリに組み込むことで、プロフェッショナルな音響体験が実現できます。アプリのブランディング、ユーザー体験向上に直結します。
