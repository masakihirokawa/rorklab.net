## はじめに：RorkでAI学習アプリを作ろう

語学学習・資格勉強・受験対策——スマホで使えるフラッシュカードアプリは、いつの時代も根強い需要があります。AnkiやQuizletのような学習アプリは世界中で何百万人ものユーザーに利用されており、個人開発者が参入する余地も十分あります。

このガイドでは、**Rork**を使ってAI搭載のフラッシュカード学習アプリをゼロから作る方法を解説します。コードを書く知識がなくても、AIと対話するだけでiOS・Android両対応のReact Nativeアプリが完成します。

このチュートリアルを読み終えると、以下のことができるようになります。

- Rorkでフラッシュカードアプリの基本構造を作成する
- カードの追加・編集・削除機能を実装する
- クイズモード（正解・不正解フィードバック）を組み込む
- 学習進捗をローカルに保存する
- AIを使ってカードを自動生成するプロンプトを活用する

## Step 1：Rorkでプロジェクトを作成する

[rork.com](https://rork.com) にアクセスし、ログイン後に「New Project」をクリックします。

最初のプロンプトで、以下のように入力してみましょう。

```
フラッシュカード学習アプリを作ってください。
機能：
- デッキ（カードのセット）を複数作成・管理できる
- 各カードに「表面（問題）」と「裏面（答え）」を持つ
- タップでカードをめくるアニメーション
- クイズモード：ランダムにカードが表示され「正解」「不正解」を記録
- 学習進捗（正解率）を表示するホーム画面
デザイン：シンプルでクリーンなUI、ダークモード対応
```

Rorkはこのプロンプトを解析し、React NativeとExpoを使ったプロジェクトの雛形を自動生成します。数秒〜数分でプレビューが表示されます。

## Step 3：カードめくりアニメーションを実装する

フラッシュカードアプリの「顔」となるのが、カードをめくるアニメーションです。Rorkに以下のように指示します。

```
カードめくりのアニメーションを実装してください。
- タップするとカードが水平方向に180度回転（flip）する
- 表面：問題テキスト（青背景）
- 裏面：答えテキスト（緑背景）
- Animated APIを使用し、パフォーマンスを最適化する
```

Rorkが生成するコードの例は以下のとおりです。

```typescript
// FlashCard.tsx — カードめくりコンポーネント
import React, { useState, useRef } from 'react';
import { View, Text, TouchableOpacity, Animated, StyleSheet } from 'react-native';

interface FlashCardProps {
  front: string;  // 問題テキスト
  back: string;   // 答えテキスト
}

export const FlashCard: React.FC<FlashCardProps> = ({ front, back }) => {
  const [isFlipped, setIsFlipped] = useState(false);
  const flipAnim = useRef(new Animated.Value(0)).current;

  // カードをめくる処理
  const flipCard = () => {
    Animated.spring(flipAnim, {
      toValue: isFlipped ? 0 : 1,
      friction: 8,
      tension: 10,
      useNativeDriver: true,
    }).start();
    setIsFlipped(!isFlipped);
  };

  // 表面・裏面それぞれの回転角度を計算
  const frontInterpolate = flipAnim.interpolate({
    inputRange: [0, 1],
    outputRange: ['0deg', '180deg'],
  });
  const backInterpolate = flipAnim.interpolate({
    inputRange: [0, 1],
    outputRange: ['180deg', '360deg'],
  });

  return (
    <TouchableOpacity onPress={flipCard} activeOpacity={1}>
      {/* 表面 */}
      <Animated.View style={[styles.card, styles.front, { transform: [{ rotateY: frontInterpolate }] }]}>
        <Text style={styles.cardText}>{front}</Text>
        <Text style={styles.hint}>タップしてめくる</Text>
      </Animated.View>
      {/* 裏面 */}
      <Animated.View style={[styles.card, styles.back, { transform: [{ rotateY: backInterpolate }] }]}>
        <Text style={styles.cardText}>{back}</Text>
      </Animated.View>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  card: {
    width: 320,
    height: 200,
    borderRadius: 16,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 24,
    backfaceVisibility: 'hidden',  // めくり前の面を非表示にする
    position: 'absolute',
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 4 },
    shadowOpacity: 0.15,
    shadowRadius: 8,
    elevation: 5,
  },
  front: { backgroundColor: '#3B82F6' },  // 青：問題面
  back:  { backgroundColor: '#10B981' },  // 緑：答え面
  cardText: { fontSize: 22, fontWeight: '700', color: '#fff', textAlign: 'center' },
  hint:     { fontSize: 12, color: 'rgba(255,255,255,0.7)', marginTop: 12 },
});

// 期待する出力：
// - 表面（青）: "apple" と表示
// - タップ後に180度回転
// - 裏面（緑）: "りんご" と表示
```

## Step 4：学習進捗をローカルに保存する

AsyncStorageを使って、学習進捗（正解数・不正解数）をデバイスに保存します。Rorkに以下のように指示します。

```
AsyncStorageを使って各カードの学習記録（正解数・不正解数・最終学習日）を
保存・読み込みする機能を追加してください。
```

```typescript
// storage.ts — 学習記録の保存・読み込みユーティリティ
import AsyncStorage from '@react-native-async-storage/async-storage';

interface CardProgress {
  cardId: string;
  correct: number;     // 正解回数
  incorrect: number;   // 不正解回数
  lastStudied: string; // ISO 8601 形式の日時
}

// 特定カードの進捗を保存
export const saveCardProgress = async (progress: CardProgress): Promise<void> => {
  const key = `progress_${progress.cardId}`;
  await AsyncStorage.setItem(key, JSON.stringify(progress));
};

// 全カードの進捗を読み込み
export const loadAllProgress = async (): Promise<Record<string, CardProgress>> => {
  const keys = await AsyncStorage.getAllKeys();
  const progressKeys = keys.filter(k => k.startsWith('progress_'));
  const pairs = await AsyncStorage.multiGet(progressKeys);

  return pairs.reduce((acc, [key, value]) => {
    if (value) {
      const cardId = key.replace('progress_', '');
      acc[cardId] = JSON.parse(value);
    }
    return acc;
  }, {} as Record<string, CardProgress>);
};

// 正解率を計算するヘルパー
export const calculateAccuracy = (correct: number, incorrect: number): number => {
  const total = correct + incorrect;
  if (total === 0) return 0;
  return Math.round((correct / total) * 100); // 例: 8/10 → 80
};
```

## Step 5：AIによるカード自動生成

Rorkの最大の強みは、AIを活用した拡張です。例えば、テーマを入力するだけでカードを自動生成する機能を追加できます。

```
OpenAI APIを使ってカード自動生成機能を追加してください。
ユーザーがテーマ（例：「英語の前置詞」）を入力すると、
そのテーマに関するフラッシュカード10枚を自動生成します。
レスポンスはJSON配列形式で [{front: "...", back: "..."}] とします。
```

```typescript
// autoGenerateCards.ts — OpenAI を使ったカード自動生成
const OPENAI_API_KEY = process.env.EXPO_PUBLIC_OPENAI_API_KEY || '';

interface GeneratedCard {
  front: string;
  back: string;
}

export const generateFlashcards = async (topic: string): Promise<GeneratedCard[]> => {
  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Authorization: `Bearer ${OPENAI_API_KEY}`,
    },
    body: JSON.stringify({
      model: 'gpt-4o-mini',
      messages: [
        {
          role: 'system',
          content: 'You are a helpful tutor. Generate flashcards in JSON array format: [{front: "question", back: "answer"}]',
        },
        {
          role: 'user',
          content: `Generate 10 flashcards about the topic: "${topic}". Return only valid JSON.`,
        },
      ],
      temperature: 0.7,
    }),
  });

  const data = await response.json();
  const content = data.choices[0].message.content;

  // JSONを抽出してパース（コードブロックが含まれる場合も考慮）
  const jsonMatch = content.match(/\[[\s\S]*\]/);
  return jsonMatch ? JSON.parse(jsonMatch[0]) : [];
};

// 期待する出力（topic: "英語の前置詞"）:
// [
//   { front: "at", back: "〜で（場所・時間の一点）" },
//   { front: "in",  back: "〜の中に（空間・期間）" },
//   ...
// ]
```

## Step 6：よくあるエラーと対処法

### エラー1：カードがめくれない

**原因**：`backfaceVisibility: 'hidden'` が正しく適用されていない。
**対処**：Rorkに「`backfaceVisibility` の適用がおかしい場合は `overflow: 'hidden'` をカードコンテナに追加してください」と指示する。

### エラー2：AsyncStorageのデータが消える

**原因**：Expo Goのキャッシュクリアやアプリ再インストール後はデータが削除される。
**対処**：本番環境ではSupabaseやFirebaseなどのクラウドDBへのデータ同期を検討する。

### エラー3：OpenAI APIの呼び出しがブロックされる

**原因**：APIキーが正しく設定されていない、またはCORSエラー。
**対処**：`EXPO_PUBLIC_OPENAI_API_KEY` が `.env` ファイルに記載されているか確認する。本番環境ではAPIキーをバックエンド（Cloudflare Workersなど）経由で呼び出す。

## Step 7：応用・発展的な活用法

### 間隔反復（SRS）アルゴリズムの導入

AnkiのようなSRS（Spaced Repetition System）を実装することで、記憶定着率を最大化できます。Rorkに以下のように依頼してください。

```
SM-2アルゴリズムを実装してください。
各カードに「次回学習日」を設定し、正解なら間隔を延ばし、
不正解なら間隔をリセットして翌日に再出題します。
```

### 音声読み上げ（Text-to-Speech）対応

React Nativeの`expo-speech`を使って、カードの答えを音声で読み上げる機能を追加できます。語学学習アプリとして差別化する有効な機能です。

### App Storeへの公開

アプリが完成したら、RorkのExportボタンからApp Store Connect経由でiOSアプリとして公開できます。詳細は App Store公開ガイド を参照してください。

## FAQ

**Q. Rorkの無料プランでフラッシュカードアプリは作れますか？**
A. 基本機能（カードめくり・クイズモード・ローカル保存）は無料プランで作れます。AIによるカード自動生成などの拡張機能はAPI費用がかかりますが、Rork自体のプランはJunior（$25/月）以上あれば十分です。

**Q. Androidにも対応できますか？**
A. はい。RorkはReact Native（Expo）を使っているため、iOSとAndroidの両方に対応したアプリが自動的に生成されます。コードを書き換える必要はありません。

**Q. Supabaseと連携してカードをクラウドに保存できますか？**
A. できます。Rorkに「Supabaseのテーブル設計とAPI連携コードを追加して」と指示するだけです。詳しくは [RorkとSupabaseでチャットアプリを作る](/articles/rork-dev/rork-supabase-chat-app) を参照してください。

**Q. 生成されたコードを自分でカスタマイズできますか？**
A. はい。RorkにはGitHub連携機能があり、生成したReact Nativeコードをそのまま自分のリポジトリにエクスポートできます。詳しくは [RorkとGitHubの連携ガイド](/articles/rork-dev/rork-github-integration) をご覧ください。
