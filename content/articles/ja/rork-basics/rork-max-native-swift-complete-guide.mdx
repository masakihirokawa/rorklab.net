# Rork Max 完全ガイド — Xcode 不要でネイティブ Swift アプリを構築する

Rork Max は、AI プロンプトだけで **ネイティブ Swift アプリを自動生成できる**開発プラットフォームです。従来は Xcode で複雑な設定を行う必要がありましたが、Rork Max ならその手間をすべて削減できます。

このガイドでは、Rork Max の仕組み、対応プラットフォーム、元の Rork（React Native）との違い、そして最初のステップまで、初心者向けに詳しく解説します。

## Rork Max とは

Rork Max は、**AI が Swift コードを自動生成するノーコード/ローコード開発プラットフォーム**です。テキストプロンプトを入力するだけで、以下の特徴を備えたネイティブアプリが完成します。

- **100% ネイティブ実装**: Swift と SwiftUI で実装され、App Store への公開も可能
- **AI による自動生成**: Rork Max のプロンプト入力で、複雑なロジックも一瞬で実装
- **Apple 全プラットフォーム対応**: iPhone、iPad、Apple Watch、Apple TV、Vision Pro をサポート
- **リアルタイムプレビュー**: 変更をすぐに確認できるシミュレータ環境

## 対応プラットフォーム

Rork Max で開発したアプリは、以下の Apple デバイスすべてで動作します。

### iPhone
最も一般的なプラットフォーム。画面サイズ自動調整により、すべての iPhone モデルに対応します。

```
ターゲット: iOS 14.0 以上
デバイス: iPhone 12, 13, 14, 15 シリーズ
画面サイズ: 5.4" ～ 6.9"
```

### iPad
大画面に最適化されたレイアウト。Split View や Slide Over の実装も AI が自動サポート。

```
ターゲット: iPadOS 14.0 以上
デバイス: iPad Pro 11-inch, 12.9-inch / iPad Air / iPad (無印)
マルチタスク: Split View、Slide Over 対応
```

### Apple Watch
ウォッチフェイスに常時表示される情報、心拍計データ取得、Siri Shortcuts の連携まで AI が実装。

```
ターゲット: watchOS 7.0 以上
画面サイズ: 38mm, 41mm, 42mm
バッテリー最適化: AI が自動最適化
```

### Apple TV
大画面での視聴体験。Siri リモコンでの操作 UI も自動生成。

```
ターゲット: tvOS 15.0 以上
解像度: 1080p、4K 対応
Siri 統合: 音声コマンド可能
```

### Vision Pro
空間コンピューティングの時代へ。3D UI、Hand Tracking、Spatial Audio の実装が AI で簡単に。

```
ターゲット: visionOS 1.0 以上
入力: Hand Tracking, Eye Tracking, Voice Commands
グラフィックス: Metal 3D レンダリング対応
```

## Rork Max vs 元の Rork（React Native の違い）

Rork Max は、元の Rork（React Native ベース）の後継です。両者の主な違いを理解しておくと、プロジェクト選定がスムーズです。

| 特徴 | Rork Max | 元の Rork |
|------|---------|----------|
| **実装言語** | Swift（ネイティブ） | JavaScript（React Native） |
| **対応 OS** | iOS、watchOS、tvOS、visionOS | iOS、Android |
| **パフォーマンス** | ネイティブ级（最高速） | 仲介層経由（中速） |
| **App Store 公開** | ✓ 対応 | ✓ 対応（ブリッジ経由） |
| **ネイティブ API 利用** | ✓ 全て利用可 | △ 限定的 |
| **AR/3D 機能** | ✓ Metal、ARKit 対応 | △ 制限あり |
| **デバイス固有機能** | ✓ HealthKit、HomeKit など | △ 対応プラグイン必要 |
| **学習曲線** | 初心者向け | 初心者向け |
| **コード修正** | Swift 知識で直接編集可 | JavaScript 修正 |

**結論**: Apple デバイスのネイティブアプリを公開する場合は **Rork Max** 一択。Android も対応したい場合は、元の Rork を選択します。

## Rork Max での開発ステップ

### ステップ 1: プロジェクト作成

Rork Lab のダッシュボードにログイン → 「新規プロジェクト」を選択 → 「Rork Max」を選択。プロジェクト名と説明を入力すれば、自動で開発環境が立ち上がります。

```
プロジェクト名: 「My First Swift App」
説明: 「シンプルな TODOリスト」
対象デバイス: 「iPhone」を選択
```

### ステップ 2: アプリ概要をプロンプトで説明

AI に、作りたいアプリの概要を自然言語で説明します。

```
「シンプルな TODOリストアプリを作ってください。
- タスク追加ボタン
- タスク一覧を表示（完了・削除機能付き）
- タスク完了時にチェックマーク表示
- ローカル保存対応」
```

AI がこれを読み、ネイティブ Swift UI を自動生成します。

### ステップ 3: リアルタイムプレビューで動作確認

Rork Max のシミュレータで、生成されたアプリをその場でテスト。プロンプトを修正すると、画面がリアルタイムで更新されます。

### ステップ 4: ネイティブ機能を追加

必要に応じて、HealthKit（歩数取得）、Core Location（位置情報）、Push Notification など、ネイティブ iOS API を追加。プロンプトで「心拍計データを表示したい」と伝えるだけで、Rork Max が HealthKit 連携コードを生成します。

### ステップ 5: App Store に公開

Rork Max が生成した Swift コードは 100% ネイティブなため、App Store への公開も通常通り。Apple の審査も問題なく通ります。

## チュートリアル：初心者向け最初の 10 分

### 1. ダッシュボードへログイン（1 分）

[Rork Lab 公式ページ](https://rorklab.net) から「ダッシュボード」をクリック。Google アカウント、Apple ID、メールアドレスから選んでログイン。

### 2. 新規プロジェクト作成（2 分）

「+ 新規プロジェクト」 → 「Rork Max」選択 → プロジェクト名入力 → 「作成」ボタン。

### 3. プロンプト入力（3 分）

エディタが開いたら、作りたいアプリを説明：

```
「音声メモアプリを作ってください。
- マイクボタンで録音開始/停止
- 録音済みメモをリスト表示
- 各メモをタップして再生
- メモを削除可能」
```

### 4. リアルタイムプレビュー確認（3 分）

右側のシミュレータ画面で、自動生成されたアプリが動作。「マイクボタンが小さい」などの修正はプロンプトで指示するだけで、即座に更新。

### 5. 公開準備（1 分以内）

「公開する」ボタンでエクスポート。App Store への登録手続きへ進みます。

## Rork Max 利用時の注意点

### メモリ効率
Vision Pro 向けで 3D UI を多用する場合、Metal レンダリングは自動最適化されますが、複雑な AR 処理が多い場合は Rork Max の「パフォーマンス最適化」オプションを有効化することをお勧めします。

### Apple 審査対応
HealthKit など個人情報を扱う API を使用する場合、プライバシーポリシーが必須。Rork Max は自動生成テンプレートを提供しています。

### バージョン管理
Rork Max で生成されたコードは GitHub に自動同期可能。チームでの開発はこの機能を活用してください。

## まとめ

Rork Max を使えば、**Xcode の複雑な設定を気にせず、プロンプト一つで ネイティブ Swift アプリを開発できます**。iPhone から Vision Pro まで、Apple 全プラットフォーム対応。初心者でも、複雑な AR・3D 機能でも、AI の力で実装が加速します。

次のステップは、実際にダッシュボードで「Hello World」からプロジェクトをスタートしてみること。10 分で最初のアプリが動作する体験が、開発の未来を変えるはずです。
