## API・SDK プラットフォーム

### Gemini API — モバイルアプリへの AI 統合

**概要**

Gemini API は Google が提供する公式 API で、モバイルアプリに AI 機能を統合するための標準的な選択肢です。Android SDK、iOS SDK（Swift）が提供され、ネイティブアプリへの直接統合が容易です。テキスト生成、画像解析、マルチモーダル処理に対応しています。

**用途**

- アプリ内 AI チャットボット
- テキスト分析・分類機能
- 画像解析・物体認識
- パーソナライズされたレコメンデーション
- ユーザーサポート自動化

**料金**

- 無料枠：毎日 50 リクエスト（2,000RPM 制限）
- 従量課金：$0.075 / 1K プロンプトトークン、$0.3 / 1K 出力トークン（目安）

**Rork との連携ポイント**

Rork で生成されたアプリに Gemini API を統合することで、ユーザーが AI の力を直接体感できるアプリが実現できます。API キー管理、レート制限、キャッシング、タイムアウト処理などを適切に実装することが重要です。

### Claude API — Anthropic の AI API

**概要**

Claude API は Anthropic の API で、複雑な推論、長文分析に定評があります。REST API、複数言語 SDK に対応し、モバイルアプリからの HTTP リクエストで利用できます。安全性と信頼性を重視した設計が特徴です。

**用途**

- テキスト分析・要約
- チャットボット機能
- ユーザーサポート自動化
- 複雑な推論が必要なタスク
- カスタムエージェント開発

**料金**

- 従量課金：$0.80 / 1M 入力トークン（Claude 3 Haiku）〜（モデルにより異なる）
- 月間最低利用額：なし

**Rork との連携ポイント**

Gemini API と Claude API の両方をアプリに統合することで、タスクに最適な AI モデルを使い分けられます。高速な応答が必要な場合は Gemini、精度・安全性が重要な場合は Claude という使い分けが可能です。

### Firebase Realtime Database / Firestore SDK

**概要**

Firebase が提供するデータベース SDK（Realtime Database、Firestore）により、モバイルアプリからサーバーコードなしでデータベースにアクセスできます。AI が生成したデータ、ユーザーデータの永続化に活用できます。

**用途**

- AI チャットボットの履歴保存
- ユーザーデータ管理
- リアルタイムデータ同期
- オフラインサポート
- マルチデバイス間のデータ同期

**料金**

- Firebase Spark Plan：無料（制限あり）
- Firebase Blaze Plan：従量課金

**Rork との連携ポイント**

Rork アプリに Firebase SDK を組み込むことで、バックエンド構築なしでデータベース機能が実装できます。Gemini API の結果を Firebase に保存し、ユーザー履歴として活用するワークフローが標準的です。

## モバイルバックエンド

### Firebase — Google のモバイル BaaS

**概要**

Firebase は Google のバックエンド・アズ・ア・サービス（BaaS）プラットフォームで、Authentication、Firestore、Realtime Database、Cloud Storage、Cloud Functions などが統合されています。モバイルアプリ開発の標準的なバックエンド選択肢です。

**用途**

- ユーザー認証・認可管理
- データベース（NoSQL）
- ファイルストレージ
- リアルタイムデータ同期
- バックエンド自動化（Cloud Functions）

**料金**

- Firebase Spark Plan：無料（制限あり）
- Firebase Blaze Plan：従量課金（データベース読み書き、ストレージなど）

**Rork との連携ポイント**

Rork で生成されたアプリに Firebase を統合することで、複雑なバックエンド実装なしで、本格的なサーバーレスアプリケーションが構築できます。Gemini API の結果を Cloud Functions で処理し、Firestore に永続化するパターンが一般的です。

### Supabase — PostgreSQL ベースの BaaS

**概要**

Supabase は Firebase の代替として、PostgreSQL ベースのデータベース、リアルタイム機能、認証、ストレージを提供します。Firebase より低コスト、カスタマイズ性が高く、Vector 検索による AI ベクトル対応が可能です。

**用途**

- PostgreSQL データベース利用
- リアルタイムデータベース機能
- ユーザー認証・認可
- ファイルストレージ
- Vector 検索（AI ベクトル機能）

**料金**

- Supabase Free：無料（基本機能、ストレージ 500MB）
- Supabase Pro：月額25ドル

**Rork との連携ポイント**

Supabase の Vector 検索を活用することで、Gemini API の出力をベクトル化して保存・検索し、より高度な AI 機能を実装できます。RAG（Retrieval-Augmented Generation）をモバイルアプリで実現する際に活用できます。

- [Vector 検索による AI 検索機能](/)

### Cloudflare Workers — エッジコンピューティング

**概要**

Cloudflare Workers は、Cloudflare の全世界エッジロケーションでサーバーレスコードを実行できるプラットフォームです。モバイルアプリからのリクエストに低遅延で応答する API プロキシ、データ変換などに最適です。

**用途**

- AI API プロキシの実装（レート制限、キャッシング）
- リクエスト前処理・ルーティング
- レスポンス変換・圧縮
- CORS ハンドリング
- エッジでのデータ処理

**料金**

- Cloudflare Workers Free：無料（100K リクエスト/日）
- Cloudflare Workers Paid：月額5ドル（以上従量課金）

**Rork との連携ポイント**

Cloudflare Workers で Gemini API へのリクエストをラッピングすることで、モバイルアプリからの API 呼び出しをより効率的に管理できます。キャッシング、レート制限、エラーハンドリングを一元管理できます。

**関連記事**

## 課金・収益化

### StoreKit 2 — Apple の課金 SDK

**概要**

StoreKit 2 は Apple が提供する iOS アプリ内課金 SDK で、サブスクリプション、ワンタイム購入、無料トライアルなどに対応しています。App Store での販売に必須のツールです。

**用途**

- iOS アプリ内サブスクリプション
- ワンタイム購入（アイテム、機能追加）
- 無料トライアル期間の設定
- App Store での販売・配信
- 課金状態の管理

**料金**

- StoreKit 2：無料（Apple の 30% 手数料）

**Rork との連携ポイント**

Rork で iOS アプリを生成する際に StoreKit 2 を統合することで、App Store での販売準備が完了します。RevenueCat などの課金管理ツールと組み合わせることで、さらに運用が効率化されます。

### RevenueCat — クロスプラットフォーム課金管理

**概要**

RevenueCat はクロスプラットフォーム（iOS、Android、Web）の課金管理プラットフォームです。StoreKit 2、Google Play Billing Library を統一インターフェースで管理でき、Analytics、A/B テスト、カスタマー管理などの機能も提供します。

**料金**

- RevenueCat Free：無料（基本機能）
- RevenueCat Pro：月額$249 ～

**Rork との連携ポイント**

RevenueCat を使用することで、Rork で iOS・Android アプリを並行開発する際に、課金システムを統一的に管理できます。プラットフォーム間での課金データを一元管理でき、ビジネス分析が効率化されます。

### Stripe — サブスクリプション・決済プラットフォーム

**概要**

Stripe はグローバルな決済プラットフォームで、クレジットカード決済、デジタルウォレット、サブスクリプション管理などに対応しています。モバイルアプリの物理商品販売、外部決済に活用できます。

**用途**

- 物理商品の決済処理
- デジタル財 販売（アプリ外）
- サブスクリプション管理
- インボイス発行
- 複数通貨対応

**料金**

- Stripe 基本手数料：2.9% + $0.30（カード決済）
- 月額利用料：なし

**Rork との連携ポイント**

Stripe は App Store 内課金（StoreKit 2）の外部の決済が必要な場合に利用します。例えば、アプリ内での物理商品販売、サードパーティサービスとの連携などが該当します。

## CI/CD・テスト・デプロイメント

EAS は Expo が提供する CI/CD サービスで、React Native アプリの自動ビルド、署名、デプロイをサポートします。ローカル環境を使わずに App Store / Google Play へのデプロイが可能です。

**用途**

- iOS・Android ビルドの自動化
- App Store・Google Play への自動デプロイ
- TestFlight・Google Play Internal Testing への配布

**料金**

- EAS Free：無料（月 30 ビルド）
- EAS Pro：月額99ドル（無制限ビルド）

**Rork との連携ポイント**

Rork で React Native アプリを生成した場合、EAS で CI/CD パイプラインを構築することで、継続的な開発・リリースが容易になります。

### Firebase App Distribution — Google の内部テスト配布

Firebase App Distribution は Firebase に統合された、Android・iOS アプリのテスト配布プラットフォームです。内部テスター、外部テスターへの配布が可能です。

**用途**

- 内部チームへのテスト配布
- 外部ベータテスター招待
- テストフィードバック収集
- クラッシュレポート自動収集
- App Store・Google Play 申請前テスト

**料金**

- Firebase App Distribution：無料（Firebase Spark Plan 以上）

**Rork との連携ポイント**

Firebase App Distribution で TestFlight と同様のテスト配布が可能です。Firebase と統合されているため、他の Firebase 機能（Analytics、Crashlytics）とシームレスに連携できます。

### Expo — React Native 開発フレームワーク

**概要**

Expo は React Native 開発を簡素化するフレームワークで、ローカル開発環境構築なしに、クラウドでアプリをビルド・実行できます。Rork が生成するコードの一部は Expo 形式です。

**用途**

- React Native 開発の簡素化
- EAS との統合
- OTA 更新
- デバイス実機テスト
- クロスプラットフォーム開発

**料金**

- Expo：無料（基本機能）
- Expo Application Services（EAS）：従量課金

**Rork との連携ポイント**

Rork で生成されたアプリは Expo ベースで構成されていることが多く、Expo の機能（OTA 更新、開発サーバーなど）をそのまま活用できます。

### React Native — モバイル開発フレームワーク

**概要**

React Native は Facebook が開発したモバイルアプリ開発フレームワークで、JavaScript / TypeScript で iOS・Android アプリを開発できます。Rork が生成するコードのベースフレームワークです。

**用途**

- iOS・Android クロスプラットフォーム開発
- JavaScript による高速開発
- ホットリロード
- ネイティブプラットフォーム機能へのアクセス

**料金**

- React Native：無料（オープンソース）

**Rork との連携ポイント**

Rork が生成するコードは React Native 形式で、React Native エコシステムのあらゆるライブラリ、ツール、リソースが活用できます。Rork 出力後のカスタマイズでも React Native の知識が重要です。
