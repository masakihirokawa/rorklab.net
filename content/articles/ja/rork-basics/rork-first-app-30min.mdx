## この記事で達成できること

この記事を読みながら手を動かすと、30分以内であなたのスマートフォンで動くアプリが完成します。プログラミングの知識はゼロで大丈夫です。

前回の記事「[コード不要！AIでアプリが作れる時代がやってきた](/articles/rork-basics/app-making-ai-era)」でRorkの魅力を知った方は、いよいよ実践の時間です。

## ステップ1：Rorkにアクセスする（5分）

ブラウザ（Google Chromeがおすすめ）で以下のURLにアクセスします。

```
https://rork.app
```

「サインアップ」または「始める」ボタンをクリックし、Googleアカウントまたはメールアドレスで登録します。

```
登録の流れ：
1. rork.app にアクセス
2.「Googleで続ける」をクリック
3. Googleアカウントを選択
4. 利用規約に同意
5. ダッシュボードが表示されたら準備完了！
```

## ステップ2：アプリの内容をAIに伝える（5分）

ダッシュボードにある「新しいプロジェクトを作成」ボタンをクリックします。すると、AIとチャットする画面が表示されます。

今日作るのは「買い物リストアプリ」です。以下のように入力してみましょう。

```
あなた：「買い物リストアプリを作ってください。
         ・商品名を入力して追加できる
         ・追加した商品にチェックマークを付けられる
         ・チェックした商品を一括削除できる
         ・シンプルで使いやすいデザイン
         ・色はパステルブルー系でお願いします」
```

## ステップ3：AIがアプリを生成するのを待つ（5〜10分）

送信ボタンを押すと、RorkのAIがアプリの生成を開始します。画面にコードやデザインが自動で構築されていく様子が表示されます。

```
AIの作業風景：
📱 アプリの画面デザインを生成中...
⚙️ 商品追加機能を実装中...
✅ チェック機能を追加中...
🗑️ 削除機能を実装中...
🎨 カラーテーマを適用中...
✨ 完成！
```

数分待つと、アプリのプレビューが表示されます。

## ステップ4：スマートフォンで動かしてみる（5分）

完成したアプリは、QRコードを使ってスマートフォンで確認できます。

```
確認の手順：
1. Rorkの画面に表示されるQRコードを見つける
2. スマートフォンのカメラでQRコードを読み取る
3. ブラウザでアプリが開く
4. 実際に操作してみる！
```

商品名を入力して「追加」ボタンを押してみてください。リストに追加されたら、タップしてチェックマークを付けてみましょう。動いていますね！

## ステップ5：気になるところを修正する（5分）

「もう少しこうしたい」と思ったら、AIに追加で指示するだけです。

```
あなた：「文字をもう少し大きくしてください。
         あと、カテゴリ分け（食品・日用品・その他）
         ができるようにしてほしいです。」
```

```
あなた：「商品を追加するときに、
         個数も入力できるようにして。」
```

AIが修正を加えて、新しいバージョンのアプリを生成してくれます。

## 完成したアプリを使ってみよう

あなたの最初のアプリが完成しました！実際にスーパーに行くときに使ってみてください。

```
完成アプリの機能：
├── ➕ 商品を追加（名前＋個数）
├── 📂 カテゴリ分け（食品・日用品・その他）
├── ✅ 買った商品にチェック
├── 🗑️ チェック済みを一括削除
└── 🎨 パステルブルーのやさしいデザイン
```

## よくある質問（FAQ）

### Q: 作ったアプリは保存されますか？

はい、Rorkのアカウントにプロジェクトとして保存されます。いつでも開いて修正したり、続きを作ったりできます。

### Q: 作ったアプリを家族に共有できますか？

はい。プレビューURLやQRコードを家族に送れば、家族のスマートフォンでも使えます。

### Q: アプリがうまく動かないときはどうすればいいですか？

AIに「○○がうまく動きません。直してください」と伝えるだけでOKです。AIが問題を特定して修正してくれます。

### Q: 無料プランでいくつアプリが作れますか？

無料プランでも数個のプロジェクトを作成できます。まずは1つ作ってみて、気に入ったら追加のプロジェクトに挑戦してみてください。
