「毎朝ジョギングする」「1日10分読書する」「水を8杯飲む」——こうした日常の習慣を続けることは、簡単なようで難しいものです。

しかし、自分専用の習慣管理アプリがあれば、モチベーションが格段に上がります。今回はRorkを使って、**30分で習慣管理アプリ**を作るチュートリアルをお届けします。プログラミングの知識は不要です。

## 今回作るアプリの概要

完成するアプリの機能：

- 習慣のリストを自分で追加・管理できる
- 毎日のチェックマーク（今日その習慣を達成したか）
- 連続達成日数（ストリーク）のカウント
- シンプルで使いやすいUI

## Rorkでの開発手順

### ステップ1：プロジェクト作成

Rork（[rork.app](https://rork.app)）にログインして「新しいプロジェクト」を作成します。

プロジェクト名に「HabitTracker」または「習慣管理アプリ」と入力してスタートしましょう。

### ステップ2：アプリの仕様をRorkに伝える

Rorkのチャット入力欄に以下のプロンプトを入力してください：

```
習慣管理アプリ（ハビットトラッカー）を作ってください。

【必要な機能】
1. 習慣の追加
   - テキスト入力で習慣名を追加
   - 習慣のカテゴリ（健康・学習・仕事・その他）を選択できる
   - カテゴリに合ったアイコンを表示

2. 今日の習慣チェック
   - ホーム画面に今日の習慣リストを表示
   - タップでチェックオン/オフを切り替え
   - チェック済みは緑色でハイライト

3. ストリーク（連続達成日数）
   - 各習慣の連続達成日数を表示
   - ストリークが7日、30日に達したら祝福メッセージを表示

4. データ保存
   - アプリを閉じても習慣とチェック履歴が消えない
   - 本日分のリセットは毎晩0時に自動実行

【デザイン】
- 清潔感のあるシンプルなデザイン
- カラーは白・グリーン系メイン
- 達成済みと未達成が一目でわかるビジュアル
```

**メインの習慣リスト画面:**

```typescript
// app/(tabs)/index.tsx
import React, { useState, useEffect } from 'react';
import {
  View, Text, FlatList, TouchableOpacity,
  StyleSheet, Alert
} from 'react-native';
import { useHabitStore } from '@/store/habitStore';
import { HabitCard } from '@/components/HabitCard';
import { getTodayString } from '@/utils/dateUtils';

export default function HomeScreen() {
  const { habits, completions, toggleCompletion } = useHabitStore();
  const today = getTodayString();

  const getCompletionStatus = (habitId: string) => {
    return completions.some(
      c => c.habitId === habitId && c.date === today && c.completed
    );
  };

  const completedCount = habits.filter(h => getCompletionStatus(h.id)).length;
  const completionRate = habits.length > 0
    ? Math.round((completedCount / habits.length) * 100)
    : 0;

  return (
    <View style={styles.container}>
      {/* 今日の達成率 */}
      <View style={styles.summaryCard}>
        <Text style={styles.dateText}>
          {new Date().toLocaleDateString('ja-JP', {
            month: 'long', day: 'numeric', weekday: 'long'
          })}
        </Text>
        <Text style={styles.progressText}>
          {completedCount} / {habits.length} 達成
        </Text>
        <View style={styles.progressBar}>
          <View
            style={[
              styles.progressFill,
              { width: `${completionRate}%` }
            ]}
          />
        </View>
        {completionRate === 100 && habits.length > 0 && (
          <Text style={styles.congratsText}>🎉 今日のすべての習慣を達成しました！</Text>
        )}
      </View>

      {/* 習慣リスト */}
      <FlatList
        data={habits}
        keyExtractor={item => item.id}
        renderItem={({ item }) => (
          <HabitCard
            habit={item}
            isCompleted={getCompletionStatus(item.id)}
            onToggle={() => toggleCompletion(item.id, today)}
          />
        )}
        ListEmptyComponent={
          <View style={styles.emptyState}>
            <Text style={styles.emptyText}>まだ習慣がありません</Text>
            <Text style={styles.emptySubtext}>「習慣を追加」から始めましょう</Text>
          </View>
        }
        contentContainerStyle={styles.listContent}
      />
    </View>
  );
}
```

## よくある質問（FAQ）

**Q1. Rorkの無料プランでこのアプリは作れますか？**

はい、このチュートリアルの内容は Rork の無料プランで全て作れます。プッシュ通知など一部のネイティブ機能は有料プランや Rork Max が必要になる場合があります。まずは基本機能から始めてみてください。

**Q2. できあがったアプリを家族や友人と共有できますか？**

Rork の基本プランでは QR コード経由で実機にインストールしてシェアできます。App Store / Google Play への公開は、Rork Max（$200/月）の2クリック公開機能か、通常の開発者アカウント審査が必要です。

**Q3. 毎日0時のリセット（チェックのリフレッシュ）はどうやって実装しますか？**

アプリ起動時に「今日の日付」と前回チェックした日付を比較する方法が最もシンプルです。Rorkに「アプリ起動時に日付が変わっていたら当日のチェックをリセットする処理を追加して」と伝えると、実装してくれます。
