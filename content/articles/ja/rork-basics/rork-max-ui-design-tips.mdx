# Rork Max UIデザインのコツ — 美しいアプリを作る10のルール

ユーザーの第一印象を決めるUIデザイン。このガイドでは、美しく使いやすいアプリを作るために即座に実践できる10のルールを紹介します。

## ルール 1: 60-30-10 の色彩比率

プロ品質のUIは、色彩の配置戦略に基づいています。60-30-10 ルールは、デザイン業界の標準です。

- **60%**: 背景色（アプリ全体の調和を取るメイン色）
- **30%**: セカンダリ色（UI要素、カード）
- **10%**: アクセント色（ボタン、強調要素）

**src/styles/colors.ts**
```typescript
export const colorPalette = {
  // 60% — Background / Primary
  primary: '#FFFFFF',
  primaryDark: '#F5F5F5',
  primaryDarkDark: '#E5E5E5',

  // 30% — Secondary
  secondary: '#6C757D',
  secondaryLight: '#ADB5BD',
  secondaryLighter: '#DEE2E6',

  // 10% — Accent
  accent: '#007AFF',
  accentDark: '#0051D5',
  accentLight: '#40B0FF',

  // Semantic colors
  success: '#34C759',
  warning: '#FF9500',
  error: '#FF3B30',
  info: '#00C7E8',
};

export const darkColorPalette = {
  primary: '#1C1C1E',
  primaryLight: '#2C2C2E',
  primaryLighter: '#3A3A3C',
  secondary: '#8E8E93',
  accent: '#0A84FF',
  success: '#30B0C0',
  warning: '#FF9500',
  error: '#FF453A',
};
```

**src/screens/Home.tsx**
```typescript

const styles = StyleSheet.create({

```

<Callout type="info">
**ツール**: Figma や Adobe Color Wheel で色彩調和を探索してから実装します。色盲の人にも認識しやすいパレットを選びます。
</Callout>

## ルール 2: タイポグラフィの階層化

テキストサイズを統一し、明確な階層を作ります。

**src/styles/typography.ts**
```typescript
import { StyleSheet } from 'react-native';

export const typography = StyleSheet.create({
  // メガ見出し：タイトル画面
  h0: {
    fontSize: 40,
    fontWeight: '800',
    lineHeight: 48,
    letterSpacing: -1,
  },

  // 大見出し：セクションタイトル
  h1: {
    fontSize: 32,
    fontWeight: '700',
    lineHeight: 40,
  },

  // 中見出し：カードタイトル
  h2: {
    fontSize: 24,
    fontWeight: '600',
    lineHeight: 32,
  },

  // 小見出し
  h3: {
    fontSize: 20,
    fontWeight: '600',
    lineHeight: 28,
  },

  // 本文：標準テキスト
  body: {
    fontSize: 16,
    fontWeight: '400',
    lineHeight: 24,
    letterSpacing: 0.5,
  },

  // 補助テキスト
  caption: {
    fontSize: 14,
    fontWeight: '400',
    lineHeight: 20,
    color: '#666',
  },

  // 超小テキスト
  tiny: {
    fontSize: 12,
    fontWeight: '400',
    lineHeight: 16,
    color: '#999',
  },

  // ボタンテキスト
  button: {
    fontSize: 16,
    fontWeight: '600',
    lineHeight: 20,
  },
});
```

**実装例**
```typescript
import { typography } from '../styles/typography';

export default function ArticleCard() {
  return (
    <View>
      <Text style={typography.h3}>Rork Max の魅力</Text>
      <Text style={typography.body}>
        AI駆動型モバイルアプリビルダー...
      </Text>
      <Text style={typography.caption}>
        2026年3月14日 · 5分で読める
      </Text>
    </View>
  );
}
```

<Callout type="warning">
**注意**: フォントファミリーは 1～2 種類のみに限定します。San serif（Roboto、Inter）推奨。システムフォント（SF Pro Display、Segoe UI）を活用。
</Callout>

## ルール 3: 余白と パディング の一貫性

余白は視覚的リズムを生み出します。8px グリッドシステムを基準に。

**src/styles/spacing.ts**
```typescript
export const spacing = {
  xs: 4,
  sm: 8,
  md: 12,
  lg: 16,
  xl: 24,
  xxl: 32,
  xxxl: 48,
};

export const createSpacedStyles = (baseSpacing = spacing.lg) =>
  StyleSheet.create({
    container: {
      padding: baseSpacing,
    },
    section: {
      marginBottom: baseSpacing * 1.5,
    },
    row: {
      marginBottom: baseSpacing / 2,
    },
  });
```

**実装**
```typescript
import { spacing } from '../styles/spacing';

const styles = StyleSheet.create({
  container: {
    flex: 1,
    padding: spacing.lg,        // 16px
  },
  cardContainer: {
    marginBottom: spacing.xl,    // 24px
  },
  cardPadding: {
    padding: spacing.lg,         // 16px
  },
  textSpacing: {
    marginBottom: spacing.md,    // 12px
  },
});
```

| オブジェクト | パディング | マージン |
|-----------|-----------|---------|
| カード | 16px | 12px bottom |
| ボタン | 12px vertical, 16px horizontal | 8px |
| セクション | 16px | 24px bottom |
| リスト item | 16px | 8px bottom |

## ルール 4: タッチターゲットサイズは最小 44×44px

Apple HIG（Human Interface Guideline）では、タッチ可能な領域は最小 44×44px を推奨。

**src/components/Button.tsx**
```typescript
import { TouchableOpacity, Text, StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  button: {
    minHeight: 44,
    minWidth: 44,
    paddingVertical: 12,
    paddingHorizontal: 16,
    borderRadius: 8,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#007AFF',
  },
  // プレスフィードバック用
  buttonPressed: {
    opacity: 0.8,
  },
  text: {
    color: '#fff',
    fontSize: 16,
    fontWeight: '600',
  },
});

export default function Button({ title, onPress }) {
  return (
    <TouchableOpacity
      style={styles.button}
      onPress={onPress}
      activeOpacity={0.8}
      hitSlop={{ top: 10, bottom: 10, left: 10, right: 10 }}
    >
      <Text style={styles.text}>{title}</Text>
    </TouchableOpacity>
  );
}
```

## ルール 5: アニメーションは機能的に

美しいアニメーションは、ユーザーの期待を伝えます。不要なアニメーションは避けます。

**src/components/FadeInView.tsx**
```typescript
import { useEffect, useRef } from 'react';
import { Animated, View } from 'react-native';

export default function FadeInView({ children }) {
  const fadeAnim = useRef(new Animated.Value(0)).current;

  useEffect(() => {
    Animated.timing(fadeAnim, {
      toValue: 1,
      duration: 400,
      useNativeDriver: true,
    }).start();
  }, [fadeAnim]);

  return (
    <Animated.View style={{ opacity: fadeAnim }}>
      {children}
    </Animated.View>
  );
}
```

**スケールアニメーション（ボタン押下）**
```typescript
import { Animated, TouchableOpacity } from 'react-native';
import { useRef } from 'react';

export default function ScaleButton({ onPress, children }) {
  const scaleAnim = useRef(new Animated.Value(1)).current;

  const handlePressIn = () => {
    Animated.spring(scaleAnim, {
      toValue: 0.95,
      useNativeDriver: true,
    }).start();
  };

  const handlePressOut = () => {
    Animated.spring(scaleAnim, {
      toValue: 1,
      useNativeDriver: true,
    }).start();
  };

  return (
    <TouchableOpacity
      onPressIn={handlePressIn}
      onPressOut={handlePressOut}
      onPress={onPress}
    >
      <Animated.View
        style={{
          transform: [{ scale: scaleAnim }],
        }}
      >
        {children}
      </Animated.View>
    </TouchableOpacity>
  );
}
```

<Callout type="info">
**ベストプラクティス**:
- ページ遷移: 200-300ms
- アイコン interaction: 200ms
- ボタン feedback: 100-150ms
- すべてのアニメーションに useNativeDriver={true} を設定
</Callout>

## ルール 6: ダークモード対応

ダークモードは iOS・Android 両方で必須。useColorScheme で自動検出。

**src/hooks/useColors.ts**
```typescript
import { useColorScheme } from 'react-native';
import { colorPalette, darkColorPalette } from '../styles/colors';

export function useColors() {
  const colorScheme = useColorScheme();
  return colorScheme === 'dark' ? darkColorPalette : colorPalette;
}
```

**src/screens/Profile.tsx**
```typescript
import { useColors } from '../hooks/useColors';

export default function Profile() {
  const colors = useColors();

  return (
    <View style={{ backgroundColor: colors.primary, flex: 1 }}>
      <Text style={{ color: colors.secondary }}>Profile</Text>
    </View>
  );
}
```

**App.tsx で DarkMode Provider 設定**
```typescript
import { useColorScheme } from 'react-native';

export default function App() {
  const colorScheme = useColorScheme();

  return (
    <NavigationContainer
      theme={colorScheme === 'dark' ? DarkTheme : DefaultTheme}
    >
      <RootStack />
    </NavigationContainer>
  );
}
```

## ルール 7: アイコンの一貫性

Material Icons か iOS SF Symbols を全体で統一。

**src/components/Icon.tsx**
```typescript
import { Platform } from 'react-native';
import MaterialIcons from '@expo/vector-icons/MaterialIcons';
import Ionicons from '@expo/vector-icons/Ionicons';

export function Icon({
  name,
  size = 24,
  color = '#000',
  platform = 'auto',
}) {
  const useSF = platform === 'auto' ? Platform.OS === 'ios' : platform === 'ios';

  if (useSF) {
    return (
      <Ionicons
        name={name}
        size={size}
        color={color}
      />
    );
  }

  return (
    <MaterialIcons
      name={name}
      size={size}
      color={color}
    />
  );
}
```

## ルール 8: 状態表現（Loading・Error・Empty）

三つの状態を適切に表現：

**src/components/StateViews.tsx**
```typescript
import { View, Text, ActivityIndicator } from 'react-native';
import { useColors } from '../hooks/useColors';

export function LoadingView() {
  const colors = useColors();
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <ActivityIndicator size="large" color={colors.accent} />
      <Text style={{ marginTop: 16, color: colors.secondary }}>
        読み込み中...
      </Text>
    </View>
  );
}

export function EmptyView({ title, message }) {
  const colors = useColors();
  return (
    <View
      style={{
        flex: 1,
        justifyContent: 'center',
        alignItems: 'center',
        paddingHorizontal: 32,
      }}
    >
      <Text style={{ fontSize: 20, fontWeight: '600', color: colors.secondary }}>
        {title}
      </Text>
      <Text
        style={{
          fontSize: 14,
          color: colors.secondary,
          marginTop: 8,
          textAlign: 'center',
        }}
      >
        {message}
      </Text>
    </View>
  );
}

export function ErrorView({ title, message, onRetry }) {
  const colors = useColors();
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text style={{ fontSize: 20, fontWeight: '600', color: colors.error }}>
        {title}
      </Text>
      <Text style={{ color: colors.secondary, marginTop: 8 }}>
        {message}
      </Text>
      <TouchableOpacity
        style={{
          marginTop: 16,
          paddingHorizontal: 24,
          paddingVertical: 8,
          backgroundColor: colors.accent,
          borderRadius: 8,
        }}
        onPress={onRetry}
      >
        <Text style={{ color: '#fff', fontWeight: '600' }}>
          再試行
        </Text>
      </TouchableOpacity>
    </View>
  );
}
```

## ルール 9: リスト・スクロール最適化

大量アイコンのレンダリングは FlatList で。

```typescript
import { FlatList, Text, View } from 'react-native';

const items = Array.from({ length: 1000 }, (_, i) => ({
  id: String(i),
  title: `Item ${i + 1}`,
}));

export default function ItemList() {
  const renderItem = ({ item }) => (
    <View style={{ padding: 16, borderBottomWidth: 1, borderColor: '#eee' }}>
      <Text>{item.title}</Text>
    </View>
  );

  return (
    <FlatList
      data={items}
      renderItem={renderItem}
      keyExtractor={(item) => item.id}
      maxToRenderPerBatch={10}
      updateCellsBatchingPeriod={50}
      removeClippedSubviews
    />
  );
}
```
