# Rork Max で StoreKit 2 の In-App Purchase を実装する

Rork Max で作ったアプリを App Store に公開し、収益を得るために欠かせないのが In-App Purchase（アプリ内課金）です。StoreKit 2 は Apple が提供する最新の課金 API で、Swift Concurrency（async/await）に対応した modern な設計になっています。

この記事では、Rork Max で構築した Swift ネイティブアプリにサブスクリプション課金を実装する手順を、ステップバイステップで解説します。

## StoreKit 2 の基本概念

StoreKit 2 は、StoreKit 1（従来の API）を置き換えるために設計されました。主な改善点は3つあります。

**async/await 対応。** 購入処理やレシート検証が async/await で書けるため、コールバック地獄から解放されます。

**JWS（JSON Web Signature）ベースのトランザクション。** サーバーサイドでの検証が容易になりました。

**Transaction.updates のストリーム。** 購入状態の変化をリアルタイムで監視できます。

## App Store Connect での設定

コードを書く前に、App Store Connect でプロダクトを定義する必要があります。

### サブスクリプショングループの作成

App Store Connect にログインし、「サブスクリプション」タブからサブスクリプショングループを作成します。グループ名は「Premium」のようなシンプルな名前で OK です。

グループ内にサブスクリプションプランを作成します。例えば、月額プラン（`premium_monthly`）と年額プラン（`premium_yearly`）の2つです。

各プランに対して、価格、期間、無料トライアル期間を設定します。日本のApp Store では、月額プランは ¥480、年額プランは ¥3,800 あたりが個人開発アプリでは一般的な価格帯です。

### プロダクト ID の命名規則

プロダクト ID は一度作成すると変更・削除できないため、慎重に命名しましょう。`com.yourcompany.appname.premium.monthly` のように、バンドル ID + プラン名の構成が推奨されます。

## Rork Max でのコード実装

Rork Max のプロンプトに、以下のような指示を出してコードを生成します。

### Store Manager の実装

```swift
import StoreKit

@Observable
class StoreManager {
    private(set) var products: [Product] = []
    private(set) var purchasedProductIDs: Set<String> = []
    private(set) var isLoading: Bool = false

    private let productIDs: Set<String> = [
        "com.yourcompany.app.premium.monthly",
        "com.yourcompany.app.premium.yearly"
    ]

    private var transactionListener: Task<Void, Error>?

    init() {
        transactionListener = listenForTransactions()
        Task { await loadProducts() }
    }

    deinit {
        transactionListener?.cancel()
    }

    // プロダクト情報の取得
    func loadProducts() async {
        isLoading = true
        defer { isLoading = false }

        do {
            products = try await Product.products(for: productIDs)
                .sorted { $0.price < $1.price }
        } catch {
            print("Failed to load products: \(error)")
        }
    }

    // 購入処理
    func purchase(_ product: Product) async throws -> Transaction? {
        let result = try await product.purchase()

        switch result {
        case .success(let verification):
            let transaction = try checkVerified(verification)
            await updatePurchasedProducts()
            await transaction.finish()
            return transaction

        case .userCancelled:
            return nil

        case .pending:
            return nil

        @unknown default:
            return nil
        }
    }

    // 購入済みプロダクトの更新
    func updatePurchasedProducts() async {
        var purchased: Set<String> = []

        for await result in Transaction.currentEntitlements {
            if let transaction = try? checkVerified(result) {
                purchased.insert(transaction.productID)
            }
        }

        purchasedProductIDs = purchased
    }

    // プレミアム会員かどうか
    var isPremium: Bool {
        !purchasedProductIDs.isEmpty
    }

    // トランザクション検証
    private func checkVerified<T>(_ result: VerificationResult<T>) throws -> T {
        switch result {
        case .unverified(_, let error):
            throw error
        case .verified(let safe):
            return safe
        }
    }

    // トランザクションのリアルタイム監視
    private func listenForTransactions() -> Task<Void, Error> {
        Task.detached {
            for await result in Transaction.updates {
                if let transaction = try? self.checkVerified(result) {
                    await self.updatePurchasedProducts()
                    await transaction.finish()
                }
            }
        }
    }
}
```

### 課金画面の UI

```swift
import SwiftUI
import StoreKit

struct PaywallView: View {
    @Environment(StoreManager.self) private var store
    @State private var selectedProduct: Product?
    @State private var isPurchasing = false
    @State private var showError = false
    @State private var errorMessage = ""

    var body: some View {
        VStack(spacing: 24) {
            // ヘッダー
            VStack(spacing: 8) {
                Text("Premium にアップグレード")
                    .font(.title.bold())
                Text("すべての機能をアンロック")
                    .foregroundStyle(.secondary)
            }
            .padding(.top, 40)

            // 特典リスト
            VStack(alignment: .leading, spacing: 16) {
                FeatureRow(icon: "star.fill", text: "広告なし")
                FeatureRow(icon: "cloud.fill", text: "クラウド同期")
                FeatureRow(icon: "wand.and.stars", text: "AI アシスタント")
                FeatureRow(icon: "paintpalette.fill", text: "カスタムテーマ")
            }
            .padding(.horizontal, 32)

            Spacer()

            // プラン選択
            if store.isLoading {
                ProgressView()
            } else {
                VStack(spacing: 12) {
                    ForEach(store.products) { product in
                        PlanButton(
                            product: product,
                            isSelected: selectedProduct?.id == product.id
                        ) {
                            selectedProduct = product
                        }
                    }
                }
                .padding(.horizontal, 24)
            }

            // 購入ボタン
            Button {
                Task { await purchase() }
            } label: {
                if isPurchasing {
                    ProgressView()
                        .tint(.white)
                } else {
                    Text("購入する")
                        .font(.headline)
                }
            }
            .frame(maxWidth: .infinity)
            .padding(.vertical, 16)
            .background(selectedProduct != nil ? Color.blue : Color.gray)
            .foregroundStyle(.white)
            .clipShape(RoundedRectangle(cornerRadius: 12))
            .disabled(selectedProduct == nil || isPurchasing)
            .padding(.horizontal, 24)

            // 利用規約リンク
            HStack {
                Link("利用規約", destination: URL(string: "https://yourapp.com/terms")!)
                Text("・")
                Link("プライバシーポリシー", destination: URL(string: "https://yourapp.com/privacy")!)
                Text("・")
                Button("購入の復元") {
                    Task { await store.updatePurchasedProducts() }
                }
            }
            .font(.caption)
            .foregroundStyle(.secondary)
            .padding(.bottom, 16)
        }
        .alert("エラー", isPresented: $showError) {
            Button("OK") {}
        } message: {
            Text(errorMessage)
        }
        .onAppear {
            selectedProduct = store.products.first
        }
    }

    private func purchase() async {
        guard let product = selectedProduct else { return }
        isPurchasing = true
        defer { isPurchasing = false }

        do {
            _ = try await store.purchase(product)
        } catch {
            errorMessage = error.localizedDescription
            showError = true
        }
    }
}

struct FeatureRow: View {
    let icon: String
    let text: String

    var body: some View {
        HStack(spacing: 12) {
            Image(systemName: icon)
                .foregroundStyle(.blue)
                .frame(width: 24)
            Text(text)
        }
    }
}

struct PlanButton: View {
    let product: Product
    let isSelected: Bool
    let action: () -> Void

    var body: some View {
        Button(action: action) {
            HStack {
                VStack(alignment: .leading) {
                    Text(product.displayName)
                        .font(.headline)
                    Text(product.displayPrice + " / " + product.subscription!.subscriptionPeriod.debugDescription)
                        .font(.subheadline)
                        .foregroundStyle(.secondary)
                }
                Spacer()
                Image(systemName: isSelected ? "checkmark.circle.fill" : "circle")
                    .foregroundStyle(isSelected ? .blue : .gray)
            }
            .padding()
            .background(isSelected ? Color.blue.opacity(0.1) : Color(.systemGray6))
            .clipShape(RoundedRectangle(cornerRadius: 12))
            .overlay(
                RoundedRectangle(cornerRadius: 12)
                    .stroke(isSelected ? Color.blue : Color.clear, lineWidth: 2)
            )
        }
        .buttonStyle(.plain)
    }
}
```

## テスト

### Xcode の StoreKit テスト環境

StoreKit 2 のテストには、Xcode の StoreKit Configuration File を使います。`.storekit` ファイルを作成し、テスト用のプロダクトを定義することで、実際の App Store Connect に接続せずにローカルでテストできます。

Rork Max でビルドしたアプリを TestFlight 経由でテストする場合は、App Store Connect のサンドボックス環境が自動的に使用されます。

### テストで確認すべきポイント

購入フローが正常に完了するか。購入後にプレミアム機能がアンロックされるか。アプリを再起動した後も購入状態が維持されるか。「購入の復元」ボタンが正しく動作するか。ネットワークエラー時の表示が適切か。

## App Store 審査のポイント

In-App Purchase を含むアプリの審査では、いくつかの特有のチェックポイントがあります。

「購入の復元」ボタンを必ず配置してください。Apple のガイドラインで義務付けられています。

利用規約とプライバシーポリシーへのリンクを、購入画面に表示してください。

サブスクリプションの自動更新について、価格と更新頻度を明確に表示してください。

無料トライアルを提供する場合、トライアル終了後に自動的に課金されることを明示してください。

## まとめ

StoreKit 2 は、Swift Concurrency ベースの modern な設計で、Rork Max の Swift ネイティブ開発と非常に相性が良いです。`StoreManager` クラスによる購入管理、美しい Paywall UI、適切なテストとApp Store ガイドラインへの準拠——これらを押さえておけば、Rork Max で作ったアプリで安定した収益を得る基盤が整います。
