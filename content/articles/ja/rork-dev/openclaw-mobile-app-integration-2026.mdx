## はじめに

2026年のアプリ開発は、AI バックエンド + ノーコードフロントエンド という組み合わせが主流になっています。その中でも、**OpenClaw**（オープンソース AI エージェント）と **Rork**（AI モバイルアプリビルダー）の組み合わせは、驚くほど簡単に本格的な AI チャットアプリを実現します。

本記事では、OpenClaw バックエンド、Rork フロントエンドを使用して、プロダクションレベルの AI チャットアプリを構築する方法を初心者向けに解説します。

## OpenClaw のモバイル対応状況（2026年版）

```

         │

         │
    ┌────▼────────┐
    │  OpenClaw   │

    └─────────────┘
```

## OpenClaw バックエンドとの接続実装

### ステップ1. WebSocket 接続の実装

```typescript
// services/OpenClawService.ts

import { useEffect, useState, useCallback } from 'react';

interface OpenClawConfig {
  baseURL: string;
  apiToken: string;
  conversationId?: string;
}

export const useOpenClawConnection = (config: OpenClawConfig) => {
  const [ws, setWs] = useState<WebSocket | null>(null);
  const [connectionStatus, setConnectionStatus] = useState<'connected' | 'disconnected'>('disconnected');
  const [messages, setMessages] = useState<Message[]>([]);

  // WebSocket 接続確立
  useEffect(() => {
    const conversationId = config.conversationId || `conv-${Date.now()}`;
    const wsURL = `${config.baseURL}/ws/chat/${conversationId}`;

    const websocket = new WebSocket(wsURL);

    websocket.onopen = () => {
      console.log('OpenClaw に接続しました');
      setConnectionStatus('connected');

      // 認証トークンを送信
      websocket.send(JSON.stringify({
        type: 'auth',
        token: config.apiToken,
      }));
    };

    websocket.onmessage = (event) => {
      const data = JSON.parse(event.data);
      console.log('受信:', data);

      if (data.type === 'message') {
        const assistantMessage: Message = {
          id: data.id,
          role: 'assistant',
          content: data.content,
          timestamp: new Date(data.timestamp),
        };
        setMessages(prev => [...prev, assistantMessage]);
      }

      if (data.type === 'error') {
        console.error('OpenClaw エラー:', data.error);
      }
    };

    websocket.onclose = () => {
      console.log('OpenClaw から切断されました');
      setConnectionStatus('disconnected');

      // 5秒後に再接続を試みる
      setTimeout(() => {
        // 再接続ロジック
      }, 5000);
    };

    websocket.onerror = (error) => {
      console.error('WebSocket エラー:', error);
      setConnectionStatus('disconnected');
    };

    setWs(websocket);

    return () => {
      if (websocket && websocket.readyState === WebSocket.OPEN) {
        websocket.close();
      }
    };
  }, [config]);

  // メッセージ送信
  const sendMessage = useCallback((content: string) => {
    if (ws && ws.readyState === WebSocket.OPEN) {
      ws.send(JSON.stringify({
        type: 'message',
        content,
        timestamp: new Date().toISOString(),
      }));
    } else {
      console.error('WebSocket に接続していません');
    }
  }, [ws]);

  return {
    connectionStatus,
    messages,
    sendMessage,
  };
};
```

### ステップ2. REST API による代替実装

WebSocket が使用できない環境向けに、REST API での実装も提供：

```typescript
// services/OpenClawAPIService.ts

import axios from 'axios';

interface APIResponse {
  conversationId: string;
  messageId: string;
  content: string;
  timestamp: string;
}

export class OpenClawAPIService {
  private baseURL: string;
  private apiToken: string;
  private conversationId: string;

  constructor(baseURL: string, apiToken: string) {
    this.baseURL = baseURL;
    this.apiToken = apiToken;
    this.conversationId = `conv-${Date.now()}`;
  }

  async sendMessage(content: string): Promise<APIResponse> {
    try {
      const response = await axios.post<APIResponse>(
        `${this.baseURL}/api/v1/message`,
        {
          conversationId: this.conversationId,
          content,
          timestamp: new Date().toISOString(),
        },
        {
          headers: {
            Authorization: `Bearer ${this.apiToken}`,
            'Content-Type': 'application/json',
          },
        }
      );

      return response.data;
    } catch (error) {
      console.error('メッセージ送信失敗:', error);
      throw error;
    }
  }

  async getConversationHistory(): Promise<Message[]> {
    try {
      const response = await axios.get(
        `${this.baseURL}/api/v1/conversation/${this.conversationId}`,
        {
          headers: {
            Authorization: `Bearer ${this.apiToken}`,
          },
        }
      );

      return response.data.messages;
    } catch (error) {
      console.error('履歴取得失敗:', error);
      throw error;
    }
  }

  async getAvailableSkills() {
    try {
      const response = await axios.get(
        `${this.baseURL}/api/v1/skills`,
        {
          headers: {
            Authorization: `Bearer ${this.apiToken}`,
          },
        }
      );

      return response.data.skills;
    } catch (error) {
      console.error('スキル取得失敗:', error);
      throw error;
    }
  }
}
```

## オフライン対応とローカルキャッシュ

### AsyncStorage を使用したメッセージ保存

```typescript
// services/MessageStorage.ts

import AsyncStorage from '@react-native-async-storage/async-storage';

export class MessageStorage {
  private storageKey = 'openclaw_messages';

  async saveMessages(messages: Message[]): Promise<void> {
    try {
      const jsonData = JSON.stringify(messages);
      await AsyncStorage.setItem(this.storageKey, jsonData);
    } catch (error) {
      console.error('メッセージ保存失敗:', error);
    }
  }

  async loadMessages(): Promise<Message[]> {
    try {
      const jsonData = await AsyncStorage.getItem(this.storageKey);
      return jsonData ? JSON.parse(jsonData) : [];
    } catch (error) {
      console.error('メッセージ読み込み失敗:', error);
      return [];
    }
  }

  async clearMessages(): Promise<void> {
    try {
      await AsyncStorage.removeItem(this.storageKey);
    } catch (error) {
      console.error('メッセージ削除失敗:', error);
    }
  }

  async saveOfflineMessage(message: Message): Promise<void> {
    const messages = await this.loadMessages();
    messages.push({
      ...message,
      synced: false, // オフラインフラグ
    });
    await this.saveMessages(messages);
  }
}
```

## プッシュ通知の設定

OpenClaw からのプロアクティブなメッセージ（例：アラート）をプッシュ通知で受け取る設定：

### Expo Notifications を使用

```typescript
// services/NotificationService.ts

import * as Notifications from 'expo-notifications';
import axios from 'axios';

export const setupPushNotifications = async (apiToken: string) => {
  // 通知パーミッション要求
  const { status } = await Notifications.requestPermissionsAsync();

  if (status === 'granted') {
    // デバイストークンを取得
    const token = (await Notifications.getExpoPushTokenAsync()).data;

    // OpenClaw にトークンを登録
    await axios.post(
      'https://your-openclaw-instance.com/api/v1/notification-token',
      { token },
      {
        headers: {
          Authorization: `Bearer ${apiToken}`,
        },
      }
    );

    console.log('プッシュ通知トークン登録完了:', token);
  }

  // 通知を受け取ったときの処理
  Notifications.setNotificationHandler({
    handleNotification: async (notification) => {
      console.log('通知受信:', notification.request.content.body);

      // チャット画面を開く、またはメッセージを追加
      return {
        shouldShowAlert: true,
        shouldPlaySound: true,
        shouldSetBadge: true,
      };
    },
  });
};
```

## よくある質問（FAQ）

### Q1. Rork で生成したコンポーネントをカスタマイズできますか？

**A.** はい。生成されたコンポーネントは完全に編集可能な React Native コードです。任意のカスタマイズが可能です。ただし、Rork で再度修正する際に上書きされないよう注意が必要です。

### Q2. OpenClaw サーバーは自分でホストする必要がありますか？

**A.** いいえ。OpenClaw はマネージドサービスとしても提供されています（月額 $50～$200）。セルフホスティングも可能です。要件に応じて選択してください。

### Q3. モバイルアプリで複数ユーザーを管理できますか？

**A.** はい。OpenClaw の API は複数のユーザーをサポートしており、ユーザー ID ごとに異なる会話を管理できます。認証システムの実装が必要です。

### Q4. バッテリー消費を最小化するには？

**A.** WebSocket 接続は常時オープンなため、バッテリー消費が増加します。以下の最適化が効果的です：
- 画面オフ時は接続を一時停止
- バックグラウンドモードでは REST API を使用
- メッセージフェッチ間隔を調整

### Q5. 複数言語対応が必要な場合は？

**A.** Rork は多言語テンプレートを提供しており、i18n ライブラリ（react-i18next）の統合も容易です。OpenClaw スキルで言語別の処理も可能です。
