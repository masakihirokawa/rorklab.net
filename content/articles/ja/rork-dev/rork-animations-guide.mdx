## なぜアニメーションが重要か

モバイルアプリの品質感を大きく左右するのがアニメーションです。ボタンを押したときの反応、画面遷移のスムーズさ、ローディング中の表現 — これらがユーザー体験の「高級感」を生み出します。

Rork Max は React Native ベースなので、`Animated` API と `react-native-reanimated` を使ったアニメーションが利用可能です。

## 基本のフェードアニメーション

Rork Max に以下のプロンプトを送ると、基本的なフェードインアニメーションが実装されます。

```
「このコンポーネントに、マウント時にフェードインするアニメーションを追加してください。
React Native の Animated API を使用してください。」
```

生成されるコードの例:

```javascript
import { useEffect, useRef } from 'react';
import { Animated, View, Text } from 'react-native';

export function FadeInCard({ children }) {
  const opacity = useRef(new Animated.Value(0)).current;

  useEffect(() => {
    Animated.timing(opacity, {
      toValue: 1,
      duration: 400,
      useNativeDriver: true,
    }).start();
  }, []);

  return (
    <Animated.View style={{ opacity }}>
      {children}
    </Animated.View>
  );
}
```

## スプリングアニメーション

バウンスの効いた自然な動きには `spring` を使います。ボタンのタップ反応などに最適です。

```javascript
const scale = useRef(new Animated.Value(1)).current;

const handlePress = () => {
  Animated.sequence([
    Animated.spring(scale, {
      toValue: 0.95,
      useNativeDriver: true,
      tension: 100,
    }),
    Animated.spring(scale, {
      toValue: 1,
      useNativeDriver: true,
      friction: 3,
    }),
  ]).start();
};
```

## スライドイン（パネル・モーダル）

下から滑り出すパネルの実装例:

```javascript
const translateY = useRef(new Animated.Value(300)).current;

const slideIn = () => {
  Animated.spring(translateY, {
    toValue: 0,
    useNativeDriver: true,
    tension: 65,
    friction: 8,
  }).start();
};

return (
  <Animated.View
    style={{
      transform: [{ translateY }],
      position: 'absolute',
      bottom: 0,
      left: 0,
      right: 0,
      backgroundColor: 'white',
      borderTopLeftRadius: 20,
      borderTopRightRadius: 20,
      padding: 24,
    }}
  >
    {/* パネルコンテンツ */}
  </Animated.View>
);
```

## Lottie アニメーションの活用

より複雑なアニメーションには **Lottie** が有効です。Rork Max にプロンプトで依頼できます。

```
「ローディング画面に Lottie アニメーションを追加してください。
https://lottiefiles.com のアニメーションファイル（.json）を使用します。」
```

LottieFiles からダウンロードしたアニメーションJSONを `assets/animations/` フォルダに配置し、Rork Max に組み込んでもらいます。

## アニメーションのパフォーマンス最適化

**`useNativeDriver: true` を必ず指定する**: JavaScript スレッドではなく UI スレッドでアニメーションが実行され、スムーズさが大幅に向上します。

**不要なレンダリングを避ける**: アニメーション値が変化するたびにコンポーネント全体が再レンダリングされないよう、`React.memo` や適切なコンポーネント分割を行いましょう。

## まとめ

Rork Max でアニメーションを実装するには:

- 基本的なフェード・スプリング・スライドは `Animated` API で対応
- Rork Max へのプロンプトで自動生成が可能
- `useNativeDriver: true` でパフォーマンスを確保
- 複雑なアニメーションには Lottie が効果的

アニメーションを適切に使うことで、Rork Max で作ったアプリの品質感が格段に上がります。
