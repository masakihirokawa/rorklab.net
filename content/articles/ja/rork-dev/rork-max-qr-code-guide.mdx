## はじめに

QRコードは現代のモバイルアプリに欠かせない機能のひとつです。決済アプリ、ポイントカード、在庫管理、チケット確認など、あらゆる場面で活用されています。Rork Maxを使えば、iOSのネイティブカメラAPI（AVFoundation）を活用した高精度なQRコードスキャナーと、CoreImageを使ったQRコード生成機能をAIプロンプトだけで実装できます。

この記事では、以下の内容を段階的に解説します：

- QRコードスキャナーの実装（AVFoundationベース）
- QRコード生成機能の実装（CoreImageベース）
- スキャン履歴の管理（UserDefaults）
- 実用的なユースケース（チェックイン、URL共有など）

## QRコードスキャナーの仕組み

iOSでQRコードをスキャンする仕組みは以下の通りです：

1. **AVCaptureSession** でカメラ入力を開始する
2. **AVCaptureMetadataOutput** でカメラ映像からQRコードを検出する
3. 検出されたQRコードのデータを処理する
4. SwiftUIビューにオーバーレイ表示でスキャン状況をフィードバックする

Rork Maxはこのすべてのネイティブコードをプロンプト一発で生成できます。

## Step 1: QRコードスキャナーの実装

### Rork Maxへのプロンプト

Rork Maxのエディタに以下のプロンプトを入力します：

```
iOSアプリにQRコードスキャナー機能を追加してください。
要件：
- AVFoundationを使ったカメラベースのQRコード読み取り
- スキャン成功時にバイブレーションとサウンドフィードバック
- スキャン結果をシート（モーダル）で表示
- URLの場合はSafariで開くボタンを表示
- スキャン履歴を最大20件まで保存（UserDefaults）
- フロントライトのオン/オフ切り替えボタン
- カメラ権限リクエスト処理を含める
```

### 生成されるコードの例

Rork Maxが生成するQRスキャナーの主要部分は以下のようになります：

```swift
import SwiftUI
import AVFoundation

// QRスキャナーのメインビュー
struct QRScannerView: View {
    @StateObject private var scanner = QRScannerViewModel()
    @State private var showResult = false
    @State private var scannedCode = ""

    var body: some View {
        ZStack {
            // カメラプレビュー
            CameraPreviewView(session: scanner.session)
                .ignoresSafeArea()

            // スキャンガイドオーバーレイ
            ScannerOverlayView()

            // コントロールボタン
            VStack {
                Spacer()
                HStack {
                    // フラッシュライトボタン
                    Button(action: { scanner.toggleTorch() }) {
                        Image(systemName: scanner.isTorchOn
                            ? "bolt.fill" : "bolt.slash.fill")
                            .foregroundColor(.white)
                            .padding(12)
                            .background(.ultraThinMaterial)
                            .clipShape(Circle())
                    }
                    Spacer()
                }
                .padding()
            }
        }
        .onAppear { scanner.startScanning() }
        .onDisappear { scanner.stopScanning() }
        .onChange(of: scanner.scannedValue) { value in
            if let value = value {
                scannedCode = value
                showResult = true
            }
        }
        .sheet(isPresented: $showResult) {
            ScanResultView(code: scannedCode) {
                scanner.resetScan()
                showResult = false
            }
        }
    }
}

// QRスキャナーのViewModel
class QRScannerViewModel: NSObject, ObservableObject, AVCaptureMetadataOutputObjectsDelegate {
    @Published var scannedValue: String?
    @Published var isTorchOn = false

    let session = AVCaptureSession()
    private let metadataOutput = AVCaptureMetadataOutput()

    func startScanning() {
        checkCameraPermission()
    }

    // QRコード検出時のコールバック
    func metadataOutput(_ output: AVCaptureMetadataOutput,
                        didOutput metadataObjects: [AVMetadataObject],
                        from connection: AVCaptureConnection) {
        guard let metadata = metadataObjects.first as? AVMetadataMachineReadableCodeObject,
              let value = metadata.stringValue else { return }

        // 触覚フィードバック
        AudioServicesPlaySystemSound(SystemSoundID(kSystemSoundID_Vibrate))

        DispatchQueue.main.async {
            self.scannedValue = value
            self.session.stopRunning() // スキャン停止（重複防止）
        }
    }
}
```

<Callout type="info">
`Info.plist` に `NSCameraUsageDescription` キーを追加する必要があります。Rork Maxは自動的にこの設定を追加します。
</Callout>

## Step 2: QRコード生成機能の実装

スキャンだけでなく、アプリ内でQRコードを生成して表示・共有する機能も重要です。

### Rork Maxへのプロンプト

```
アプリにQRコード生成機能を追加してください。
要件：
- テキストフィールドで任意の文字列を入力
- CoreImageフィルターを使ってQRコードを生成・表示
- 生成したQRコードを写真ライブラリに保存
- UIActivityViewControllerでシェア機能
- QRコードのサイズを256x256ピクセルで生成
```

### 生成されるQRコード生成ロジック

```swift
import CoreImage
import UIKit

// QRコード生成ユーティリティ
struct QRCodeGenerator {
    /// 文字列からQRコード画像を生成する
    /// - Parameters:
    ///   - string: QRコードに埋め込む文字列
    ///   - size: 出力画像のサイズ（デフォルト: 256x256）
    /// - Returns: 生成されたQRコードのUIImage（失敗時はnil）
    static func generate(from string: String,
                         size: CGSize = CGSize(width: 256, height: 256)) -> UIImage? {
        guard let data = string.data(using: .utf8) else { return nil }

        // CIQRCodeGeneratorフィルターでQRコードを生成
        let filter = CIFilter(name: "CIQRCodeGenerator")
        filter?.setValue(data, forKey: "inputMessage")
        filter?.setValue("M", forKey: "inputCorrectionLevel") // 中程度の誤り訂正

        guard let ciImage = filter?.outputImage else { return nil }

        // スケーリング（高解像度化）
        let scaleX = size.width / ciImage.extent.size.width
        let scaleY = size.height / ciImage.extent.size.height
        let scaledImage = ciImage.transformed(by: CGAffineTransform(
            scaleX: scaleX, y: scaleY))

        // CIImageをUIImageに変換
        let context = CIContext()
        guard let cgImage = context.createCGImage(
            scaledImage, from: scaledImage.extent) else { return nil }

        return UIImage(cgImage: cgImage)
    }
}

// 期待出力: 256x256ピクセルの白黒QRコード画像
// 入力例: "https://rorklab.net" → URLをエンコードしたQRコード
```

### SwiftUIでの表示

```swift
struct QRGeneratorView: View {
    @State private var inputText = ""
    @State private var qrImage: UIImage?

    var body: some View {
        VStack(spacing: 24) {
            TextField("テキストまたはURLを入力", text: $inputText)
                .textFieldStyle(.roundedBorder)
                .padding(.horizontal)
                .onChange(of: inputText) { text in
                    // リアルタイムにQRコードを更新
                    qrImage = QRCodeGenerator.generate(from: text)
                }

            if let qrImage = qrImage {
                Image(uiImage: qrImage)
                    .resizable()
                    .interpolation(.none) // ピクセル補間なし（シャープな表示）
                    .frame(width: 256, height: 256)
                    .padding()
                    .background(Color.white)
                    .cornerRadius(12)
                    .shadow(radius: 8)

                // 共有ボタン
                ShareLink(item: Image(uiImage: qrImage),
                          preview: SharePreview("QRコード",
                                               image: Image(uiImage: qrImage))) {
                    Label("QRコードを共有", systemImage: "square.and.arrow.up")
                        .frame(maxWidth: .infinity)
                        .padding()
                        .background(Color.blue)
                        .foregroundColor(.white)
                        .cornerRadius(10)
                        .padding(.horizontal)
                }
            }
        }
        .navigationTitle("QRコード生成")
    }
}
```

<Callout type="tip">
`.interpolation(.none)` を指定することで、QRコードの小さなピクセルがぼやけずにシャープに表示されます。
</Callout>

## Step 3: スキャン履歴の管理

スキャンしたQRコードの履歴を保存・管理する機能を実装します。

```swift
// スキャン履歴の管理（UserDefaults使用）
class ScanHistoryManager: ObservableObject {
    private let maxHistoryCount = 20
    private let storageKey = "qr_scan_history"

    @Published var history: [ScanRecord] = []

    struct ScanRecord: Codable, Identifiable {
        let id: UUID
        let value: String       // スキャンした文字列
        let scannedAt: Date     // スキャン日時
        var type: CodeType      // コードの種類

        enum CodeType: String, Codable {
            case url, text, email, phone
        }
    }

    init() {
        loadHistory()
    }

    /// 新しいスキャン結果を履歴に追加
    func add(_ value: String) {
        let record = ScanRecord(
            id: UUID(),
            value: value,
            scannedAt: Date(),
            type: detectType(value)
        )

        history.insert(record, at: 0) // 最新を先頭に追加

        // 最大件数を超えた場合は古いものを削除
        if history.count > maxHistoryCount {
            history = Array(history.prefix(maxHistoryCount))
        }

        saveHistory()
    }

    private func detectType(_ value: String) -> ScanRecord.CodeType {
        if value.hasPrefix("http://") || value.hasPrefix("https://") {
            return .url
        } else if value.hasPrefix("mailto:") {
            return .email
        } else if value.hasPrefix("tel:") {
            return .phone
        }
        return .text
    }

    private func saveHistory() {
        if let encoded = try? JSONEncoder().encode(history) {
            UserDefaults.standard.set(encoded, forKey: storageKey)
        }
    }

    private func loadHistory() {
        guard let data = UserDefaults.standard.data(forKey: storageKey),
              let decoded = try? JSONDecoder().decode([ScanRecord].self, from: data) else {
            return
        }
        history = decoded
    }
}
```

```swift
// イベントチェックイン用のQRスキャナー活用例
struct EventCheckInView: View {
    @State private var checkInStatus: CheckInStatus = .waiting

    enum CheckInStatus {
        case waiting, success(String), failed(String)
    }

    var body: some View {
        VStack {
            // QRスキャナーを組み込む
            QRScannerView()
                .frame(height: 300)

            // チェックイン結果を表示
            switch checkInStatus {
            case .waiting:
                Text("チケットQRコードをスキャンしてください")
                    .foregroundColor(.secondary)
            case .success(let name):
                Label("\(name) さん、チェックイン完了！", systemImage: "checkmark.seal.fill")

            case .failed(let reason):
                Label(reason, systemImage: "xmark.seal.fill")
                    .foregroundColor(.red)
            }
        }
    }
}
```

```swift
// WiFi設定をQRコードで共有する
extension QRCodeGenerator {
    /// WiFiのQRコードを生成（Androidも読み取り可能な標準形式）
    static func generateWiFiQR(ssid: String,
                                password: String,
                                security: String = "WPA") -> UIImage? {
        // WiFi QRコードの標準フォーマット
        let wifiString = "WIFI:T:\(security);S:\(ssid);P:\(password);;"
        return generate(from: wifiString)
    }
}
```

## パフォーマンス最適化のヒント

QRコードスキャン機能を実装する際の注意点をまとめます：

**バッテリー消費の削減**: スキャン成功後や画面が非表示になった際は必ず `session.stopRunning()` を呼び出す。Rork Maxは `onDisappear` に自動的にこの処理を追加します。

**誤検知の防止**: 同じQRコードが連続して読み取られないよう、スキャン成功後に一定時間（0.5秒程度）のクールダウンを設ける。

**UI応答性の確保**: AVFoundationのコールバックはバックグラウンドスレッドで呼び出されます。UI更新は必ず `DispatchQueue.main.async` 内で行います。

## FAQ

**Q1. Rork（無料版）でもQRコードスキャナーは実装できますか？**

Rork（無料版）はExpo React Nativeベースのため、`expo-barcode-scanner` ライブラリを使用したQRスキャナーが実装できます。ただし、AVFoundationを直接使ったネイティブSwift実装はRork Max（ネイティブSwiftコード生成）専用です。

**Q2. QRコード以外のバーコード（JANコードなど）もスキャンできますか？**

はい、できます。Rork Maxへのプロンプトに「EAN-13、QRコード、Code128に対応したマルチフォーマットスキャナー」と指定することで、複数のバーコード形式に対応したスキャナーが生成されます。`metadataObjectTypes` プロパティに読み取り形式を追加するだけです。

**Q3. スキャンしたURLを自動的にSafariで開くことはできますか？**

はい。スキャン結果がURLの場合に自動的に `UIApplication.shared.open(url)` を呼び出す処理をRork Maxに追加してもらうことができます。ただし、自動オープンはユーザー体験上好ましくないケースもあるため、確認ダイアログを挟むのがベストプラクティスです。
