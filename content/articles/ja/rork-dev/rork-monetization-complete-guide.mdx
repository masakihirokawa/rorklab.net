## はじめに：Rork アプリの収益化戦略

Rork で構築したアプリを収益化する手段は主に3つあります：

1. **サブスクリプション** — 月額・年額の継続課金（LTV が高い）
2. **買い切り（One-Time Purchase）** — 一度払えば永久アクセス
3. **広告（AdMob）** — 無料ユーザーからの収益

本記事では、**RevenueCat を使ったサブスクリプション + 買い切り** の実装を中心に、ストア審査を通過するための実装パターンまで解説します。

<Callout type="info">
Rork は React Native ベースのため、RevenueCat の React Native SDK がそのまま利用できます。Apple Developer Program と Google Play Console のアカウントが必要です。
</Callout>

## 1. RevenueCat のセットアップ

### インストール

Rork の Prompt から RevenueCat を追加するよう指示します：

```
Add RevenueCat subscription to this app.
Install: react-native-purchases
Configure with API keys for iOS and Android
```

または直接パッケージを追加：

```bash
npm install react-native-purchases
```

### 初期設定（App.tsx）

```typescript
// App.tsx
import { useEffect } from 'react';
import Purchases, { LOG_LEVEL } from 'react-native-purchases';
import { Platform } from 'react-native';

const REVENUECAT_IOS_KEY = 'appl_xxxxxxxxxxxxxxxxxxxxxxxx';
const REVENUECAT_ANDROID_KEY = 'goog_xxxxxxxxxxxxxxxxxxxxxxxx';

export default function App() {
  useEffect(() => {
    const initRevenueCat = async () => {
      Purchases.setLogLevel(LOG_LEVEL.DEBUG); // 開発時のみ

      if (Platform.OS === 'ios') {
        await Purchases.configure({ apiKey: REVENUECAT_IOS_KEY });
      } else {
        await Purchases.configure({ apiKey: REVENUECAT_ANDROID_KEY });
      }
    };

    initRevenueCat();
  }, []);

  return <RootNavigator />;
}
```

## 2. 購入状態の管理（Context + Hook）

```typescript
// contexts/PurchaseContext.tsx
import React, { createContext, useContext, useState, useEffect, useCallback } from 'react';
import Purchases, {
  CustomerInfo,
  PurchasesOffering,
  PurchasesPackage,
} from 'react-native-purchases';

interface PurchaseContextType {
  customerInfo: CustomerInfo | null;
  currentOffering: PurchasesOffering | null;
  isPro: boolean;
  isPremium: boolean;
  isLoading: boolean;
  purchasePackage: (pkg: PurchasesPackage) => Promise<boolean>;
  restorePurchases: () => Promise<void>;
  refreshCustomerInfo: () => Promise<void>;
}

const PurchaseContext = createContext<PurchaseContextType | null>(null);

export function PurchaseProvider({ children }: { children: React.ReactNode }) {
  const [customerInfo, setCustomerInfo] = useState<CustomerInfo | null>(null);
  const [currentOffering, setCurrentOffering] = useState<PurchasesOffering | null>(null);
  const [isLoading, setIsLoading] = useState(true);

  // エンタイトルメント（権限）の確認
  const isPro = Boolean(
    customerInfo?.entitlements.active['pro']
  );
  const isPremium = Boolean(
    customerInfo?.entitlements.active['premium']
  );

  useEffect(() => {
    initPurchases();
    // 購入状態の変更をリッスン
    const listener = Purchases.addCustomerInfoUpdateListener(setCustomerInfo);
    return () => listener.remove();
  }, []);

  const initPurchases = async () => {
    try {
      const [info, offerings] = await Promise.all([
        Purchases.getCustomerInfo(),
        Purchases.getOfferings(),
      ]);
      setCustomerInfo(info);
      setCurrentOffering(offerings.current);
    } catch (error) {
      console.error('RevenueCat 初期化エラー:', error);
    } finally {
      setIsLoading(false);
    }
  };

  const purchasePackage = useCallback(async (pkg: PurchasesPackage): Promise<boolean> => {
    try {
      const { customerInfo } = await Purchases.purchasePackage(pkg);
      setCustomerInfo(customerInfo);
      return true;
    } catch (error: any) {
      if (error.userCancelled) return false;
      throw error;
    }
  }, []);

  const restorePurchases = useCallback(async () => {
    const info = await Purchases.restorePurchases();
    setCustomerInfo(info);
  }, []);

  const refreshCustomerInfo = useCallback(async () => {
    const info = await Purchases.getCustomerInfo();
    setCustomerInfo(info);
  }, []);

  return (
    <PurchaseContext.Provider value={{
      customerInfo,
      currentOffering,
      isPro,
      isPremium,
      isLoading,
      purchasePackage,
      restorePurchases,
      refreshCustomerInfo,
    }}>
      {children}
    </PurchaseContext.Provider>
  );
}

export function usePurchase() {
  const ctx = useContext(PurchaseContext);
  if (!ctx) throw new Error('usePurchase は PurchaseProvider 内で使用してください');
  return ctx;
}
```

## 3. ペイウォール UI の実装

```typescript
// components/Paywall.tsx
import React, { useState } from 'react';
import {
  View,
  Text,
  TouchableOpacity,
  StyleSheet,
  ActivityIndicator,
  Alert,
  ScrollView,
} from 'react-native';
import { PurchasesPackage } from 'react-native-purchases';
import { usePurchase } from '../contexts/PurchaseContext';

interface PaywallProps {
  onDismiss: () => void;
  onSuccess: () => void;
}

export function Paywall({ onDismiss, onSuccess }: PaywallProps) {
  const { currentOffering, purchasePackage, restorePurchases, isLoading } = usePurchase();
  const [purchasing, setPurchasing] = useState<string | null>(null);
  const [restoring, setRestoring] = useState(false);

  const packages = currentOffering?.availablePackages ?? [];

  // RevenueCat の Offering からパッケージを種別で取得
  const monthlyPkg = packages.find(p => p.packageType === 'MONTHLY');
  const annualPkg = packages.find(p => p.packageType === 'ANNUAL');
  const lifetimePkg = packages.find(p => p.packageType === 'LIFETIME');

  const handlePurchase = async (pkg: PurchasesPackage) => {
    setPurchasing(pkg.identifier);
    try {
      const success = await purchasePackage(pkg);
      if (success) {
        onSuccess();
      }
    } catch (error: any) {
      Alert.alert(
        '購入エラー',
        error.message || '購入処理中にエラーが発生しました。もう一度お試しください。',
      );
    } finally {
      setPurchasing(null);
    }
  };

  const handleRestore = async () => {
    setRestoring(true);
    try {
      await restorePurchases();
      Alert.alert('復元完了', '購入の復元が完了しました。');
      onDismiss();
    } catch {
      Alert.alert('復元エラー', '購入の復元に失敗しました。');
    } finally {
      setRestoring(false);
    }
  };

  if (isLoading) {
    return (
      <View style={styles.loadingContainer}>
        <ActivityIndicator size="large" color="#4CE8A0" />
      </View>
    );
  }

  return (
    <ScrollView contentContainerStyle={styles.container}>
      <Text style={styles.title}>プレミアムにアップグレード</Text>
      <Text style={styles.subtitle}>
        すべての機能を無制限でお使いいただけます
      </Text>

      {/* 特典リスト */}
      <View style={styles.benefitsContainer}>
        {BENEFITS.map((benefit, idx) => (
          <View key={idx} style={styles.benefitRow}>
            <Text style={styles.benefitIcon}>✓</Text>
            <Text style={styles.benefitText}>{benefit}</Text>
          </View>
        ))}
      </View>

      {/* プランボタン */}
      {annualPkg && (
        <PurchaseButton
          pkg={annualPkg}
          label={`年額プラン — ${annualPkg.product.priceString}/年`}
          badge="おすすめ"
          isPurchasing={purchasing === annualPkg.identifier}
          onPress={() => handlePurchase(annualPkg)}
        />
      )}
      {monthlyPkg && (
        <PurchaseButton
          pkg={monthlyPkg}
          label={`月額プラン — ${monthlyPkg.product.priceString}/月`}
          isPurchasing={purchasing === monthlyPkg.identifier}
          onPress={() => handlePurchase(monthlyPkg)}
        />
      )}
      {lifetimePkg && (
        <PurchaseButton
          pkg={lifetimePkg}
          label={`永久プラン — ${lifetimePkg.product.priceString} 一括`}
          isPurchasing={purchasing === lifetimePkg.identifier}
          onPress={() => handlePurchase(lifetimePkg)}
        />
      )}

      {/* 復元・規約 */}
      <TouchableOpacity onPress={handleRestore} disabled={restoring}>
        <Text style={styles.restoreText}>
          {restoring ? '復元中...' : '以前の購入を復元'}
        </Text>
      </TouchableOpacity>

      <Text style={styles.legalText}>
        購入はiTunes/Google Playアカウントに請求されます。{'\n'}
        サブスクリプションは期間終了の24時間前までに自動更新されます。
      </Text>
    </ScrollView>
  );
}

function PurchaseButton({
  label,
  badge,
  isPurchasing,
  onPress,
}: {
  pkg: PurchasesPackage;
  label: string;
  badge?: string;
  isPurchasing: boolean;
  onPress: () => void;
}) {
  return (
    <TouchableOpacity
      style={[styles.purchaseButton, badge && styles.featuredButton]}
      onPress={onPress}
      disabled={isPurchasing}
    >
      {badge && <Text style={styles.badge}>{badge}</Text>}
      {isPurchasing ? (
        <ActivityIndicator color="#fff" />
      ) : (
        <Text style={styles.purchaseButtonText}>{label}</Text>
      )}
    </TouchableOpacity>
  );
}

const BENEFITS = [
  'すべてのプレミアム機能が使い放題',
  '広告なし',
  'クラウド同期（無制限デバイス）',
  '優先カスタマーサポート',
];

const styles = StyleSheet.create({
  container: { padding: 24, paddingBottom: 48 },
  loadingContainer: { flex: 1, alignItems: 'center', justifyContent: 'center' },
  title: { fontSize: 24, fontWeight: '700', color: '#fff', textAlign: 'center', marginBottom: 8 },
  subtitle: { fontSize: 15, color: '#aaa', textAlign: 'center', marginBottom: 24 },
  benefitsContainer: { marginBottom: 28 },
  benefitRow: { flexDirection: 'row', alignItems: 'center', marginBottom: 10 },
  benefitIcon: { color: '#4CE8A0', fontSize: 16, marginRight: 10, fontWeight: '700' },
  benefitText: { color: '#ddd', fontSize: 14 },
  purchaseButton: {
    backgroundColor: '#1e1e1e',
    borderWidth: 1,
    borderColor: '#4CE8A020',
    borderRadius: 12,
    padding: 16,
    marginBottom: 12,
    alignItems: 'center',
  },
  featuredButton: { borderColor: '#4CE8A0', backgroundColor: '#4CE8A010' },
  purchaseButtonText: { color: '#fff', fontSize: 15, fontWeight: '600' },
  badge: {
    backgroundColor: '#4CE8A0',
    color: '#000',
    fontSize: 11,
    fontWeight: '700',
    paddingHorizontal: 8,
    paddingVertical: 2,
    borderRadius: 10,
    marginBottom: 6,
  },
  restoreText: { color: '#4CE8A0', fontSize: 13, textAlign: 'center', marginTop: 8 },
  legalText: { color: '#555', fontSize: 11, textAlign: 'center', marginTop: 16, lineHeight: 16 },
});
```

## まとめ

Rork アプリの収益化は RevenueCat を使うことで、iOS/Android の課金処理の複雑さを大幅に抽象化できます。重要なのは以下の3点です：

1. **エンタイトルメント管理** — RevenueCat のエンタイトルメントで機能解放を制御
2. **購入復元** — Apple の必須要件。必ずペイウォールに実装
3. **ストア審査対策** — キャンセルポリシー・自動更新情報の明示

RevenueCat のダッシュボードでリアルタイムの収益・解約率・LTV を追跡しながら、プランの価格とコンテンツを最適化していきましょう。
