# Rork Maxでのディープリンク・ユニバーサルリンク実装

ディープリンクは現代的なアプリ・グロースの礎です。適切なディープリンク実装がないと、莫大なポテンシャルを失っています。広告から流入するユーザーは正しいコンテンツにたどり着けず、紹介プログラムはソースを追跡できず、シームレスな複数プラットフォーム体験は不可能です。

Rork Maxがネイティブ・Swiftコードを生成することで、Webとアプリにまたがってシームレスに動作する高度なディープリンクおよびユニバーサルリンク・システムを実装できます。このガイドでは、基本的な実装から高度なグロース帰属パターンまでをカバーします。

## グロースにおけるディープリンクの重要性

ディープリンクは以下を可能にします。

- **シームレスなユーザー体験**: ユーザーはアプリ・ホームではなく、関連コンテンツに直接ランディング
- **正確な帰属追跡**: どのキャンペーンまたはチャネルが各ユーザーをもたらしたかを追跡
- **高いコンバージョン率**: 正しい画面にランディングするユーザーのコンバージョン率は約5倍向上
- **紹介プログラム**: ウイルス的グロースのためのビルトイン・メカニズム
- **クロス・プラットフォーム一貫性**: 同じリンクがWebとアプリ内ブラウザとネイティブ・アプリで動作

<Callout type="tip">
ディープリンクを実装したアプリはユーザー・エンゲージメントが30%向上し、リテンション率が4倍向上するという調査結果があります。さらに重要なのは、ディープリンクにより帰属追跡が可能になることです。これはIDFA廃止後のマーケティングROI測定に不可欠です。
</Callout>

## Rork Maxアプリのディープリンク・アーキテクチャ

### 3つの構成要素

適切なディープリンク・アーキテクチャには3つのレイヤーの調整が必要です。

1. **URLスキーム・ハンドラー**: ディープリンクURLをインターセプト
2. **ルートパーサーとナビゲーター**: URLをパースし正しい画面に移動
3. **ペイロード・マネージャー**: ディープリンク・データを画面に渡す

### 実装戦略

```swift
// URLスキーム定義
enum DeepLinkScheme {
    case native(NativeDeepLink)
    case web(WebDeepLink)
    case attributionTracking(AttributionData)

    static func parse(_ url: URL) -> DeepLinkScheme? {
        if url.scheme == "myapp" {
            return parseNativeDeepLink(url)
        } else if url.host == "myapp.com" {
            return parseWebDeepLink(url)
        }
        return nil
    }

    private static func parseNativeDeepLink(_ url: URL) -> DeepLinkScheme? {
        let components = URLComponents(url: url, resolvingAgainstBaseURL: true)
        guard let host = components?.host else { return nil }

        switch host {
        case "product":
            if let productId = components?.queryItems?.first(where: { $0.name == "id" })?.value {
                return .native(.product(id: productId))
            }
        case "user":
            if let username = components?.queryItems?.first(where: { $0.name == "username" })?.value {
                return .native(.user(username: username))
            }
        case "invite":
            if let inviteCode = components?.queryItems?.first(where: { $0.name == "code" })?.value {
                return .native(.invite(code: inviteCode))
            }
        default:
            return nil
        }
        return nil
    }
}

enum NativeDeepLink {
    case product(id: String)
    case user(username: String)
    case invite(code: String)
    case home
}

enum WebDeepLink {
    case product(slug: String)
    case article(slug: String)
    case profile(username: String)
}

// ディープリンク・ナビゲーション・ハンドラー
@main
struct MyApp: App {
    @StateObject private var deepLinkRouter = DeepLinkRouter()

    var body: some Scene {
        WindowGroup {
            ContentView()
                .environmentObject(deepLinkRouter)
                .onOpenURL { url in
                    handleDeepLink(url)
                }
                .onContinueUserActivity(NSUserActivityTypeBrowsingWeb) { userActivity in
                    if let url = userActivity.webpageURL {
                        handleDeepLink(url)
                    }
                }
        }
    }

    private func handleDeepLink(_ url: URL) {
        // ディープリンクをパースしてルーティング
        if let scheme = DeepLinkScheme.parse(url) {
            deepLinkRouter.navigate(to: scheme)
        }
    }
}
```

## ユニバーサルリンク — シームレスなWebからアプリへのディープリンク

ユニバーサルリンクは、WebサイトのURLをアプリで直接開く機能です。ユーザーがアプリをインストールしていない場合、URLはブラウザで開きます。

### ユニバーサルリンクのセットアップ

ユニバーサルリンクには2つの要素が必要です。

1. **apple-app-site-association ファイル** をドメイン上に配置
2. **Entitlements設定** をアプリで設定

```swift
// ステップ1：apple-app-site-associationファイルを作成
// 配置先: https://yourdomain.com/.well-known/apple-app-site-association

{
  "applinks": {
    "apps": [],
    "details": [
      {
        "appID": "com.example.myapp",
        "paths": [
          "/products/*",
          "/users/*",
          "/articles/*"
        ]
      }
    ]
  },
  "webcredentials": {
    "apps": ["com.example.myapp"]
  }
}

// ステップ2：Rork Maxでentitlementsを設定
// プロンプトで以下をリクエスト：
// "ユニバーサルリンク向けのassociated domainsを追加してください：
//  yourdomain.com, www.yourdomain.com"

// ステップ3：SwiftUIでユニバーサルリンクを処理
class UniversalLinkHandler {
    func parseUniversalLink(_ url: URL) -> DeepLinkScheme? {
        // 末尾のスラッシュを削除して一貫したパースを実現
        var urlPath = url.path
        while urlPath.hasSuffix("/") {
            urlPath.removeLast()
        }

        // ルート・パターンにマッチさせる
        if urlPath.starts(with: "/products/") {
            let productSlug = String(urlPath.dropFirst("/products/".count))
            return .native(.product(id: productSlug))
        }

        if urlPath.starts(with: "/users/") {
            let username = String(urlPath.dropFirst("/users/".count))
            return .native(.user(username: username))
        }

        if urlPath.starts(with: "/articles/") {
            let articleSlug = String(urlPath.dropFirst("/articles/".count))
            // スラッグから記事IDを取得する必要があるかもしれません
            return .native(.article(slug: articleSlug))
        }

        return nil
    }
}
```

<Callout type="warning">
ユニバーサルリンクはAppleとドメイン間の暗号化されたハンドシェイクで検証されます。テストには以下が必要です。

1. ドメインはHTTPSである必要があります
2. apple-app-site-associationファイルは`/.well-known/apple-app-site-association`で公開アクセス可能である必要があります
3. associated domains entitlementを持つ適切なプロビジョニング・プロファイルが必要です

開発中は、`xcode-select --install`でユニバーサルリンク検証を無効化し、App Clipsを無効にした実機でテストしてください。
</Callout>

```swift
            navigateTo(deepLink)
        } else {

            UIApplication.shared.open(url)
        }
```

## 遅延ディープリンク — グロースの究極ツール

遅延ディープリンクにより、ユーザーがアプリをインストールしていなくても、ディープリンクが機能します。ユーザーがストアからアプリをインストールして初めて起動すると、意図した画面にルーティングされます。

<Callout type="tip">
遅延ディープリンクは紹介プログラムとウイルス・キャンペーンに変革をもたらします。ユーザーは特定の商品へのリンクを共有でき、友人はインストール後、その商品に直接ランディングします。サーバー・ロジック不要です。
</Callout>

### 帰属追跡を伴う遅延ディープリンク実装

```swift
// 帰属追跡を伴う遅延ディープリンク・サービス
class DeferredDeepLinkService {
    private let apiBaseURL = URL(string: "https://api.yourdomain.com")!
    private let deviceIdentifier = UIDevice.current.identifierForVendor?.uuidString ?? UUID().uuidString

    // 最初のアプリ起動時に遅延ディープリンクを保存
    func fetchDeferredDeepLink() async throws -> DeferredDeepLinkData? {
        var request = URLRequest(url: apiBaseURL.appendingPathComponent("/deferred-links/fetch"))
        request.httpMethod = "POST"

        let payload = DeferredLinkRequest(
            deviceId: deviceIdentifier,
            appVersion: Bundle.main.appVersion,
            osVersion: UIDevice.current.systemVersion,
            isFirstLaunch: !UserDefaults.standard.bool(forKey: "hasLaunchedBefore")
        )

        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.httpBody = try JSONEncoder().encode(payload)

        let (data, _) = try await URLSession.shared.data(for: request)
        let decoder = JSONDecoder()
        let response = try decoder.decode(DeferredLinkResponse.self, from: data)

        return response.deepLinkData
    }

    // 帰属データをログ
    func logAttribution(source: String, campaign: String, medium: String) async throws {
        var request = URLRequest(url: apiBaseURL.appendingPathComponent("/attribution"))
        request.httpMethod = "POST"

        let attribution = AttributionEvent(
            deviceId: deviceIdentifier,
            source: source,
            campaign: campaign,
            medium: medium,
            timestamp: Date()
        )

        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.httpBody = try JSONEncoder().encode(attribution)

        _ = try await URLSession.shared.data(for: request)
    }
}

struct DeferredLinkRequest: Codable {
    let deviceId: String
    let appVersion: String
    let osVersion: String
    let isFirstLaunch: Bool
}

struct DeferredDeepLinkData: Codable {
    let deepLink: String
    let source: String // "referral", "ad_campaign"など
    let campaign: String
    let userId: String? // リンクを共有したユーザーのID
}

struct AttributionEvent: Codable {
    let deviceId: String
    let source: String
    let campaign: String
    let medium: String
    let timestamp: Date
}
```

### 起動時の遅延ディープリンク処理

```swift
class AppStartupManager {
    @StateObject private var deferredLinkService = DeferredDeepLinkService()

    func setupAppLaunchFlow() async {
        // アプリ起動をマーク
        if !UserDefaults.standard.bool(forKey: "hasLaunchedBefore") {
            UserDefaults.standard.set(true, forKey: "hasLaunchedBefore")

            // 最初の起動時に遅延ディープリンクを取得
            do {
                if let deferredLink = try await deferredLinkService.fetchDeferredDeepLink() {
                    // 帰属をログ
                    try await deferredLinkService.logAttribution(
                        source: deferredLink.source,
                        campaign: deferredLink.campaign,
                        medium: "deferred_deep_link"
                    )

                    // ディープリンクにルーティング
                    handleDeepLinkString(deferredLink.deepLink)

                    // オプション：紹介関係を保存
                    if let referrerId = deferredLink.userId {
                        UserDefaults.standard.set(referrerId, forKey: "referredBy")
                    }
                }
            } catch {
                // エラーをログしても、アプリ起動をブロックしない
                print("遅延ディープリンク取得失敗: \(error)")
            }
        }
    }

    private func handleDeepLinkString(_ deepLinkString: String) {
        guard let url = URL(string: deepLinkString) else { return }
        if let scheme = DeepLinkScheme.parse(url) {
            // 適切な画面にナビゲート
            navigateToDeepLink(scheme)
        }
    }
}
```

```swift
class AttributionTracker {
    struct Campaign {
```

### UTMパラメーター・パース

```swift
class UTMParameterParser {
    struct UTMParameters: Codable {
        let source: String?      // utm_source
        let medium: String?      // utm_medium
        let campaign: String?    // utm_campaign
        let content: String?     // utm_content
        let term: String?        // utm_term
    }

    static func extractUTM(from url: URL) -> UTMParameters {
        let components = URLComponents(url: url, resolvingAgainstBaseURL: true)
        let queryItems = components?.queryItems ?? []

        return UTMParameters(
            source: queryItems.first(where: { $0.name == "utm_source" })?.value,
            medium: queryItems.first(where: { $0.name == "utm_medium" })?.value,
            campaign: queryItems.first(where: { $0.name == "utm_campaign" })?.value,
            content: queryItems.first(where: { $0.name == "utm_content" })?.value,
            term: queryItems.first(where: { $0.name == "utm_term" })?.value
        )
    }

    static func createDeepLinkWithUTM(
        baseDeepLink: String,
        utm: UTMParameters
    ) -> String? {
        guard var components = URLComponents(string: baseDeepLink) else {
            return nil
        }

        var queryItems = components.queryItems ?? []

        if let source = utm.source {
            queryItems.append(URLQueryItem(name: "utm_source", value: source))
        }
        if let medium = utm.medium {
            queryItems.append(URLQueryItem(name: "utm_medium", value: medium))
        }
        if let campaign = utm.campaign {
            queryItems.append(URLQueryItem(name: "utm_campaign", value: campaign))
        }
        if let content = utm.content {
            queryItems.append(URLQueryItem(name: "utm_content", value: content))
        }
        if let term = utm.term {
            queryItems.append(URLQueryItem(name: "utm_term", value: term))
        }

        components.queryItems = queryItems
        return components.url?.absoluteString
    }
}
```

```swift
class ReferralProgram {
    let baseDeepLinkURL = "myapp://invite"

    func generateReferralLink(for userId: String, withReward reward: Double) -> URL? {
        let referralCode = generateCode(userId: userId, reward: reward)

        var components = URLComponents(string: baseDeepLinkURL)
        components?.queryItems = [
            URLQueryItem(name: "code", value: referralCode),
            URLQueryItem(name: "referrer_id", value: userId)
        ]

        return components?.url
    }

    // ユニバーサルリンク版も生成
    func generateUniversalReferralLink(for userId: String, withReward reward: Double) -> URL? {
        let referralCode = generateCode(userId: userId, reward: reward)

        components?.queryItems = [
            URLQueryItem(name: "code", value: referralCode),
            URLQueryItem(name: "referrer_id", value: userId)
        ]

        return components?.url
    }

    private func generateCode(userId: String, reward: Double) -> String {
        let timestamp = Int(Date().timeIntervalSince1970)
        let data = "\(userId):\(timestamp):\(reward)"
        return data.base64Encoded()
    }
}
```

```swift
class ReferralRewardService {
    private let apiURL = URL(string: "https://api.yourdomain.com")!

    // 紹介コードを検証し両者にリワード
    func processReferral(
        code: String,
        refereeId: String
    ) async throws -> ReferralResult {
        var request = URLRequest(url: apiURL.appendingPathComponent("/referrals/validate"))
        request.httpMethod = "POST"

        let payload = ReferralValidation(code: code, refereeId: refereeId)
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.httpBody = try JSONEncoder().encode(payload)

        let (data, response) = try await URLSession.shared.data(for: request)

        guard let httpResponse = response as? HTTPURLResponse,
              httpResponse.statusCode == 200 else {
            throw ReferralError.invalidCode
        }

        let result = try JSONDecoder().decode(ReferralResult.self, from: data)
        return result
    }
}

    let refereeReward: Double
    let referrerReward: Double
    let referrerBalance: Double

    case alreadyUsed
    case expired
```

```swift
class SecureDeepLinkValidator {
    private let validHosts = ["myapp.com", "www.myapp.com"]

    func validateDeepLink(_ url: URL) -> Bool {
        // Webリンクはhttpsのみ受け入れ
        if let scheme = url.scheme, scheme != "https" {
            return false
        }

        // ホストを検証
        if let host = url.host {
            guard validHosts.contains(host) else { return false }
        }

        // URL構造を検証
        let components = URLComponents(url: url, resolvingAgainstBaseURL: true)
        guard let path = components?.path else { return false }

        // 有効なパスをホワイトリスト
        let validPaths = ["/products", "/users", "/articles", "/invite"]
        let isValidPath = validPaths.contains { path.starts(with: $0) }

        return isValidPath
    }

    // 紹介コードを暗号的に検証
    func validateReferralCode(_ code: String) -> Bool {
        // HMAC署名を検証
        guard let decodedCode = Data(base64Encoded: code) else { return false }

        let signature = decodedCode.prefix(32)
        let payload = decodedCode.suffix(from: 32)

        let expectedSignature = HMAC(
            key: "your-secret-key".data(using: .utf8)!,
            data: payload,
            algorithm: .sha256
        ).digest

        return signature == expectedSignature
    }
}
```

## まとめ

ディープリンクとユニバーサルリンクは現代的なアプリ・グロースに必須です。Rork Maxがネイティブ・Swiftコードを生成することで、以下のシステムを実装できます。

1. **ユーザーを正確にルーティング** して期待するコンテンツに案内
2. **帰属を正確に追跡** して全マーケティング・チャネルでROI測定
3. **遅延ディープリンク** で紹介プログラムとウイルス・キャンペーンを有効化

- ユーザー・エンゲージメントとリテンション向上

- ユーザー紹介とグロース・メカニズムのビルトイン
- WebとネイティブのクロスプラットフォームX一貫性

基本的なディープリンク実装から始め、アプリが成長するにしたがってユニバーサルリンク、遅延ディープリンク、帰属追跡を段階的に追加してください。このガイドの例を使えば、Rork Maxでグロース重視のアプリを構築するのに必要なすべてを持っています。
