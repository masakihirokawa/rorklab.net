## Rork 開発を「一人チーム」に進化させる

Rork でアプリを作り、App Store に公開する——この流れは AI のおかげでかつてないほどシンプルになりました。しかし「アプリ本体」以外にも、開発者には多くの作業が伴います。

- アプリの説明文や ASO（App Store 最適化）テキストの生成
- スクリーンショット用のデザイン作成
- ドキュメントや PDF の自動生成
- SEO 対策コンテンツの執筆

これらを手動でこなすには時間がかかります。**skills.sh** は、こうした周辺作業を AI エージェントに委任するためのスキルマーケットプレイスです。Rork と組み合わせることで、アプリ開発全体を「一人チーム」で回せるようになります。

---

## skills.sh とは

**skills.sh** は、Claude Code（Anthropic の AI コーディングツール）向けの**オープンソーススキルマーケットプレイス**です。コミュニティが開発した AI エージェントスキルを検索・インストールして使用できます。

主なスキルカテゴリ：
- **フロントエンドデザイン**: React/HTML コンポーネントの生成
- **SEO 監査**: サイトの SEO 問題を自動検出
- **PDF 処理**: PDF の生成・編集・フォーム入力
- **スプレッドシート**: Excel/CSV のデータ処理
- **プレゼンテーション**: PowerPoint の自動生成

> **注意**: skills.sh のスキルは Claude Code（CLI ツール）を通じて使用します。Rork は Claude Code の技術基盤を共有しているため、Rork Max モードや Claude Code と Rork を組み合わせたワークフローで活用できます。

## インストール方法

### Claude Code を使ったインストール

```bash
# スキルのインストール
claude install skills.sh/frontend-design
claude install skills.sh/seo-audit
claude install skills.sh/pdf
claude install skills.sh/pptx
```

### インストール済みスキルの確認

```bash
claude skills list
```

## Rork 開発で活用できるスキル TOP 5

### 1. frontend-design — UI コンポーネントの雛形生成

Rork でアプリの大枠を作った後、個別の UI コンポーネントをさらに磨きたいときに使います。

```
@frontend-design ダッシュボード画面のカードコンポーネントを作成してください。
- データ：売上金額、前月比、折れ線グラフ
- スタイル：ダークモード対応、角丸、グラデーション
- React Native (NativeWind) 形式で出力
```

### 2. seo-audit — ランディングページの SEO 確認

App Store の外に Web ランディングページを持っている場合、SEO スコアを自動チェックできます。

```
@seo-audit https://myapp.example.com のSEOを監査してください。
モバイルスコア、OGP設定、ページ速度の問題点をリストアップして。
```

### 3. pdf — 利用規約・プライバシーポリシーの PDF 化

アプリ公開に必要な法的文書を PDF として自動生成できます。

```
@pdf 以下のプライバシーポリシー文章を、
会社名「Dolice」、アプリ名「FitTrack」として
A4 縦形式の PDF に整形して出力してください。

[プライバシーポリシーのテキスト]
```

### 4. pptx — 投資家・クライアント向けピッチデッキ

アプリの事業計画や機能紹介のスライドを自動生成します。

```
@pptx フィットネスアプリ「FitTrack」の投資家向けピッチデッキを作成してください。
10スライド構成：課題、解決策、市場規模、機能紹介、収益モデル、チーム、ロードマップ
```

### 5. xlsx — ASO データの分析・管理

キーワードランキングや競合分析のデータを Excel で整理します。

```
@xlsx App Store のキーワードランキングデータを分析してください。
添付の CSV から：キーワード別の順位変動グラフ、
競合との比較表、改善優先度マトリクスを作成して。
```

## 実践ワークフロー: アプリリリースの自動化

### Week 1: アプリ開発（Rork）

```
Rork Max でフィットネストラッキングアプリを作成する。
機能：ワークアウト記録、進捗グラフ、通知リマインダー
```

### Week 2: リリース準備（Claude Code + skills.sh）

**App Store 説明文の生成:**
```bash
# Claude Code で実行
claude "このアプリの App Store 向け説明文を日本語・英語で生成してください。
キーワード：フィットネス、ワークアウト、健康管理、習慣化
文字数：4000文字（日本語）、4000文字（英語）"
```

**スクリーンショット用デザイン:**
```
@frontend-design App Store のスクリーンショット用バナーを 5 種類作成してください。
サイズ: iPhone 15 Pro Max (1290×2796px)
デザイン: アプリの各主要画面を背景に、キャッチコピーを重ねる
```

**プライバシーポリシーの PDF 化:**
```
@pdf アプリのプライバシーポリシーを PDF で生成してください。
```

**ピッチデッキ作成:**
```
@pptx アプリの紹介資料（10スライド）を作成してください。
```

## skills.sh のカスタムスキル作成

自分だけのスキルを作り、繰り返し使うことも可能です。

### SKILL.md の基本構造

```markdown
# App Store Description Generator

このスキルは Rork で開発したアプリの App Store 説明文を自動生成します。

## 使い方
1. アプリの機能一覧を提供してください
2. ターゲットユーザーを指定してください
3. キーワード（ASO 用）を最大 10 個指定してください

## 出力
- 日本語説明文（4000文字以内）
- 英語説明文（4000文字以内）
- キーワードリスト（日本語・英語各 100 文字）
```

### スキルの公開

作ったスキルは GitHub リポジトリとして公開し、skills.sh のマーケットプレイスに登録できます。

## まとめ

skills.sh は Rork 開発のエコシステムを大幅に拡張します。

| 作業 | 従来 | skills.sh 活用後 |
|------|------|-----------------|
| App Store 説明文 | 1〜2時間 | 5分 |
| プライバシーポリシー PDF | 半日 | 10分 |
| ピッチデッキ | 1〜2日 | 30分 |
| SEO 監査 | 外注 or 専用ツール | 即座に無料 |

Rork でアプリを作り、skills.sh でリリース周りを自動化する——この組み合わせで、個人開発者でもプロフェッショナルなリリースプロセスが実現します。
