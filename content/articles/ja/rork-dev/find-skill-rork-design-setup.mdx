## はじめに

Rork でアプリを開発する際、最も時間がかかるのは **「セットアップ」** です。必要なデザインツール、テストツール、デプロイメント環境を手動で整備するのは、初心者にはハードルが高くなります。

そこで活躍するのが **「Find skill」** という機能です。Find skill は、あなたのプロジェクトに必要なスキルを自動検出し、ワンクリックでインストールできる革新的なツールです。

本記事では、Find skill の基本から、Rork 開発における実践的な活用法まで、**初心者でも理解できる形で**解説します。

## Find skill とは？

### 概要

Find skill は、以下の役割を果たします：

1. **プロジェクト分析**：あなたの Rork プロジェクトに最適なスキルを自動判定
2. **スキル推奨**：「このプロジェクトには、これらのスキルがおすすめです」と提案
3. **ワンクリックインストール**：推奨されたスキルを、まとめてインストール

### 従来の方法との違い

**従来（Find skill なし）:**

```
1. 何のスキルが必要か考える（時間がかかる）
2. Rork Community で一つずつ検索
3. 各スキルを手動でインストール
4. 実装開始 ← 既に 30 分経過している
```

**Find skill を使った方法:**

```
1. Find skill を起動
2. 「推奨スキル」をクリック
3. 必要なスキルが自動インストール
4. 実装開始 ← 5 分で完了
```

## Find skill の実装原理

### どうやってスキルを判定する？

Find skill は、以下の情報を分析します：

- **プロジェクトの種類**（Web アプリ、モバイルアプリ、etc.）
- **使用している技術スタック**（React、Vue、Flutter など）
- **デザインツール**（Figma 連携の有無など）
- **ターゲットプラットフォーム**（iOS / Android / Web）

これらの情報から機械学習モデルが、**最適なスキルの組み合わせを推奨**します。

### 推奨スキルの例

Rork でモバイルアプリを開発する場合、典型的に以下が推奨されます：

| スキル名 | 役割 | 優先度 |
|----------|------|--------|
| UI Pro Max | UI/UX デザイン高度化 | 高 |
| frontend-design | フロントエンド実装サポート | 高 |
| sleek-design-mobile-apps | モバイル UI 最適化 | 中 |
| test-automation-mobile | 自動テスト | 中 |
| app-deployment-swift | iOS デプロイ | 中 |

## Rork 開発で必要な主要スキル

### 1. UI Pro Max — 最優先スキル

**役割**：Figma で生成した UI をアプリに統合、UI/UX を洗練する

**具体的な機能**：

- デザインシステム管理
- レスポンシブ対応の自動化
- ダークモード対応
- アニメーション制御

**使用シーン**：

```
[ステップ 1] Rork で UI 自動生成
    ↓
[ステップ 2] Figma で色・フォント調整
    ↓
[ステップ 3] UI Pro Max で実装コードを生成
    ↓
[ステップ 4] アプリにそのまま統合
```

UI Pro Max がないと、Figma での調整結果をアプリに反映させるのに数時間かかります。Find skill で最初にインストールすることを強く推奨します。

### 2. frontend-design — UI 実装の最適化

**役割**：React / Vue などのコンポーネント設計をサポート

**具体的な機能**：

- コンポーネント分割の提案
- CSS-in-JS の最適化
- パフォーマンス改善（不要な再レンダリング排除）

**使用シーン**：

UI Pro Max で出力されたコンポーネントを、さらに最適化したい場合に活躍します。

### 3. sleek-design-mobile-apps — モバイル特化

**役割**：iPhone / Android それぞれの設計ガイドラインに準拠

**具体的な機能**：

- iOS HIG（Human Interface Guidelines）への準拠チェック
- Material Design（Android）への準拠チェック
- 安全領域（Safe Area）の自動設定

**使用シーン**：

モバイルアプリ開発では必須です。デスクトップ視点で設計したものを、モバイル視点に自動変換してくれます。

### 4. test-automation-mobile — 自動テスト

**役割**：UI テスト、機能テストの自動化

**具体的な機能**：

- スクリーンショット比較テスト
- タップ・スワイプ操作の自動検証
- 複数デバイスでの同時テスト

**使用シーン**：

アプリの品質確保に欠かせません。手動テストより 10 倍高速です。

### 5. app-deployment-swift / app-deployment-android — デプロイメント

**役割**：App Store / Google Play への公開を自動化

**具体的な機能**：

- ビルド設定の自動化
- TestFlight / Google Play Console への自動アップロード
- バージョン管理の自動化

**使用シーン**：

最終的なストア公開時に活躍します。

## Find skill の使い方

### ステップ 1: Find skill を起動

Rork のプロジェクトダッシュボードで、右上の「スキル」メニューを選択。「Find skill を使う」ボタンをクリックします。

### ステップ 2: プロジェクト情報を入力

以下の情報を入力（または自動検出）：

- **プロジェクト名**（デフォルト：既存のプロジェクト名）
- **アプリの種類**（モバイルアプリ / Web アプリ / etc.）
- **ターゲットプラットフォーム**（iOS / Android / Web / 複数）
- **使用技術**（React Native / Flutter / Swift など）

### ステップ 3: 推奨スキルを確認

Find skill が、プロジェクトに最適なスキル 5-10 個を提案します。各スキルの説明を読み、本当に必要かを判断します。

```
推奨スキル一覧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ UI Pro Max （強く推奨）
   Figma 連携で UI/UX を高度化します

✅ frontend-design （推奨）
   React コンポーネントの最適化

✅ sleek-design-mobile-apps （推奨）
   iOS / Android ガイドラインへの準拠

⭕ test-automation-mobile （オプション）
   自動テストが必要な場合に

⭕ app-deployment-swift （オプション）
   iOS デプロイメントが必要な場合に
```

### ステップ 4: スキルを一括インストール

「推奨スキルをすべてインストール」ボタンを押すと、数秒で完了。各スキルは自動的に Rork 環境に統合されます。

### ステップ 5: スキルの確認

インストール完了後、「マイスキル」ページで、インストール済みスキルが表示されます。各スキルには、簡潔なドキュメント（日本語）と、実装例が記載されています。

## 落とし穴と対策

### 落とし穴 1: 推奨スキルをむやみにインストール

Find skill が推奨したスキルが、**必ずしも自分に必要とは限りません**。

**対策**: 推奨スキル一覧の説明を読み、実際のプロジェクトに合わせて判断します。

### 落とし穴 2: スキルを入れっぱなし

インストール後、使わないスキルが残ったままになることがあります。

**対策**: 定期的（月 1 回）に「マイスキル」を確認し、不要なスキルをアンインストール。環境をすっきり保つことで、パフォーマンス向上・トラブルシューティングが容易になります。

### 落とし穴 3: ドキュメント未読

スキルをインストールしても、使い方を理解していないと意味がありません。

**対策**: 各スキルの日本語ドキュメントを必ず読む。特に最初の 1-2 個のスキルは、実装例を参考に「動かしながら理解する」ことを推奨。

## トラブルシューティング

### Q: Find skill が推奨スキルを提案できません

**A**: 以下を確認：
- Rork への インターネット接続がある
- プロジェクト設定（技術スタック、ターゲットプラットフォーム）が正確に入力されている
- Rork を最新バージョンに更新

### Q: スキルをインストールしたが、Rork に反映されません

**A**: キャッシュクリア：
```
Rork メニュー → 設定 → キャッシュをクリア → 再起動
```

### Q: 特定のスキルがない

**A**: Rork Community で手動検索。Find skill は推奨スキルのみを表示するため、ニッチなスキルは手動でインストール可能。
