# Rork MaxでAppleネイティブAPIを活用する完全ガイド

Rork MaxはネイティブSwiftコードを生成するため、React Nativeベースの従来のRorkでは利用できなかったApple独自のAPIにフルアクセスできます。このガイドでは、Rork Maxで活用できる主要なネイティブAPIと、それぞれの実装方法を解説します。

## React Nativeでは使えなかったAPIが使える理由

従来のRorkはReact Nativeベースのため、Appleのネイティブフレームワークへのアクセスにはブリッジ層が必要でした。Rork MaxはSwiftUIで直接コードを生成するため、Appleが提供するすべてのAPIに直接アクセスできます。

これにより、以下のような高度な機能を自然言語の指示だけで実装できるようになりました。

## ARKit — 拡張現実アプリの構築

ARKitはAppleの拡張現実フレームワークで、カメラとセンサーを組み合わせて現実世界にデジタルコンテンツを重ね合わせます。

### Rork Maxでの活用例

- 家具配置シミュレーター：部屋にバーチャル家具を配置して試せるアプリ
- ARナビゲーション：カメラ越しに道案内を表示するアプリ
- 3Dモデルビューア：製品やアートを空間に表示するアプリ
- AR測定ツール：カメラで物体のサイズを測定するアプリ

### プロンプト例

```
ARKitを使って、カメラで部屋を映すと家具を配置できるアプリを作ってください。
家具は回転・拡大縮小ができ、複数の家具を同時に配置できるようにしてください。
```

### LiDARスキャナーの活用

iPhone Pro / iPad ProシリーズにはLiDARスキャナーが搭載されており、Rork Maxではこれを利用した高精度な空間認識アプリも構築できます。LiDARにより、部屋の3Dスキャン、正確な距離測定、リアルタイムのメッシュ生成が可能です。

## HealthKit — ヘルスケアアプリの開発

HealthKitはAppleの健康データフレームワークで、ユーザーの健康関連データを安全に読み書きできます。

### アクセス可能なデータ

- 歩数、距離、消費カロリー
- 心拍数、血圧、血中酸素濃度
- 睡眠データ
- ワークアウト記録
- 体重、BMI、体脂肪率

### プロンプト例

```
HealthKitと連携して、毎日の歩数と消費カロリーをグラフで表示するフィットネスアプリを作ってください。
週間・月間の推移も確認でき、目標設定機能もつけてください。
```

### プライバシーへの配慮

HealthKitのデータはユーザーの明示的な許可が必要です。Rork Maxで生成されるコードには、適切な権限リクエストダイアログが自動的に含まれます。App Storeの審査では、健康データの使用目的を明確に説明する必要があります。

## Dynamic Island — iPhone 15 Pro以降の没入型UI

Dynamic Islandは、画面上部のノッチ領域をインタラクティブなUIに変換する機能です。リアルタイムの情報表示や、アプリからの通知をエレガントに表示できます。

### 活用シーン

- 配達追跡：配達状況をDynamic Islandにリアルタイム表示
- スポーツスコア：試合のスコアをライブ更新
- タイマー：料理タイマーや集中タイマーの残り時間表示
- 音楽再生：再生中の曲情報とコントロール

### プロンプト例

```
料理タイマーアプリを作ってください。タイマーを設定すると、Dynamic Islandに
残り時間が表示され、完了時に通知が届くようにしてください。
複数のタイマーを同時に管理できるようにしてください。
```

## Live Activities — ロック画面のリアルタイム更新

Live ActivitiesはDynamic Islandと連携して、ロック画面にもリアルタイム情報を表示する機能です。

### ユースケース

- フードデリバリーの注文状況追跡
- スポーツの試合経過
- ライドシェアの車両到着状況
- フライト情報のリアルタイム更新

### プロンプト例

```
フードデリバリーの注文追跡アプリを作ってください。
Live Activitiesで注文状況（準備中→配達中→到着）をロック画面に表示し、
配達員の現在位置をマップ上にリアルタイムで表示してください。
```

## Core ML — オンデバイス機械学習

Core MLはAppleのオンデバイス機械学習フレームワークで、クラウドに送信せずにデバイス上でAI推論を実行できます。

### 実装可能な機能

- 画像認識・分類
- 自然言語処理
- 音声認識
- オブジェクト検出
- スタイル変換

### プロンプト例

```
Core MLを使って、カメラで撮影した植物を自動識別するアプリを作ってください。
植物の名前、特徴、育て方のヒントを表示してください。
```

### プライバシーのメリット

Core MLはすべての処理をデバイス上で行うため、ユーザーのデータがサーバーに送信されることはありません。プライバシーを重視するアプリに最適です。

## App Clips — インストール不要の軽量アプリ体験

App ClipsはフルアプリをインストールせずにNFCタグ、QRコード、またはSafariリンクから即座に起動できる軽量バージョンのアプリです。

### 活用シーン

- レストランのメニュー注文
- 駐車場の支払い
- 自転車シェアリングの解錠
- イベントチケットの表示

### プロンプト例

```
カフェのモバイルオーダーアプリを作ってください。
App Clipとして、QRコードをスキャンすると注文画面が開き、
Apple Payで支払いができるようにしてください。
```

## HomeKit — スマートホーム連携

HomeKitを利用すると、Apple HomeKit対応のスマートホームデバイスを制御するアプリを構築できます。

### 制御可能なデバイス

- スマート照明（明るさ、色温度）
- サーモスタット（温度設定）
- セキュリティカメラ（ライブ映像）
- スマートロック（施錠・解錠）
- スマートプラグ（オン・オフ）

### プロンプト例

```
HomeKit対応のスマートホームダッシュボードアプリを作ってください。
部屋ごとにデバイスを管理でき、シーン設定（おはよう、おやすみ等）で
複数のデバイスを一括操作できるようにしてください。
```

## ウィジェット — ホーム画面のカスタマイズ

SwiftUIベースのWidgetKitを使って、ホーム画面やスタンバイモードに情報を表示するウィジェットを作成できます。

### ウィジェットの種類

- Small（2x2）：コンパクトな情報表示
- Medium（4x2）：詳細な情報表示
- Large（4x4）：リッチなコンテンツ表示
- Lock Screen：ロック画面ウィジェット
- StandBy：スタンバイモード対応

### プロンプト例

```
天気予報アプリにウィジェットを追加してください。
Smallウィジェットで現在の気温と天気アイコン、
Mediumウィジェットで3時間ごとの予報、
ロック画面ウィジェットで気温を表示してください。
```

## Siri Intents — 音声操作の統合

SiriKitを利用して、Siriからアプリの機能を音声で操作できるようにします。

### 対応可能なインテント

- メッセージの送信
- ワークアウトの開始・終了
- 支払い・送金
- リストの管理
- メディアの再生

### プロンプト例

```
タスク管理アプリにSiriショートカットを追加してください。
「今日のタスクを教えて」でタスク一覧を読み上げ、
「買い物リストに牛乳を追加」で項目を追加できるようにしてください。
```

## Metal — 高性能グラフィックス

MetalはAppleの低レベルグラフィックスAPIで、GPUを直接制御して高性能な3Dグラフィックスやコンピュテーションを実現します。

### 活用例

- 3Dゲーム開発
- リアルタイム画像・動画エフェクト
- 科学的可視化
- カスタムレンダリングエンジン

### プロンプト例

```
Metalを使った3Dの太陽系シミュレーターを作ってください。
各惑星がリアルな軌道で回転し、ピンチで拡大縮小、
スワイプで視点を変更できるようにしてください。
```

## まとめ

Rork Maxの最大の強みは、ネイティブSwift生成によりAppleのすべてのAPIにアクセスできることです。ARKit、HealthKit、Dynamic Island、Core MLなどの高度な機能も、自然言語でプロンプトを書くだけで実装できます。

従来のReact Nativeベースのアプリでは困難だった機能が、Rork Maxなら数分で実現可能です。まずは興味のあるAPIから試して、ネイティブならではの体験をユーザーに届けましょう。

ネイティブAPIを組み合わせることで、よりリッチでパフォーマンスの高いアプリを構築できます。例えば、ARKitとCore MLを組み合わせたAR物体認識アプリや、HealthKitとウィジェットを組み合わせた健康ダッシュボードなど、可能性は無限大です。

TypeScriptでの堅牢な開発についてより実践的な知識を身につけたい方には、「[プロを目指す人のためのTypeScript入門](https://www.amazon.co.jp/dp/4297127474/?tag=pinocchio-22)」（鈴木僚太 著）が参考になります。
