## AI スタイリストアプリとは

毎朝「今日は何を着よう？」と悩んだ経験は誰にでもあるはずです。AI スタイリストアプリは、クローゼットの服をカメラで撮影・登録し、天気やシーン、好みに合わせてコーディネートを提案してくれるアプリです。

Rork を使えば、こうした画像認識 + AI レコメンデーション機能を持つアプリを、コードをほとんど書かずに構築できます。本記事では、Rork のプロンプト機能を活用して、実際に動作する AI スタイリストアプリをステップバイステップで作成していきます。

これは Rork Lab のプレミアム記事で扱うような実践チュートリアルの一部を、無料で体験いただける見本記事です。

## プロジェクト構成の設計

まず Rork でプロジェクトを作成し、画面構成をプロンプトで指示します。

```
# Rork への指示例
AIスタイリストアプリを作成してください。

画面構成:
1. ホーム画面: 今日のおすすめコーディネートを表示
2. クローゼット画面: 登録した服の一覧（カテゴリ別タブ）
3. 服の登録画面: カメラで撮影 → AI が自動分類
4. コーディネート生成画面: 条件を選んでAIにコーデを提案してもらう

- タブナビゲーション（5画面）
- AsyncStorage でローカル保存

- 画像ピッカー（expo-image-picker）
```

Rork はこのプロンプトから、React Native + Expo Router ベースのプロジェクト構成を自動生成します。タブナビゲーション、画面遷移、基本的な UI コンポーネントがすべて揃った状態でスタートできます。

## 服の撮影と画像認識

カメラで撮影した服を AI が自動的にカテゴリ（トップス・ボトムス・アウター・シューズ・アクセサリー）に分類する仕組みを実装します。

```typescript
// types/wardrobe.ts
export type ClothingCategory =
  | "tops"
  | "bottoms"
  | "outerwear"
  | "shoes"
  | "accessories";

export type ClothingColor =
  | "black" | "white" | "gray" | "navy"
  | "red" | "blue" | "green" | "brown"
  | "beige" | "pink" | "other";

export interface ClothingItem {
  id: string;
  imageUri: string;
  category: ClothingCategory;
  color: ClothingColor;
  tags: string[];        // カジュアル, フォーマル, スポーツ等
  addedAt: string;
  wearCount: number;
  lastWornAt?: string;
}
```

```typescript
// utils/clothing-classifier.ts
// Gemini API を使った服の自動分類
export async function classifyClothing(
  imageBase64: string
): Promise<{
  category: ClothingCategory;
  color: ClothingColor;
  tags: string[];
}> {
  const response = await fetch(
    "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=YOUR_GEMINI_API_KEY",
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        contents: [{
          parts: [
            {
              inlineData: {
                mimeType: "image/jpeg",
                data: imageBase64,
              },
            },
            {
              text: `この服の画像を分析し、以下のJSON形式で回答してください。
{
  "category": "tops" | "bottoms" | "outerwear" | "shoes" | "accessories",
  "color": "black" | "white" | "gray" | "navy" | "red" | "blue" | "green" | "brown" | "beige" | "pink" | "other",
  "tags": ["casual", "formal", "sporty" 等のスタイルタグ（最大3つ）]
}
JSON以外のテキストは出力しないでください。`,
            },
          ],
        }],
      }),
    }
  );

  const data = await response.json();
  const text = data.candidates[0].content.parts[0].text;
  return JSON.parse(text);
}
```

ポイントは、Gemini のマルチモーダル API を活用して画像から服のカテゴリ・色・スタイルタグを自動抽出している点です。ユーザーは服を撮影するだけで、面倒な手動入力なしにクローゼットを構築できます。

## クローゼットのデータ管理

登録した服のデータは AsyncStorage でローカルに保存します。Rork のプロンプトでカスタムフックの生成を指示できます。

```typescript
// hooks/useWardrobe.ts
import { useState, useEffect, useCallback } from "react";
import AsyncStorage from "@react-native-async-storage/async-storage";
import { ClothingItem, ClothingCategory } from "@/types/wardrobe";

const STORAGE_KEY = "@wardrobe_items";

export function useWardrobe() {
  const [items, setItems] = useState<ClothingItem[]>([]);
  const [loading, setLoading] = useState(true);

  // 初回読み込み
  useEffect(() => {
    loadItems();
  }, []);

  const loadItems = async () => {
    try {
      const stored = await AsyncStorage.getItem(STORAGE_KEY);
      if (stored) setItems(JSON.parse(stored));
    } catch (error) {
      console.error("Failed to load wardrobe:", error);
    } finally {
      setLoading(false);
    }
  };

  const addItem = useCallback(async (item: ClothingItem) => {
    const updated = [...items, item];
    setItems(updated);
    await AsyncStorage.setItem(STORAGE_KEY, JSON.stringify(updated));
  }, [items]);

  const getByCategory = useCallback(
    (category: ClothingCategory) =>
      items.filter((item) => item.category === category),
    [items]
  );

  const recordWear = useCallback(async (id: string) => {
    const updated = items.map((item) =>
      item.id === id
        ? {
            ...item,
            wearCount: item.wearCount + 1,
            lastWornAt: new Date().toISOString(),
          }
        : item
    );
    setItems(updated);
    await AsyncStorage.setItem(STORAGE_KEY, JSON.stringify(updated));
  }, [items]);

  return { items, loading, addItem, getByCategory, recordWear };
}
```

## AI コーディネート提案エンジン

クローゼット内の服を組み合わせて、条件に合ったコーディネートを AI が提案する機能を実装します。

```typescript
// utils/outfit-recommender.ts
import { ClothingItem } from "@/types/wardrobe";

interface OutfitRequest {
  occasion: "casual" | "business" | "date" | "sports";
  weather: "hot" | "warm" | "cool" | "cold";
  preference?: string; // ユーザーの追加リクエスト
}

export interface OutfitSuggestion {
  items: ClothingItem[];
  reasoning: string;
  styleScore: number; // 1-10
}

export async function generateOutfit(
  wardrobe: ClothingItem[],
  request: OutfitRequest
): Promise<OutfitSuggestion[]> {
  // クローゼット内容をテキストに変換
  const wardrobeDesc = wardrobe.map((item) =>
    `[ID:${item.id}] ${item.category} - ${item.color} (${item.tags.join(", ")})`
  ).join("\n");

  const response = await fetch(
    "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=YOUR_GEMINI_API_KEY",
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        contents: [{
          parts: [{
            text: `あなたはプロのスタイリストです。
以下のクローゼット内容から、条件に合うコーディネートを3つ提案してください。

クローゼット:
${wardrobeDesc}

条件:
- シーン: ${request.occasion}
- 天気: ${request.weather}
${request.preference ? `- 追加リクエスト: ${request.preference}` : ""}

JSON配列で回答してください:
[{
  "item_ids": ["使用するアイテムのID配列"],
  "reasoning": "このコーデを選んだ理由（日本語）",
  "style_score": 8
}]`,
          }],
        }],
      }),
    }
  );

  const data = await response.json();
  const text = data.candidates[0].content.parts[0].text;
  const suggestions = JSON.parse(text);

  // IDからアイテムを復元
  return suggestions.map((s: { item_ids: string[]; reasoning: string; style_score: number }) => ({
    items: s.item_ids
      .map((id: string) => wardrobe.find((w) => w.id === id))
      .filter(Boolean) as ClothingItem[],
    reasoning: s.reasoning,
    styleScore: s.style_score,
  }));
}
```

## FAQ

**Q: 画像認識の精度はどの程度ですか？**

Gemini 2.0 Flash のマルチモーダル機能を使った場合、一般的な衣類のカテゴリ分類で90%以上の精度が期待できます。ただし、似た色味のアイテムや複雑な柄物は誤分類されることがあるため、ユーザーが手動で修正できる UI を用意しておくことを推奨します。

**Q: オフラインでも使えますか？**

クローゼットのデータと過去のコーディネート結果は AsyncStorage にローカル保存されるため、閲覧は可能です。ただし、新しい服の自動分類やコーディネート生成には API 通信が必要なので、オンライン接続が必要です。

**Q: API の利用料金はどのくらいかかりますか？**

Gemini 2.0 Flash は画像1枚あたりのコストが非常に低く、個人利用であれば無料枠内で十分に収まります。1日10回程度のコーディネート生成なら、月額の API コストはほぼゼロに抑えられます。

**Q: Rork でこのアプリを作るのにどのくらい時間がかかりますか？**

基本的な機能（服の登録・一覧表示・AI コーディネート提案）であれば、Rork のプロンプト機能を活用して2〜3時間で動作するプロトタイプが完成します。UI の調整やエッジケースの対応を含めても、1日あれば十分です。
