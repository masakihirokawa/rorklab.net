## ディープリンクとは

**ディープリンク**は、URLからアプリの特定の画面を直接開く機能です。例えば `myapp://product/123` というリンクをタップすると、アプリが起動して商品 ID 123 の詳細画面が表示される — というものです。

ディープリンクを実装することで、以下が可能になります:
- プッシュ通知からアプリ内の特定画面に遷移
- Web サイトからアプリへの誘導
- アプリ間の連携
- マーケティングリンクのトラッキング

## URL Scheme vs Universal Links

| 方式 | 形式 | 特徴 |
|------|------|------|
| **URL Scheme** | `myapp://path` | 実装が簡単。アプリ未インストール時は何も起きない |
| **Universal Links** (iOS) | `https://myapp.com/path` | Webとアプリをシームレスに切り替え。Apple の推奨 |
| **App Links** (Android) | `https://myapp.com/path` | AndroidのUniversal Links相当 |

## Rork Max での設定

### app.json の設定

```json
{
  "expo": {
    "scheme": "myapp",
    "ios": {
      "associatedDomains": ["applinks:myapp.com"]
    },
    "android": {
      "intentFilters": [
        {
          "action": "VIEW",
          "autoVerify": true,
          "data": [
            {
              "scheme": "https",
              "host": "myapp.com",
              "pathPrefix": "/app"
            }
          ],
          "category": ["BROWSABLE", "DEFAULT"]
        }
      ]
    }
  }
}
```

### React Navigation との統合

Rork Max に以下のプロンプトで依頼できます:

```
「アプリに以下のディープリンクを追加してください:
- myapp://home → ホーム画面
- myapp://product/:id → 商品詳細画面
- myapp://profile → プロフィール画面
React Navigation を使って実装してください。」
```

生成されるリンク設定例:

```javascript
const linking = {
  prefixes: ['myapp://', 'https://myapp.com'],
  config: {
    screens: {
      Home: 'home',
      ProductDetail: 'product/:id',
      Profile: 'profile',
    },
  },
};

// NavigationContainer に渡す
<NavigationContainer linking={linking}>
  {/* ... */}
</NavigationContainer>
```

## プッシュ通知との連携

プッシュ通知のタップでアプリ内の特定画面を開くには、通知データにディープリンク URL を含めます。

```javascript
// 通知の送信側（サーバーサイド）
const notification = {
  title: "新しいメッセージ",
  body: "田中さんからメッセージが届きました",
  data: {
    url: "myapp://chat/user/tanaka"
  }
};

// アプリ側での処理
import * as Notifications from 'expo-notifications';

Notifications.addNotificationResponseReceivedListener((response) => {
  const url = response.notification.request.content.data?.url;
  if (url) {
    Linking.openURL(url);
  }
});
```

## まとめ

ディープリンクはアプリのユーザー体験を大幅に向上させる機能です。

- `app.json` でスキームと Associated Domains を設定
- React Navigation の `linking` 設定でルートを定義
- プッシュ通知と組み合わせて通知からの直接遷移を実現

Rork Max へのプロンプトで基本的な実装は自動化できます。
