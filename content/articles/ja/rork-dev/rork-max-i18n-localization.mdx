# Rork Max 多言語対応（i18n）完全ガイド

グローバルユーザーを対象としたアプリ開発では、多言語対応は必須です。このガイドでは、Rork Max アプリに react-i18next と expo-localization を組み合わせた本格的な多言語機能を実装する方法を、段階的に解説します。

## 準備：必要なパッケージのインストール

```bash
npm install i18next react-i18next expo-localization

# TypeScript を使う場合
npm install --save-dev @types/i18next
```

<Callout type="info">
**推奨構成**: react-i18next は翻訳管理を、expo-localization はデバイスの言語設定を取得するのに役立ちます。二つを組み合わせることで、ユーザーの環境に自動で適応するアプリが作成できます。
</Callout>

## ステップ 2: i18n の初期化

**src/i18n/config.ts**
```typescript
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import * as Localization from 'expo-localization';

// 翻訳ファイルのインポート
import jaCommon from '../locales/ja/common.json';
import jaHome from '../locales/ja/home.json';
import enCommon from '../locales/en/common.json';
import enHome from '../locales/en/home.json';

const resources = {
  ja: {
    common: jaCommon,
    home: jaHome,
  },
  en: {
    common: enCommon,
    home: enHome,
  },
};

// デバイスの言語設定を取得
const deviceLanguage = Localization.getLocales()[0]?.languageCode || 'en';

i18n
  .use(initReactI18next)
  .init({
    resources,
    lng: deviceLanguage,
    fallbackLng: 'en',
    defaultNS: 'common',
    ns: ['common', 'home'],
    interpolation: {
      escapeValue: false, // React でフォーマット処理されるため不要
    },
    react: {
      useSuspense: false, // Suspense 使用時は true に設定
    },
  });

export default i18n;
```

**src/App.tsx**
```typescript
import './i18n/config';
import { I18nextProvider } from 'react-i18next';
import i18n from './i18n/config';
import Home from './screens/Home';

export default function App() {
  return (
    <I18nextProvider i18n={i18n}>
      <Home />
    </I18nextProvider>
  );
}
```

<Callout type="warning">
**重要**: App.tsx の最上部で i18n config をインポートしないと、翻訳が機能しません。
</Callout>

## ステップ 3: useTranslation フックの使用

**src/screens/Home.tsx**
```typescript
import React from 'react';
import { View, Text, TouchableOpacity } from 'react-native';
import { useTranslation } from 'react-i18next';

export default function Home() {
  const { t, i18n } = useTranslation();

  const greeting = t('home:greeting', { name: 'Alice' });

  return (
    <View style={{ flex: 1, padding: 20, justifyContent: 'center' }}>
      <Text style={{ fontSize: 24, fontWeight: 'bold' }}>
        {t('app_name')}
      </Text>
      <Text style={{ marginTop: 16, fontSize: 18 }}>
        {greeting}
      </Text>

      {/* 言語切替ボタン */}
      <TouchableOpacity
        style={{
          marginTop: 24,
          padding: 12,
          backgroundColor: '#007AFF',
          borderRadius: 8,
        }}
        onPress={() => {
          const nextLang = i18n.language === 'en' ? 'ja' : 'en';
          i18n.changeLanguage(nextLang);
        }}
      >
        <Text style={{ color: '#fff', fontSize: 16, textAlign: 'center' }}>
          {t('common:next')} ({i18n.language === 'en' ? '日本語' : 'English'})
        </Text>
      </TouchableOpacity>
    </View>
  );
}
```

## ステップ 4: 複数形・日付・数値のフォーマット

翻訳ファイルで複数形をサポート：

**locales/ja/home.json**
```json
{
  "tasks_count": {
    "zero": "タスクなし",
    "one": "1つのタスク",
    "other": "{{count}}個のタスク"
  },
  "last_updated": "最終更新: {{date, datetime}}",
  "price": "¥{{amount, number}}"
}
```

**src/screens/TaskList.tsx**
```typescript
import React from 'react';
import { Text } from 'react-native';
import { useTranslation } from 'react-i18next';
import * as Localization from 'expo-localization';

export default function TaskList() {
  const { t, i18n } = useTranslation(['home', 'common']);
  const taskCount = 5;
  const lastUpdated = new Date();
  const price = 19.99;

  return (
    <>
      {/* 複数形対応 */}
      <Text>
        {t('home:tasks_count', { count: taskCount })}
      </Text>

      {/* 日付フォーマット - i18next の Intl バックエンド */}
      <Text>
        {new Intl.DateTimeFormat(i18n.language, {
          year: 'numeric',
          month: 'long',
          day: 'numeric',
          hour: '2-digit',
          minute: '2-digit',
        }).format(lastUpdated)}
      </Text>

      {/* 数値フォーマット */}
      <Text>
        {new Intl.NumberFormat(i18n.language, {
          style: 'currency',
          currency: i18n.language === 'ja' ? 'JPY' : 'USD',
        }).format(price)}
      </Text>
    </>
  );
}
```

## ステップ 5: 動的言語切替コンポーネント

ユーザーが設定画面で言語を変更できるようにします。

**src/screens/Settings.tsx**
```typescript
import React, { useState } from 'react';
import { View, Text, ScrollView, TouchableOpacity } from 'react-native';
import { useTranslation } from 'react-i18next';
import AsyncStorage from '@react-native-async-storage/async-storage';

const LANGUAGE_KEY = 'user_language_preference';

export default function Settings() {
  const { t, i18n } = useTranslation();
  const [selectedLanguage, setSelectedLanguage] = useState(i18n.language);

  const languages = [
    { code: 'ja', label: '日本語' },
    { code: 'en', label: 'English' },
    { code: 'es', label: 'Español' },
    { code: 'fr', label: 'Français' },
  ];

  const handleLanguageChange = async (langCode: string) => {
    setSelectedLanguage(langCode);
    await i18n.changeLanguage(langCode);
    // ユーザーの言語選択を永続化
    await AsyncStorage.setItem(LANGUAGE_KEY, langCode);
  };

  return (
    <ScrollView style={{ flex: 1, padding: 20 }}>
      <Text style={{ fontSize: 24, fontWeight: 'bold', marginBottom: 20 }}>
        {t('settings:language')}
      </Text>

      {languages.map((lang) => (
        <TouchableOpacity
          key={lang.code}
          style={{
            padding: 16,
            marginBottom: 8,
            backgroundColor: selectedLanguage === lang.code ? '#E3F2FD' : '#F5F5F5',
            borderRadius: 8,
            borderWidth: selectedLanguage === lang.code ? 2 : 0,
            borderColor: '#007AFF',
          }}
          onPress={() => handleLanguageChange(lang.code)}
        >
          <Text style={{ fontSize: 16 }}>{lang.label}</Text>
        </TouchableOpacity>
      ))}
    </ScrollView>
  );
}
```

**src/App.tsx** で永続化を復元：
```typescript
useEffect(() => {
  const loadSavedLanguage = async () => {
    const saved = await AsyncStorage.getItem(LANGUAGE_KEY);
    if (saved) {
      await i18n.changeLanguage(saved);
    }
  };
  loadSavedLanguage();
}, []);
```

## ステップ 6: RTL（右から左）対応

アラビア語やヘブライ語などの RTL 言語に対応します。

**src/utils/rtl.ts**
```typescript
import { I18n } from 'i18next';
import { I18nManager } from 'react-native';

export const RTL_LANGUAGES = ['ar', 'he', 'ur', 'fa'];

export function setupRTL(i18n: I18n) {
  i18n.on('languageChanged', (lng) => {
    const isRTL = RTL_LANGUAGES.includes(lng);
    I18nManager.forceRTL(isRTL);
  });

  // 初期化
  const isRTL = RTL_LANGUAGES.includes(i18n.language);
  I18nManager.forceRTL(isRTL);
}
```

**src/App.tsx**
```typescript
import { setupRTL } from './utils/rtl';

useEffect(() => {
  setupRTL(i18n);
}, []);
```

<Callout type="info">
**RTL レイアウト**: React Native は I18nManager.forceRTL() で自動的に flexDirection を反転します。カスタムレイアウトの場合は、width/marginLeft など方向性に依存するプロパティを避け、flex を活用してください。
</Callout>

## ベストプラクティス

### 1. ネームスペース分割
```typescript
// 大規模アプリは画面ごとにネームスペースを分ける
const { t } = useTranslation(['home', 'profile', 'common']);
// home:key または t('key') で home ネームスペースから取得
```

### 2. 型安全な翻訳キー
```typescript
// src/types/i18n.ts
export const translationKeys = {
  common: ['app_name', 'welcome', 'back'],
  home: ['title', 'greeting', 'today_tasks'],
} as const;

type TranslationKey = typeof translationKeys[keyof typeof translationKeys][number];
```

### 3. キーの漏れ防止
```typescript
// src/scripts/validate-i18n.js
const ja = require('../locales/ja/common.json');
const en = require('../locales/en/common.json');

const jaKeys = Object.keys(ja).sort();
const enKeys = Object.keys(en).sort();

if (JSON.stringify(jaKeys) !== JSON.stringify(enKeys)) {
  console.error('翻訳キーが一致しません');
  process.exit(1);
}
```

## まとめ

Rork Max での多言語対応は、react-i18next と expo-localization の組み合わせで効率的に実装できます。

- **翻訳ファイル**: JSON で構造化し、ネームスペースで分割
- **自動言語検出**: デバイス設定から自動取得
- **RTL対応**: 言語に応じて自動調整
- **永続化**: ユーザーの言語選択を保存
- **型安全**: TypeScript でキーを検証

これらの実装により、グローバル展開に対応した高品質なアプリを構築できます。
