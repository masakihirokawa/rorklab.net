Rork Maxで構築したアプリにAI機能を搭載したいとき、クラウドバックエンドの選択は極めて重要です。本ガイドではCloudflare Workersを使用した、スケーラブルで低遅延、かつ低コストなAIバックエンド構築方法を解説します。

## アーキテクチャ概要

Rork MaxアプリからCloudflare Workersへのリクエストフローを理解することが出発点です。

```
Rork Max App (React Native)
        ↓
    HTTPS Request
        ↓
Cloudflare Workers
        ↓
    ┌─────────────────────────┐
    │ AI Gateway              │ ← LLM API コスト削減・観測
    │ ↓                       │
    │ OpenAI / Anthropic API  │
    │ または                  │
    │ Workers AI (Llama等)    │
    └─────────────────────────┘
        ↓
    D1 SQLite
   (ユーザーデータ)
```

Cloudflare Workersを選ぶ理由：
- **エッジコンピューティング**: グローバルネットワークで低遅延応答
- **スケーラビリティ**: サーバーレスで自動スケール、管理不要
- **AI Gateway**: LLM API呼び出しをキャッシング・ルーティング、コスト最適化
- **Workers AI**: エッジで推論実行、レイテンシ削減
- **D1統合**: SQLiteデータベースがネイティブ統合、チャット履歴管理が簡単

## Cloudflare Workers セットアップ

まずプロジェクトを初期化します。

```bash
npm create cloudflare@latest rork-ai-backend -- --typescript
cd rork-ai-backend
```

生成された `wrangler.toml` ファイルを編集し、AI GatewayとD1バインディングを設定します：

```toml
name = "rork-ai-backend"
main = "src/index.ts"
compatibility_date = "2026-03-10"

[env.production]
name = "rork-ai-backend-prod"

[[d1_databases]]
binding = "DB"
database_name = "rork_data"
database_id = "xxxxxxx"

[[vars]]
AI_GATEWAY_URL = "https://gateway.ai.cloudflare.com/v1/YOUR-ACCOUNT-ID/YOUR-GATEWAY-ID"
JWT_SECRET = "your-jwt-secret-key"
ALLOWED_ORIGINS = "https://yourdomain.com"
```

D1データベースを作成します：

```bash
wrangler d1 create rork_data
```

出力から `database_id` をコピーして `wrangler.toml` に追加します。

## AI Gateway セットアップ

Cloudflareダッシュボードで「AI」→「AI Gateway」にアクセスし、新しいGatewayを作成します。

```
Gateway Name: rork-ai-gateway
Provider: OpenAI（またはAnthropicなど）
```

AI Gatewayの役割：
- **キャッシング**: 同じプロンプトに対する応答をキャッシュ、API呼び出し削減
- **ルーティング**: 複数のLLMプロバイダーへの振り分け
- **観測**: リクエスト・レスポンスのログ記録、トークン使用量の追跡
- **レート制限**: DDoS保護、API利用上限設定

## Workers AI 統合

Llama 2、Mistral 7Bなどのモデルをエッジで推論実行できます。

```typescript
// src/ai.ts
import { Ai } from "@cloudflare/ai";

export async function generateWithWorkersAI(
  ai: Ai,
  prompt: string
): Promise<string> {
  const response = await ai.run("@cf/meta/llama-2-7b-chat-int8", {
    prompt: prompt,
    max_tokens: 512,
  });

  return response.response;
}

// ストリーミングレスポンス対応
export async function generateWithWorkersAIStream(
  ai: Ai,
  prompt: string
): Promise<ReadableStream<string>> {
  const response = await ai.run("@cf/meta/llama-2-7b-chat-int8", {
    prompt: prompt,
    max_tokens: 512,
    stream: true,
  });

  return response as ReadableStream<string>;
}
```

```sql
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  email TEXT UNIQUE NOT NULL,
  name TEXT NOT NULL,
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL
);

CREATE TABLE conversations (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  title TEXT,
  model TEXT DEFAULT 'gpt-3.5-turbo',
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE chat_messages (
  id TEXT PRIMARY KEY,
  conversation_id TEXT NOT NULL,
  role TEXT NOT NULL,
  content TEXT NOT NULL,
  tokens_used INTEGER DEFAULT 0,
  created_at INTEGER NOT NULL,
  FOREIGN KEY (conversation_id) REFERENCES conversations(id)
);

CREATE INDEX idx_user_conversations ON conversations(user_id);
CREATE INDEX idx_conversation_messages ON chat_messages(conversation_id);
```

マイグレーションを実行：

```bash
wrangler d1 migrations create rork_data init
# migrations/001_init.sql にスキーマを記述
wrangler d1 migrations apply rork_data --remote
```

## JWT 認証ミドルウェア

Rork Maxアプリからの安全なリクエストを実現します：

```typescript
// src/auth.ts
import { sign, verify } from "jsonwebtoken";

const JWT_SECRET = "your-secret-key";

export function createToken(userId: string): string {
  return sign(
    {
      userId,
      iat: Math.floor(Date.now() / 1000),
      exp: Math.floor(Date.now() / 1000) + 86400 * 7, // 7日間有効
    },
    JWT_SECRET
  );
}

export function verifyToken(token: string): { userId: string } | null {
  try {
    const decoded = verify(token, JWT_SECRET) as any;
    return { userId: decoded.userId };
  } catch (error) {
    return null;
  }
}

// ミドルウェア
export async function authenticateRequest(request: Request): Promise<string | null> {
  const authHeader = request.headers.get("Authorization");
  if (!authHeader?.startsWith("Bearer ")) {
    return null;
  }

  const token = authHeader.slice(7);
  const payload = verifyToken(token);
  return payload?.userId ?? null;
}
```

## API エンドポイント設計

RESTful APIを設計します。Rork Maxからのリクエストを処理します：

```typescript
// src/index.ts
import { Router } from "itty-router";
import { json } from "itty-router-extra";
import { authenticateRequest } from "./auth";
import { generateWithWorkersAI } from "./ai";

const router = Router();

// CORS対応
const corsHeaders = {
  "Access-Control-Allow-Origin": "*",
  "Access-Control-Allow-Methods": "GET, POST, OPTIONS",
  "Access-Control-Allow-Headers": "Content-Type, Authorization",
};

// AIチャット実行エンドポイント
router.post("/api/chat", async (request, env) => {
  const userId = await authenticateRequest(request);
  if (!userId) {
    return json({ error: "Unauthorized" }, { status: 401 });
  }

  const { message, conversationId, model = "gpt-3.5-turbo" } = await request.json();

  // チャット履歴を取得
  const history = await env.DB.prepare(
    `SELECT role, content FROM chat_messages
     WHERE conversation_id = ?
     ORDER BY created_at DESC LIMIT 20`
  )
    .bind(conversationId)
    .all();

  // AI Gateway経由でLLMに問い合わせ
  const aiGatewayUrl = env.AI_GATEWAY_URL;
  const aiResponse = await fetch(`${aiGatewayUrl}/chat/completions`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${env.OPENAI_API_KEY}`,
    },
    body: JSON.stringify({
      model,
      messages: [
        ...history.map((h: any) => ({
          role: h.role,
          content: h.content,
        })),
        { role: "user", content: message },
      ],
      temperature: 0.7,
      max_tokens: 1024,
    }),
  });

  const aiData = await aiResponse.json();
  const assistantMessage = aiData.choices[0].message.content;
  const tokensUsed = aiData.usage.total_tokens;

  // チャット履歴をD1に保存
  const messageId = crypto.randomUUID();
  await env.DB.prepare(
    `INSERT INTO chat_messages
     (id, conversation_id, role, content, tokens_used, created_at)
     VALUES (?, ?, ?, ?, ?, ?)`
  )
    .bind(messageId, conversationId, "user", message, 0, Date.now())
    .run();

  const assistantId = crypto.randomUUID();
  await env.DB.prepare(
    `INSERT INTO chat_messages
     (id, conversation_id, role, content, tokens_used, created_at)
     VALUES (?, ?, ?, ?, ?, ?)`
  )
    .bind(assistantId, conversationId, "assistant", assistantMessage, tokensUsed, Date.now())
    .run();

  return json({
    id: assistantId,
    conversationId,
    content: assistantMessage,
    tokensUsed,
    timestamp: Date.now(),
  });
});

// OPTIONSリクエスト処理（CORS preflight）
router.options("*", () => {
  return new Response(null, {
    headers: corsHeaders,
  });
});

export default {
  fetch: router.handle,
};
```

## Rork Max からの接続

Rork MaxアプリからWorkers APIを呼び出します：

```typescript
// Rork生成コード内での実装例
import { useState } from "react";
import { View, Text, TextInput, Button, ScrollView } from "react-native";

export default function ChatScreen() {
  const [messages, setMessages] = useState<any[]>([]);
  const [input, setInput] = useState("");
  const [loading, setLoading] = useState(false);
  const [token, setToken] = useState("");

  const sendMessage = async () => {
    if (!input.trim()) return;

    setLoading(true);
    try {
      const response = await fetch(
        "https://rork-ai-backend.workers.dev/api/chat",
        {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
            Authorization: `Bearer ${token}`,
          },
          body: JSON.stringify({
            message: input,
            conversationId: "conv_123",
            model: "gpt-3.5-turbo",
          }),
        }
      );

      if (!response.ok) {
        throw new Error(`API Error: ${response.status}`);
      }

      const result = await response.json();
      setMessages((prev) => [
        ...prev,
        { role: "user", content: input },
        { role: "assistant", content: result.content },
      ]);
      setInput("");
    } catch (error) {
      console.error("Chat error:", error);
      alert("メッセージ送信に失敗しました");
    } finally {
      setLoading(false);
    }
  };

  return (
    <View style={{ flex: 1, padding: 16 }}>
      <ScrollView style={{ flex: 1, marginBottom: 16 }}>
        {messages.map((msg, idx) => (
          <View
            key={idx}
            style={{
              padding: 8,
              marginVertical: 4,
              backgroundColor: msg.role === "user" ? "#e3f2fd" : "#f5f5f5",
              borderRadius: 8,
            }}
          >
            <Text style={{ fontWeight: "600" }}>{msg.role}</Text>
            <Text>{msg.content}</Text>
          </View>
        ))}
      </ScrollView>
      <TextInput
        value={input}
        onChangeText={setInput}
        placeholder="メッセージを入力..."
        style={{ borderWidth: 1, padding: 8, marginBottom: 8 }}
      />
      <Button title={loading ? "送信中..." : "送信"} onPress={sendMessage} disabled={loading} />
    </View>
  );
}
```

**Rork MaxでのAPI認証ヒント**: アプリ起動時にバックエンド認証エンドポイントを呼び出してJWTトークンを取得し、AsyncStorageに保存。その後すべてのAPI呼び出しにAutorizationヘッダーとして含めます。

## ストリーミング レスポンス

Workers AI でのストリーミング対応により、ユーザーは即座にテキスト出現を見られます：

```typescript
// ストリーミング対応エンドポイント
router.post("/api/chat/stream", async (request, env) => {
  const userId = await authenticateRequest(request);
  if (!userId) {
    return json({ error: "Unauthorized" }, { status: 401 });
  }

  const { message, conversationId } = await request.json();
  const ai = new Ai(env.AI);

  // ReadableStreamを使用したレスポンス
  const stream = await generateWithWorkersAIStream(ai, message);

  return new Response(stream, {
    headers: {
      "Content-Type": "text/event-stream",
      "Cache-Control": "no-cache",
      Connection: "keep-alive",
      ...corsHeaders,
    },
  });
});
```

Rork Max側でServer-Sent Eventsを受け取る：

```typescript
const eventSource = new EventSource(
  "https://rork-ai-backend.workers.dev/api/chat/stream"
);
eventSource.onmessage = (event) => {
  const chunk = event.data;
  setMessages((prev) => {
    const last = prev[prev.length - 1];
    return [
      ...prev.slice(0, -1),
      { ...last, content: last.content + chunk },
    ];
  });
};
```

## コスト分析

**Workers AI vs OpenAI API:**
- Workers AI: 推論実行 $0.05/100万トークン
- OpenAI API (GPT-3.5): $0.0005/1Kトークン ($0.5/100万トークン)

ただしWorkers AIはエッジで実行されるため、レイテンシが大幅に短く、キャッシング効果も高いです。

**AI Gatewayキャッシング**: 同じプロンプトの50%がキャッシュされた場合、API呼び出しが半減し、月額数千円の削減が期待できます。

**D1 コスト**: 最初の5GBまで無料。超過時は$0.75/GB。チャット履歴が月500万メッセージでも約10MB程度のため、実質無料枠内です。
