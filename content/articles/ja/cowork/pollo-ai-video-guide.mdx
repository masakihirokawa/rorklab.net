## Pollo AI とは

Pollo AI は、テキストや画像から高品質な AI 動画を生成できるオールインワンプラットフォームです。Kling AI、Hailuo AI、Runway、PixVerse、Vidu AI、Luma AI など、業界をリードする複数の動画生成モデルを1つのインターフェースに統合しており、目的やスタイルに合わせて最適なモデルを選択できます。

さらに、Pollo AI 独自のモデル「Pollo 2.5」も搭載されており、高いプロンプト忠実度とコストパフォーマンスが特徴です。2026 年現在、40 種類以上の AI ビデオエフェクトとテンプレートも提供されており、初心者でもプロ品質の動画を手軽に作成できます。

主な生成モードは以下のとおりです。

- **Text to Video**: テキストプロンプトから動画を生成
- **Image to Video**: 静止画に動きを加えて動画化
- **Video to Video**: 既存の動画をスタイル変換
- **AI アニメーション**: キャラクターアニメーションの自動生成
- **一貫性キャラクター**: 同一キャラクターを維持した連続シーン生成

日本語プロンプトにも対応しているため、英語に自信がない方でも利用しやすいのもポイントです。

## なぜ Claude と組み合わせるのか

Pollo AI の動画生成クオリティは入力プロンプトの質に大きく左右されます。Claude と組み合わせることで、以下の 3 つのメリットが得られます。

まず、**プロンプトの精度向上**です。Claude は映像表現に関する豊富な知識を持っているため、曖昧なイメージを具体的なシーン記述に変換できます。カメラアングル、照明、色彩、動きの方向性など、プロの映像制作者が考慮するポイントを網羅したプロンプトを生成します。

次に、**バッチ生成の効率化**です。複数シーンの動画を制作する場合、Claude に全体の構成を設計してもらい、各シーンのプロンプトを一括で生成できます。手作業でプロンプトを書く手間を大幅に削減します。

そして、**API 連携による自動化**です。Claude Code から Pollo AI の API を呼び出すスクリプトを作成すれば、プロンプト生成から動画制作までをプログラムで自動化できます。大量の動画を定期的に制作するワークフローの構築も可能です。

## 基本ワークフロー：Claude でプロンプトを書いて Pollo AI で動画にする

### Step 1: 動画のコンセプトを Claude に伝える

まず、作りたい動画のイメージを Claude に伝えます。漠然としたアイデアでも、Claude が具体的な映像表現に変換してくれます。

```
商品紹介用の 5 秒動画を作りたいです。
白い背景にスニーカーが 360 度回転するような映像をイメージしています。
高級感のある雰囲気で、照明は柔らかいスタジオライトを使ったような感じにしたいです。
```

Claude はこのような入力から、Pollo AI に最適化されたプロンプトを生成します。

### Step 2: Pollo AI 向けプロンプトを生成する

Claude に Pollo AI 用のプロンプトを依頼する際は、以下のポイントを指示に含めると効果的です。

```
以下の条件で Pollo AI 用の動画生成プロンプトを英語で作成してください：
- シーンの構図と主被写体を明確に記述
- カメラの動き（パン、ズーム、回転など）を指定
- 照明条件と色調を指定
- 動きの速度と方向を指定
- 雰囲気やムードを表す形容詞を含める
```

Claude が生成するプロンプト例：

```
A pristine white sneaker slowly rotating 360 degrees on a glossy
white surface, soft studio lighting with subtle reflections,
smooth cinematic rotation, premium product photography style,
clean minimalist background, subtle shadow beneath the shoe,
warm highlights on the leather texture, 4K quality
```

### Step 3: モデルを選択して生成する

プロンプトが完成したら、Pollo AI のウェブインターフェースまたは API で動画を生成します。用途に応じてモデルを使い分けるのがコツです。

**製品プロモーション動画**には Kling AI や Runway が向いています。写実的な表現に優れ、物体の動きを忠実に再現します。**アート系・MV 風の動画**には Hailuo AI がおすすめです。クリエイティブな映像表現が得意で、独特の雰囲気を持つ映像を生成できます。**コスト重視**の場合は Pollo 2.5 が最適です。高いプロンプト忠実度を維持しながら、クレジット消費を抑えられます。

## 実践テクニック：連続シーンの構成

### シーンプランニングをClaude に依頼する

短編動画やプロモーション映像では、複数のシーンを組み合わせる必要があります。Claude にシーン構成を依頼すると、各カットのプロンプトを一括で設計できます。

```
カフェの紹介動画（30 秒、6 シーン構成）のシーンプランを作ってください。
各シーンについて以下を出力してください：
1. シーン番号と尺（秒数）
2. Pollo AI 用プロンプト（英語）
3. 推奨モデル
4. 推奨アスペクト比
```

Claude の出力例：

```
Scene 1 (5 sec): 外観ショット
Prompt: "Warm sunlight illuminating the entrance of a cozy
Japanese cafe, wooden facade with green plants, gentle breeze
moving leaves, inviting atmosphere, golden hour lighting,
slow dolly-in camera movement, cinematic 4K"
Model: Kling AI
Aspect: 16:9

Scene 2 (5 sec): インテリア
Prompt: "Interior of a minimalist Japanese cafe, natural wood
furniture, soft ambient lighting, steam rising from a coffee
cup on the counter, shallow depth of field, slow pan right,
warm color grading, peaceful atmosphere"
Model: Kling AI
Aspect: 16:9

...（以下略）
```

このようにシーンごとの設計をまとめておくと、Pollo AI での生成作業がスムーズに進みます。

## Claude Code × Pollo AI API で自動化する

### API キーの取得

Pollo AI の API プラットフォーム（pollo.ai/api-platform）からアカウントを作成し、API キーを取得します。API ではテキストから動画生成（Text to Video）、画像から動画生成（Image to Video）など、Web UI と同等の機能をプログラムから利用できます。

### Claude Code で生成スクリプトを作成する

Claude Code に API 呼び出しスクリプトの作成を依頼します。

```
Pollo AI の API を使って、テキストプロンプトから動画を生成する
Python スクリプトを書いてください。
以下の機能を含めてください：
- 環境変数からの API キー読み込み
- プロンプトとモデル名を引数で指定
- 生成ステータスのポーリング
- 完成動画のダウンロード
```

以下は Claude Code が生成する基本的なスクリプトの骨組みです。

```python
import os
import time
import requests

API_KEY = os.environ.get("POLLO_API_KEY")
BASE_URL = "https://api.pollo.ai/v1"

def generate_video(prompt: str, model: str = "pollo-2.5",
                   aspect_ratio: str = "16:9",
                   duration: int = 5) -> str:
    """Pollo AI API でテキストから動画を生成する"""
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    }

    # 生成リクエスト送信
    response = requests.post(
        f"{BASE_URL}/video/generate",
        headers=headers,
        json={
            "prompt": prompt,
            "model": model,
            "aspect_ratio": aspect_ratio,
            "duration": duration,
        }
    )
    response.raise_for_status()
    task_id = response.json()["task_id"]

    # ステータスポーリング
    while True:
        status = requests.get(
            f"{BASE_URL}/video/status/{task_id}",
            headers=headers
        ).json()

        if status["state"] == "completed":
            return status["video_url"]
        elif status["state"] == "failed":
            raise RuntimeError(f"生成失敗: {status.get('error')}")

        print(f"生成中... ({status['state']})")
        time.sleep(10)


def download_video(url: str, output_path: str) -> None:
    """生成された動画をダウンロードする"""
    response = requests.get(url, stream=True)
    response.raise_for_status()
    with open(output_path, "wb") as f:
        for chunk in response.iter_content(chunk_size=8192):
            f.write(chunk)
    print(f"保存完了: {output_path}")


if __name__ == "__main__":
    prompt = (
        "A serene Japanese garden with cherry blossoms "
        "gently falling, koi pond reflecting sunlight, "
        "slow camera pan, cinematic 4K quality"
    )

    print("動画を生成中...")
    video_url = generate_video(prompt)
    download_video(video_url, "output.mp4")
```

### バッチ生成パイプラインの構築

複数のプロンプトを一括で処理するパイプラインも Claude Code で構築できます。

```python
import json

def batch_generate(scene_plan_path: str,
                   output_dir: str = "./output") -> list:
    """シーンプランの JSON からバッチで動画を生成する"""
    os.makedirs(output_dir, exist_ok=True)

    with open(scene_plan_path) as f:
        scenes = json.load(f)

    results = []
    for i, scene in enumerate(scenes):
        print(f"\nScene {i + 1}/{len(scenes)}: {scene['description']}")

        video_url = generate_video(
            prompt=scene["prompt"],
            model=scene.get("model", "pollo-2.5"),
            aspect_ratio=scene.get("aspect_ratio", "16:9"),
            duration=scene.get("duration", 5),
        )

        output_path = os.path.join(
            output_dir, f"scene_{i + 1:02d}.mp4"
        )
        download_video(video_url, output_path)
        results.append(output_path)

    print(f"\n全 {len(results)} シーンの生成が完了しました")
    return results
```

シーンプランは以下のような JSON 形式で準備します。

```json
[
  {
    "description": "カフェ外観",
    "prompt": "Warm sunlight illuminating a cozy cafe entrance...",
    "model": "kling-ai",
    "aspect_ratio": "16:9",
    "duration": 5
  },
  {
    "description": "インテリア",
    "prompt": "Interior of a minimalist cafe with natural lighting...",
    "model": "kling-ai",
    "aspect_ratio": "16:9",
    "duration": 5
  }
]
```

この JSON ファイルの作成も Claude に依頼できるので、テーマを伝えるだけで生成パイプライン全体を構築できます。

## プロンプトエンジニアリングのコツ

Pollo AI で高品質な動画を生成するためのプロンプト設計には、いくつかの重要なポイントがあります。Claude にプロンプトを依頼する際に、これらのポイントを踏まえた指示を出すと結果が大きく向上します。

### 構図の明確化

「何が」「どこに」「どのように」あるかを具体的に記述します。曖昧な表現は避け、カメラからの距離感や被写体の配置を示す言葉を使いましょう。

良い例: `A single red rose in a glass vase, centered in frame, placed on a white marble table`

悪い例: `A flower on a table`

### カメラの動きを指定する

静止ショットか動きのあるショットかを明示することで、生成される動画の品質が安定します。

使用できるカメラワーク表現: `slow dolly-in`（ゆっくりカメラが近づく）、`smooth pan right`（滑らかに右にパン）、`gentle zoom out`（穏やかにズームアウト）、`360 orbit`（360度回転）、`static shot`（固定ショット）、`tracking shot`（追跡ショット）。

### 照明と色調を指定する

映像の雰囲気を大きく左右するのが照明と色彩です。

照明の種類: `golden hour sunlight`（夕方の暖かい光）、`soft studio lighting`（スタジオの柔らかい光）、`dramatic side lighting`（ドラマチックなサイドライト）、`neon glow`（ネオンの輝き）。

色彩: `warm color grading`（暖色系カラーグレーディング）、`cool blue tones`（クールな青色トーン）、`desaturated cinematic`（彩度を抑えたシネマティック）。

### 解像度とスタイルキーワード

最後にクオリティとスタイルを示すキーワードを加えます: `cinematic 4K`、`photorealistic`、`anime style`、`oil painting style`、`slow motion` など。

## 料金体系とモデル別コスト

Pollo AI はクレジット制を採用しています。無料プランでは毎日一定のクレジットが付与され、有料プランでは追加クレジットの購入が可能です。

モデルによって消費クレジットが異なるため、用途に応じて使い分けることが重要です。プロトタイプや検証段階では低コストの Pollo 2.5 を使い、本番用の動画には Kling AI や Runway を選ぶという戦略が効率的です。

Claude に「低コストで効果的なモデルとパラメータの組み合わせを提案して」と依頼すると、予算内で最大の効果を得るための戦略を立ててくれます。

## Cowork での活用シーン

Claude Desktop の Cowork モードでは、ファイル操作とブラウザ操作を組み合わせた統合ワークフローが可能です。

**プレゼン素材の動画化**では、PowerPoint のスライド内容を Claude に読み込ませ、各スライドに合ったプロモーション動画のプロンプトを生成し、Pollo AI で動画を作成してプレゼンに挿入する、という一連の作業を Cowork 内で完結できます。

**SNS コンテンツの量産**では、商品写真のフォルダを Cowork に読み込ませ、Claude が各画像に最適な動画プロンプトを設計し、Pollo AI の Image to Video で動画化するバッチワークフローを構築できます。

**ブランドガイドラインに基づく統一動画**では、ブランドカラー、フォント、トーンを CLAUDE.md に記述しておくことで、すべての動画プロンプトにブランドの世界観を反映させることができます。

## よくある質問（FAQ）

### Pollo AI は無料で使えますか？

はい、無料プランが用意されています。毎日付与されるクレジットの範囲内で動画を生成できます。ただし、高品質なモデルや長尺の動画にはより多くのクレジットが必要なため、本格的な制作には有料プランの検討をおすすめします。

### 日本語プロンプトは使えますか？

Pollo AI は日本語プロンプトに対応しています。ただし、多くの動画生成 AI は英語プロンプトで最も高い精度を発揮します。Claude に英語プロンプトの生成を依頼すれば、日本語でイメージを伝えながら英語の最適化されたプロンプトを得られるので、両方の利点を活かせます。

### 生成した動画の著作権はどうなりますか？

Pollo AI で生成した動画の使用権はプランによって異なります。有料プランでは商用利用が可能です。詳細は Pollo AI の利用規約を確認してください。

### Claude Code で API 連携するにはどうすればよいですか？

Pollo AI の API プラットフォーム（pollo.ai/api-platform）でアカウント登録後に API キーを取得し、環境変数 `POLLO_API_KEY` に設定します。本記事の「Claude Code × Pollo AI API で自動化する」セクションにサンプルコードを掲載していますので、参考にしてください。

Pollo AI と Claude の組み合わせは、AI 動画制作のワークフローを大幅に効率化します。Claude が得意とするプロンプト設計・構成立案の能力と、Pollo AI が提供する多彩な動画生成モデル群を掛け合わせることで、初心者でもプロ品質の動画コンテンツを制作できます。

まずは Claude にイメージを伝えてプロンプトを生成してもらうところから始めてみてください。慣れてきたら API 連携やバッチ生成パイプラインの構築にも挑戦し、動画制作ワークフローの全自動化を目指しましょう。
