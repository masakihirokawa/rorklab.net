# Cowork × Excel / PowerPoint 連携実践

Cowork モードでは、Claude がローカル環境で直接ファイルを操作できるため、Excel スプレッドシートの作成・分析や PowerPoint プレゼンテーションの自動生成が可能です。このガイドでは、実践的なユースケースとテクニックを紹介します。

## Cowork の Office ファイル対応

Cowork は軽量な Linux VM 上で動作し、Python ライブラリを活用して Office ファイルを直接操作できます。主に以下のライブラリが使用されます：

- **openpyxl**: Excel (.xlsx) ファイルの読み書き
- **python-pptx**: PowerPoint (.pptx) ファイルの作成・編集
- **pandas**: データ分析と Excel 連携

## Excel 連携の実践

### スプレッドシートの作成

Cowork に「売上レポートの Excel を作って」と依頼するだけで、フォーマット済みのスプレッドシートが生成されます。

依頼例：

```
月次売上データを元に、Excel レポートを作成してください。
ヘッダーには色付きの書式を適用し、合計行と前月比の列を追加してください。
```

Cowork は以下のような処理を自動で行います：

- データの構造化とセルへの配置
- ヘッダーの書式設定（フォント、背景色、罫線）
- 数式の挿入（SUM、前月比計算など）
- 列幅の自動調整
- 条件付き書式の適用

### データ分析と可視化

既存の Excel ファイルを Cowork に渡して分析を依頼できます：

```
このExcelファイルの売上データを分析して、
月別のトレンドグラフと上位10商品のランキング表を追加してください。
```

Cowork が行える分析と可視化：

- ピボットテーブルの作成
- 折れ線グラフ、棒グラフ、円グラフの挿入
- 統計値の計算（平均、中央値、標準偏差）
- データの並べ替えとフィルタリング
- 条件付き書式によるハイライト

### CSV からの変換

CSV ファイルを書式付き Excel に変換するのも簡単です：

```
このCSVファイルをExcelに変換してください。
テーブル形式にして、数値列にはカンマ区切りの書式を適用してください。
```

## PowerPoint 連携の実践

### プレゼンテーションの自動生成

Cowork はテーマに基づいてプレゼンテーションを一から作成できます：

```
「2026年Q1事業報告」のPowerPointを作成してください。
タイトルスライド、目次、業績サマリー、部門別実績、今後の計画の
5セクション構成でお願いします。
```

生成されるスライドには以下が含まれます：

- タイトルスライドとセクション区切り
- テキストコンテンツの自動配置
- 表やグラフの挿入
- 統一されたデザインテーマ
- スピーカーノートの追加

### 既存スライドの編集

既存の PowerPoint ファイルを編集することも可能です：

```
このPowerPointの3ページ目のグラフデータを更新してください。
また、最後にまとめスライドを追加してください。
```

対応する編集操作：

- テキストの修正・追加
- スライドの追加・削除・並べ替え
- 画像の挿入・差し替え
- テーブルの更新
- デザインの調整

### データから直接スライド生成

Excel データや CSV から直接グラフ入りスライドを生成できます：

```
この売上データのExcelから、経営会議用のプレゼンテーションを作成してください。
主要KPIのダッシュボードスライドと部門別の詳細スライドを含めてください。
```

## 実践ワークフロー例

### 例1: 月次レポートの自動化

1. 「先月の売上CSVから月次レポートExcelを作成して」と依頼
2. Cowork がデータを読み込み、書式付き Excel を生成
3. グラフとサマリーが自動的に追加される
4. 完成したファイルをダウンロード

### 例2: 会議資料の準備

1. 議題と要点を Cowork に伝える
2. PowerPoint スライドが自動生成される
3. 「3ページ目にグラフを追加して」など修正を指示
4. 完成したプレゼンテーションをダウンロード

### 例3: データ集約と報告

1. 複数の Excel ファイルを Cowork に渡す
2. 「これらのデータを集約してサマリーレポートを作成して」
3. 統合された Excel とプレゼン資料が生成される

## Claude in Excel アドイン

Cowork とは別に、Claude in Excel アドインも利用可能です。これは Excel 内で直接 Claude を使用できる機能で、以下の操作が可能です：

- セル内でのデータ分析と要約
- ピボットテーブルの編集
- 条件付き書式の設定
- 数式の提案と修正

Claude in Excel は Opus 4.6 モデルを使用し、Excel のネイティブ操作をサポートしています。

## Tips とベストプラクティス

### 効果的な依頼のコツ

- **具体的に指示する**: 「きれいに」ではなく「ヘッダーは青背景白文字、数値は右揃え」と指定
- **完成イメージを伝える**: 誰が見るのか、何を伝えたいのかを説明
- **段階的に作成する**: 複雑なファイルは一度に全て依頼せず、段階的に修正
- **フォルダを選択する**: 作業フォルダを事前に選択しておくと、ファイルの読み書きがスムーズ

### 注意点

- 大きなファイル（数万行以上）は処理に時間がかかる場合がある
- マクロ（VBA）を含む .xlsm ファイルは、マクロ部分の編集に制限がある
- PowerPoint のアニメーション設定は限定的なサポート
- フォントは環境に依存するため、特殊フォントが必要な場合は注意

## まとめ

Cowork の Excel / PowerPoint 連携を活用することで、定型的なオフィス業務を大幅に効率化できます。CSV やデータの分析から、書式付きレポートの生成、プレゼンテーションの自動作成まで、自然言語の指示だけで実現できるのが Cowork の強みです。日常業務の中で少しずつ試してみてください。
