# Cowork で繰り返しタスクを自動化する実践テクニック

Claude Desktop の **Cowork モード**は、2026年1月に発表されたリサーチプレビュー機能です。Claude Code のエージェント能力をコーディング以外のナレッジワークに拡張し、ドキュメント作成、ファイル整理、リサーチの統合といった日常業務を「結果だけ伝えて、あとはお任せ」というスタイルで処理できます。

この記事では、Cowork の中でも特に実務で威力を発揮する**スキル**と**スケジュールタスク**の二つの機能にフォーカスし、繰り返しの作業をどう自動化するかを具体的に解説します。

## Cowork モードとは何か

Cowork は Claude Desktop 上で動作する、いわば「デジタルな同僚」です。通常のチャットとは異なり、以下のような特徴があります。

**ローカルファイルへの直接アクセス。** フォルダを選択すれば、Claude がその中のファイルを読み書きできます。手動でのアップロードやダウンロードは不要です。

**サブエージェントによる並列処理。** 複雑な作業を小さなタスクに分割し、それぞれを並行して実行します。大量のファイル処理やリサーチの横断検索が高速に完了します。

**プロフェッショナルな成果物。** 数式付きの Excel スプレッドシート、フォーマットの整った PowerPoint プレゼンテーション、レイアウト済みの Word ドキュメントなど、そのまま提出できる品質のファイルを生成します。

**長時間タスクへの対応。** 通常の会話セッションにはないレベルで、長時間にわたる複雑な処理を中断なく続けられます。

## スキル — Claude に「やり方」を教える

### スキルとは

スキルは、Claude に繰り返し可能なワークフローを教えるための仕組みです。特定のタスクにおけるベストプラクティスを `SKILL.md` ファイルとして定義すると、Claude がそのルールに従って作業を行います。

たとえば、Word ドキュメントの作成には `docx` スキル、プレゼンテーションの作成には `pptx` スキル、PDF の処理には `pdf` スキルがあらかじめ用意されています。Claude は作業の種類を認識すると、該当するスキルファイルを自動で読み込んでから作業に着手します。

### プリインストールされた主要スキル

Cowork には最初からいくつかのスキルが搭載されています。

**ドキュメント系スキル。** `docx`（Word文書）、`pptx`（プレゼンテーション）、`xlsx`（スプレッドシート）、`pdf`（PDF処理）の4つが基本セットです。いずれもファイルの新規作成、読み取り、編集に対応しています。

**フロントエンドデザイン。** `frontend-design` スキルを使うと、HTML / CSS / React のコンポーネントを洗練されたデザインで生成できます。ランディングページやダッシュボードの構築に威力を発揮します。

**SEO 監査。** `seo-audit` スキルはウェブサイトの技術的な SEO 問題を診断し、改善提案を出してくれます。

**スキルファインダー。** 必要なスキルが見つからないときは `find-skills` スキルで検索できます。「〇〇ができるスキルはある？」と聞くだけで、インストール可能なスキルを探してくれます。

### カスタムスキルの作り方

既存のスキルだけでは足りない場合、自分だけのスキルを作ることもできます。`skill-creator` スキルを使うか、手動で以下の構造のフォルダを作成します。

```
skills/
  my-custom-skill/
    SKILL.md
```

`SKILL.md` の中には、Claude にどのように振る舞ってほしいかを自然言語で記述します。ポイントは次の3つです。

**トリガー条件を明確にする。** どんなキーワードや文脈でこのスキルが発動すべきかを書きます。「ユーザーが『レポート作成』と言ったら」「`.csv` ファイルが添付されたら」といった具合です。

**手順をステップバイステップで。** 曖昧な指示ではなく、具体的な手順を書きます。「まずデータを読み取り、次に集計し、最後にグラフを生成する」のように。

**出力フォーマットを指定する。** 成果物のファイル形式、命名規則、保存場所などを明記します。

## スケジュールタスク — 繰り返し作業を「仕掛けて放置」

### 概要

2026年に追加されたスケジュールタスク機能は、Cowork の自動化能力を次のレベルに引き上げます。タスクの内容を一度記述し、実行頻度を設定すれば、あとは Cowork が自動で処理を回してくれます。

毎日、毎週、平日のみ、毎時間、あるいはオンデマンドなど、柔軟なスケジュール設定が可能です。各スケジュールタスクは独立した Cowork セッションとして起動し、接続済みのツール、プラグイン、MCP サーバーのすべてにアクセスできます。

### 設定方法

スケジュールタスクの設定は2つの方法があります。

**チャットから設定する場合。** 任意のタスク内で `/schedule` と入力すると、Claude が対話形式で設定をガイドしてくれます。タスク名、説明、プロンプト、頻度を順番に聞いてくるので、答えていくだけで完了します。

**サイドバーから設定する場合。** サイドバーの「Scheduled」をクリックし、新規タスクを作成します。名前、説明、プロンプト、頻度、モデル、作業フォルダを自分で指定できます。

### cron 式の基本

スケジュールの頻度は cron 式で指定します。よく使うパターンをいくつか紹介します。

```
0 9 * * *       毎日 9:00
0 9 * * 1-5     平日 9:00
30 8 * * 1      毎週月曜 8:30
0 0 1 * *       毎月1日 0:00
0 */3 * * *     3時間ごと
```

cron 式の各フィールドは「分 時 日 月 曜日」の順です。`*` は「すべて」を意味し、`1-5` は「1から5」、`*/3` は「3ごと」というように書きます。タイムゾーンはマシンのローカル時間が使われるため、UTC変換は不要です。

### 実践例

**日次レポートの自動生成。** 毎朝9時に特定のフォルダ内のデータファイルを集計し、サマリーレポートを生成する。売上データの日次集計や、プロジェクト進捗の自動まとめに活用できます。

**定期的なファイル整理。** 週に一度、ダウンロードフォルダ内のファイルを種類別にサブフォルダへ振り分ける。デスクトップやダウンロードフォルダが散らかりがちな人におすすめです。

**コンテンツの定期チェック。** ウェブサイトのコンテンツを定期的にチェックし、更新が必要な箇所をリストアップする。ブログやドキュメントサイトの運営に便利です。

**バックアップと整理。** 重要なドキュメントの定期バックアップを作成し、日付付きのアーカイブフォルダに整理する。

## プラグイン — スキルをさらに拡張する

2026年1月末にリリースされたプラグイン機能は、スキル・コネクタ・スラッシュコマンド・サブエージェントをひとまとめにしたパッケージです。

### プリビルトプラグイン

ローンチ時点で11のオープンソースプラグインが公開されています。生産性向上、エンタープライズ検索、マーケティング、セールスなど、主要なビジネス領域をカバーしています。

### プラグインの導入方法

Cowork の設定画面からインストールするか、カスタムプラグインをアップロードすることもできます。さらに `Plugin Create` プラグインを使えば、新しいプラグインを対話形式で構築できます。

## コネクタで外部サービスと連携する

Cowork は MCP（Model Context Protocol）を通じて、さまざまな外部サービスと接続できます。

**Google 系。** Gmail、Google Drive との連携により、メールの確認やドキュメントの読み書きが可能です。

**Microsoft 系。** Microsoft 365 との連携で、Excel や PowerPoint を横断したワークフローを構築できます。

**プロジェクト管理。** Asana、Slack、Notion など、チームで使うツールとの統合もサポートされています。

**デザインツール。** Canva との連携で、チャットからデザインの作成・編集・エクスポートまで完結します。

コネクタの追加は、Cowork セッション内で Claude に依頼するか、設定の「Customize」セクションから行えます。

## 効果的に使うための5つのヒント

### 1. 具体的に指示する

「いい感じにまとめて」ではなく、「A4縦で、見出しは3階層まで、フォントは游ゴシックで」のように具体的に指示しましょう。Claude は曖昧な指示でもそれなりに動きますが、具体的であるほど期待通りの結果が返ってきます。

### 2. フォルダ構成を先に整える

作業フォルダ内のファイル構成が整理されていると、Claude の処理精度が上がります。入力ファイルと出力先を分けておくだけでも効果があります。

### 3. スキルを育てる

最初に完璧なスキルを作る必要はありません。使いながら `SKILL.md` を修正し、少しずつ精度を上げていくアプローチが実践的です。

### 4. スケジュールタスクは小さく始める

いきなり複雑な自動化を組むのではなく、まずは単純なタスク（フォルダ内のファイルをリストアップするなど）から始めて、動作を確認してから拡張していきましょう。

### 5. プレビューで確認する癖をつける

ファイルの編集や削除を伴うタスクでは、実行前にプレビューや確認ステップを挟むようにしましょう。Cowork は安全設計がされていますが、自分の目で確認する習慣は大切です。

## まとめ

Cowork のスキルとスケジュールタスクを組み合わせることで、これまで手作業で行っていた繰り返しタスクの多くを自動化できます。最初の設定には少し時間がかかりますが、一度仕組みを作ってしまえば、毎日の作業時間を大幅に短縮できるはずです。

まずは身近な繰り返し作業を一つ選んで、Cowork に任せてみてください。「こんなことまでできるのか」という驚きが、きっとあるはずです。

Cowork は現在リサーチプレビューとして、Pro・Max・Team・Enterprise プランで利用可能です。macOS と Windows の Claude Desktop アプリから使えます。
