## Rork Max とは

Rork Max は、2026年2月にリリースされた **AI 駆動のネイティブ Swift アプリビルダー**です。テキストプロンプトを入力するだけで、ネイティブの Swift / SwiftUI アプリが生成されます。

従来の Rork が React Native（クロスプラットフォーム）ベースだったのに対し、Rork Max は Apple エコシステムに特化。AR、Metal グラフィックス、ウィジェットなど、React Native では実現できなかったネイティブ機能をフルに活用できるのが最大の特徴です。

<Callout type="info">
**Rork Max の注目ポイント:**
- AI がネイティブ Swift アプリを自動生成する唯一のビルダー
- ブラウザだけで完結 — Xcode も Mac も不要
- ローンチ3日で ARR $1.5M を達成
- 単一プロンプトで AR 対応のポケモン GO クローンを構築したデモが話題に
</Callout>

## 対応プラットフォーム

Rork Max は Apple エコシステムの主要プラットフォームすべてに対応しています。

### iPhone / iPad（iOS / iPadOS）

最も基本的なターゲットです。SwiftUI ベースの UI で、iPhone と iPad 両方に最適化されたレイアウトを自動生成します。

### Apple Watch（watchOS）

ヘルスケアアプリ、通知アプリ、コンパニオンアプリなど、watchOS 向けのアプリを生成できます。HealthKit との連携もネイティブでサポートされています。

### Apple TV（tvOS）

大画面向けのメディアアプリ、ゲーム、ストリーミングインターフェースを構築できます。tvOS の Focus Engine に対応したナビゲーションが自動的に実装されます。

### Vision Pro（visionOS）

空間コンピューティング向けのアプリを生成できます。3D オブジェクトの配置、イマーシブな AR 体験、ウィンドウベースのインターフェースをプロンプトだけで構築可能です。

### iMessage アプリ

iMessage 内で動作するステッカーパック、ミニゲーム、インタラクティブなコンテンツを生成できます。

## ネイティブ機能へのアクセス

Rork Max の最大の強みは、Apple プラットフォームのネイティブ API にフルアクセスできることです。

### ARKit / LiDAR スキャン

```
プロンプト例:
「部屋の家具を AR で配置確認できるアプリを作って。
LiDAR で部屋をスキャンし、3D モデルをリアルに配置する」
```

ARKit と LiDAR センサーを活用した拡張現実アプリを生成します。家具の配置シミュレーション、AR ゲーム、3D スキャンアプリなどが数分で構築可能です。

### Metal グラフィックス

```
プロンプト例:
「3D のフライトシミュレーターを作って。
Metal で滑らかな 3D レンダリングを実現して」
```

Apple の Metal API を使った高性能な 3D グラフィックスに対応。ゲーム、ビジュアライゼーション、クリエイティブツールなどに活用できます。

### ホーム画面ウィジェット / Dynamic Island / Live Activities

```
プロンプト例:
「天気アプリを作って。ホーム画面ウィジェット、
Dynamic Island での気温表示、
Live Activities での天気アラートに対応させて」
```

iOS 16 以降のウィジェットシステムと、iPhone 14 Pro 以降の Dynamic Island、iOS 16.1 以降の Live Activities をサポートします。

### Siri Intents

```
プロンプト例:
「タスク管理アプリを作って。
Siri で『タスクを追加して』と言うとタスクが追加される機能を付けて」
```

Siri ショートカットとの統合により、音声コマンドでアプリの機能を呼び出せます。

### HealthKit / HomeKit / NFC

- **HealthKit**: 歩数、心拍数、睡眠データなどにアクセス
- **HomeKit**: スマートホームデバイスの制御
- **NFC**: NFC タグの読み書き、非接触決済

### Core ML

```
プロンプト例:
「写真から食べ物を認識してカロリーを推定するアプリを作って。
Core ML でオンデバイス推論を行って」
```

Apple の機械学習フレームワーク Core ML を使い、デバイス上で高速な AI 推論を実行できます。画像分類、自然言語処理、音声認識などに対応。

### App Clips

インストール不要の小さなアプリ体験を構築できます。QR コードや NFC タグからすぐに起動する、軽量なアプリです。

## 2クリックで App Store に公開

Rork Max の最も革新的な機能は、**ブラウザだけで App Store への公開が完結する**ことです。

### 従来のワークフロー

1. Mac を購入（10万円〜）
2. Xcode をインストール（数GB）
3. Apple Developer Program に登録（$99/年）
4. コードを書く
5. Provisioning Profile、Certificate を設定
6. Archive → App Store Connect にアップロード
7. 審査待ち

### Rork Max のワークフロー

1. ブラウザでプロンプトを入力
2. クラウド Mac でネイティブコンパイル
3. ストリーミングシミュレータでリアルタイムプレビュー（タッチ操作可能）
4. QR コードでデバイスにインストールして実機テスト
5. **2クリックで App Store に提出**

<Callout type="warning">
**Apple Developer Program は必要:** App Store に公開するには Apple Developer Program のメンバーシップ（$99/年）が必要です。ただし、Xcode のダウンロードや Mac の所有は不要です。
</Callout>

## Rork Max × Claude Code 連携

Rork Max で生成したアプリを Claude Code でさらにブラッシュアップできます。

### エクスポートと準備

Rork Max から Swift プロジェクトをエクスポートし、ローカルに展開します。

```bash
# プロジェクトを開く
cd MyRorkMaxApp
git init

# CLAUDE.md を作成してコンテキストを提供
claude "Create a CLAUDE.md file describing this Swift project"

# Claude Code で開発開始
claude .
```

### Claude Code でできる改善

```bash
# MVVM アーキテクチャの適用
claude "Refactor to MVVM architecture with proper view models"

# Swift Concurrency の導入
claude "Convert all async operations to use Swift Concurrency (async/await)"

# テストスイートの生成
claude "Generate XCTest test suite with at least 70% coverage"

# Core Data でデータ永続化
claude "Add Core Data persistence layer for user data"

# アクセシビリティの改善
claude "Audit and improve VoiceOver accessibility for all screens"

# パフォーマンスの最適化
claude "Profile and optimize: reduce memory usage, optimize image loading"
```

### SwiftData との統合

iOS 17 以降の新しいデータ永続化フレームワーク SwiftData を、Claude Code で追加できます。

```bash
claude "Replace Core Data with SwiftData for modern persistence.
Use @Model macro, @Query property wrapper, and modelContainer"
```

## 実践ワークフロー

### ステップ 1: アイデアからプロトタイプ（30分）

Rork Max にプロンプトを入力。

```
「ランニング記録アプリを作って。
- GPS でルートを記録
- HealthKit で心拍数を取得
- Apple Watch のコンパニオンアプリ
- ホーム画面ウィジェットに今週の走行距離を表示
- ダークモード対応」
```

### ステップ 2: シミュレータで検証（15分）

ストリーミングシミュレータでアプリを操作。タッチ操作がリアルタイムで反映されるので、実際のユーザー体験を確認できます。

### ステップ 3: 実機テスト（15分）

QR コードをスキャンして iPhone にインストール。GPS やセンサーを使った機能を実機で検証します。

### ステップ 4: Claude Code でブラッシュアップ（2〜3時間）

エクスポートした Swift プロジェクトを Claude Code で改善。ビジネスロジックの追加、テスト作成、パフォーマンス最適化を行います。

### ステップ 5: App Store に公開（30分）

Rork Max の2クリック公開機能で App Store に提出。審査は通常24〜48時間です。

<Callout type="success">
**トータル所要時間:** アイデアから App Store 提出まで、約半日で完了できます。
</Callout>

## Rork（React Native）vs Rork Max（Swift）

| 項目 | Rork（React Native） | Rork Max（Swift） |
|------|---------------------|-------------------|
| プラットフォーム | iOS + Android | Apple のみ（iPhone, iPad, Watch, TV, Vision Pro） |
| 言語 | JavaScript / TypeScript | Swift / SwiftUI |
| ネイティブ機能 | 限定的 | フルアクセス（ARKit, Metal, HealthKit 等） |
| ウィジェット | 非対応 | 対応 |
| Dynamic Island | 非対応 | 対応 |
| Vision Pro | 非対応 | 対応 |
| App Store 公開 | 可能（Expo 経由） | 2クリック |
| 料金 | 無料〜$30/月 | 無料〜$200/月 |
| Android 対応 | ✅ | ❌ |

**選び方のポイント:** Android ユーザーもターゲットにするならRork（React Native）、Apple エコシステムに集中してネイティブ機能をフル活用するなら Rork Max を選びましょう。

## トラブルシューティング

### 生成されたアプリが期待通りでない場合

プロンプトをより具体的にしましょう。画面構成、データモデル、使用する Apple フレームワークを明記すると精度が上がります。

### コンパイルエラーが発生する場合

Rork Max のクラウド Mac で自動コンパイルされるため、通常はエラーが少ないですが、複雑な機能の組み合わせでは発生することがあります。プロンプトを分割して段階的に機能を追加すると安定します。

### App Store の審査に落ちた場合

Apple のガイドライン違反が原因です。Claude Code で以下を確認しましょう：

```bash
claude "Review this app for Apple App Store guideline compliance.
Check: privacy policy, permissions usage descriptions,
content guidelines, and minimum functionality requirements"
```

## まとめ

Rork Max は、ネイティブ Swift アプリ開発の敷居を劇的に下げるツールです。プログラミング経験がなくても、テキストプロンプトだけで ARKit、Metal、HealthKit を活用したアプリを構築し、App Store に公開できます。

Claude Code と組み合わせることで、生成されたコードをプロダクション品質にブラッシュアップし、テスト・最適化まで完了させる「AI × AI」のワークフローが実現します。

## FAQ

### Rork Max を使うのに Mac は必要ですか？

いいえ。すべてブラウザ上で完結します。コンパイルはクラウド上の Mac で行われるため、Windows や Linux からでも利用可能です。

### Android アプリは作れますか？

Rork Max は Apple プラットフォーム専用です。Android アプリが必要な場合は、React Native ベースの従来の Rork を使いましょう。

### 無料プランでどこまでできますか？

週5プロンプト程度でお試しできます。本格的な開発には Max プラン（$200/月）が必要です。

### Claude Code なしでも使えますか？

はい。Rork Max 単体でアプリの生成から App Store 公開まで完結できます。ただし、ビジネスロジックの追加やコード品質の向上には Claude Code の併用を強くおすすめします。

### Vision Pro のアプリも本当に作れますか？

はい。visionOS 向けの空間コンピューティングアプリを生成できます。3D オブジェクト配置、イマーシブ体験、ウィンドウベースのインターフェースに対応しています。
