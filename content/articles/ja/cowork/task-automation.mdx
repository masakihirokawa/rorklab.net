## Cowork とは

Cowork は Claude のデスクトップ機能で、ファイル操作やタスクの自動化に特化したツールです。コードを書かなくても、自然言語で指示するだけで複雑なファイル処理やデータ変換を実行できます。

<Callout type="info">
Cowork は Claude Desktop アプリの機能として提供されています。現在はリサーチプレビューとして利用可能です。
</Callout>

## 基本的な使い方

Cowork では、ローカルのフォルダを選択して Claude にアクセス権を付与します。Claude はそのフォルダ内のファイルを読み書きできるようになり、さまざまなタスクを自動で処理します。

### できること

- **ファイル整理** — フォルダ内のファイルを種類別に分類
- **データ変換** — CSV から Excel、Markdown から Word などの形式変換
- **レポート生成** — データを分析して文書やプレゼンテーションを自動生成
- **コード実行** — Python スクリプトの作成・実行
- **画像処理** — リサイズ、フォーマット変換、メタデータ抽出

## スケジュールタスク

Cowork では定期的に実行するタスクをスケジュールできます。cron 形式で実行間隔を指定します。

### スケジュールの設定例

| 用途 | cron 式 | 説明 |
|------|---------|------|
| 毎朝の日報準備 | `0 9 * * 1-5` | 平日9:00に実行 |
| 週次レポート | `0 17 * * 5` | 毎週金曜17:00 |
| 日次バックアップ | `0 0 * * *` | 毎日深夜0:00 |
| 月次集計 | `0 10 1 * *` | 毎月1日の10:00 |

<Callout type="tip">
cron 式はローカルマシンのタイムゾーンで評価されます。UTC変換の必要はありません。
</Callout>

## スキルシステム

スキルは Cowork の機能を拡張する仕組みです。特定のタスクに最適化されたプロンプトと処理手順がまとめられています。

### 組み込みスキルの例

- **docx** — Word 文書の作成・編集
- **xlsx** — Excel スプレッドシートの操作
- **pptx** — PowerPoint プレゼンテーション作成
- **pdf** — PDF の生成・結合・分割
- **frontend-design** — フロントエンド UI の構築

### スキルの使い方

スキルは自動的に発動します。例えば「Excelでまとめて」と言えば xlsx スキルが、「プレゼン資料を作って」と言えば pptx スキルが自動的に読み込まれます。

## 実践例

### 1. ダウンロードフォルダの整理

```
ダウンロードフォルダ内のファイルを以下のルールで整理してください：
- PDF → Documents/PDF/
- 画像（jpg, png, gif） → Pictures/
- Excel/CSV → Documents/Data/
- それ以外 → Documents/Other/
```

### 2. CSVデータの分析レポート

```
sales_data.csv を分析して、以下を含むレポートをExcelで作成してください：
- 月別売上の推移グラフ
- トップ10製品の売上ランキング
- 前年同月比の比較テーブル
```

### 3. 定期メール下書きの自動生成

```
毎週金曜日の16:00に、今週の作業ログ（work_log.md）を要約して、
週次レポートのメール下書きをMarkdownで生成してください。
```

## コネクタとの連携

Cowork は MCP コネクタを通じて外部サービスとも連携できます：

- **Slack** — メッセージの検索や要約
- **Asana / Jira** — タスク管理との連携
- **Google Drive** — ファイルの読み書き
- **GitHub** — リポジトリの操作

<Callout type="warning">
コネクタの利用には各サービスの認証が必要です。Cowork の設定画面からコネクタを接続してください。
</Callout>

## Chrome 連携

Cowork は Claude in Chrome と組み合わせることで、ブラウザ操作の自動化も可能です。ウェブサイトの情報収集、フォーム入力、スクリーンショット取得などが自然言語で指示できます。
