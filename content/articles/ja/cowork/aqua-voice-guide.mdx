## Aqua Voice とは

Aqua Voice は、Mac と Windows に対応した AI 音声入力ツールです。Y Combinator（W24）出身のスタートアップが開発しており、独自の音声認識モデル「Avalon」を搭載しています。

一般的な音声入力ツールとの最大の違いは、**画面のコンテキストを読み取って認識精度を高める**点です。プログラミング用語やテクニカルタームの認識精度は Siri や Google 音声入力の約 17 倍とされており、開発者やテクニカルライターに特に人気があります。

## なぜ Claude と相性が良いのか

Claude の操作は、Cowork でも Claude Code でも「テキスト入力」が基本です。複雑な指示を出すほど、タイピング量は増えていきます。

Aqua Voice を使えば、カーソルがあるテキストフィールドに直接音声でテキストが入力されます。つまり、Cowork のチャット欄でも、Claude Code のターミナルでも、ブラウザの入力欄でも、そのまま使えるということです。

話すスピードは平均 150 語/分（タイピングの約 3〜4 倍）なので、長いプロンプトを書くときほど時間短縮の効果が大きくなります。

## セットアップ

### 1. インストール

公式サイト（aquavoice.com）からダウンロードしてインストールします。Mac（Apple Silicon / Intel）と Windows 10/11 に対応しています。

### 2. 起動と基本操作

インストール後、メニューバー（Mac）またはタスクトレイ（Windows）に常駐します。設定したショートカットキーを押すだけで音声入力が開始されます。

デフォルトのショートカットキーはカスタマイズ可能です。起動は 50ms 以下、テキスト挿入は約 1 秒（最速 450ms）と非常に高速です。

### 3. カスタム辞書の設定

技術用語や固有名詞をカスタム辞書に登録できます。たとえば、以下のような用語を登録しておくと認識精度が上がります。

- `Claude Code` — AI コーディングツール名
- `Cowork` — Claude デスクトップ機能名
- `MCP` — Model Context Protocol の略
- `SKILL.md` — スキル定義ファイル名

## 実践的な使い方

### Cowork での活用

Cowork のチャット欄にカーソルを合わせて Aqua Voice を起動し、指示を話しかけるだけです。

たとえば、「このフォルダ内の PDF をすべて読み込んで、内容を要約した Excel ファイルを作成してください」といった長い指示も、話すだけで入力できます。

Aqua Voice は文脈に応じて句読点や改行を自動的に挿入するため、話した内容がそのまま整ったテキストとして入力されます。

### Claude Code での活用

Claude Code はターミナル上で動作するため、ターミナルのプロンプト入力欄にそのまま音声入力できます。

コマンドやファイルパスなどのテクニカルタームも、画面コンテキストの読み取りにより高精度で認識されます。`useEffect`、`kubectl`、`PyTorch` といったプログラミング用語の認識精度は 97% を達成しています。

### ペーストショートカット

最後に入力したテキストを再度ペーストするショートカットも用意されています。

- **Mac**: `Cmd + Ctrl + V`
- **Windows**: `Ctrl + Shift + →`

同じ指示を別のウィンドウに送りたいときに便利です。

## Claude 組み込み音声モードとの比較

Claude にはすでにいくつかの音声入力手段があります。

### Claude.ai の音声モード

Claude.ai（Web / モバイル）には音声での会話モードが組み込まれています。リアルタイムの音声会話が可能ですが、Web 版に限られます。

### Claude Code の /voice コマンド

Claude Code には `/voice` コマンドが組み込まれており、スペースキーを長押しして話し、離すとテキストが入力されます。手軽ですが、認識精度は汎用的な音声認識に依存します。

### Aqua Voice の優位性

Aqua Voice は以下の点で差別化されています。

1. **アプリを問わない**: Cowork、Claude Code、ブラウザ、メール、Slack — カーソルがあればどこでも使える
2. **テクニカルターム認識**: 画面コンテキストを読み取り、プログラミング用語を正確に変換する
3. **49 言語対応**: 日本語と英語の切り替えも自然に対応
4. **低レイテンシ**: 50ms 以下で起動し、約 1 秒でテキスト挿入

一方、Claude Code の `/voice` はインストール不要で手軽に使えるため、簡単な指示にはそちらで十分です。長いプロンプトや正確なテクニカルターム入力が必要な場面では、Aqua Voice のメリットが際立ちます。

## まとめ

Aqua Voice は、Claude との組み合わせで大きな生産性向上をもたらすツールです。特に以下のような場面でその価値を発揮します。

- 長いプロンプトを頻繁に入力する
- プログラミング用語を含む指示を正確に伝えたい
- 手がふさがっている状態で Claude を操作したい

タイピングのスピードに制限されることなく、思考の速さで Claude に指示を出せる — それが Aqua Voice と Claude の組み合わせが実現する開発体験です。

## FAQ

### Aqua Voice は無料で使えますか？

はい、基本機能が使える Free プランがあります。フル機能を使いたい場合は月額 $8 の Pro プランにアップグレードできます。

### Claude Code の /voice との違いは何ですか？

Claude Code の /voice は組み込みの音声入力で手軽ですが、Aqua Voice は画面コンテキストを読み取る独自モデルによりテクニカルタームの認識精度が大幅に高く、あらゆるアプリで使える汎用性があります。

### 日本語の認識精度はどうですか？

Aqua Voice は 49 言語に対応しており、日本語も高い精度で認識します。カスタム辞書に専門用語を登録することで、さらに精度を高めることができます。
